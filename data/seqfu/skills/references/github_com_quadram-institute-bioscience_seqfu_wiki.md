They have been originally built these principles: Reading both FASTQ and FASTA sequences with the same parser Parsing both name and comments from sequence headers (i.e. &gt;Seq_name length=1200 ) Supporting .gz input files, and possibly other compression formats Supporting streams (standard input / standard output) Native support for Illumina Paired-End libraries when needed New scripts also adopt BioX::Seq .
