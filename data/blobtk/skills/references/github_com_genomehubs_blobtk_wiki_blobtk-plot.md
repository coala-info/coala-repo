# blobtk plot

Process an existing BlobDir and generate static plots.

```
blobtk plot --help
```

## Usage

```
blobtk plot [OPTIONS] --blobdir <BLOBDIR> --view <VIEW>
```

Common views:

* `blob`
* `cumulative`
* `legend`
* `snail`

## Filter syntax

Preferred syntax:

```
<field>:<type>=<value>
```

Examples:

* `--filter length:min=1000`
* `--filter gc:max=0.6`
* `--filter buscogenes_phylum:keys=Arthropoda,Chordata`
* `--filter buscogenes_phylum:inv=Arthropoda,Chordata`

Legacy BlobToolKit URL-compatible syntax is also accepted:

```
<field>--<type>=<value>
```

Type names are case-insensitive.

## Examples

### Blob plot

```
blobtk plot \
  --blobdir /path/to/blobdir \
  --view blob \
  --output blob.png
```

### Cumulative plot

```
blobtk plot \
  --blobdir /path/to/blobdir \
  --view cumulative \
  --output cumulative.png
```

### Snail plot from existing BlobDir

```
blobtk plot \
  --blobdir /path/to/blobdir \
  --view snail \
  --output snail.svg
```

### Snail plot with reference assembly and filters

```
blobtk plot \
  --blobdir /path/to/blobdir \
  --reference /path/to/reference.blobdir \
  --view snail \
  --filter length:min=1000 \
  --filter phylum:inv=Arthropoda,Chordata \
  --output snail_ref_filtered.svg
```

### Legend-only output

```
blobtk plot \
  --blobdir /path/to/blobdir \
  --view legend \
  --output legend.svg
```

### Custom category palette

```
blobtk plot \
  --blobdir /path/to/blobdir \
  --view blob \
  --palette viridis \
  --cat-count 14 \
  --color 12=ff7001 \
  --color 13=01ffff \
  --output blob_custom_palette.png
```

For building snail plots directly from FASTA + BUSCO, see [blobtk snail](blobtk-snail).
