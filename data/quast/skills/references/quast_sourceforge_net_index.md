QUAST QUAST &#8211; Quality Assessment Tool for Genome Assemblies The project aim is to create easy-to-use tools for genome assemblies evaluation and comparison. Currently, we are working on four tools which are distributed inside one package : QUAST for regular genome assemblies MetaQUAST for metagenome assemblies QUAST-LG for large genome (e.g. mammalian) assemblies Icarus for contig alignment visualization About us GenomeWeb BioStar Homologus H1 Connect (former Faculty Opinions) Milestones November 9, 2024&nbsp; —&nbsp;the version 5.3 is released !
