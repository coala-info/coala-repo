# rp2paths

## Overview
The rp2paths tool is a specialized utility designed to process the "metabolic space" or "scope" files generated by the RetroPath2.0 retrosynthesis workflow. While RetroPath2.0 identifies the broad network of possible reactions, rp2paths performs the critical task of decomposing this network into individual, linear, or branched pathways. Use this skill to automate the extraction of pathways for specific target compounds, generate visual pathway maps, and produce structured data for downstream flux analysis or strain design.

## Command Line Usage

The primary interface for rp2paths is the `all` command, which executes the full extraction and visualization pipeline in a single step.

### Basic Execution
To process a RetroPath2.0 scope file and output results to a specific directory:
```bash
python -m rp2paths all <retropath2_scope_csv> --outdir <output_directory>
```

### Arguments
- `all`: The recommended mode to run all pathway retrieval tasks (preprocessing, enumeration, and formatting) at once.
- `<retropath2_scope_csv>`: The CSV file produced by the RetroPath2.0 workflow.
- `--outdir`: (Optional) The directory where results will be saved. If omitted, the tool may use default naming conventions in the current directory.

## Output Interpretation

After a successful run, the output directory will contain:
1. **out_paths.csv**: A comprehensive list of all enumerated pathways.
2. **Pathway Visualizations**: For every identified pathway, the tool generates:
   - A `.dot` file: Graphviz-compatible representation of the pathway graph.
   - A `.svg` file: A rendered image of the pathway for immediate inspection.

## Troubleshooting and Best Practices

### Handling "No Scope Matrix" Warnings
If you encounter the warning: `WARNING: *** Scope Task: no scope matrix was produced, exit`, it indicates that no reactions were identified as being part of a complete pathway.
- **Cause**: This typically happens when the retrosynthesis exploration in RetroPath2.0 failed to reach the defined "sink" compounds (the starting building blocks available in the chassis).
- **Solution**: Re-evaluate the RetroPath2.0 parameters, specifically the search depth or the list of available sink compounds.

### Visualization Tips
- Use the `.svg` files for quick manual validation of the chemistry.
- Use the `.dot` files if you need to programmatically manipulate the graph structure or re-render the pathways with custom styling using Graphviz.

### Environment Management
Ensure the tool is executed within a configured Conda environment to manage dependencies like `brs_utils`.
```bash
conda activate <your_env_name>
python -m rp2paths all ...
```
