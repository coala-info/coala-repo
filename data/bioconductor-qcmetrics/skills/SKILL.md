# bioconductor-qcmetrics

## Overview

Use the Bioconductor R package **qcmetrics** for: The package provides a framework for generic quality control of data. It permits to create, manage and visualise individual or sets of quality control metrics and generate quality control reports in various formats.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("qcmetrics")
```
