## `frag-length-bins`

### Let's retrieve the 10bp binned (`--bin-size 10`) distribution of fragment lengths (`frag-length-bins`) from fragments that have a MAPQ score greater than 30 (`-q 30`), a minimum length of 50bp (`-min 50`), a maximum length of 350bp (`-max 350`) and whose midpoints (`-p midpoint`) are on the interval 22:40000000-41000000 (`-c 22`, `-S 40000000`, `-E 41000000` ). We will output (`-o`) the TSV to 22.s40Mb.e41Mb.b10.tsv and the histogram (`--histogram_path`) to 22.s40Mb.e41Mb.b10.png. We will print summary statistics to the output (`-stats`), including a short fraction <= 150 (`-sf 150`). We will run the command in verbose mode (`-v`).

```
finaletoolkit frag-length-bins BH01.hg19.mdups.22.bam \
  -q 30 \
  --bin-size 10 \
  -p midpoint \
  -min 50 \
  -max 350 \
  -c 22 \
  -S 40000000 \
  -E 41000000 \
  -o 22.s40Mb.e41Mb.b10.tsv \
  -stats \
  -sf 150 \
  --histogram-path 22.s40Mb.e41Mb.b10.png \
  -v
```

## `frag-length-intervals`

### Let's retrieve the fragment length statistics (`frag-length-intervals`) from fragments that have a MAPQ score greater than 30 (`-q 30`), a minimum length of 50bp (`-min 50`), a maximum length of 350bp (`-max 350`) and whose midpoints (`-p midpoint`) are on 100kb intervals across 22. We will output (`-o`) the data to 100kb.22.bed, using one worker (`-w 1`) and verbose mode (`-v`).

```
finaletoolkit frag-length-intervals BH01.hg19.mdups.22.cram 100kb.22.bins \
  -q 30 \
  -p midpoint \
  -min 50 \
  -max 350 \
  -o 100kb.22.bed \
  -w 1 \
  -v
```

### The specific fraction of short fragments can be changed from the default value (150).

For instance, if we are interested in S130 along with all of the previous parameters, we can use the following command:

```
finaletoolkit frag-length-intervals BH01.hg19.mdups.22.cram 100kb.22.bins \
  -q 30 \
  -p midpoint \
  -min 50 \
  -max 350 \
  -o 100kb.22.bed \
  -w 1 \
  -s 130
  -v
```
