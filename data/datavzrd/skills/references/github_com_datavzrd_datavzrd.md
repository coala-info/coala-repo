Reports include automatically generated vega-lite histograms per column. Plots can be fully customized by users via a config file. These also allow the user to add linkouts to other websites or link between multiple tables. Documentation is available via the official website: https://datavzrd.github.io/ An example report can be viewed online with the corresponding config file .
