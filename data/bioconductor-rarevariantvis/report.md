# bioconductor-rarevariantvis Bioconductor Package Report

## RareVariantVis

### Package Description
The package includes homozygous region caller and allows to analyse whole human genomes in less than 30 minutes on a desktop computer. RareVariantVis disclosed novel causes of several rare monogenic disorders, including one with non-coding causative variant - keratolythic winter erythema.
