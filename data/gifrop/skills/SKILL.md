# gifrop

## Overview
The `gifrop` tool (Genomic Islands From Roary Pangenomes) is designed to extract and characterize clusters of accessory genes that form genomic islands. It operates by analyzing the gene order and presence/absence patterns generated by the Roary pangenome pipeline. It is particularly effective for comparative genomics of highly similar strains (e.g., a single Salmonella serovar) where mobile genetic elements represent the primary source of variation.

## Core Workflows

### Standard Island Identification
To run the basic identification and clustering workflow, navigate to the directory containing your Roary output (`gene_presence_absence.csv`) and the corresponding Prokka-annotated `.gff` files.

```bash
gifrop --get_islands --threads 8
```

### Reference-Based Extraction
If you want to identify islands specifically absent from a high-quality reference genome, use the reference flag. This removes all genes present in the reference from the pangenome before identifying islands.

```bash
gifrop --get_islands --reference <reference_isolate_name>
```

### End-to-End Pipeline (pan_pipe)
If starting from raw nucleotide FASTA files, use the `pan_pipe` wrapper. This automates annotation (Prokka), pangenome calculation (Roary), and island extraction (gifrop).

```bash
pan_pipe --prokka_args "--genus Salmonella" --roary_args "-p 8 -s" --gifrop_args "--get_islands"
```

## Expert Tips and Best Practices

### Input Requirements
* **File Location**: Ensure all `.gff` files used for the Roary run are in the same directory where you execute `gifrop`.
* **Strain Relatedness**: This tool is optimized for closely related strains. Using it on highly divergent genomes (e.g., different species) will result in fragmented and messy island clusters.

### Understanding Clustering Levels
`gifrop` provides four levels of clustering in `clustered_island_info.csv`:
1. **Primary**: Any overlap (Simpson similarity > 0).
2. **Secondary**: Moderate overlap (Simpson similarity > 0.5).
3. **Tertiary**: High overlap (Simpson similarity > 0.75).
4. **Quaternary**: High Jaccard similarity (> 0.75), meaning the islands have nearly identical total gene content.

### Handling Draft Assemblies
Be aware that `gifrop` cannot bridge contig breaks. If a genomic island is split across two contigs in a draft assembly, the tool will identify it as two separate, smaller islands.

### Key Outputs
* **clustered_island_info.csv**: The primary file for downstream analysis, containing detailed descriptions and cluster assignments for every island.
* **pan_with_island_info.csv**: A version of the Roary presence/absence matrix annotated with island data.
* **All_islands.fasta**: A multi-fasta file containing the sequences of all identified islands, useful for further BLAST or phylogenetic analysis.
