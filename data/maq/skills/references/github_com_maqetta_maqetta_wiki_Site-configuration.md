# Site configuration

[Jump to bottom](#wiki-pages-box)

Bill Reed edited this page Jun 4, 2013
·
[2 revisions](/maqetta/maqetta/wiki/Site-configuration/_history)

Maqetta includes some features that allow particular sites to define site-specific customizations to the running application.

The site-specific customizations are found in the `siteConfig` folder. Within the core development system, the folder that contains Maqetta's defaults is `davinci.releng/orion/builder/siteConfi`. Here are the config files:

* [defaultThemeSet.json](https://github.com/maqetta/maqetta/blob/master/releng/davinci.releng/orion/builder/rootfiles/siteConfig/defaultThemeSet.json) - This defines the default set of themes that should be used when the application starts up.
* [widgetPalette.json](/maqetta/maqetta/wiki/widgetpalette.json) - Controls which widgets appear in the widget palette, along with grouping and ordering of the widgets.
