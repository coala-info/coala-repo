Try gbdraw Web App! Features Circular and linear diagrams: Generates both circular and linear representations of genome structures. Multiple Input Formats: Supports standard GenBank/DDBJ files as well as GFF3 + FASTA file pairs. Dual Interface: Available as a powerful command-line tool and an interactive local/web GUI. Various output formats: Vector and raster graphics suitable for publication and further editing. Flexible Label Control: Provides advanced control over feature labels, including priority rules, blacklists, and whitelists. Comparative genomics: Visualizes sequence similarity between genomes using BLAST results. Accurate text metrics: CLI uses kerning-aware font measurements for improved label placement and legend sizing. 📖 Documentation &amp; Tutorials For detailed instructions on how to use gbdraw , please visit our main documentation page.
