#### Application Description

maze visualizes exact matches between two DNA sequences as a
dotplot. Longer match lengths reduce noise; shorter lengths reveal
smaller local matches.

#### Usage

* Enter or paste two DNA sequences (FASTA headers are supported).
* Set the match length and click *Launch Analysis*.
* Use zoom/pan on the plot to inspect regions of interest.
* Download the visualization as PNG or SVG from the Results tab.

#### Accepted Input

Raw DNA sequences or FASTA files (plain text or gzip). Only the DNA
alphabet {A, C, G, T} is accepted.
