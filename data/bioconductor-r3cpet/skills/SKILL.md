# bioconductor-r3cpet

## Overview

Use the Bioconductor R package **R3CPET** for: The package provides a method to infer the set of proteins that are more probably to work together to maintain chormatin interaction given a ChIA-PET experiment results.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("R3CPET")
```
