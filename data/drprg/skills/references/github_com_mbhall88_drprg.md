It can be used for any species, provided an index is available for that species. The documentation outlines which species have prebuilt indices and also a guide for how to create your own. Quick Installation conda install -c bioconda drprg Linux is currently the only supported platform; however, there is a Docker container that can be used on other platforms. See the installation guide for more options.
