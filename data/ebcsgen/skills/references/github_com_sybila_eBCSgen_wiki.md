BCSL is a rule-based language with focus on the human-readability and compactness of the description. The tool is designed as a command line tool, but a user-friendly front-end is provided by an instance of Galaxy Platform . All supported functionality is available online (among other tools developed by Sybila team ). To get started with the usage of Galaxy Platform and eBCSgen, we recommend this tutorial . The BCSL models format is specified here . The editing of the model is enabled in a modified Ace editor with automatic syntax highlighting and syntax error messaging. This editor can be used directly in Galaxy platform.
