# The khmer developer documentation[¶](#the-khmer-developer-documentation "Permalink to this headline")

(or would like to contribute to) the khmer project codebase, either by
contributing code or by helping improve the documentation.

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.html).
By participating in the development of this project you agree to abide by its
terms.
