Secrets and variables &gt; Actions روی دکمه New repository secret کلیک کنید و سه secret جدید با نام‌های دقیق زیر بسازید و مقادیر مربوطه را در آن‌ها قرار دهید: TELEGRAM_BOT_TOKEN مقدار: توکن ربات خود را که از BotFather گرفته‌اید، اینجا وارد کنید. TELEGRAM_CHAT_ID مقدار: آیدی کانال اصلی خود را (با @ ) اینجا وارد کنید. TELEGRAM_CHANNEL_ID مقدار: آیدی کانال دوم (کانال کانفیگ‌ها) را (با @ ) اینجا وارد کنید. مرحله ۲: فعال‌سازی در فایل Workflow حالا باید به GitHub Actions بگویید که قابلیت ارسال به تلگرام را فعال کند. فایل .github/workflows/main.yml را در ریپازیتوری خود باز کنید. در بخش Run Main Extractor Script ، مقدار SEND_TO_TELEGRAM را به true تغییر دهید. - name : Run Main Extractor Script env : # ---------------------------------------------------- # برای فعال‌سازی، این مقدار را true قرار دهید SEND_TO_TELEGRAM : true # ---------------------------------------------------- TELEGRAM_BOT_TOKEN : ${{ secrets.TELEGRAM_BOT_TOKEN }} TELEGRAM_CHAT_ID : ${{ secrets.TELEGRAM_CHAT_ID }} TELEGRAM_CHANNEL_ID : ${{ secrets.TELEGRAM_CHANNEL_ID }} run : python main.py با انجام این دو مرحله، پس از هر بار اجرای موفق workflow، نتایج به صورت خودکار به کانال‌های تلگرام شما نیز ارسال خواهد شد. 🌟 ساب لینک‌های کانفیگ کالکتور 🌟 برای استفاده، لینک‌های زیر رو در کلاینت وارد کن: 🔹 mix : https://raw.githubusercontent.com/arshiacomplus/v2rayExtractor/refs/heads/main/mix/sub.html 🔹 vless : https://raw.githubusercontent.com/arshiacomplus/v2rayExtractor/refs/heads/main/vless.html 🔹 vmess : https://raw.githubusercontent.com/arshiacomplus/v2rayExtractor/refs/heads/main/vmess.html 🔹 trojan : https://raw.githubusercontent.com/arshiacomplus/v2rayExtractor/refs/heads/main/trojan.html 🔹 ss : https://raw.githubusercontent.com/arshiacomplus/v2rayExtractor/refs/heads/main/ss.html 🔹 hy2 : https://raw.githubusercontent.com/arshiacomplus/v2rayExtractor/refs/heads/main/hy2.html 🌍 در این پیج می‌تونی کانفیگ‌ها رو بر اساس منطقه مشاهده کنی: https://arshiacomplus.github.io/V2rayExtractor-page/ About telegram config collector sublinks t.me/v2ray_extractor Topics config freeconfigs sublink Resources Readme Uh oh!
