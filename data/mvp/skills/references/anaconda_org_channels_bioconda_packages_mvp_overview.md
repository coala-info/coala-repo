Since motifs can be recognition sites for biological functions such as regulation or DNA modification, their gain and loss can have additional consequences. Using a list of variants in variant call format, the corresponding reference sequence, and a set of motifs to search for,mvp (motif-variant probe) identifies variants responsible for changing the number of occurrences of these motifs in the sequence. mvp can process both nucleotide and amino acid sequences.
