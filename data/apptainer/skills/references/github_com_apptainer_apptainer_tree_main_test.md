# Test Corpus

## Images

Various SIF images are available for testing in the [images](/apptainer/apptainer/blob/main/test/images) directory:

* [empty.sif](/apptainer/apptainer/blob/main/test/images/empty.sif): image with no data objects.
* [one-group.sif](/apptainer/apptainer/blob/main/test/images/one-group.sif): image with two data objects in one
  object group.
* [one-group-signed-dsse.sif](/apptainer/apptainer/blob/main/test/images/one-group-signed-dsse.sif): image with a
  DSSE envelope containing Ed25519 and RSA signatures.
* [one-group-signed-legacy-all.sif](/apptainer/apptainer/blob/main/test/images/one-group-signed-legacy-all.sif):
  image with a legacy `--all` signature.
* [one-group-signed-legacy-group.sif](/apptainer/apptainer/blob/main/test/images/one-group-signed-legacy-group.sif):
  image with a legacy `--groupid` signature.
* [one-group-signed-legacy.sif](/apptainer/apptainer/blob/main/test/images/one-group-signed-legacy.sif): image with
  a legacy signature.
* [one-group-signed-pgp.sif](/apptainer/apptainer/blob/main/test/images/one-group-signed-pgp.sif): image with a PGP
  signature.

The above images originated from the
[SIF test imagecorpus](https://github.com/apptainer/sif/tree/main/test/images).
When adding new images, please consider updating
[gen\_sifs.go](https://github.com/apptainer/sif/blob/main/test/images/gen_sifs.go)
in the SIF repository.

## Keys

Cryptographic key pairs using various algorithms are available for testing in
the [keys](/apptainer/apptainer/blob/main/test/keys) directory:

* ECDSA: [ecdsa-public.pem](/apptainer/apptainer/blob/main/test/keys/ecdsa-public.pem)/
  [ecdsa-private.pem](/apptainer/apptainer/blob/main/test/keys/ecdsa-private.pem)
* Ed25519: [ed25519-public.pem](/apptainer/apptainer/blob/main/test/keys/ed25519-public.pem)/
  [ed25519-private.pem](/apptainer/apptainer/blob/main/test/keys/ed25519-private.pem)
* RSA: [rsa-public.pem](/apptainer/apptainer/blob/main/test/keys/rsa-public.pem)/
  [rsa-private.pem](/apptainer/apptainer/blob/main/test/keys/rsa-private.pem)

The above key pairs originated from the
[SIF test key corpus](https://github.com/apptainer/sif/tree/main/test/keys).
When adding new key pairs, please consider updating
[gen\_keys.go](https://github.com/apptainer/sif/blob/main/test/keys/gen_keys.go)
in the SIF repository.

A PGP key pair is also available for testing:

* [pgp-public.asc](/apptainer/apptainer/blob/main/test/keys/pgp-public.asc)/
  [pgp-private.asc](/apptainer/apptainer/blob/main/test/keys/pgp-private.asc)
