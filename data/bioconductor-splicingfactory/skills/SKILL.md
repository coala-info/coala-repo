# bioconductor-splicingfactory

## Overview

Use the Bioconductor R package **SplicingFactory** for: the package analyzes the isoform diversity data, looking for significant changes between conditions.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("SplicingFactory")
```
