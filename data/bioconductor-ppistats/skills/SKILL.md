# bioconductor-ppistats

name: bioconductor-ppistats
description: Analysis and description of protein-protein interaction (PPI) data from bait-prey technologies (Y2H, AP-MS). Use this skill to summarize interaction graphs, identify systematic biases in proteins, estimate stochastic error rates (false positives/negatives), and calculate corrected node degrees.

## Overview

The `ppiStats` package provides statistical tools for analyzing directed graphs generated by bait-prey experiments. It distinguishes between "Viable Baits" (VB), "Viable Preys" (VP), and proteins that are both (VBP). The package is essential for cleaning PPI data by identifying proteins with systematic degree imbalances and for estimating the underlying stochastic error probabilities using gold-standard datasets.

## Core Workflow

### 1. Data Preparation and Summarization
Most functions require `graphNEL` objects. Use `ppiStats` to define the viable population and generate summary statistics.

```R
library(ppiStats)
# GavinGraphs is a list of graphNEL objects
summaryTables = createSummaryTables(GavinGraphs)

# Identify specific viable proteins
viableProts = idViableProteins(graph) # Returns list with VB, VP, VBP
```

### 2. Identifying Systematic Bias
Proteins with extreme imbalances between in-degree (detected as prey) and out-degree (detecting others as bait) may be systematically biased.

```R
# Visualize in/out degree distributions
inOutScatterCharts(GavinGraphs)

# Identify and remove systematic VBPs
vbpProts = idViableProteins(graph)$VBP
systematicNodes = idSystematic(graph, vbpProts, bpGraph=TRUE)
unbiasedNodes = setdiff(vbpProts, systematicNodes)

# Create a cleaned subgraph
cleanSubgraph = subGraph(unbiasedNodes, graph)
```

### 3. Estimating Error Probabilities
Use a gold standard (e.g., curated complexes like `ScISIC`) to estimate false positive ($pFP$) and false negative ($pFN$) rates.

```R
library(ScISI)
data(ScISIC)

# Convert graph to matrix for estimation
adjMat = as(cleanSubgraph, "matrix")

# Estimate rates
errorRates = estimateCCMErrorRates(adjMat, ScISIC)
pTP = errorRates$globalpTP # True Positive rate (1 - pFN)
pFP = errorRates$globalpFP # False Positive rate
```

### 4. Estimating True Node Degree
After calculating error probabilities, estimate the "true" number of interacting partners for proteins, accounting for experimental noise.

```R
# Calculate observed stats (reciprocated vs unreciprocated edges)
obsStats = calcInOutDegStats(cleanSubgraph)

# Estimate true degree
estDegrees = degreeEstimates(adjMat, pTP, pFP)
```

## Key Functions
- `createSummaryTables`: Generates counts for VB, VP, VBP, and Total Interactions (TI).
- `idViableProteins`: Returns character vectors of proteins categorized by viability.
- `inOutScatterCharts`: Produces scatter plots to visualize degree bias.
- `idSystematic`: Statistically identifies nodes that fail the Binomial model for stochastic error.
- `estimateCCMErrorRates`: Method of moments approach to find $pFP$ and $pFN$ using a gold standard.
- `degreeEstimates`: Parametric model to estimate true node degree from observed bait-prey data.
