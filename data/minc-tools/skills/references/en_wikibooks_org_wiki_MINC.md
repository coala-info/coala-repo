[![Medical Image NetCDF](//upload.wikimedia.org/wikipedia/commons/6/6e/MINC_logo1.jpg)](/wiki/File%3AMINC_logo1.jpg "Medical Image NetCDF")MINC is a Medical Imaging file format and Toolbox for use in medical imaging, this wikibook contains Tutorials, API Reference and Examples

[Introduction](/wiki/MINC/Introduction "MINC/Introduction") - The How, What and Where of MINC

[History](/wiki/MINC/History "MINC/History") - The Who and When of MINC

[MincWorld](/wiki/MINC/MincWorld "MINC/MincWorld") - Who uses and develops software based on MINC

[Installation](/wiki/MINC/Installation "MINC/Installation") - Tutorials for installing the MINC tools and associated software

[FAQ](/w/index.php?title=MINC/FAQ&action=edit&redlink=1 "MINC/FAQ (does not exist)") - MINC Frequently asked questions

[Tools](/wiki/MINC/Tools "MINC/Tools") - MINC extra tools (registration, B0 correction, classification, etc)

[VisualTools](/wiki/MINC/VisualTools "MINC/VisualTools") - Visual MINC tools (manual segmentation, registration, data visualisation)

[Scripts](/wiki/MINC/Scripts "MINC/Scripts") - Extra scripts (typically perl) that have been collected over time

[SoftwareDevelopment](/wiki/MINC/SoftwareDevelopment "MINC/SoftwareDevelopment") - Information on developing MINC software

[Tutorials](/wiki/MINC/Tutorials "MINC/Tutorials") - Software development tutorials

[Resources](/wiki/MINC/Resources "MINC/Resources") - Example data sets, library files, color maps, etc.

[Atlases](/wiki/MINC/Atlases "MINC/Atlases") - The various atlases that are available from the BIC in the MINC format

[Authors](/wiki/MINC/Authors "MINC/Authors") - The Authors of MINC and how to reference the tools
