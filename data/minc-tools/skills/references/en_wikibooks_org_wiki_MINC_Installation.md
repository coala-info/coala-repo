## Installing and configuring MINC

[[edit](/w/index.php?title=MINC/Installation&veaction=edit&section=1 "Edit section: Installing and configuring MINC") | [edit source](/w/index.php?title=MINC/Installation&action=edit&section=1 "Edit section's source code: Installing and configuring MINC")]

### Current Version

[[edit](/w/index.php?title=MINC/Installation&veaction=edit&section=2 "Edit section: Current Version") | [edit source](/w/index.php?title=MINC/Installation&action=edit&section=2 "Edit section's source code: Current Version")]

Currently, MINC tools are distributed as part of minc-toolkit, details on installation are located at [bic-mni.github.io](http://bic-mni.github.io/), all the source code is now hosted on [GitHUB](https://github.com/BIC-MNI/)

### Historical information (out-of date)

[[edit](/w/index.php?title=MINC/Installation&veaction=edit&section=3 "Edit section: Historical information (out-of date)") | [edit source](/w/index.php?title=MINC/Installation&action=edit&section=3 "Edit section's source code: Historical information (out-of date)")]

The minc tools and related software can be installed under Unix, Linux, Windows and Mac OS X. Below is the list of detailed installation types depending on the operating system used. You can download all available software at the ["MNI software repository"](http://packages.bic.mni.mcgill.ca/)

* Source means having the source code in a tar.gz compression format, to be compiled and installed manually. See [SourceInstall](/wiki/MINC/Tutorials/SourceInstall "MINC/Tutorials/SourceInstall") for more information. You will have to [configure](/wiki/MINC/Tutorials/AddingPaths "MINC/Tutorials/AddingPaths") your system and to [install](/wiki/MINC/Tutorials/InstallingPerl "MINC/Tutorials/InstallingPerl") additional supplementary PERL modules.
* Binary means there are installable binaries for this system. It can take the form of a package (e.g. a .deb format for Ubuntu) or a tar.gz with executables. See [BinaryInstalls](/wiki/MINC/Tutorials/BinaryInstalls "MINC/Tutorials/BinaryInstalls") for more information.
* Note that "MINC-Bundle" means there is a bundle with all the MINC tools & associated software available, as opposed to installing each package separately.
