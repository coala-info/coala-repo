## 0.24.0 (2026-03-21)

* Migrate from `setup.py` to `pyproject.toml` (thanks to [@nsoranzo](https://github.com/nsoranzo)). [Pull Request 142](https://github.com/galaxyproject/gxformat2/pull/142)
* Use \_unlabeled\_input\_ sentinel to preserve label=None on round-trip (thanks to [@jmchilton](https://github.com/jmchilton)). [Pull Request 144](https://github.com/galaxyproject/gxformat2/pull/144)
* Fix / in step labels breaking source reference parsing (thanks to [@jmchilton](https://github.com/jmchilton)). [Pull Request 148](https://github.com/galaxyproject/gxformat2/pull/148)
* Add pick\_value and comments to Format2 schema (thanks to [@jmchilton](https://github.com/jmchilton)). [Pull Request 147](https://github.com/galaxyproject/gxformat2/pull/147)
* Fix unlabeled tool steps getting dangling numeric source refs after roundtrip (thanks to [@jmchilton](https://github.com/jmchilton)). [Pull Request 149](https://github.com/galaxyproject/gxformat2/pull/149)
* Add uv dependency groups for lint, mypy, test; update dev docs (thanks to [@jmchilton](https://github.com/jmchilton)). [Pull Request 150](https://github.com/galaxyproject/gxformat2/pull/150)
* More dev process updates - including history generation stuff, better uv usage, etc (thanks to [@jmchilton](https://github.com/jmchilton)). [Pull Request 152](https://github.com/galaxyproject/gxformat2/pull/152)
