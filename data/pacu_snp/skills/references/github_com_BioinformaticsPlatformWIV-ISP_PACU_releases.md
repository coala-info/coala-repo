Long overdue version bump to v1.0.0!

## Improvements

* Mapping: skipped creation of Bowtie2 / Minimap2 indices when they already exist.
* PACU: allowed use of reference FASTA containing multiple sequences when `Gubbins` is disabled (`--skip-gubbins`).
* General: improved handling of relative paths for input/output.
* Build: migrated project configuration from setup.py to pyproject.toml.

No major functional changes; this release primarily includes quality-of-life enhancements for CLI workflows.

* Added the option `--skip-gubbins` to skip recombination detection with Gubbins (can be used to work with fragmented reference genomes)
