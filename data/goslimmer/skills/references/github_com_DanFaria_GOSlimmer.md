It requires as input: A full Gene Ontology file in either OBO or OWL A slim Gene Ontology file in either OBO or OWL (any GOSlim version) An Annotation file, which can be in GAF format (from the Gene Ontology website), BLAST2GO format, or in tabular format (with gene ids in the first column and GO term ids in the second one) It produces as output: A tabular Slim Annotation file listing all non-redundant GO Slim annotations of gene products in the original Annotation file The JAR and XML files are setup for the Galaxy platform ( https://usegalaxy.org/ ) About No description, website, or topics provided.
