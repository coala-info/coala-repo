# bioconductor-hellorangesdata

## Overview
HelloRangesData is a Bioconductor experiment data package that provides datasets for the HelloRanges tutorial. It contains a collection of BED files representing DnaseI hypersensitivity hotspots across various human tissues, as well as standard genomic annotation tracks (CpG islands, exons, GWAS SNPs) and genome metadata for the hg19 assembly.

## Accessing Data
The package does not contain R data objects (like GRanges) directly. Instead, it provides raw files in the package's `extdata` directory. Use `system.file` to retrieve the local paths to these files for use with functions like `import()` from `rtracklayer` or functions in `HelloRanges`.

### List all available files
```r
dir(system.file("extdata", package = "HelloRangesData"))
```

### Locate specific files
To get the path to a specific file (e.g., CpG islands):
```r
cpg_path <- system.file("extdata", "cpg.bed", package = "HelloRangesData")
```

## Available Datasets
The package includes the following file types:

*   **DnaseI Hypersensitivity**: 20 BED files containing hotspots for various tissues (e.g., `fBrain`, `fHeart`, `fMuscle`, `fLung`).
*   **Annotations**:
    *   `cpg.bed`: CpG islands from UCSC.
    *   `exons.bed`: Refseq exon coordinates.
    *   `gwas.bed`: Disease-associated SNPs from GWAS catalogs.
    *   `hesc.chromHmm.bed`: Functional annotations for human embryonic stem cells (hESC) generated by chromHMM.
*   **Genome Metadata**:
    *   `hg19.genome`: A text file containing chromosome names and lengths for the hg19 genome build, often used to define the "genome" argument in range operations.

## Typical Workflow
Use these files to practice genomic interval manipulations.

### Loading a BED file into R
```r
library(rtracklayer)
cpg_path <- system.file("extdata", "cpg.bed", package = "HelloRangesData")
cpg_ranges <- import(cpg_path)
```

### Using the genome file
The `hg19.genome` file is useful for operations requiring chromosome boundaries:
```r
genome_path <- system.file("extdata", "hg19.genome", package = "HelloRangesData")
genome_info <- read.table(genome_path)
```
