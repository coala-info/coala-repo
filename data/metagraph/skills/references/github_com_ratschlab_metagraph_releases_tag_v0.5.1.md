## What's Changed

### Performance improvements

* map sequences to graph with dynamic omp schedule for better performance by [@karasikov](https://github.com/karasikov) in [#587](https://github.com/ratschlab/metagraph/pull/587)
* refactoring and optimizations in annotation queries by [@karasikov](https://github.com/karasikov) in [#582](https://github.com/ratschlab/metagraph/pull/582)
* updated sdsl-lite: removed memory overhead in select\_support\_mcl loaded with mmap by [@karasikov](https://github.com/karasikov) in [#603](https://github.com/ratschlab/metagraph/pull/603)

### Workflow and API changes

* changed default query mode: report hits with num k-mer matches by [@karasikov](https://github.com/karasikov) in [#588](https://github.com/ratschlab/metagraph/pull/588)
* fix graph augmentation and added tests by [@karasikov](https://github.com/karasikov) in [#580](https://github.com/ratschlab/metagraph/pull/580)
* Inline indexing of suffix ranges in BOSS construction by [@karasikov](https://github.com/karasikov) in [#590](https://github.com/ratschlab/metagraph/pull/590)
* Server queries in multi-graph regime with full loading to RAM by [@karasikov](https://github.com/karasikov) in [#569](https://github.com/ratschlab/metagraph/pull/569)

### Other updates

* Fix catching errors when lsof doesn't exist or fails by [@karasikov](https://github.com/karasikov) in [#583](https://github.com/ratschlab/metagraph/pull/583)
* Don't compile sshash executable by [@karasikov](https://github.com/karasikov) in [#584](https://github.com/ratschlab/metagraph/pull/584)
* Add citation section to README by [@adamant-pwn](https://github.com/adamant-pwn) in [#585](https://github.com/ratschlab/metagraph/pull/585)
* check workflows on linux by [@karasikov](https://github.com/karasikov) in [#592](https://github.com/ratschlab/metagraph/pull/592)
* updated KMC: removed unused variables by [@karasikov](https://github.com/karasikov) in [#602](https://github.com/ratschlab/metagraph/pull/602)
* Upgrade sshash to v5.0.0 with API compatibility changes by [@adamant-pwn](https://github.com/adamant-pwn) in [#594](https://github.com/ratschlab/metagraph/pull/594)
