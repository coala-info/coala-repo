## Overview

`nwkit prune` removes specified tips (or leaves). Tips are selected by a regular expression (`--pattern`). With `--invert_match yes`, unmatched tips are removed instead (i.e., only matched tips are kept).

## Examples

### 1. Remove tips matching a pattern

#### Command

```
nwkit prune --infile input.nwk --outfile output.nwk --pattern "B1|C2"
```

#### input.nwk

```
(((A1:2.0,(B1:1.0,B2:1.0):1.0):1.0,(A2:1.0,C1:1.0):2.0):1.0,C2:4.0):0.25;
```

Tips `B1` and `C2` are removed. The remaining tips and branch lengths are preserved.

### 2. Keep only matching tips with --invert\_match

With `--invert_match yes`, tips that do **not** match the pattern are removed, effectively keeping only the matched tips.

#### Command

```
nwkit prune --infile input.nwk --outfile output.nwk --pattern "A.*|B.*" --invert_match yes
```

#### input.nwk

```
(((A1:2.0,(B1:1.0,B2:1.0):1.0):1.0,(A2:1.0,C1:1.0):2.0):1.0,C2:4.0):0.25;
```

#### output.nwk

```
(((A1:2,(B1:1,B2:1)1:1)1:1,A2:3)1:1)1:0.25;
```

Tips `C1` and `C2` are removed because they do not match the pattern `A.*|B.*`. Only tips whose names start with `A` or `B` are kept.
