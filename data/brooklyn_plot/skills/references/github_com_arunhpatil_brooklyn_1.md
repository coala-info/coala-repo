GitHub - arunhpatil/brooklyn: Brooklyn plot for single cell/nuclei RNA sequencing data. This package enables a Pearson pairwise (ie., many to many) gene comparison and outputs co-expression patterns of genes across the genome. It generates a Brooklyn plot for visualization.
