Query a metaplasmidome database to identify and annotate plasmids in metagenomes

This workflow query metagenomic raw data against a metaplasmidome database to identify plasmids and annotate them with genes, KO, PFAM ## Associated Tutorial This workflows is part of the tutorial [Query an annotated mobile genetic element database to identify and annotate genetic elements (e.g. plasmids) in metagenomics data](https://training.galaxyproject.org/training-material/topics/microbiome/tutorials/metaplasmidome_query/tutorial.html), available in the [GTN](https://training.galaxyproject.org) ## Features * Includes [Galaxy Workflow Tests](https://training.galaxyproject.org/training-material/faqs/gtn/workflow_run_test.html) * Includes a [Galaxy Workflow Report](https://training.galaxyproject.org/training-material/faqs/galaxy/workflows_report_view.html) ## Thanks to... **Workflow Author(s)**: Bérénice Batut, Nadia Goué **Tutorial Author(s)**: [Bérénice Batut](https://training.galaxyproject.org/training-material/hall-of-fame/bebatut/), [Nadia Goué](https://training.galaxyproject.org/training-material/hall-of-fame/nagoue/), [Didier Debroas](https://training.galaxyproject.org/training-material/hall-of-fame/debroas/) **Tutorial Contributor(s)**: [Björn Grüning](https://training.galaxyproject.org/training-material/hall-of-fame/bgruening/), [Bérénice Batut](https://training.galaxyproject.org/training-material/hall-of-fame/bebatut/) [![gtn star logo followed by the word workflows](https://training.galaxyproject.org/training-material/assets/branding/gtn-workflows.png)](https://training.galaxyproject.org/training-material/)
