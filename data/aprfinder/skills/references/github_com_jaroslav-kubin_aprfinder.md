The apr stands for a-phased repeats. Installation aprfinder can be easily installed within conda enviroment. Usage aprfinder - tool for a-phased repeat search. Usage: aprfinder [--help] [--input=&lt;string&gt;] [--output=&lt;string&gt;] [--upper-bound=&lt;int&gt;] [--lower-bound=&lt;int&gt;] [--max-AT=&lt;int&gt;] [--min-AT=&lt;int&gt;] [--max-tracks=&lt;int&gt;] [--min-tracks=&lt;int&gt;] [--memory-size=&lt;int&gt;] [--exact-bound=&lt;int&gt;] [--exact-tracks=&lt;int&gt;] [--exact-AT=&lt;int&gt;] options --help Prints information about the software. --input &lt;string&gt; Input FASTA or MULTIFASTA file. --output &lt;string&gt; Defines output file name. Note that &lt;string&gt; should be in format &lt;name&gt;.gff. If not defined, output is set to result.gff. (Be carefull - running multiple times without this option can overwrite the previous result). --upper-bound &lt;int&gt; Maximum size of spacer. --lower-bound &lt;int&gt; Minimum size of spacer. --exact-bound &lt;int&gt; Sets lower-bound and upper-bound to the same value. --max-AT &lt;int&gt; Maximum number of consecutive A/T nucleotides. --min-AT &lt;int&gt; Minimum number of consecutive A/T nucleotides. --exact-AT &lt;int&gt; Sets max-AT and min-AT to the same value. --min-tracks &lt;int&gt; Minimum number of tracks to consider a sequence to be an a-phased repeat --max-tracks &lt;int&gt; Minimum number of tracks to consider a sequence to be an a-phased repeat --exact-tracks &lt;int&gt; Sets min-tracks and max-tracks to the same value About A-phased repeat software Resources Readme License MIT license Uh oh!
