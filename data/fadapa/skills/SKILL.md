# fadapa

## Overview
`fadapa` (FAstqc DAta PArser) is a specialized Python tool designed to handle the structured text output generated by FastQC. It eliminates the need for manual string parsing or complex regular expressions when interacting with sequencing quality control data. The tool allows you to load a FastQC result file and immediately access its various modules—such as "Per base sequence quality" or "Adapter Content"—in both raw and cleaned formats.

## Usage Instructions

### Installation
Ensure the package is installed in your environment:
```bash
pip install fadapa
```

### Basic Workflow
To begin parsing, import the `Fadapa` class and point it to your FastQC text output:

```python
from fadapa import Fadapa

# Load the FastQC data file
f = Fadapa('/path/to/fastqc_data.txt')
```

### Extracting Summaries
Use the `summary()` method to get a high-level overview of all QC modules and their respective statuses (PASS, WARN, or FAIL). This is the most efficient way to check if a sample meets quality thresholds.

```python
# Returns a list of lists: [['Module Name', 'Status'], ...]
qc_summary = f.summary()
```

### Accessing Module Data
FastQC data is organized into modules. You can retrieve data for a specific module using its exact name as it appears in the FastQC report.

1.  **Cleaned Data**: Use `clean_data('Module Name')` to get the data as a list of lists, with empty elements filtered out. This is preferred for most analysis tasks.
2.  **Raw Data**: Use `raw_data('Module Name')` to get the data exactly as it appears in the file, including headers and formatting characters.

```python
# Example: Extracting per base sequence quality
base_quality = f.clean_data('Per base sequence quality')

# Example: Extracting adapter content
adapters = f.clean_data('Adapter Content')
```

### Common Module Names
When calling `clean_data` or `raw_data`, use these standard FastQC module strings:
- `Basic Statistics`
- `Per base sequence quality`
- `Per tile sequence quality`
- `Per sequence quality scores`
- `Per base sequence content`
- `Per sequence GC content`
- `Per base N content`
- `Sequence Length Distribution`
- `Sequence Duplication Levels`
- `Overrepresented sequences`
- `Adapter Content`
- `Kmer Content`

## Expert Tips
- **Case Sensitivity**: Module names passed to `clean_data()` and `raw_data()` are case-sensitive and must match the FastQC headers exactly.
- **Full Content**: If you need the entire file content as a single list of strings, use `f.content()`.
- **File Metadata**: Access the source filename via `f.file_name` to keep track of samples during batch processing.
