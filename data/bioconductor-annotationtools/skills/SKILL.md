# bioconductor-annotationtools

name: bioconductor-annotationtools
description: Annotation of DNA microarray experiments and cross-platform/cross-species integration of gene expression profiles. Use this skill when you need to map Affymetrix probe sets to gene symbols, Entrez IDs, or GO terms, or when performing cross-species analysis by finding orthologous genes and probe sets using HomoloGene or NCBI ortholog databases.

## Overview
The `annotationTools` package provides functions for annotating microarray data using flat-file annotation tables (typically from Affymetrix) and mapping genes across species. It is particularly useful for translating probe set IDs between different array platforms or finding orthologous probe sets in different species (e.g., Human to Mouse) to compare gene expression studies.

## Core Workflows

### 1. Basic Annotation
To annotate probe sets, you must first load an Affymetrix annotation CSV file as a data frame.

```R
library(annotationTools)

# Load annotation file (ensure colClasses='character' to prevent ID corruption)
annot <- read.csv('HG-U133_Plus_2_annot.csv', colClasses='character', comment.char='#')

myPS <- c('117_at', '1007_s_at')

# Retrieve Gene Symbols
symbols <- getGENESYMBOL(myPS, annot)

# Retrieve GO Terms (specifics=2 returns text descriptions only)
go_terms <- getGENEONTOLOGY(myPS, annot, specifics=2)

# Generic lookup for any column in the annotation file
entrez_ids <- getGENEID(myPS, annot)
```

### 2. Finding Orthologs via HomoloGene
To find orthologs, you need a HomoloGene data file (available from NCBI).

```R
# Load HomoloGene data
homologene <- read.delim('homologene.data', header=FALSE)

# Find mouse orthologs (Taxon ID 10090) for human Gene IDs
human_genes <- c(5982, 93587)
mouse_orthologs <- getHOMOLOG(human_genes, 10090, homologene)
```

### 3. Cross-Species Probe Set Mapping
A common task is finding the equivalent probe set on a different species' array.

```R
# 1. Get Human Gene ID from Human Probe Set
gid_human <- getGENEID('1053_at', annot_human)

# 2. Get Mouse Orthologous Gene ID
gid_mouse <- getHOMOLOG(unlist(gid_human), 10090, homologene)

# 3. Get Mouse Probe Set from Mouse Gene ID
ps_mouse <- getPROBESET(unlist(gid_mouse), annot_mouse)
```

### 4. Building Ortholog Mapping Tables
The `ps2ps` function automates the creation of a mapping table between two different Affymetrix platforms using homology information.

```R
# Create a table mapping all probe sets from human array to mouse array
orthoTable <- ps2ps(annot_human, annot_mouse, homologene, 10090)

# Save for future use
write.table(orthoTable, file='HGU133Plus2_to_Mouse4302.txt', sep='\t', row.names=F)
```

### 5. Cross-Species Expression Comparison
Use `getOrthologousProbesets` to map a list of differentially expressed genes from one species to another and select the best representative probe set (e.g., based on highest fold change).

```R
# ps_mouse: vector of mouse probe set IDs
# table_human: data frame of human expression results
# ortho: mapping table generated by ps2ps
# selection: function to pick among multiple orthologs (e.g., max)

orthops <- getOrthologousProbesets(ps_mouse, table_human, ortho, max, forceProbesetSelection=TRUE)
```

## Tips and Troubleshooting
- **Data Loading**: Always use `colClasses='character'` when reading annotation or homology files to ensure IDs containing leading zeros or special characters are not mangled.
- **List Output**: Most functions return a list where each element corresponds to an input ID. Use `unlist()` or `compactList()` to handle cases where one probe set maps to multiple genes.
- **Diagnosis**: If a lookup fails, set `diagnose=TRUE` in annotation functions to see if the ID was missing from the file or simply lacked an entry.
- **NCBI gene_orthologs**: For newer NCBI ortholog files, set `tableType="gene_orthologs"` in `getHOMOLOG`.
