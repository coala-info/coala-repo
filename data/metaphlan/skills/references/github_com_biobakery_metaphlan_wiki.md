# Home

[Jump to bottom](#wiki-pages-box)

Vitor Heidrich edited this page Feb 20, 2024
·
[22 revisions](/biobakery/MetaPhlAn/wiki/Home/_history)

MetaPhlAn is a computational tool for species-level microbial profiling (bacteria, archaea, eukaryotes, and viruses) from metagenomic shotgun sequencing data. StrainPhlAn (available within MetaPhlAn) allows strain-level microbial population genomics.

## MetaPhlAn documentation

You can find [here](https://github.com/biobakery/MetaPhlAn/wiki/MetaPhlAn-4.1) the documentation for the latest version of MetaPhlAn (v4.1). The documentation for version 4.0 is also available [here](https://github.com/biobakery/MetaPhlAn/wiki/MetaPhlAn-4).
