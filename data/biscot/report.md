```text
usage: py --cmap-ref CMAP_REF_PATH --cmap-1 CMAP_1_PATH --xmap-1 XMAP_1_PATH
          --key KEY_PATH --contigs CONTIGS_PATH [--cmap-2 CMAP_2_PATH]
          [--xmap-2 XMAP_2_PATH] [--xmap-2enz XMAP_TWO_ENZYMES] [--aggressive]
          [--only-confirmed-pos] [--output OUTPUT_DIRECTORY] [--debug]
          [--help]

Reorganizes scaffolds generated by the Bionano scaffolding tool to remove small gaps that are unnecessary

Mandatory arguments:
  --cmap-ref CMAP_REF_PATH
                        Path to the reference CMAP file (example : '...NGS_contigs_HYBRID_Export_r.cmap')
  --cmap-1 CMAP_1_PATH  Path to a query CMAP file (example : 'E_CTTAAG_Q_NGScontigs_A_HYBRID_q.cmap')
  --xmap-1 XMAP_1_PATH  Path to a XMAP file
  --key KEY_PATH        Path to the Bionano 'key' file giving the correspondance between maps and contigs
  --contigs CONTIGS_PATH
                        Path to the contigs file that was scaffolded by the Bionano Access

Secondary map argument:
  --cmap-2 CMAP_2_PATH  Path to a query cmap (example : 'E_GCTCTTC_Q_NGScontigs_A_HYBRID_q.cmap')
  --xmap-2 XMAP_2_PATH  Path to a query xmap (example : 'E_GCTCTTC_Q_NGScontigs_A_HYBRID.xmap')

Two enzymes xmap args:
  --xmap-2enz XMAP_TWO_ENZYMES
                        Path to a xmap file containing mappings for both enzymes

Optional arguments:
  --aggressive          Enables Blat phase
  --only-confirmed-pos  To be retained, an alignment position in --xmap-1 or --xmap-2 has to be present in --xmap-2enz
  --output OUTPUT_DIRECTORY, -o OUTPUT_DIRECTORY
                        Name of the output directory (default : biscot)
  --debug               Activates debug level logs (default : False)
  --help, -h            show this help message and exit
```
