[2.1](/pdimens/djinn/releases/tag/2.1)

## New

* migrate `assign-mi` from harpy to `djinn sam`
  + assigns `MI` and `BX` tags for unique molecules based on an alignment distance threshold
* migrate `concat` from harpy to `djinn sam`
  + concatenate SAM/BAM files while preserving within-sample molecule uniqueness
* `djinn sam` commands have a `-S/--sam` option to output SAM format instead of BAM
* specify where interleaved files wont work

## Fixes

* fixed `sam standardize` incorrectly outputting only the first occurrence of a new barcode when `-s` style was invoked

**Full Changelog**: [2.0...2.1](https://github.com/pdimens/djinn/compare/2.0...2.1)

[2.0](/pdimens/djinn/releases/tag/2.0)

## Changes

* commands now need to prefixed with `sam` or `fastq` depending on the type of input file
* `fastq` commands accept one single-end or two paired-end files
* `fastq` commands have `--threads` option to using system `pigz` (if available) to speed up writing
* better fastq singleton filtering that accounts for the source of a barcode being one single-end or two paired-end reads
* `downsample` renamed `sample`

**Full Changelog**: [1.1...2.0](https://github.com/pdimens/djinn/compare/1.1...2.0)

# New

* `filter-singletons` to remove reads with singleton barcodes
* `filter-invalid` to remove invalid barcodes
* fastq file writing has been replaced with a fancy new internal cache object that can be configured with the hidden option `--cache-size` to help control gzip output speed
  + this also means no more writing uncompressed fastq and compressing after-- all fastq output is gzipped-on-write

[1.0](/pdimens/djinn/releases/tag/1.0)

This is the first official release of Djinn, which is the much-smaller package born from [Harpy](https://www.github.com/pdimens/harpy) to absorb the novel file-manipulation features from Harpy.

These features will be removed from the next Harpy release, but the (conda) Harpy installation will include Djinn in the future, although Djinn is also a perfectly good standalone package that is available through Bioconda.
