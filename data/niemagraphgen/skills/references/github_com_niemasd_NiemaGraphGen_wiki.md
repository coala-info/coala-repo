NiemaGraphGen (NGG) is a collection of efficient C++ implementations of graph generators for use in sampling contact networks in global-scale pandemic simulations. Importantly, NGG avoids actually storing the graph in memory and instead is intended to be used in a data streaming pipeline.

## Cookbook

In bash, run a NiemaGraphGen (FAVITES Output Format) executable, and save the output file.

```
ngg_complete 5 > complete_n5.tsv
```

In bash, run a NiemaGraphGen (FAVITES Output Format) executable, and gzip-compress and save the output file.

```
ngg_complete 5 | gzip > complete_n5.tsv.gz
```

In bash, run a NiemaGraphGen (Compact Output Format) executable, and save the output file.

```
ngg_complete 5 > complete_n5.ngg
```

In bash, run a NiemaGraphGen (Compact Output Format) executable, and gzip-compress and save the output file.

```
ngg_complete 5 | gzip > complete_n5.ngg.gz
```

In Python, load a contact network (FAVITES Output Format) into NetworkX.

```
from networkx import Graph
g = Graph()
with open("complete_n5.tsv") as f:
    for line in f:
        if line.startswith("NODE"):
            g.add_node(int(line.split('\t')[1]))
        else:
            u, v = line.split('\t')[1:3]
            g.add_edge(int(u), int(v))
```

In Python, load a gzip-compressed contact network (FAVITES Output Format) into NetworkX.

```
import gzip
from io import BufferedReader, TextIOWrapper
from networkx import Graph
g = Graph()
with TextIOWrapper(BufferedReader(gzip.open("complete_n5.tsv.gz"))) as f:
    for line in f:
        if line.startswith("NODE"):
            g.add_node(int(line.split('\t')[1]))
        else:
            u, v = line.split('\t')[1:3]
            g.add_edge(int(u), int(v))
```

In Python, load a contact network (Compact Output Format) into NetworkX.

```
from struct import unpack
from networkx import Graph
g = Graph()
with open("complete_n5.ngg", 'rb') as f:
    header = unpack('B', f.read(1))[0]
    int_size = {0b00: 1, 0b01: 2, 0b10: 4, 0b11: 8}[header & 0b00000011]
    int_size_letter = {1: 'B', 2: 'H', 4: 'I', 8: 'L'}[int_size]
    n = unpack(int_size_letter, f.read(int_size))[0]
    for i in range(n):
        g.add_node(i)
    while True:
        chunk = f.read(int_size)
        if len(chunk) == 0:
            break # reached end of file
        u = unpack(int_size_letter, chunk)[0]
        v = unpack(int_size_letter, f.read(int_size))[0]
        g.add_edge(u,v)
```
