# bioconductor-lrbase.hsa.eg.db

## Overview

LRBase.Hsa.eg.db is a specialized Bioconductor annotation package that serves two primary purposes:
1. **Class Definition**: It defines the `LRBaseDb` class, which provides a unified interface for interacting with Ligand-Receptor interaction databases.
2. **Package Construction**: It provides tools to build custom LRBase-type packages (e.g., `FANTOM5.Hsa.eg.db`) from user-provided ligand-receptor lists.

This package is a foundational component for bioinformaticians studying cell-cell communication, particularly when using the `scTensor` framework.

## Core Workflows

### Loading the Package
```R
library(LRBase.Hsa.eg.db)
```

### Querying LRBase Objects
All LRBase-type packages (including those generated by this package) use standard AnnotationDbi-style methods to access data.

*   **List available data columns:**
    ```R
    columns(LRBase.Hsa.eg.db)
    ```
*   **List searchable key types:**
    ```R
    keytypes(LRBase.Hsa.eg.db)
    ```
*   **Retrieve specific keys:**
    ```R
    # Get all receptor gene IDs
    keys_r <- keys(LRBase.Hsa.eg.db, keytype="GENEID_R")
    ```
*   **Perform a join/selection:**
    ```R
    # Map specific receptors to their corresponding ligands
    select(LRBase.Hsa.eg.db, 
           keys = keys_r[1:5], 
           columns = c("GENEID_L", "GENEID_R"), 
           keytype = "GENEID_R")
```

### Creating a Custom LRBase Package
You can create your own annotation package if you have a custom list of Ligand-Receptor pairs.

1.  **Prepare Data**: You need a data frame with `GENEID_L` and `GENEID_R` (NCBI Gene IDs).
2.  **Prepare Metadata**: A data frame describing the source and version of the LR list.
3.  **Run Generator**:
    ```R
    # Example workflow
    makeLRBasePackage(lrlist, metadata, outputDir, packageVignette)
    ```

### Metadata and Database Info
Use these functions to inspect the database properties:
*   `species(db)`: Common name of the species.
*   `nomenclature(db)`: Scientific name.
*   `listDatabases(db)`: Sources of the LR interaction data.
*   `dbInfo(db)`: General package information.
*   `dbconn(db)`: Returns the underlying SQLite connection.

## Tips for Success
*   **Gene IDs**: This package primarily uses NCBI Gene IDs (Entrez IDs). If you have Gene Symbols or Ensembl IDs, use `org.Hs.eg.db` to convert them to Entrez IDs before querying or building a package.
*   **Integration**: This package is designed to work seamlessly with `scTensor`. If you are performing tensor decomposition for cell-cell communication, ensure your LRBase package is correctly loaded.
*   **Interactive Help**: Use `example('makeLRBasePackage')` to see a live demonstration of package construction using FANTOM5 demo data.
