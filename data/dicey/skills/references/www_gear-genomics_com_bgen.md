##### How to use bgen

1. Select the desired barcode length (4–16).
2. Set how many barcodes to generate (2–1000).
3. Optionally paste candidate sequences (one per line, A/C/G/T only, all same length).
4. Click **Launch Analysis** to generate barcodes.

Switch to the **Results** tab to see generated sequences, summary stats, and the base frequency chart.

Use **Show Example** to load a sample run.
