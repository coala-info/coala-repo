It is designed to help run the artic bioinformatics protocols; an SOP for running command-line viral amplicon sequencing bioinformatics is available on the ARTIC website. . Features include: read filtering primer trimming amplicon coverage normalisation variant calling consensus building Installation Via conda conda install -c bioconda -c conda-forge artic Please make sure you are using either mamba or conda version &gt;= 23.10.0 where libmamba solver was made the default conda solver. Via source 1. downloading the source: git clone https://github.com/artic-network/fieldbioinformatics cd fieldbioinformatics 2. installing dependencies: The artic pipeline has several software dependencies . You can solve these dependencies using the minimal conda environment we have provided, Please make sure you are using either mamba or conda version &gt;= 23.10.0 where libmamba solver was made the default conda solver. : conda env create -f environment.yml conda activate artic 3. installing the pipeline: pip install . 4. testing the pipeline: First check the pipeline can be called. artic -v You can try the pipeline tests. ./test-runner.sh clair3 For further tests, such as the variant validation tests, check the documentation .
