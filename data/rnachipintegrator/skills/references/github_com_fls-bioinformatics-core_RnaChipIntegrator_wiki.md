The individual analyses differ from each other according to the criteria that are used to calculate the distances between peaks and transcripts. The program was originally written specifically for ChIP-Seq and RNA-Seq data but can be used with any set of genomic features e.g. canonical genes, CpG islands etc, or expression data e.g. microarrays. See the project's homepage at http://fls-bioinformatics-core.github.com/RnaChipIntegrator/ for download, installation and usage information.
