# bioconductor-nanoporernaseq

## Overview
The `NanoporeRNASeq` package provides a curated collection of long-read RNA-Seq data generated by the SG-NEx project. It includes direct RNA and cDNA sequencing data from K562 and MCF7 cell lines, specifically aligned to human chromosome 22 (GRCh38). This package serves as an experiment data resource for demonstrating long-read analysis workflows, particularly for transcript discovery and isoform quantification using tools like `bambu`.

## Core Workflows

### 1. Accessing Sample Metadata and Data
The package provides metadata for the SG-NEx samples and uses `ExperimentHub` to manage large BAM and FASTA files.

```r
library(NanoporeRNASeq)
library(ExperimentHub)
library(Rsamtools)

# View sample metadata
data("SGNexSamples")
print(SGNexSamples)

# Query and retrieve BAM files from ExperimentHub
eh <- ExperimentHub()
nanopore_query <- query(eh, c("NanoporeRNA", "GRCh38", "Bam"))
# Example: Loading specific Hub IDs into a BamFileList
bam_files <- BamFileList(nanopore_query[["EH3808"]], nanopore_query[["EH3809"]])
```

### 2. Loading Annotations
Pre-processed annotations for human chromosome 22 are included for use with the `bambu` package.

```r
data("HsChr22BambuAnnotation")
# This returns a GRangesList object
HsChr22BambuAnnotation
```

### 3. Visualizing Coverage and Alignments
You can visualize specific genes using `ggbio` and `BSgenome` resources.

```r
library(ggbio)
library(BSgenome.Hsapiens.NCBI.GRCh38)

# Select a gene range (e.g., MAPK1)
gene_range <- HsChr22BambuAnnotation$ENST00000215832

# Create tracks
tx_track <- autoplot(gene_range, aes(col = strand), group.selfish = TRUE)
cov_track <- autoplot(bam_files[[1]], aes(col = coverage), which = gene_range)

# Combine tracks
tracks(annotation = tx_track, coverage = cov_track, heights = c(1, 3)) + theme_minimal()
```

### 4. Integration with Bambu
The primary use case for this data is to run transcript discovery and quantification workflows.

```r
library(bambu)

# Get genome sequence
fasta_query <- query(eh, c("NanoporeRNA", "GRCh38", "FASTA"))
genome_seq <- fasta_query[["EH7260"]]

# Run bambu analysis
se <- bambu(reads = bam_files, 
            annotations = HsChr22BambuAnnotation, 
            genome = genome_seq)

# Visualize results (annotated and novel isoforms)
plotBambu(se, type = "annotation", gene_id = "ENSG00000099968")
```

## Tips and Best Practices
- **Memory Management**: When working with `BamFileList`, remember that these point to remote or cached files. Ensure you have sufficient disk space in your `ExperimentHub` cache directory.
- **Chromosome Limitation**: Note that the data in this package is subsetted to **Chromosome 22**. Analysis attempted on other chromosomes using these specific BAM files will yield no results.
- **Object Types**: The `bambu` function returns a `SummarizedExperiment` object. Use `assays(se)` to access counts, CPM, and full-length read data.
