# manormfast

## Overview

MAnormFast is a robust tool for the quantitative comparison of ChIP-Seq experiments. It addresses the challenge of comparing binding intensities across different cell types or states by normalizing signal levels between samples. This allows researchers to identify cell-specific binding events and correlate these changes with gene expression or specific regulatory activity.

## Command Line Usage

The primary interface is the `MAnormFast` command (or `python MAnorm.py`).

### Basic Syntax
```bash
MAnormFast --p1 <peak_file1> --r1 <read_file1> --p2 <peak_file2> --r2 <read_file2> -o <output_folder>
```

### Required Arguments
- `--p1`, `--p2`: Peak files for sample 1 and sample 2.
- `--r1`, `--r2`: Read files (mapped tags) for sample 1 and sample 2.
- `-o`: Name of the directory where results will be stored.

## Peak File Preparation

MAnormFast is flexible with input formats but requires specific column ordering:

1.  **Standard BED**: Supported by default.
2.  **3-Column Tab-Split**: Must be ordered as `chromosome`, `start`, `end`.
3.  **4-Column Tab-Split**: Must be ordered as `chromosome`, `start`, `end`, `summit`.
4.  **MACS Output**: Supports the `.xls` files generated by the MACS peak caller.

### Format Example (3-column)
```text
chr1    2345    4345
chr1    3456    5456
chr2    6543    8543
```

## Best Practices and Tips

- **Summit Information**: If your peak caller provides summit positions, use the 4-column format. Including the summit in the fourth column improves the precision of the comparison.
- **Read Files**: Ensure that the read files (`--r1`, `--r2`) correspond exactly to the samples used to generate the peak files (`--p1`, `--p2`).
- **Help Command**: Use `MAnormFast --help` to view additional parameters for fine-tuning the normalization model.
- **Installation**: If the command is not found, ensure you have run `python setup.py install` and that the `/bin/MAnormFast` script is in your system PATH with executable permissions (`chmod +x`).
