Multiple fields can be searched simultaneously by adding more search rows '+'. To reset search fields, hit the 'Reset' button. To execute the search, hit the 'Advanced Search' button. Partial matches are allowed, though matches need to be exact. Multiple search queries are strict AND. Empty fields will be ignored in the search.
