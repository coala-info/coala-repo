[`4eb8b34`](/gifford-lab/deepaccess-package/commit/4eb8b34acab682072bcd33b45ee43c5522eb469a)

This is the first stable release of deepaccess. The binaries released here were packaged with [pyinstaller](https://www.pyinstaller.org/) and include all Python dependencies including Python itself. The only dependency you will need is bedtools. These binaries have been tested for Ubuntu 18.04.3 LTS and are unlikely to work for other operating systems.

The deepaccess package is also available as a python package on [PyPI](https://pypi.org/project/deepaccess/), and [bioconda](https://anaconda.org/bioconda/deepaccess). The DeepAccess system provide a code base for training and interpreting an ensemble of convolutional neural networks for multi-task prediction of functional epigenomic signal. Installation and instructions for running the deepaccess package are available at <https://cgs.csail.mit.edu/deepaccess-package/>.
