# WF5_Data_Interpretation_Worklow

## Overview

This workflow provides a rigorous statistical framework for interpreting quantitative proteomic datasets generated by MaxQuant. By leveraging the [MSstatsTMT](https://toolshed.g2.bx.psu.edu/repos/galaxyp/msstatstmt/msstatstmt/2.0.0+galaxy1) toolset, it enables researchers to perform sophisticated protein signaling and expression analysis across various experimental conditions.

The pipeline requires several key inputs, including MaxQuant protein groups, evidence files, and quantified peptides, as well as a comparison matrix and an annotation file. These data sources are processed through filtering steps using selection tools to ensure only high-quality, relevant data is passed to the statistical modeling phase.

Beyond statistical testing, the workflow integrates [Unipept](https://toolshed.g2.bx.psu.edu/repos/galaxyp/unipept/unipept/4.5.1) for functional and taxonomic analysis of the identified peptides. This comprehensive approach allows for deep biological insights into TMT (Tandem Mass Tag) experiments, following established [GTN](https://training.galaxyproject.org/) methodologies. ## How to run (Planemo)

### Local Galaxy (default engine)

```bash
planemo run wf5-data-interpretation-worklow.ga job.yml --download_outputs --output_directory . --output_json output.json
```

Optional: `--galaxy_root /path/to/galaxy` speeds up spinning up a local Galaxy.

### Docker Galaxy

Append e.g. `--engine docker_galaxy --ignore_dependency_problems` (requires Docker).

### External Galaxy (usegalaxy.org / EU / …)

```bash
planemo run wf5-data-interpretation-worklow.ga job.yml --engine external_galaxy --galaxy_url SERVER_URL --galaxy_user_key YOUR_API_KEY
```

Create a reusable profile: `planemo profile_create myprofile --galaxy_url ... --galaxy_user_key ... --engine external_galaxy` then `planemo run wf5-data-interpretation-worklow.ga job.yml --profile myprofile`.

### Job file

- Generate a template: `planemo workflow_job_init wf5-data-interpretation-worklow.ga -o job.yml`
- Lint: `planemo workflow_lint wf5-data-interpretation-worklow.ga`

See **[Running Galaxy workflows](https://planemo.readthedocs.io/en/master/running.html)** in the Planemo documentation.
