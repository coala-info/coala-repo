The framework is based on the pyspark library for working with big data. As part of the simulation process, the framework includes data generators, response functions and other tools that allow flexible use of the simulator. Synthetic tabular data generation is not a purpose of the Sit4Rec framework. Sim4Rec provides an API and wrappers to run simulations with synthetic data, but the method of synthetic data generation is determined by the user. The SDV package is imported for illustration purposes and can be replaced by another synthetic data generation solution. Thus, synthetic data generation functionality and quality evaluation is provided by the SDV library, namely SDVDataGenerator from generator.py and evaluate_synthetic from evaluation.py should only be used for non-production purposes according to the SDV licence.
