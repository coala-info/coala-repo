# PhyloDeep Data

This folder contains data used for the Figures and Tables in the PhyloDeep manuscript:

## Preprint

Voznica J, Zhukova A, Boskova V, Saulnier E, Lemoine F, Moslonka-Lefebvre M, Gascuel O (2021)
**Deep learning from phylogenies to uncover the transmission dynamics of epidemics**. [bioRxiv](https://www.biorxiv.org/content/10.1101/2021.03.11.435006v1)

## Figures

### Fig\_3 :

* you will find the test trees (100 per BD, BDEI and BDSS) in newick format in the folder [Fig\_3/test\_trees](/evolbioinfo/phylodeep/blob/main/data_publication/Fig_3/test_trees)
* you will find the predicted values (100 per model (BD, BDEI and BDSS) and per method (BEAST2, FFNN-SS, CNN-CBLV together with target values)) in the folder [Fig\_3/predicted\_values](/evolbioinfo/phylodeep/blob/main/data_publication/Fig_3/predicted_values)
* you will find all the BEAST2 logs (100 per model) in the folder [Fig\_3/BEAST2\_log](/evolbioinfo/phylodeep/blob/main/data_publication/Fig_3/BEAST2_log)

### Fig\_4 :

* Fig4 has its own [README](/evolbioinfo/phylodeep/blob/main/data_publication/Fig_4/README.md)

### Fig\_5 :

* you will find the BEAST2 logs for HIV phylogeny for each model in the folder [Fig\_5/BEAST2\_logs](/evolbioinfo/phylodeep/blob/main/data_publication/Fig_5/BEAST2_logs)
* you will find the values inferred with FFNN-SS and CNN-CBLV together with CI values in the folder [Fig\_5/inferred\_values](/evolbioinfo/phylodeep/blob/main/data_publication/Fig_5/inferred_values)

### Supp\_Fig\_2 :

* you will find the test trees (100 per model (BD, BDEI)) in newick format in the folder [Supp\_Fig\_2/test\_trees](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_2/test_trees)
* you will find the predicted values (100/10,000 per model (BD, BDEI) and per method (BEAST2, FFNN-SS, CNN-CBLV together with target values)) in the folder [Supp\_Fig\_2/predicted\_values](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_2/predicted_values)
* you will find all the BEAST2 logs (100 per model (BD, BDEI)) in the folder [Supp\_Fig\_2/BEAST2\_logs](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_2/BEAST2_logs)

### Supp\_Fig\_3 :

* you will find predicted and target values (1,000 sets per network (FFNN-SS, CNN-CBLV) and per model (BD, BDEI and BDSS)) in the folder [Supp\_Fig\_3](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_3)

### Supp\_Fig\_4 :

* Supp\_Fig\_4 has its own [README](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_4/README.md)

### Supp\_Fig\_5 :

* you will find predicted and target values for the BDSS model (10,000 sets per network (CNN-CBLV, CNN-CRV and FFNN-CBLV) and per training set size setting (10K, 100K, 1M and 4M)
  in the folder [Supp\_Fig\_5/BDSS\_large](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_5/BDSS_large)

### Supp\_Fig\_6 :

* you will find predicted and target values (10,000 per model (BD, BDEI, BDSS) and method (FFNN-SS, CNN-CBLV, FFNN-SS-NULL and linear regression)
  with different training set size (10K, 100K, 1M and 4M)) together with predicted values by BEAST2 (100 sets) in the folder [Supp\_Fig\_6/predicted\_values](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_6/predicted_values)

### Supp\_Fig\_7 :

* you will find predicted values for the BDSS model (10,000 sets per network (CNN-CBLV, FFNN-SS and FFNN-origSS) in the folder [Supp\_Fig\_7](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Fig_7)

## Tables

### Supp\_Table\_1\_2 :

* you will find the test trees (10,000 per model (BD, BDEI, BDSS) and per tree size (small and big trees)) in newick format in the folder [Supp\_Table\_1\_2/test\_trees](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Table_1_2/test_trees)
* you will find the predicted values (100/10,000 per model (BD, BDEI, BDSS) and per method (BEAST2, FFNN-SS, CNN-CBLV, linear regression, FFNN-CBLV, FFNN-SS-NULL together with target values))
  in the folder [Supp\_Table\_1\_2/predicted\_values](/evolbioinfo/phylodeep/blob/main/data_publication/Supp_Table_1_2/test_trees)
