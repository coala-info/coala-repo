# Install GRAFIMO via Bioconda (Linux users only)

[Jump to bottom](#wiki-pages-box)

ManuelTgn edited this page Dec 10, 2020
·
[2 revisions](/pinellolab/GRAFIMO/wiki/Install-GRAFIMO-via-Bioconda-%28Linux-users-only%29/_history)

To install GRAFIMO via Bioconda the user should first have the `conda` package installed. **If the user have an Anaconda Python installation, `conda` is already available**, otherwise it can be installed with **[Miniconda](https://conda.io/en/latest/miniconda.html)** package. For further details (<https://bioconda.github.io/user/install.html>).

Once `conda` is available, to install GRAFIMO type:

```
conda install grafimo
```

To update GRAFIMO:

```
conda update grafimo
```
