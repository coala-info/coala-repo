To use jcvi as a generic Bioinformatics library, please follow the instructions on the project page . This wiki hosts standalone applications within the library that are capable of solving domain-specific problems. Applications ALLMAPS ALLMAPS anchors genomic scaffolds onto linkage groups to reconstruct chromosomal assemblies. MCscan MCscan facilitates pairwise and multiple genome syntenic comparisons and visualizations. GRABSEEDS GRABSEEDS facilitates computerized extraction of object features and text labels from digital images in large phenotyping experiments. More Case studies Genome features and landscape plotting A suite of visualization routines in the package that might be useful to create customized plots. Genome build quality control A suite of tools including genome size survey, genetic map and Hi-C heatmap to check for quality of genome build. Pedigree and genome diversity A suite of tools including pedigree analysis and genomic variations between varieties. FAQ Plotting failed with error message "No such file or directory: 'latex'" Please take a look at Get LaTeX for various options for each major OS.
