# raiss.ld\_matrix[](#module-raiss.ld_matrix "Link to this heading")

Function set to compute LD correlation from a reference panel
in predefined Region

LD matrix are then transformed to the pandas sparse format

Functions

|  |  |
| --- | --- |
| `generate_genome_matrices`(region\_files, ...) | go through region files and compute LD matrix for each transform and save the results in a pandas sparse dataframe |
| `generate_sparse_matrix`(plink\_ld, ref\_chr\_df) | Extract correlation matrix from the plink correlation file generated by ld\_matrix.launch\_plink\_ld read plink results create a sparse dataframe LD-matrix then save it to a zipped pickle |
| `launch_plink_ld`(startpos, endpos, chr, ...) | launch plink linkage desiquilibrium correlation and save the ouput |
