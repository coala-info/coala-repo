### Developers

* Correct IDEA configuration files are now created by Mill so we can run test coverage in IntelliJ ([#83](https://github.com/clintval/cvbio/pull/83))
* Checksum calculating input streams ([#75](https://github.com/clintval/cvbio/pull/75)) and implicits ([#80](https://github.com/clintval/cvbio/pull/80)) for creating checksum hashes as side-effects to consuming byte streams
