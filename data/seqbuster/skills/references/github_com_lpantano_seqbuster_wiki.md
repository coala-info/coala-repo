Ignore this section. Go to the main README for the current working tool Welcome to the seqbuster wiki! small RNA analysis of deep sequencing data. SeqBuster offers three tools to analyze all your small RNA data coming from Illumina sequencing. Installation JAVA visual interface version: seqbusterviz &amp; Quick Start adapter removal : remove 3' adapter miraligner : miRNA annotation using miRBase database R package isomiRs to visualize isomiRs and miRNA abundances in a group of samples ( new ) seqcluster analyzes the rest of small RNAs and annotate to multiple databases in bed format ( new ) SeqBuster team updates periodically the tools with more information about how to be used and new applications. If you have any comment please use: mailing-list . Also visit our blog or facebook or twitter to be up-to-day of the new information added in the wiki.
