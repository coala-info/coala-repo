# cdskit intersection

[Jump to bottom](#wiki-pages-box)

Kenji Fukushima edited this page Dec 20, 2024
·
[2 revisions](/kfuku52/cdskit/wiki/cdskit-intersection/_history)

`cdskit intersection` drops non-overlapping sequence labels between two sequences files or between a sequence file and a gff file.

### Example 1: Sequence vs sequence intersection

#### Command

```
cdskit intersection --seqfile input1.fasta --seqfile2 input2.fasta --outfile output1.fasta --outfile2 output2.fasta
```

#### input1.fasta

```
>seq1
AAAAAAAAAA
>seq2
TTTTTTTTTT
>seq3
GGGGGGGGGG
>seq4
CCCCCCCCCC
```

#### input2.fasta

```
>seq3
GGGGGGGGGGCCCCCCCCCC
>seq4
CCCCCCCCCCATATATATAT
>seq5
ATATATATATAGAGAGAGAG
>seq6
AGAGAGAGAGGGGGGGGGGG
```

#### output1.fasta

```
>seq3
GGGGGGGGGG
>seq4
CCCCCCCCCC
```

#### output2.fasta

```
>seq3
GGGGGGGGGGCCCCCCCCCC
>seq4
CCCCCCCCCCATATATATAT
```

### Example 2: Sequence vs gff intersection

#### Command

```
cdskit intersection --seqfile input.fasta --ingff input.gff --outfile output.fasta --outgff output.gff
```

#### input.fasta

```
>seq1
AAAAAAAAAA
>seq2
TTTTTTTTTT
>seq3

>seq4
CCCCCCCCCC
```
