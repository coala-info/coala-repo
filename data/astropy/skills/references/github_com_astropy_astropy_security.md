But if that does not work for whatever reason, please report to the @astropy/security-team that is also listed on the Astropy Team listing . We will respond as soon as we are able. If vulnerability is accepted, we will work on a hotfix. The timing depends on the available resources and the severity/risk of the vulnerability.
