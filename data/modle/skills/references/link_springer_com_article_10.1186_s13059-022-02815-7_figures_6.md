# Fig. 6

From: [MoDLE: high-performance stochastic modeling of DNA loop extrusion interactions](/article/10.1186/s13059-022-02815-7)

![Fig. 6](//media.springernature.com/full/springer-static/image/art%3A10.1186%2Fs13059-022-02815-7/MediaObjects/13059_2022_2815_Fig6_HTML.png)

Optimization of individual barriers and computation of barrier and LEF profiles. **A** Micro-C (hESC) data from a 5Mb region on chromosome 1 (20–25 Mbp). **B** MoDLE output for the same region, where individual barriers are optimized from Micro-C data. **C** MoDLE output for the same region using Rad21 ChIP-seq data as input, **D** Computed barrier occupancy profile from MoDLE trained on Micro-C data (normalized with *P**UU* = 0.7). **E** Computed LEF occupancy profile from MoDLE trained on Micro-C data. **F** CTCF ChIP-seq data from the same region. **G** Rad21 ChIP-seq data from the same region
