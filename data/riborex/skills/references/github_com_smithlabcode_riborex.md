Online Paper | PDF | Supplementary File DEPENDENCIES DESeq2 edgeR fdrtool INSTALLATION We strongly recommend that you install Riborex via conda: conda install -c bioconda r-riborex To install locally, please make sure you have DESeq2 and edgeR installed. Then start R and enter: # # try http:// if https:// URLs are not supported source( " https://bioconductor.org/biocLite.R " ) biocLite( " DESeq2 " ) biocLite( " edgeR " ) biocLite( ' fdrtool ' ) To install Riborex, download the latest version "riborex-x.x.x.tar.gz" from releases at https://github.com/smithlabcode/riborex , start a terminal and CD into the directory where you downloaded Riborex, start R and enter install.packages( " riborex-x.x.x.tar.gz " , repos = NULL , type = " source " ) Alternatively, you could also install devtools package and then install riborex to get latest changes : install.packages( ' devtools ' ) library( devtools ) options( unzip = ' internal ' ) devtools :: install_github( ' smithlabcode/riborex ' ) DOCUMENTATION Please refer to vignettes/riborex.pdf for how to use riborex.
