## Installation

This is an [R](https://www.r-project.org/) package.

```
install.packages(
    pkgs = "AcidGenerics",
    repos = c(
        "https://r.acidgenomics.com",
        getOption("repos")
    ),
    dependencies = TRUE
)
```
