Source code and binaries can be found at &lt;https://www.github.com/szpiech/selscan&gt;. selscan currently implements EHH, iHS, XP-EHH, nSL, and XP-nSL. To calculate EHH: ./selscan --ehh &lt;locusID&gt; --vcf &lt;vcf&gt; --map &lt;mapfile&gt; --out &lt;outfile&gt; To calculate iHS: ./selscan --ihs --vcf &lt;vcf&gt; --map &lt;mapfile&gt; --out &lt;outfile&gt; To calculate nSL: ./selscan --nsl --vcf &lt;vcf&gt; --out &lt;outfile&gt; To calculate XP-nSL: ./selscan --xpnsl --vcf &lt;vcf&gt; --vcf-ref &lt;vcf&gt; --out &lt;outfile&gt; To calculate iHH12: ./selscan --ihh12 --vcf &lt;vcf&gt; --map &lt;mapfile&gt; --out &lt;outfile&gt; To calculate XP-EHH: ./selscan --xpehh --vcf &lt;vcf&gt; --vcf-ref &lt;vcf&gt; --map &lt;mapfile&gt; --out &lt;outfile&gt; ----------Command Line Arguments---------- --alt &lt;bool&gt;: Set this flag to calculate homozygosity based on the sum of the squared haplotype frequencies in the observed data instead of using binomial coefficients. Default: false --cutoff &lt;double&gt;: The EHH decay cutoff. Default: 0.05 --ehh &lt;string&gt;: Calculate EHH of the '1' and '0' haplotypes at the specified locus. Output: &lt;physical dist&gt; &lt;genetic dist&gt; &lt;'1' EHH&gt; &lt;'0' EHH&gt; Default: __NO_LOCUS__ --ehh-win &lt;int&gt;: When calculating EHH, this is the length of the window in bp in each direction from the query locus. Default: 100000 --gap-scale &lt;int&gt;: Gap scale parameter in bp. If a gap is encountered between two snps &gt; GAP_SCALE and &lt; MAX_GAP, then the genetic distance is scaled by GAP_SCALE/GAP. Default: 20000 --hap &lt;string&gt;: A hapfile with one column per haplotype, and one row per variant. Variants should be coded 0/1 Default: __hapfile1 --help &lt;bool&gt;: Prints this help dialog. Default: false --ihh12 &lt;bool&gt;: Set this flag to calculate iHH12. Default: false --ihs &lt;bool&gt;: Set this flag to calculate iHS. Default: false --ihs-detail &lt;bool&gt;: Set this flag to write out left and right iHH scores for '1' and '0' in addition to iHS. Default: false --keep-low-freq &lt;bool&gt;: Include low frequency variants in the construction of your haplotypes. Default: false --maf &lt;double&gt;: If a site has a MAF below this value, the program will not use it as a core snp. Default: 0.05 --map &lt;string&gt;: A mapfile with one row per variant site. Formatted &lt;chr#&gt; &lt;locusID&gt; &lt;genetic pos&gt; &lt;physical pos&gt;. Default: __mapfile --max-extend &lt;int&gt;: The maximum distance an EHH decay curve is allowed to extend from the core. Set &lt;= 0 for no restriction. Default: 1000000 --max-extend-nsl &lt;int&gt;: The maximum distance an nSL haplotype is allowed to extend from the core. Set &lt;= 0 for no restriction. Default: 100 --max-gap &lt;int&gt;: Maximum allowed gap in bp between two snps. Default: 200000 --multi-param &lt;string&gt;: Specify a JSON file with multiple parameter sets. Each set should match the structure of command-line arguments. The program will run the analysis for each set, generating separate outputs. Useful for batch processing and exploring different configurations. Default: __jsonFile --nsl &lt;bool&gt;: Set this flag to calculate nSL. Default: false --out &lt;string&gt;: The basename for all output files. Default: outfile --pi &lt;bool&gt;: Set this flag to calculate mean pairwise sequence difference in a sliding window. Default: false --pi-win &lt;int&gt;: Sliding window size in bp for calculating pi. Default: 100 --pmap &lt;bool&gt;: Use physical map instead of a genetic map.
