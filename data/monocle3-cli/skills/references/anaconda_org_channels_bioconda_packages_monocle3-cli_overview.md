monocle3-cli - bioconda | Anaconda.org CMD + K Sign In Sign Up bioconda / monocle3-cli monocle3-cli Community A set of wrappers for individual components of the monocle3 package. Functions R packages are hard to call when building workflows outside of R, so this package adds a set of simple wrappers with robust argument parsing. Intermediate steps are currently monocle3 R objects, but the ultimate objective is to have language-agnostic intermediate formats allowing composite workflows using a variety of software packages.
