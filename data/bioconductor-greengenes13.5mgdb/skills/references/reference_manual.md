greengenes13.5MgDb-package

Greengenes 16S rRNA database MgDb Package.

Description

This package loads a MgDb-class object with sequence and taxonomic data for the greengenes
database version 13.5, http://greengenes.secondgenome.com. Vignettes for working with MgDb-class
objects are in the metagenomeFeatures package.
