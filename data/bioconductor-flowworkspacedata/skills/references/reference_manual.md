flowWorkspaceData-package

A flowJo XML Workspace and Associated Data for testing the flow-
Workspace package.

Description

This package contains fcs data files and xml workspaces for testing the flowWorkspace and open-
Cyto packages and building the their vignettes. These can be located in the inst/extdata directory of
the flowWorkspaceData package. The sample files and workspaces contain data from whole blood.
The primary purpose of this data set is to test the import code and export code in the flowWorkspace
package and running automated gating pipeline in the openCyto package.
