## Description

This workflow query metagenomic raw data against a metaplasmidome database to identify plasmids and annotate them with genes, KO, PFAM

## Associated Tutorial

This workflows is part of the tutorial [Query an annotated mobile genetic element database to identify and annotate genetic elements (e.g. plasmids) in metagenomics data](https://training.galaxyproject.org/training-material/topics/microbiome/tutorials/metaplasmidome_query/tutorial.html), available in the [GTN](https://training.galaxyproject.org)
