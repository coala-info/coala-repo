It's also very fast and memory friendly, just so you know. For an introduction and further documentation, see doc/main.txt . For installation information, see INSTALL.txt . For issue tracker, see https://bugs.launchpad.net/lxml Support the project lxml has been downloaded from the Python Package Index millions of times and is also available directly in many package distributions, e.g. for Linux or macOS. Most people who use lxml do so because they like using it. You can show us that you like it by blogging about your experience with it and linking to the project website.
