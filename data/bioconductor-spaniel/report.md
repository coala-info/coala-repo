# bioconductor-spaniel Bioconductor Package Report

## Spaniel

### Package Description
The package contains functions to create a SingleCellExperiment Seurat object and provides a method of loading a histologial image into R. The spanielPlot function allows visualisation of metrics contained within the S4 object overlaid onto the image of the tissue.
