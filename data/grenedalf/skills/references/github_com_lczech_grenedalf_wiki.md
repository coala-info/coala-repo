grenedalf is a collection of commands for working with population genetic data.

Many commands in grenedalf are re-implementations of commands of the
[PoPoolation](https://sourceforge.net/projects/popoolation/) and
[PoPoolation2](https://sourceforge.net/projects/popoolation2/) tools.
However, being written in C++, grenedalf is much faster and needs less memory.

# Command Line Interface

grenedalf is used via its command line interface, with commands for each task.
The commands have the general structure:

```
grenedalf <command> [options]
```

Use `grenedalf --help` to get an overview of all commands, and use `grenedalf <command> --help` for the list of options of each command. The pages below also list these options, along with additional information on each command.

# Commands

| Command | Description |
| --- | --- |
| [cathedral-plot](wiki/../wiki/Subcommand%3A-cathedral-plot) | Create a cathedral plot, using the pre-computated cathedral data. |
| [citation](wiki/../wiki/Subcommand%3A-citation) | Print references to be cited when using grenedalf. |
| [diversity](wiki/../wiki/Subcommand%3A-diversity) | Compute pool-sequencing corrected diversity measures Theta Pi, Theta Watterson, and Tajima's D. |
| [frequency](wiki/../wiki/Subcommand%3A-frequency) | Create a table with per-sample and/or total base counts and/or frequencies at positions in the genome. |
| [fst](wiki/../wiki/Subcommand%3A-fst) | Compute pool-sequencing corrected measures of FST. |
| [fst-cathedral](wiki/../wiki/Subcommand%3A-fst-cathedral) | Compute the data for an FST cathedral plot. |
| [license](wiki/../wiki/Subcommand%3A-license) | Show the license of grenedalf. |
| [simulate](wiki/../wiki/Subcommand%3A-simulate) | Create a file with simulated random frequency data. |
| [sync](wiki/../wiki/Subcommand%3A-sync) | Create a sync file that lists per-sample base counts at each position in the genome. |
| [version](wiki/../wiki/Subcommand%3A-version) | Extended version information about grenedalf. |
