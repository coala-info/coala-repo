## Description

Galaxy Workflow created on Galaxy-E european instance, ecology.usegalaxy.eu, related to the Galaxy training tutorial "[Metabarcoding/eDNA through Obitools](https://training.galaxyproject.org/training-material/topics/ecology/tutorials/Obitools-metabarcoding/tutorial.html)" .

This workflow allows to analyze DNA metabarcoding / eDNA data produced on Illumina sequencers using the OBITools.
