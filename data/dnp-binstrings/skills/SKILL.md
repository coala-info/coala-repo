# dnp-binstrings

## Overview
`dnp-binstrings` is a high-performance C++ utility designed for genomic sequence preprocessing. It transforms standard FASTA sequences into a binary format where a '1' represents the occurrence of a specific dinucleotide pair and a '0' represents its absence. This tool serves as the foundational step in the `dnpatterntools` workflow, enabling researchers to investigate nucleosome positioning sequence (NPS) patterns by converting raw sequence data into a format suitable for statistical signal processing.

## Installation
The tool is most easily installed via the Bioconda channel:
```bash
conda install -c bioconda dnp-binstrings
```

## Command Line Usage
The tool requires a FASTA file as input and the specification of a target dinucleotide.

### Basic Syntax
```bash
dnp-binstrings [OPTIONS] "input.fa"
```

### Identifying Specific Dinucleotides
Use the `-di` or `--dinucleotide` flag to specify which pair to map.
```bash
# Generate a binary string for AA dinucleotides
dnp-binstrings -di AA sequences.fa

# Generate a binary string for CG dinucleotides
dnp-binstrings -di CG sequences.fa
```

### Supported Dinucleotides
The tool accepts all standard DNA dinucleotide combinations:
- AA, AC, AG, AT
- CA, CC, CG, CT
- GA, GC, GG, GT
- TA, TC, TG, TT

## Expert Tips and Best Practices
- **Disable Version Checks**: In automated pipelines or batch environments, use `--version-check 0` to prevent the tool from attempting to check for updates, which can cause delays or failures in restricted network environments.
- **Pipeline Integration**: `dnp-binstrings` outputs the binary string directly to standard output. To use this in a workflow, redirect the output to a file:
  ```bash
  dnp-binstrings -di AA input.fa > aa_binary_map.txt
  ```
- **Data Preparation**: Ensure your FASTA sequences are cleaned of non-ACGT characters (like Ns) if possible, as the tool is designed to identify specific dinucleotide matches within the standard genomic alphabet.
- **Downstream Compatibility**: The binary output generated by this tool is specifically formatted to be consumed by other tools in the suite, such as `dnp-diprofile` for computing frequency profiles across aligned sequences.
