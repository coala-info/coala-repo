# bioconductor-tenxpbmcdata

## Overview

The `TENxPBMCData` package provides a collection of ten single-cell RNA-seq (scRNA-seq) and CITE-seq datasets of peripheral blood mononuclear cells (PBMC) generated by 10X Genomics. Data is delivered as `SingleCellExperiment` objects. To minimize memory usage, the count matrices are stored on disk as HDF5 files and accessed via `DelayedArray` objects, loading only required subsets into memory.

## Loading Datasets

Use the `TENxPBMCData()` function to download and cache datasets from ExperimentHub.

```r
library(TENxPBMCData)

# Load a specific dataset (e.g., 4k cells)
sce <- TENxPBMCData(dataset = "pbmc4k")
```

### Available Datasets
The `dataset` argument accepts the following identifiers:
- `pbmc3k`, `pbmc4k`, `pbmc6k`, `pbmc8k`: Standard scRNA-seq datasets.
- `pbmc33k`, `pbmc68k`: Larger scRNA-seq datasets (68k is often used for scaling tests).
- `frozen_pbmc_donor_a`, `frozen_pbmc_donor_b`, `frozen_pbmc_donor_c`: Donor-specific frozen samples.
- `pbmc5k-CITEseq`: Includes both transcriptomics and antibody capture data.

## Working with the Data

### Memory Management
Because the data is HDF5-backed, you can control the amount of data loaded into memory during operations by setting the `DelayedArray` block size.

```r
# Set block size to 1 GB for faster processing of large datasets
options(DelayedArray.block.size = 1e9)
```

### Accessing Counts
The primary assay is `counts`. For CITE-seq data, use `altExp` to access the antibody protein data.

```r
# Access RNA counts
rna_counts <- counts(sce)

# Access Antibody counts (for pbmc5k-CITEseq)
cite_sce <- TENxPBMCData(dataset = "pbmc5k-CITEseq")
prot_counts <- counts(altExp(cite_sce))
```

### Basic Exploration
Standard `SummarizedExperiment` and `SingleCellExperiment` methods apply:
- `rowData(sce)`: Contains Ensembl IDs and Gene Symbols.
- `colData(sce)`: Contains cell-level metadata (Sample, Barcode, etc.).
- `counts(sce)`: Returns a `DelayedMatrix`.

## Common Workflows

### Calculating Summary Statistics
Perform calculations without loading the entire matrix into memory:

```r
# Library sizes
lib_sizes <- colSums(counts(sce))

# Number of genes expressed per cell
n_genes <- colSums(counts(sce) != 0)

# Average expression per gene
avg_expr <- rowMeans(counts(sce))
```

### Saving and Loading
When saving the object, R stores the metadata and a pointer to the HDF5 file.

```r
# Save the object
saveRDS(sce, file = "pbmc_data.rds")

# Load in a new session (requires the original HDF5 cache to be present)
sce <- readRDS("pbmc_data.rds")
```
