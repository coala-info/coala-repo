affylmGUI Package Vignette

James Wettenhall

April 6, 2004

affylmGUI is a Graphical User Interface for doing Affymetrix analysis with the limma
package (LInear Models for MicroArray Data). For Affymetrix analysis with the limma
package, see limmaGUI.

The affylmGUI Documentation can be accessed online or by installing the affylmGUI

package locally. Then at the R prompt you can type

> if (interactive())
+

browseURL("http://bioinf.wehi.edu.au/affylmGUI/Doc/")

or alternatively

> if (interactive())
+ {
+
+
+ }

library(affylmGUI)
affylmGUIhelp()

or alternatively

> if (interactive())
help.start()
+

and follow the links the affylmGUI package help.
