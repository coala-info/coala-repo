Strudel is currently available for: Windows (32 bit) or Windows (64 bit) Linux (32 bit) or Linux (64 bit) OS X Follow the instructions provided by the graphical installer to install Strudel. Note that Strudel should not require administrator privileges to install or run, but if you have any problems, don’t hesitate to contact us at strudel@hutton.ac.uk . Strudel is primarily developed on Windows 10, and gets additional regular testing on OS X and CentOS. Archived versions You can download a limited selection of older copies of Strudel from our archive . If you do need to return to an older version (for whatever reason), please let us know why.
