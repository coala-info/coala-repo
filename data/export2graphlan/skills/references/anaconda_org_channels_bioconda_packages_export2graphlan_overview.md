It automatically generate the input tree and the annotation file for GraPhlAn, starting from the input/output of MetaPhlAn, LEfSe, and HUMAnN. It supports also the biom file format. The annotation file will highlight specific sub-trees/clades automatically inferred from input file(s) provided. The two output file of export2graphlan should then be used with GraPhlAn.
