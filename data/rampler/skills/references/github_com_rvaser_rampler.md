It supports two modes, random subsampling of sequencing data to a desired depth (given the reference length) and file splitting to desired size in bytes. Usage To build rampler run the following commands: git clone --recursive https://github.com/rvaser/rampler.git rampler cd rampler &amp;&amp; mkdir build &amp;&amp; cd build cmake -DCMAKE_BUILD_TYPE=Release .. &amp;&amp; make ./bin/rampler which will display the following usage: usage: rampler [options ...] &lt; mode &gt; &lt; mode &gt; subsample &lt; sequences &gt; &lt; reference length &gt; &lt; coverage &gt; [ &lt; coverage &gt; ...] &lt; sequences &gt; input file in FASTA/FASTQ format (can be compressed with gzip) &lt; reference length &gt; integer denoting length of the reference genome (or assembly) &lt; coverage &gt; integer denoting desired coverage of the subsampled sequences split &lt; sequences &gt; &lt; chunk size &gt; &lt; sequences &gt; input file in FASTA/FASTQ format (can be compressed with gzip) containing sequences which will be split into smaller chunks &lt; chunk size &gt; integer denoting the desired chunk size in bytes options: -o, --out-directory &lt; string &gt; default: current directory path in which sampled files will be created --version prints the version number -h, --help prints out the help Dependencies gcc 4.8+ or clang 4.0+ cmake 3.9+ zlib Acknowledgment This work has been supported in part by Croatian Science Foundation under the project UIP-11-2013-7353.
