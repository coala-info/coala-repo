# The chemfp command-line tools[](#the-chemfp-command-line-tools "Link to this heading")

See [Command-line examples for binary fingerprints](tools.html#using-tools) and [Command-line examples for sparse count fingerprints](count_tools.html#using-count-tools) for examples of
how to use these commands.

The chemfp command-line tools are:

> * [fpcat](fpcat_command.html#fpcat) - merge multiple fingerprint files into one
> * [cdk2fps](cdk2fps_command.html#cdk2fps) - use CDK to generate fingerprints
> * [ob2fps](ob2fps_command.html#ob2fps) - use Open Babel to generate fingerprints
> * [oe2fps](oe2fps_command.html#oe2fps) - use OEChem/OEGraphSim to generate fingerprints
> * [rdkit2fps](rdkit2fps_command.html#rdkit2fps) - use RDKit to generate fingerprints
> * [sdf2fps](sdf2fps_command.html#sdf2fps) - extract fingerprints from an SD file
> * [simsearch](simsearch_command.html#simsearch) - search a fingerprint file for similar fingerprints
> * [chemfp](chemfp_command.html#chemfp-command) - top-level command for additional chemfp commands

The [chemfp](chemfp_command.html#chemfp-command) command implements many subcommands:

> * [chemfp butina](chemfp_butina_command.html#chemfp-butina) - cluster using the Butina algorithm
> * [chemfp csv2fps](chemfp_csv2fps_command.html#chemfp-csv2fps) - generate fingerprints from a CSV file
> * [chemfp fpb\_text](chemfp_fpb_text_command.html#chemfp-fpb-text) - show the TEXT sections of an FPB file
> * [chemfp fpc2fps](chemfp_fpc2fps_command.html#chemfp-fpc2fps) - convert sparse count fingerprints to binary fingerprints
> * [chemfp fps2fpc](chemfp_fps2fpc_command.html#chemfp-fps2fpc) - convert binary fingerprints to count fingerprints
> * [chemfp heapsweep](chemfp_heapsweep_command.html#chemfp-heapsweep) – heapsweep diversity picking
> * [chemfp license](chemfp_license_command.html#chemfp-license) - show the chemfp license status
> * [chemfp maxmin](chemfp_maxmin_command.html#chemfp-maxmin) - MaxMin diversity picking
> * [chemfp rdkit2fpc](chemfp_rdkit2fpc_command.html#chemfp-rdkit2fpc) - generate count fingerprints using RDKit
> * [chemfp report](chemfp_report_command.html#chemfp-report) - report chemfp configuration details
> * [chemfp shardsearch](chemfp_shardsearch_command.html#chemfp-shardsearch) - similarity search of multiple target files
> * [chemfp simarray](chemfp_simarray_command.html#chemfp-simarray) - full all-by-all array generation
> * [chemfp simhistogram](chemfp_simhistogram_command.html#chemfp-simhistogram) - generate histograms from the similarity scores.
> * [chemfp spherex](chemfp_spherex_command.html#chemfp-spherex) - spherical exclusion diversity picking
> * [chemfp toolkits](chemfp_toolkits_command.html#chemfp-toolkits) - list the available cheminformatics toolkits
> * [chemfp translate](chemfp_translate_command.html#chemfp-translate) - translate between two structure formats
