# Standards

The Arvados community is collaborating closely with several standards efforts.

## Common Workflow Language

The goal of the [CWL Project](https://www.commonwl.org/) is to create specifications that enable data scientists to describe analysis tools and workflows that are powerful, easy to use, portable, and support reproducibility. CWL is the native workflow language for Arvados.

![Common Workflow Language logo](/images/cwl/CWL-Logo-Header-Cropped.png)

## Global Alliance

The Global Alliance for Genomics and Health (GA4GH) is a global standards body defining data formats and APIs for precision medicine.
