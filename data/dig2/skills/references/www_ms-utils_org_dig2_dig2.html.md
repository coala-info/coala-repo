About dig2 dig2 is a simple but flexible in silico digester of protein sequences in the FASTA format. It allows for almost any enzyme to be simulated, including "MS/MS" enzymes to generate CID or ECD/ETD fragments. Download dig2 here . (Right-click and choose "Save Lin k As...".) Compiling dig2 Compile dig2 with: gcc -O3 -o dig2 dig2.c -lm Using dig2 Usage: dig2 &lt; FASTA file &gt; &lt; settings &gt; where the &lt; FASTA file &gt; contains the sequences to be digested and &lt; settings &gt; defines the enzyme and modifications. dig2 -h shows additional information. magnus.palmblad@gmail.com
