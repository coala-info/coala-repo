Cite Us ProMod3 modelling - Protein Modelling ¶ High-level functionality for protein modelling. The goal is to model a given target sequence (or list of sequences for oligomers) given some template data. Commonly, the template does not cover the full target. This module offers capabilities to extract useful template data for the target and to fill the remaining structural data to create a full model of the target. In its simplest form, you can use a target-template alignment and a template structure to create a model fully automatically as follows: from ost import io from promod3 import modelling # get raw model tpl = io . LoadPDB ( &#39;data/1crn_cut.pdb&#39; ) aln = io . LoadAlignment ( &#39;data/1crn.fasta&#39; ) aln . AttachView ( 1 , tpl . CreateFullView ()) mhandle = modelling . BuildRawModel ( aln ) # build final model final_model = modelling . BuildFromRawModel ( mhandle ) io . SavePDB ( final_model , &#39;model.pdb&#39; ) The various steps involved in protein modelling are described here: Modelling Pipeline Build Raw Modelling Handle The Default Pipeline Modelling Steps Alignment Fiddling Model Checking Detecting Ring Punches Detecting Non-Planar Rings Model Checking With MolProbity Handling Gaps Gap classes Gap Extender classes Gap Handling Functions Handling Loop Candidates The LoopCandidates class Keeping track of loop candidate scores Example: loop scoring in modelling Fitting Loops Into Gaps CCD KIC Relaxing Backbones Relaxing All Atom Loops Generating Loops De Novo SampleMonteCarlo() Sampler Object Closer Object Scorer Object Cooler Object Sidechain Reconstruction Reconstruct Function SidechainReconstructor Class The SidechainReconstructionData class Modelling Algorithms Rigid Blocks De Novo Modelling Motif Finder AFDB Modelling Search Go Enter search terms or a module, class or function name.
