# ezaai

## Overview
EzAAI is a specialized bioinformatics suite designed for rapid and accurate calculation of Average Amino Acid Identity (AAI) between prokaryotic genomes. It improves upon traditional AAI methods by utilizing MMseqs2 for high-speed protein searches and Prodigal for automated protein extraction. This skill enables users to manage the full comparative genomics workflow—from raw genome sequences to clustered dendrograms—using a modular CLI approach.

## Core Workflows

### 1. Protein Extraction
Before calculating AAI, you must extract protein sequences from your prokaryotic genome files (FASTA).
- **Command**: `ezaai extract -i <INPUT_GENOMES> -o <OUTPUT_DB>`
- **Best Practice**: Use a directory as input (`-i`) to batch process multiple genomes into a single protein database directory.

### 2. AAI Calculation
Calculate the identity between two protein databases or sets of databases.
- **Command**: `ezaai calculate -i <DB_1> -j <DB_2> -o <RESULT_FILE>`
- **Expert Tip**: If you have multiple genomes, point `-i` and `-j` to the same directory of databases to perform an all-by-all pairwise comparison.

### 3. Database Conversion
If you already have Coding Sequence (CDS) files, convert them directly into the MMseqs2 format used by EzAAI.
- **Command**: `ezaai convert -i <CDS_FASTA> -s <type> -o <OUT_DB>`
- **Parameters**: Set `-s` to `nucl` for nucleotide sequences or `prot` for amino acid sequences.

### 4. Hierarchical Clustering
Generate a clustering analysis based on the AAI results to understand taxonomic relationships.
- **Command**: `ezaai cluster -i <AAI_TABLE> -o <CLUSTER_OUTPUT>`
- **Requirement**: The input must be a result file containing all-by-all pairwise AAI values generated by the `calculate` module.

## CLI Reference Summary

| Module | Purpose | Key Arguments |
| --- | --- | --- |
| `extract` | Genome to Protein DB | `-i` (Input), `-o` (Output) |
| `calculate` | Compute AAI | `-i` (DB 1), `-j` (DB 2), `-o` (Output) |
| `convert` | FASTA to DB | `-i` (CDS), `-s` (nucl/prot), `-o` (Output) |
| `cluster` | Taxa Clustering | `-i` (AAI Table), `-o` (Output) |



## Subcommands

| Command | Description |
|---------|-------------|
| ezaai calculate | Calculate AAI value from protein databases |
| ezaai cluster | Hierarchical clustering of taxa with AAI values |
| ezaai convert | Convert CDS FASTA file into protein DB |
| ezaai convertdb | Convert protein DB into FASTA file |
| ezaai extract | Extract protein DB from prokaryotic genome sequence(s) using Prodigal |
