Select encoding scale: Sanger Solexa Illumina 1.3+ Illumina 1.5+ Illumina 1.8+ See the FASTQ format documentation for more details on encoding scales. Enter the ASCII string or Phred value you want to convert in the box below: Multiple Phred values can be entered separated by commas.
