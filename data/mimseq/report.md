 Modification-induced misincorporation analysis of tRNA sequencing data

usage: mimseq [options] sample data

Custom high-throughput tRNA sequencing alignment and quantification pipeline
based on modification induced misincorporation cDNA synthesis.

positional arguments:
  sampledata            Sample data sheet in text format, tab-separated.
                        Column 1: full path to fastq (or fastq.gz). Column 2:
                        condition/group.

optional arguments:
  -h, --help            show this help message and exit
  --version             Show version number and exit

Input files:
  -s species, --species species
                        Species being analyzed for which to load pre-packaged
                        data files (prioritized over -t, -o and -m). Options
                        are: Hsap, Hsap19, Mmus, Rnor, Scer, Spom, Dmel, Drer,
                        Cele, Ecol, Atha
  -t genomic tRNAs, --trnas genomic tRNAs
                        Genomic tRNA fasta file, e.g. from gtRNAdb or
                        tRNAscan-SE. Already avalable in data folder for a few
                        model organisms.
  -o tRNA out file, --trnaout tRNA out file
                        tRNA.out file generated by tRNAscan-SE (also may be
                        available on gtRNAdb). Contains information about tRNA
                        features, including introns.
  -m mitochondrial tRNAs, --mito-trnas mitochondrial tRNAs
                        Mitochondrial tRNA fasta file(s). Should be downloaded
                        from mitotRNAdb for species of interest. Already
                        available in data folder for a few model organisms.
  -p plastid tRNAs, --plastid-trnas plastid tRNAs
                        Plastid tRNA fasta file(s). Should be downloaded from
                        PtRNAdb for species of interest. Already available in
                        data folder for a few model organisms.

Program options:
  --pretRNAs            Input reference sequences are pretRNAs. Enabling this
                        option will disable the removal of intron sequences
                        and addition of 3'-CCA to generate mature tRNA
                        sequences. Useful for mapping and discovering pretRNA
                        sequence reads.
  --no-cluster          Disable usearch sequence clustering of tRNAs by
                        isodecoder which drastically reduces the rate of
                        multi-mapping reads. Default is enabled.
  --cluster-id [clustering identity threshold]
                        Identity cutoff for usearch clustering between 0 and
                        1. Default is 0.97.
  --deconv-cov-ratio [deconvolution coverage threshold]
                        Threshold for ratio between coverage at 3' end and
                        mismatch used for deconvolution. Coverage reductions
                        greater than the threshold will result in non-
                        deconvoluted sequences. Default is 0.5 (i.e. less than
                        50% reduction required for deconvolution).
  --threads thread number
                        Set processor threads to use during read alignment and
                        read counting.
  --posttrans-mod-off   Disable post-transcriptional modification of tRNAs,
                        i.e. addition of 3'-CCA and 5'-G (His) to mature
                        sequences. Disable for certain prokaryotes (e.g. E.
                        coli) where this is genomically encoded. Leave enabled
                        (default) for all eukaryotes.
  --control-condition control condition
                        Name of control/wild-type condition as per user
                        defined group specified in sample data input. This
                        must exactly match the group name specified in sample
                        data. This is used for differential expression
                        analysis so that results are always in the form
                        mutant/treatment vs WT/control. REQUIRED
  --no-cca-analysis     Disable analysis of 3'-CCA ends. When enabled, this
                        calculates proportions of CC vs CCA ending reads per
                        cluster and performs DESeq2 analysis. Useful for
                        comparing functional to non-functional mature tRNAs.
                        Default is enabled.
  --double-cca          Enable analysis of 3'-CCACCA tagging for tRNA
                        degradation pathway. Note that this will alter the
                        output of the CCA analysis pipeline.
  --local-modomics      Disable retrieval of Modomics data from online.
                        Instead use older locally stored data. Warning - this
                        leads to usage of older Modomics data!
  --p-adj P_ADJ         Adjusted p-value threshold for DESeq2 pairwise
                        condition differential epxression dot plots. tRNAs
                        with DESeq2 adjusted p-values equal to or below this
                        value will be displayed as green or orange triangles
                        for up- or down-regulated tRNAs, respectively. Default
                        p-adj <= 0.05
  --crosstalks          Enable analysis of crosstalks between pairs of
                        modifications and modification-charging. Full details
                        of this method in:
                        https://doi.org/10.1093/nar/gkac1185

GSNAP alignment options:
  --max-mismatches allowed mismatches
                        Maximum mismatches allowed. If specified between 0.0
                        and 1.0, then treated as a fraction of read length.
                        Otherwise, treated as integer number of mismatches.
                        Default is an automatic ultrafast value calculated by
                        GSNAP; see GSNAP help for more info.
  --remap-mismatches allowed mismatches for remap
                        Maximum number of mismatches allowed during remapping
                        of all reads. Treated similarly to --max-mismatches.
                        This is important to control misalignment of reads to
                        similar clusters/tRNAs Note that the SNP index will be
                        updated with new SNPs from the first round of
                        alignment and so this should be relatively small to
                        prohibit misalignment.
  --no-snp-tolerance    Disable GSNAP SNP-tolerant read alignment, where known
                        modifications from Modomics are mapped as SNPs.
                        Default is enabled.

Output options:
  -n experiment name, --name experiment name
                        Name of experiment. Note, output files and indices
                        will have this as a prefix. REQUIRED
  --out-dir output directory
                        Output directory. Default is current directory. Cannot
                        be an existing directory.
  --keep-temp           Keeps multi-mapping and unmapped bam files from GSNAP
                        alignments. Default is false.

Bedtools coverage options:
  --min-cov Minimum coverage per cluster
                        Minimum coverage per cluster required to include this
                        cluster in coverage plots, modification analysis, and
                        3'-CCA analysis. Can be a fraction of total mapped
                        reads between 0 and 1, or an integer of absolute
                        coverage. Any cluster not meeting the threshold in 1
                        or more sample will be excluded. Note that all
                        clusters are included for differential expression
                        analysis with DESeq2. Default = 0.0005 (0.05% mapped
                        reads).
  --max-multi Bedtools coverage multithreading
                        Maximum number of bam files to run bedtools coverage
                        on simultaneously. Increasing this number reduces
                        processing time by increasing number of files
                        processed simultaneously. However, depending on the
                        size of the bam files to process and available memory,
                        too many files processed at once can cause termination
                        of mim-tRNAseq due to insufficient memory. If mim-
                        tRNAseq fails during coverage calculation, lower this
                        number. Increase at your own discretion. Default is 3.

Analysis of unannotated modifications and realignment:
  --remap               Enable detection of unannotated (potential)
                        modifications from misincorporation data. These are
                        defined as having a total misincorporation rate higher
                        than the threshold set with --misinc-thresh. These
                        modifications are then appended to already known ones,
                        and read alignment is reperformed. Very useful for
                        poorly annotated species in Modomics. Due to
                        realignment and misincorporation parsing, enabling
                        this option slows the analysis down considerably.
  --misinc-thresh [threshold for unannotated mods]
                        Threshold of total misincorporation rate at a position
                        in a cluster used to call unannotated modifications.
                        Value between 0 and 1, default is 0.1 (10%
                        misincorporation).
