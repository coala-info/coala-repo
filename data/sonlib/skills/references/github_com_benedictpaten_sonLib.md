Author Benedict Paten Description For basic data-structures in C it contains: array lists (automatically resizing) hashes sets (backed by hashes) sorted sets (backed by an AVL tree) containers (for putting primitives into the above data-structures) better string functions simple exception mechanism logging functions For bioinformatic data-structures it contains (or will contain): Fasta I/O functions MAF I/O functions Newick tree I/O functions For more details see the Full README in doc/README About Small general purpose library for C and Python with focus on bioinformatics.
