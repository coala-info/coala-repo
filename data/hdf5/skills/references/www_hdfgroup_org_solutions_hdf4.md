#### HDF4® Library

|  |  |
| --- | --- |
| Technical Details | |
| License: | [BSD-style](https://www.hdfgroup.org/licenses/) |
| Official Media Type: | [application/vnd.hdfgroup.hdf4](https://www.iana.org/assignments/media-types/application/vnd.hdfgroup.hdf4) |
| Standard File Extension: | .h4, .hdf4, .hdf |

HDF4® is a library and multi-object file format for storing and managing data between machines. HDF4 is a very different technology than [HDF5](/hdf5/). The HDF Group recommends that, other than for working with existing HDF4 data, new applications use HDF5 since HDF5 addresses important deficiencies of HDF4.

Although The HDF Group maintains HDF4 by fixing defects and porting it to new operating environments, there is no longer any new development occurring. HDF4 is considered deprecated and new products should be developed in [HDF5](https://www.hdfgroup.org/solutions/hdf5/).

If you’re interested in converting your data from HDF4 to HDF5, our engineers can help. Learn more about our [consulting service](/solutions/consulting/) or free and open source [h4toh5 conversion tools](https://support.hdfgroup.org/downloads/h4h5tools/h4h5tools_2_2_5.html).

NASA maintains a network of [Earth Observing Satellites](/eos_vignette/) that transmit huge volumes of new imagery data daily in HDF4.  As a result, HDF4 is used extensively in the Earth Sciences. The HDF-EOS Website, <http://www.hdfeos.org> has a wealth of information on these uses.

You can [download HDF4](https://support.hdfgroup.org/downloads/#downloads-hdf4), explore HDF4-related utilities and access HDF4 documentation [here](https://support.hdfgroup.org/documentation/#hdf4). Licensing and copyrights for HDF4 are [here](https://hdfgroup.org/licenses/).
