## Get Started

Apptainer provides an abundance of resources to help you get started.

GitHub Project
:   Grab the latest stable release of Apptainer, submit issues, and contribute to the project at the official GitHub
    repository.

    [Visit Project →](https://github.com/apptainer/apptainer)

Quick Start Guide
:   Check out the quick start documentation to get Apptainer installed and set up in a hurry.

    [Get Started →](https://apptainer.org/docs/user/main/quick_start.html)

Binary Packages
:   Pre-built packages are available for download as Assets under each GitHub release for some operating systems.

    [View Releases →](https://github.com/apptainer/apptainer/releases)

Support
:   Apptainer has both freely available support resources, as well as commercial support offerings to help.

    [Get Support →](/support)
