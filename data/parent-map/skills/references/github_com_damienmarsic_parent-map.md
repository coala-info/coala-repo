Run parent-map with arguments to start in console mode. Run parent-map.py -h for help and list of arguments. Install from Bioconda: conda install -c bioconda parent-map Install from PyPI: pip install parent-map Note that you may need to individually install some dependencies. If you get an error message about missing modules when trying to run parent-map after installing it with pip, install them using pip install.
