# Text Analysis

* [bleve - Modern text indexing library for go.](/text-analysis/bleve-blevesearch-github/)
* [go-adaptive-radix-tree - Go implementation of Adaptive Radix Tree.](/text-analysis/go-adaptive-radix-tree-plar-github/)
* [go-edlib - Go string comparison and edit distance algorithms library (Levenshtein, LCS, Hamming, Damerau levenshtein, Jaro-Winkler, etc.) compatible with Unicode.](/text-analysis/go-edlib-hbollon-github/)
* [levenshtein - Levenshtein distance and similarity metrics with customizable edit costs and Winkler-like bonus for common prefix.](/text-analysis/levenshtein-agext-github/)
* [levenshtein - Implementation to calculate levenshtein distance in Go.](/text-analysis/levenshtein-agnivade-github/)
* [mspm - Multi-String Pattern Matching Algorithm for information retrieval.](/text-analysis/mspm-blackrabbitt-github/)
* [parsefields - Tools for parse JSON-like logs for collecting unique fields and events.](/text-analysis/parsefields-monaxgt-github/)
* [ptrie - An implementation of prefix tree.](/text-analysis/ptrie-viant-github/)
* [trie - Trie implementation in Go.](/text-analysis/trie-derekparker-github/)
* [graphlib - Topological sort lib,Sorting and pruning of DAG graphs.](/text-analysis/graphlib-aio-arch-github/)
* [hashsplit - Split byte streams into chunks, and arrange chunks into trees, with boundaries determined by content, not position.](/text-analysis/hashsplit-bobg-github/)
* [merkle - Space-efficient computation of Merkle root hashes and inclusion proofs.](/text-analysis/merkle-bobg-github/)
* [skiplist - Very fast Go Skiplist implementation.](/text-analysis/skiplist-mauricegit-github/)
* [skiplist - Skiplist implementation in Go.](/text-analysis/skiplist-gansidui-github/)
* [treemap - Generic key-sorted map using a red-black tree under the hood.](/text-analysis/treemap-igrmk-github/)
* [ordered-concurrently - Go module that processes work concurrently and returns output in a channel in the order of input.](/text-analysis/ordered-concurrently-tejzpr-github/)
* [parapipe - FIFO Pipeline which parallels execution on each stage while maintaining the order of messages and results.](/text-analysis/parapipe-nazar256-github/)
* [pipeline - An implementation of pipelines with fan-in and fan-out.](/text-analysis/pipeline-hyfather-github/)
* [pipelines - Generic pipeline functions for concurrent processing.](/text-analysis/pipelines-nxdir-s-github/)
