# 1 Installation

You can install the current release version of NetActivityData with:

```
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

BiocManager::install("NetActivityData")
```

You can install the development version of NetActivity from [GitHub](https://github.com/) with:

```
# install.packages("devtools")
devtools::install_github("yocra3/NetActivityData")
```

```
library(NetActivityData)
```

# 2 Introduction

This package contains pre-trained models for their use with *[NetActivity](https://github.com/yocra3/NetActivity)*. The package currently contains two models based on GO Biological Processes and KEGG pathways. One model was trained using GTEx data (`gtex_gokegg`) and the other with TCGA data (`tcga_gokegg`). Each model contains two objects: the matrix weights and the gene set annotation.
