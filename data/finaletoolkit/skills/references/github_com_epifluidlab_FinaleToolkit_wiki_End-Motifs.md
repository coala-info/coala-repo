## `end-motifs`

### Let's retrieve the 5' end 5-mer (`-k 5`) motif frequency (`end-motifs`) from fragments that have a MAPQ score greater than 30 (`-q 30`). We will output (`-o`) to BH01.22.em.tsv, using one worker (`-w 1`) and verbose mode (`-v`).

```
finaletoolkit end-motifs BH01.hg19.mdups.22.bam hg19.2bit \
  -q 30 \
  -k 5 \
  -o BH01.22.em.tsv \
  -w 1 \
  -v
```

#### Our TSV output should look something like this:

```
...
AAATT	0.0009592995774273066
AACAA	0.0012111236856193225
AACAC	0.0005582025168567324
AACAG	0.0007852540545502932
...
```

### Now let's say we want to do this but only on one strand (`-B`) and on the negative strand (`-n`)

```
finaletoolkit end-motifs BH01.hg19.mdups.22.bam hg19.2bit \
  -q 30 \
  -k 5 \
  -Bn \
  -o BH01.22.neg.em.tsv \
  -w 1 \
  -v
```

Our results should look like this:

```
AAAAA   0.0037266275227639937
AAAAC   0.0008130111658572833
AAAAG   0.0012784866972448238
AAAAT   0.0017660930355443527
...
```

## `interval-end-motifs`

### Let's retrieve the 5' end 4-mer (`-k 4`) motif frequency across intervals (`interval-end-motif`) of size 100kb across chromosome 22 from fragments that have a MAPQ score greater than 30 (`-q 30`), are longer than 50 bp (`-min 50`), and shorter than 250bp (`-max 250`). We will output (`-o`) the file to BH01.100kb.22.em.tsv, using one worker (`-w 1`) and verbose mode (`-v`).

```
finaletoolkit interval-end-motifs BH01.hg19.mdups.22.frag.gz hg19.2bit 100kb.22.bins \
  -q 30 \
  -k 4 \
  -min 50 \
  -max 250 \
  -o BH01.100kb.22.em.tsv \
  -w 1 \
  -v
```
