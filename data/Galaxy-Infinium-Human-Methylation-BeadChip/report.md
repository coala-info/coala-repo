## Description

The workflow containes the step to perpare the data for and run Galaxy Infinium Human Methylation BeadChip tool that represents many popular algorithms for detecting somatic genetic alterations from genome and exome data

## Associated Tutorial

This workflows is part of the tutorial [Infinium Human Methylation BeadChip](https://training.galaxyproject.org/training-material/topics/epigenetics/tutorials/ewas-suite/tutorial.html), available in the [GTN](https://training.galaxyproject.org)
