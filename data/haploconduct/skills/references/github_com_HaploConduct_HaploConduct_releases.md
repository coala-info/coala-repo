[HaploConduct version 0.1 / SAVAGE 0.4.1](/HaploConduct/HaploConduct/releases/tag/v0.4.1)

Minor fixes in SAVAGE pipeline:

* fixed random seed when splitting data into patches
* allow paired-end read tags /1 /2 at the end of sequence identifiers
* print bwa errors to stdout instead of /dev/null
* handle empty overlap sets and empty contig sets
* optionally, store output in user-specified directory instead of working directory
