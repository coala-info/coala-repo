Converts BC/QT barcode SAM tags into haplotag RX, QX and BX tags. See https://github.com/evolgenomics/haplotagging or https://www.fml.tuebingen.mpg.de/chan-group/haplotagging/?L=1 for details on haplotagging. Also comes with a couple of tools: '10xSpoof' for converting haplotag barcodes into 10x compatible barcodes '16BaseBCGen' for converting haplotag barcodes into generic 16-base barcodes with 7-base joins (useful for passing to programs like ema ) Bioconda SamHaplotag is available on bioconda . &gt; conda install samhaplotag Example Usage &gt; samtools view -h@ 16 -F 0xF00 reads.cram | SamHaplotag | samtools view -@ 16 -o tagged_reads.cram &gt; samtools fastq -@ 16 -nT BX -0 /dev/null -s /dev/null tagged_reads.cram | 10xSpoof SamHaplotag_Clear_BC | bgzip -@ 16 &gt; 10x_spoofed_reads.fq.gz &gt; samtools fastq -@ 16 -nT BX -0 /dev/null -s /dev/null tagged_reads.cram | 16BaseBCGen | bgzip -@ 16 &gt; 16BaseBC_reads.fq.gz &gt; cut -f 2 HaploTag_to_16BaseBCs | tail -n +2 &gt; 16BaseBCs Installation Requires: clang &gt;= 11.0.0 meson &gt;= 0.57.1 &gt; env CXX=clang meson setup --buildtype=release --prefix= &lt; installation prefix &gt; builddir &gt; cd builddir &gt; meson compile &gt; meson test &gt; meson install About Process haplotag barcodes in SAM format Resources Readme License MIT license Uh oh!
