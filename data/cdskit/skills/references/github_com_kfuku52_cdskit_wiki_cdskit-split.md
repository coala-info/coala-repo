# cdskit split

[Jump to bottom](#wiki-pages-box)

Kenji Fukushima edited this page Dec 26, 2024
·
[1 revision](/kfuku52/cdskit/wiki/cdskit-split/_history)

`cdskit split` splits 1st, 2nd, and 3rd codon positions of the input codon sequences.

### Example

#### Command

```
cdskit split --seqfile input.fasta
```
