It is currently written in C++ and Python , with support for R . You can download binaries compiled for different systems. Straw works by reading the .hic file, finding the appropriate matrix and slice of data, and printing the text in sparse upper triangular format. It is not as fully featured as the Java Juicebox command line tools ; in particular, it doesn't store the pointer data for all the matrices, only the one queried, and currently we are only supporting matrices (not vectors). Future versions may look more like the Java implementation. Special thanks to Yue Wu for the initial Python port. For questions, please use the Google Group .
