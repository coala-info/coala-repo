# Dependencies:

* python 3.7+
* biopython v.1.78+
* wget v.1.19+ (downloading reference datasets)
* ncbi-blast v.2.10.1+ (aligning nucleotide sequences)
* diamond v.2.0.6+ (aligning amino acid sequences)
* hmmer v.3.3.1+ (detecting conserved marker genes)
* barrnap v.0.9+ (detecting ribosomal RNA genes)
* aragorn v 1.2.38+ (detecting tRNA genes)
* prodigal v 2.6.3 (ORF/CDS-prediction)
  MDMcleaner only runs on UNIX plattforms

# Installing MDMcleaner (and all dependencies) Option 1: ![](https://camo.githubusercontent.com/fb0f49bd0665a2369ec8b541fb3d1fa00f50874bae95ea256a05169e7d3573e9/68747470733a2f2f616e61636f6e64612e6f72672f62696f636f6e64612f6d646d636c65616e65722f6261646765732f696e7374616c6c65722f636f6e64612e737667)

**Creating a dedicated environment:**

* run `conda create -n mdmcleaner_env -c bioconda mdmcleaner`
* then run ``` conda activate mdmcleaner_env``´ to activate and ``` conda deactivate``` to deactivate
  **Installing into currently active conda environment:**
* run `conda install -c bioconda mdmcleaner`

# Installing MDMcleaner Option 2: installing via pip [![PyPI version](https://camo.githubusercontent.com/57656d9a3b0498bae43418d4db11ec033e81d67f91e90d931d58541d2a2faa98/68747470733a2f2f62616467652e667572792e696f2f70792f6d646d636c65616e65722e737667)](https://badge.fury.io/py/mdmcleaner)

**A.)** install the [dependencies](#Dependencies).

**B.)** run  `pip install mdmcleaner`

# Installing MDMcleaner Option 3: downloading or cloning this repository (to get the most recent development version) [![GitHub version](https://camo.githubusercontent.com/6bff5877ccbafd8ef9e02b6ae77eb6e4dccb5fae89aaf80aec905e344a0201dd/68747470733a2f2f62616467652e667572792e696f2f67682f4b49542d4942472d352532466d646d636c65616e65722e737667)](https://badge.fury.io/gh/KIT-IBG-5/mdmcleaner)

**A.) first download and install the [above listed dependencies](#Dependencies)**

**B.) download the mdmcleaner source code**

The basic mdmcleaner pipeline can simply be downloaded from the github\_repository.

This can be done, e.g. by downloading the most recent version as [zip-archive](https://github.com/KIT-IBG-5/mdmcleaner/archive/refs/heads/master.zip) and then uncompressing it:

```
wget https://github.com/KIT-IBG-5/mdmcleaner/archive/refs/heads/master.zip && unzip master.zip && rm master.zip && mv mdmcleaner-master mdmcleaner
```

*Alternatively* you could simply clone this repository using git:

```
git clone https://github.com/KIT-IBG-5/mdmcleaner.git
```

**C.) create local wheel and install from that**

* enter the repository folder: `cd mdmcleaner`
* create wheel: `python setup.py bdist_wheel`
* install from local wheel file: `pip install dist/*.whl`
