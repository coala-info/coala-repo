# blue-crab release v0.5.0

Supports the addition of `open_pore_level` aux value
Provides some flexibility for when all the scaling values are deprecated in a future pod5 version
Changed how and where the package is installed in site-packages. Will require a clean install.
Tested with python3.10-14

**Please uninstall any previous version of blue-crab or create a new venv as the packaging method has changed.**
