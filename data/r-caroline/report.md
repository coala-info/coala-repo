# caroline: A Collection of Database, Data Structure, Visualization, and
Utility Functions for R

The caroline R library contains dozens of functions useful
 for: database migration (dbWriteTable2), database style joins &amp;
 aggregation (nerge, groupBy, &amp; bestBy), data structure
 conversion (nv, tab2df), legend table making (sstable &amp; leghead),
 automatic legend positioning for scatter and box plots (),  
 plot annotation (labsegs &amp; mvlabs), data visualization 
 (pies, sparge, confound.grid &amp; raPlot), character string manipulation (m &amp; pad),
 file I/O (write.delim), bat...
