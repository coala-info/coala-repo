# ucsc-psltobed

## Overview
The `pslToBed` utility is a specialized bioinformatics tool designed to bridge the gap between alignment data and interval-based analysis. PSL files, typically generated by the BLAT aligner, provide a detailed record of how a sequence matches a genome, including match counts and block coordinates. This skill allows for the transformation of these complex alignment records into the more streamlined BED format, which is the standard for representing genomic features and is compatible with a wide array of downstream analysis tools and genome browsers.

## Command Line Usage

### Basic Conversion
The primary usage pattern involves specifying the input PSL file and the desired output BED filename.
```bash
pslToBed input.psl output.bed
```

### Integration with Sorting
BED files are most useful when sorted by chromosome and start position. It is a best practice to sort the output immediately if you plan to use it with tools like `bedtools` or convert it to `bigBed`.
```bash
pslToBed input.psl stdout | bedSort stdin output.sorted.bed
```

## Expert Tips and Best Practices

### Handling Blocked Alignments
`pslToBed` preserves the internal structure of alignments. The resulting BED file is typically in BED12 format, where the "blocks" in the PSL (representing exons or alignment segments) are converted into BED blocks. If your analysis requires individual segments as separate entries, you should follow this tool with `bedToExons`.

### Environment Setup
If you have downloaded the binary directly from the UCSC Genome Browser repository rather than installing via a package manager like Bioconda, you must ensure the binary has the correct permissions to run:
```bash
chmod +x ./pslToBed
./pslToBed input.psl output.bed
```

### Validation
Before conversion, you can verify the integrity of your PSL file using `pslCheck` (another UCSC utility) to ensure that the coordinates and block counts are consistent, preventing errors during the BED transformation.

### Large Scale Processing
For high-throughput workflows, `pslToBed` is highly efficient and can be used within shell loops or parallel processing frameworks to convert thousands of alignment files rapidly, as it does not require a heavy runtime environment or database connection for standard file-to-file conversion.
