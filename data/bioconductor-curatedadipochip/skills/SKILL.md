# bioconductor-curatedadipochip

## Overview

Use the Bioconductor R package **curatedAdipoChIP** for: The package document the data collection, pre-processing and processing of the data. In addition to the documentation, the package contains the scripts that was used to generated the data.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("curatedAdipoChIP")
```
