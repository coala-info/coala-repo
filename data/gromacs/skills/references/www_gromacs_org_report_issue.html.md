Report an issue &#8212; GROMACS webpage https://www.gromacs.org documentation Report an issue ¶ GROMACS has a completely open policy concerning bugs and issues. All open and closed issues can be found on GitLab . When you (think you) have found an issue, follow the following steps: Check if you are using the latest version. If not, check the release notes of all later versions to see if your issue has already been fixed. The information is available at https://manual.gromacs.org/ . Check if your issues has been reported in one of the open or closed issues. You can search GROMACS issues on GitLab . You can search the user discussion section of the GROMACS forum to see if your issue is already being discussed. If you are not using the latest (minor) GROMACS release, download it here and try it to see if the issue has already been fixed, but has not been reported or has been reported with different symptoms. If you issue has not been reported yet and it is in a still supported release (or is a critical correctness issue in an older release), please report it on GitLab . Please carefully follow the instructions provided there, so others now which versions are affected and developers can find and fix the issue.
