## minclookup

[[edit](/w/index.php?title=MINC/Resources&veaction=edit&section=1 "Edit section: minclookup") | [edit source](/w/index.php?title=MINC/Resources&action=edit&section=1 "Edit section's source code: minclookup")]

Here is an example of a double ended lookup table that you can use with files output from programs like mincblob to make pretty images.

```
0.000 0.8 0.8 1.0
0.125 0.4 0.8 1.0
0.250 0.0 0.4 1.0
0.375 0.0 0.0 0.5
0.500 0.0 0.0 0.0
0.625 0.5 0.0 0.0
0.750 1.0 0.4 0.0
0.825 1.0 0.8 0.4
1.000 1.0 0.8 0.8
```

Note that in order to use this you will have to ensure that your input file to minclookup has a symmetric range (ie: ranges from -x to x).
