This repository contains mofapy2 Python library source code. For the downstream analysis in Python please check the mofax package: https://github.com/bioFAM/mofax For the downstream analysis in R please check the MOFA2 package: https://github.com/bioFAM/MOFA2 Please visit our website for details, tutorials, and much more.
