seurat-scripts 4.x.y for Seurat 4.4.0 In order to wrap Seurat's internal workflow in any given workflow language, it's important to have scripts to call each of those steps, which is what this package provides. This version of seurat-scripts uses native conversions to Loom, SCE, Seurat H5 and AnnData. Please note that given the little man power available to maintain this, we dropped MacOS support for Seurat version 4.4.0 onwards (not all tests were passing and there was no bandwidth to take care of the dependencies issues on both linux and macOS). Install The recommended method for usage is through the biocontainer produced by Bioconda. If needed, scripts installation is via a Bioconda recipe called seurat-scripts. With the Bioconda channels configured the latest release version of the package can be installed via the regular conda install command: conda install seurat-scripts Test installation There is a test script included: r-seurat-scripts-post-install-tests.sh This downloads a number of datasets and executes all of the scripts described below and more. Commands Currently wrapped Seurat functions are described below. Each script has usage instructions available via --help, consult function documentation in Seurat for further details. These instructions might be currently outdated for Seurat 4.0.0, the best source of information is --help on each command. Read10X(): read 10X data and create Seurat object from the matrix seurat-read-10x.R -d &lt;10x data directory&gt; -o &lt;output matrix object in .rds format&gt; FilterCells(): filter out poor-quality cells seurat-filter-cells.R -i &lt;raw Seurat object in .rds format&gt; -s nGene,nUMI -l &lt;min_genes&gt;,&lt;min_umi&gt; -o &lt;output Seurat object in .rds format&gt; NormalizeData(): normalise the expression values seurat-normalise-data.R -i &lt;filtered Seurat object in .rds format&gt; -a &lt;assay type&gt; -n &lt;normalisation method&gt; -s &lt;scale_factor&gt; -o &lt;normalised Seurat object in .rds format&gt; FindVariableGenes(): find variable genes seurat-find-variable-genes.R -i &lt;normalised Seurat object in .rds format&gt; -m &lt;mean function&gt; -d &lt;dispersion function&gt; -l &lt;fvg x low cutoff&gt; -j &lt;fvg x high cutoff&gt; -y &lt;fvg y low cutoff&gt; -z &lt;fvg y high cutoff&gt; -o &lt;output Seurat object in .rds format&gt; -t &lt;variable genes list in text format&gt; ScaleData(): scale expression values seurat-scale-data.R -i &lt;Seurat object with variable genes, in .rds format&gt; -e &lt;test genes&gt; -v &lt;variables to regress&gt; -m &lt;model to use&gt; -u &lt;use umi&gt; -s &lt;do scale&gt; -c &lt;do center&gt; -x &lt;scale max&gt; -b &lt;block size&gt; -d &lt;min cells to block&gt; -a &lt;assay type&gt; -n &lt;check for norm&gt; -o &lt;output Seurat object in .rds format&gt; RunPCA(): run a principal components analysis seurat-run-pca.R -i &lt;Seurat object with scaled expression valus in .rds format&gt; -e &lt;test genes&gt; -p &lt;pcs to compute&gt; -m &lt;use imputed&gt; -o &lt;output Seurat object in .rds format&gt; -b &lt;pca embedding in text format&gt; -l &lt;pca loadings file in format&gt; -s &lt;pca stdev file, text format&gt; FindTransferAnchors(): project a reference on a query (integration) seurat-find-transfer-anchors.R -i &lt;seurat object with computed dimension reduction used as query, .rds format&gt; -r &lt;seurat object with computed dimension reduction used as reference, .rds format&gt; -o &lt;anchorSet object with anchor matrix, .rds format&gt; -n &lt;normalization method: pcaproject or cca&gt; -f &lt;features to use for dimensional reduction&gt; -d &lt;which dimensions to use from the reduction to specify the neighbor search space, a:b format&gt; DimPlot(): plot dimension reductions seurat-dim-plot.r -i &lt;Seurat object with computed dimension reductions, .rds format&gt; -r &lt;dimension reduction, e.g. pca&gt; -a &lt;dim 1&gt; -b &lt;dim 2&gt; -p &lt;pt size&gt; -l &lt;label size&gt; -d &lt;do label&gt; -f &lt;group by&gt; -t &lt;title&gt; -w &lt;png width&gt; -j &lt;png height&gt; -o &lt;image file&gt; RunTSNE(): run-tSNE analysis seurat-run-tsne.r -i &lt;Seurat object with computed PcA, .rds format&gt; -r &lt;reduction type&gt; -d &lt;dims to use&gt; -e &lt;file with genes to use&gt; -f &lt;do fast tsne&gt; -o &lt;output Seurat object in .rds format&gt; -b &lt;tsne embeddings in csv format&gt; FindClusters(): generate cell clusters from expression values seurat-find-clusters.r -i &lt;&lt;Seurat object with computed dimension reductions, .rds format&gt;&gt; -e &lt;test genes&gt; -u &lt;dimension reduction, e.g. pca&gt; -d &lt;dims to use&gt; -k &lt;k value&gt; -r &lt;resolution&gt; -a &lt;cluster algorithm&gt; -m &lt;cluster tmp file location&gt; -o &lt;output Seurat object in .rds format&gt; -t &lt;clusters in txt format&gt; Accessory scripts Get a random set of genes seurat-get-random-genes.R &lt;Seurat object in .rds format&gt; &lt;output text file&gt; &lt;ngenes&gt; Convert formats Multiple format conversion through Seurat 3: Possible input formats: Seurat AnnData (versions contemporary to Seurat 3). Loom (versions contemporary to Seurat 3) SingleCellExperiment Possible output formats: Seurat Loom SingleCellExperiment seurat-convert.R -i inputfile.
