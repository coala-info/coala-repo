# What is GDAL?[](#what-is-gdal "Link to this heading")

GDAL is a translator library for raster and vector geospatial data formats that is released under an MIT style Open Source [License](license.html#license) by the [Open Source Geospatial Foundation](http://www.osgeo.org/). As a library, it presents a single raster abstract data model and single vector abstract data model to the calling application for all supported formats. It also comes with a variety of useful command line utilities for data translation and processing. The [NEWS](https://github.com/OSGeo/gdal/blob/v3.12.3/NEWS.md) page describes the March 2026 GDAL/OGR 3.12.3 release.

This documentation is also available as a [PDF](/_/downloads/en/stable/pdf/) or a [ZIP of individual HTML pages](/_/downloads/en/stable/htmlzip/) for offline browsing.
