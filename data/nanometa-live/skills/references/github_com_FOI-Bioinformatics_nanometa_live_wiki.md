Nanometa Live is a real-time metagenomic analysis tool designed for Oxford Nanopore sequencing data. This robust tool features both a backend pipeline and a frontend graphical user interface (GUI), providing immediate insights into microbial communities as your sequencing data is generated. From setting up a new project to in-depth exploration of your data, this wiki serves as your comprehensive resource for understanding and utilizing all the features that Nanometa Live has to offer. In this wiki, you'll find comprehensive guides on: Setting Up Your Project : Everything from installation to initial setup. Commands : Learn about the various terminal commands that control Nanometa Live. Usage : A step-by-step guide to using the tool effectively. Config File Settings : Understand how to customize your project settings. Setting Species of Interest : Receive guidance on using both names and taxonomy IDs for specifying your species of interest. Graphical User Interface : Get to know the user-friendly dashboard where your data comes to life. Whether you're new to Nanopore sequencing or a seasoned researcher, Nanometa Live simplifies the process of data analysis by offering real-time insights into your samples.
