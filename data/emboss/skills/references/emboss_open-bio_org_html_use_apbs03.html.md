## B.3. EMBASSY Packages (release R6)

Table B.2. EMBASSY Packages (release R6)

| Package | Description |
| --- | --- |
| *[Cbstools](http://emboss.open-bio.org/rel/rel6/apps/cbstools.html)* | The CBSTOOLS package is a set of wrappers to selected applications from the CBS group in Denmark. |
| *[Domainatrix](http://emboss.open-bio.org/rel/rel6/apps/domainatrix.html)* | The DOMAINATRIX programs were developed by Jon Ison and colleagues at MRC HGMP for their protein domain research. They are included as an EMBASSY package as a work in progress. |
| *[Domalign](http://emboss.open-bio.org/rel/rel6/apps/domalign.html)* | The DOMALIGN programs were developed by Jon Ison and colleagues at MRC HGMP for their protein domain research. They are included as an EMBASSY package as a work in progress. |
| *[Domsearch](http://emboss.open-bio.org/rel/rel6/apps/domsearch.html)* | The DOMSEARCH programs were developed by Jon Ison and colleagues at MRC HGMP for their protein domain research. They are included as an EMBASSY package as a work in progress. |
| *[Emnu](http://emboss.open-bio.org/rel/rel6/apps/emnu.html)* | The EMNU package is a simple EMBOSS menu system written by Gary Williams at HGMP. |
| *[Esim4](http://emboss.open-bio.org/rel/rel6/apps/esim4.html)* | The ESIM4 package is an EMBOSS conversion of the SIM4 package from Liliana Florea. |
| *[Hmmernew](http://emboss.open-bio.org/rel/rel6/apps/hmmernew.html)* | EMBASSY HMMERNEW is a suite of application wrappers to the original hmmer v2.3.2 applications written by Sean Eddy. hmmer v2.3.2 must be installed on the same system as EMBOSS and the location of the hmmer executables must be defined in your path for EMBASSY HMMER to work. |
| *[Iprscan](http://emboss.open-bio.org/rel/rel6/apps/iprscan.html)* | The IPRSCAN package is a wrapper for the interproscan program. |
| *[Memenew](http://emboss.open-bio.org/rel/rel6/apps/memenew.html)* | The EMBASSY MEME package contains 'wrapper' applications providing an EMBOSS-style interface to the applications in the original MEME package version 4.0.0 developed by Timothy L. Bailey. |
| *[Mira](http://emboss.open-bio.org/rel/rel6/apps/mira.html)* | The MIRA package is a fragment assembly program from Bastien Chevreux. The program was converted to EMBOSS by Alan Bleasby as two applications, one for EST assembly and one for shotgun fragment assembly |
| *[Mse](http://emboss.open-bio.org/rel/rel6/apps/mse.html)* | The MSE package is a multiple sequence editor. The program was contributed to the EMBOSS package by the author, Will Gilbert, as one of the first EMBASSY programs. |
| *[Myembossdemo](http://emboss.open-bio.org/rel/rel6/apps/myembossdemo.html)* | The MYEMBOSSDEMO package contains example applications using EMBOSS data types |
| *[Phylipnew](http://emboss.open-bio.org/rel/rel6/apps/phylipnew.html)* | The PHYLIPNEW programs are EMBOSS conversions of the programs in Joe Felsenstein's PHYLIP package, version 3.69. |
| *[Signature](http://emboss.open-bio.org/rel/rel6/apps/signature.html)* | The SIGNATURE programs were developed by Jon Ison and colleagues at MRC HGMP for their protein domain research. They are included as an EMBASSY package as a work in progress. |
| *[Structure](http://emboss.open-bio.org/rel/rel6/apps/structure.html)* | The STRUCTURE programs were developed by Jon Ison and colleagues at MRC HGMP for their protein domain research. They are included as an EMBASSY package as a work in progress. |
| *[Topo](http://emboss.open-bio.org/rel/rel6/apps/topo.html)* | The TOPO package is a graphics program to display membrane protein topology by Susan Jean Johns. |
| *[Vienna](http://emboss.open-bio.org/rel/rel6/apps/vienna.html)* | These programs are adapted from the VIENNA RNA package. |
