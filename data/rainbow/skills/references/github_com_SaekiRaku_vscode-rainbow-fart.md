It will check the keywords in your code to play relative sounds. Rainbow Fart is a literal translation word from Chinese, which means giving somebody exaggerated compliment that even seems a little fake. It is not connected to any specific group or individuals. Documentation For more information, please read the documentation . Quick Start Install extension from VSCode Marketplace . Press Ctrl + Shift + P （MacOS Command + Shift + P ）to bring up Command Panel . Type &gt; Enable Rainbow Fart into Command Panel . A notification should be popped up at the corner. Click Open on that notification. Click Authorization on the opened page. Enjoy coding! Try to type function in your VSCode.
