# Community Support and Contributions

To contribute to this project, we welcome pull requests and the raising of issues.
We welcome contributions to the Meta-Omics Pipeline! There are two main ways to contribute: by creating pull requests or raising issues.

## Pull Requests 🔀

If you have made changes to the code that you would like to propose for inclusion in the main project, you can create a pull request. This allows the project maintainers to review your changes and decide whether to merge them into the project.

To create a pull request, follow these steps:

1. [Fork the repository](https://github.com/iquasere/MOSCA/fork). This creates a copy of the repository under your own account.
2. Clone the repository to your local machine.
3. Create a new branch for your changes.
4. Make your changes and commit them to your branch.
5. Push your changes to your fork of the repository.
6. Visit the original repository and click the "New pull request" button.
7. Select your new branch as the source for the pull request.
8. Review your changes and add any additional information that may be helpful for the project maintainers.
9. Submit the pull request.

The project maintainers will review your changes and either merge them into the project or request further changes before merging.

## Raising Issues 💡

If you have found a bug or have an idea for a new feature, you can raise an issue to let the project maintainers know.

To raise an issue, follow these steps:

1. Visit the repository's ["Issues" tab](https://github.com/iquasere/MOSCA/issues).
2. Click the "New issue" button.
3. Give your issue a descriptive title and add any additional details or examples in the description field.
4. Click the "Submit new issue" button.

The project maintainers will review your issue and take appropriate action.

Thank you for considering contributing to the Meta-Omics Pipeline! Every contribution, no matter how small, helps us improve and make the pipeline even better. 🙌

## Troubleshooting

**AttributeError: module 'lib' has no attribute 'X509\_V\_FLAG\_CB\_ISSUER\_CHECK'**

As indicated [here](https://stackoverflow.com/questions/73830524/attributeerror-module-lib-has-no-attribute-x509-v-flag-cb-issuer-check), run the following:

```
pip install pip --upgrade
pip install pyopenssl --upgrade
```
