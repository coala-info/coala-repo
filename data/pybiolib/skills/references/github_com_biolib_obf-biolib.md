The basic idea is to provide infrastructure that allows sharing code rather than reinventing the wheel every time between different language implementations. Biolib provides a minimalistic layer to external C and C++ libraries - using SWIG as the primary interface provider. For more information see: http://biolib.open-bio.org/ The release notes are in ./doc/RELEASE_NOTES.txt For dependencies see ./doc/DEPENDENCIES.txt If you wish to contribute subscribe to the mailing list.
