# bioconductor-prolocgui

## Overview

Use the Bioconductor R package **pRolocGUI** for: The package pRolocGUI comprises functions to interactively visualise spatial proteomics data on the basis of pRoloc, pRolocdata and shiny.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("pRolocGUI")
```
