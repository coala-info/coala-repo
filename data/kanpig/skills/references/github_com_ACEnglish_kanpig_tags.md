Dec 10, 2024 64f7c78 zip tar.gz Notes Downloads v1.0.0 Toggle v1.0.0&#39;s commit message Faster, Smaller, More Accurate Genotyping Kanpig now has sub-commands `gt` and `plup`. The new `plup` command will extract reads and their SVs from a bam file into a small file that's useful for long-term storage of reads. The `gt` command can now parse these plup files much more quickly than parsing a bam. The `gt` command will now (optionally) parse PS and HP tags in a bam to increase genotyping accuracy as well as record long-range phasing information in SV genotypes. The `gt` command now uses kmedoid clustering instead of kmeans, resulting in a modest improvement to genotyping accuracy.
