# Evaluation on HG002 with GiaB draft truth set

[Jump to bottom](#wiki-pages-box)

David Heller edited this page Jun 29, 2021
·
[5 revisions](/eldariont/svim/wiki/Evaluation-on-HG002-with-GiaB-draft-truth-set/_history)

The Genome in a Bottle (GiaB) Consortium generated a draft SV truth set for the HG002 individual that we can use to evaluate the calling performance of SVIM. At this point, the truth set contains only deletions and insertions. Although SVIM detects three different types of insertions (novel element insertions, tandem and interspersed duplications), we can output all of them as insertions and compare against the insertions in the truth set. Below, we adapted a similar [evaluation](https://github.com/PacificBiosciences/sv-benchmark) by PacBio.

We analyze the publicly available HG002 PacBio CCS dataset which we align to the reference with PacBio's minimap2 frontend [pbmm2](https://github.com/PacificBiosciences/pbmm2).

### Step by step

1. Create directory structure:

```
mkdir fastqs ref alns alns_merged svim_wd giab
```

2. Download GiaB truth set:

```
FTPDIR=ftp://ftp-trace.ncbi.nlm.nih.gov/giab/ftp/data/AshkenazimTrio/analysis/
curl -s ${FTPDIR}/NIST_SVs_Integration_v0.6/HG002_SVs_Tier1_v0.6.bed > giab/HG002_SVs_Tier1_v0.6.bed
curl -s ${FTPDIR}/NIST_SVs_Integration_v0.6/HG002_SVs_Tier1_v0.6.vcf.gz > giab/HG002_SVs_Tier1_v0.6.vcf.gz
curl -s ${FTPDIR}/NIST_SVs_Integration_v0.6/HG002_SVs_Tier1_v0.6.vcf.gz.tbi > giab/HG002_SVs_Tier1_v0.6.vcf.gz.tbi
```

3. Obtain truvari tool for performance evaluation (results shown for version 2.0.2):

```
pip install truvari==2.0.2
```

4. Download hg19 reference with decoys:

```
curl -s ftp://ftp-trace.ncbi.nih.gov/1000genomes/ftp/technical/reference/phase2_reference_assembly_sequence/hs37d5.fa.gz > ref/human_hs37d5.fasta.gz
gunzip ref/human_hs37d5.fasta.gz
```
