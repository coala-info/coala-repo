## Overview
Mimick is a powerful tool for simulating linked-read sequencing data in FASTQ format. It's designed to generate synthetic data that mimics real-world linked-read libraries, making it invaluable for testing and developing bioinformatics tools and pipelines. Mimick supports a range of linked-read chemistries and offers extensive control over simulation parameters, including the number of haplotypes, overall coverage, molecule coverage, and barcode characteristics.

## Usage

Mimick is primarily used via its command-line interface. The core functionality involves specifying input reference genomes and desired simulation parameters to generate FASTQ files.

### Basic Simulation

The fundamental command structure involves providing an input FASTA file for the reference genome and specifying output parameters.

```bash
mimick simulate --fasta <input.fasta> --output <output.fastq> [options]
```

### Key Simulation Parameters

*   **`--fasta <input.fasta>`**: Path to the reference genome in FASTA format. This is a mandatory argument.
*   **`--output <output.fastq>`**: Path for the output FASTQ file.
*   **`--haplotypes <N>`**: Number of haplotypes to simulate. Defaults to 1.
*   **`--coverage <depth>`**: Overall coverage depth for the simulation.
*   **`--molecule_coverage <N>`**: Average molecule coverage (molecules per barcode).
*   **`--molecules_per_barcode <N>`**: Number of molecules per barcode.
*   **`--read_length <N>`**: Length of individual reads.
*   **`--insert_size <mean,stddev>`**: Mean and standard deviation of the insert size distribution.
*   **`--circular`**: Flag to indicate if the DNA is circular (useful for prokaryotes).
*   **`--vcf <input.vcf>`**: Path to a VCF file to introduce variants for multi-sample simulation.

### Multi-Sample Simulation (Version 3.0+)

For simulating data from multiple samples, Mimick can take a FASTA and a VCF file as input. Sample haplotypes are generated by applying variants from the VCF to the FASTA.

```bash
mimick simulate --fasta <input.fasta> --vcf <input.vcf> --output <output.fastq> [options]
```

### Supported Linked-Read Chemistries

Mimick is generalized to support various linked-read chemistries, including:

*   10X
*   Haplotagging
*   stLFR
*   TELLseq

The specific chemistry is implicitly handled by the simulation parameters and the underlying simulation engine.

### Expert Tips

*   **Realistic Parameters**: To generate realistic data, carefully consider parameters like `--coverage`, `--molecule_coverage`, and `--insert_size` based on experimental protocols.
*   **Circular DNA**: For simulating data from organisms with circular genomes (e.g., bacteria), use the `--circular` flag.
*   **Multi-sample simulation**: Leverage the `--vcf` option for simulating complex scenarios involving multiple samples with genetic variations.
*   **Barcode Sharing**: Mimick's barcode-first simulation approach allows barcodes to be shared across chromosomes, contigs, and haplotypes, which is a common phenomenon in real linked-read libraries.
