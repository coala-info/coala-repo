# About

BlobTk contains a set of core functions used by BlobToolKit tools. Implemented in Rust, these functions are intended to be accessible from the command line, as python modules and will include web assembly code for use in javascript.

# Installation

A conda package is available to install the BlobTk command line tool and python package. Alternate installation methods are also available for individual components.

## Conda

```
conda install tolkit::blobtk
```

## Pip

The python module can be installed using pip:

```
pip install blobtk
```

## Command line binary

Prebuilt Linux/macOS binaries are published on each GitHub release:

<https://github.com/blobtoolkit/blobtk/releases/latest>

With GitHub CLI, you can download assets directly, e.g.:

```
gh release download --repo blobtoolkit/blobtk --pattern 'blobtk-*'
```

## Docker / Singularity

BlobTk can be run in a container by pulling the Docker image:

```
docker pull genomehubs/blobtk:latest
docker run --rm genomehubs/blobtk blobtk --version
```

Singularity/Apptainer can pull from the same Docker image:

```
singularity pull blobtk.sif docker://genomehubs/blobtk:latest
singularity exec blobtk.sif blobtk --version
```

## Build from source

First install rust if you haven't already (BlobTk requires at least rust v1.70):

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Then clone this repository and build BlobTk:

```
git clone https://github.com/blobtoolkit/blobtk
cd blobtk/rust && cargo build --release
# executable is in ./target/release/blobtk
./target/release/blobtk plot --help

# or to install into your path
cd blobtk/rust && cargo install --path .
blobtk plot --version
```

Build the python package from source using maturin

```
cd blobtk/rust && maturin build --release
```
