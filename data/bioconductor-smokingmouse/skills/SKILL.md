# bioconductor-smokingmouse

## Overview

The `smokingMouse` package provides access to comprehensive bulk RNA-sequencing data generated by the Lieber Institute for Brain Development (LIBD). The dataset explores the molecular impact of prenatal and adult exposure to nicotine and cigarette smoke. It includes expression counts for four feature levels (genes, transcripts, exons, and exon-exon junctions) across 208 samples (brain and blood) from both pups and adult mice. Additionally, it includes differential expression (DE) statistics from human studies for cross-species comparison.

## Data Retrieval

Data is hosted on Bioconductor's `ExperimentHub`. To access the datasets, you must use the `ExperimentHub` interface.

```R
library(smokingMouse)
library(ExperimentHub)
library(SummarizedExperiment)

# Connect to ExperimentHub
ehub <- ExperimentHub()

# Query for smokingMouse records
myfiles <- query(ehub, "smokingMouse")

# Available Records:
# EH8313 | rse_gene_mouse_RNAseq_nic-smo
# EH8314 | rse_tx_mouse_RNAseq_nic-smo
# EH8315 | rse_jx_mouse_RNAseq_nic-smo
# EH8316 | rse_exon_mouse_RNAseq_nic-smo
# EH8317 | de_genes_prenatal_human_brain_smoking
# EH8318 | de_genes_adult_human_brain_smoking

# Download a specific object (e.g., mouse gene data)
rse_gene <- myfiles[["EH8313"]]
```

## Working with Mouse RSE Objects

The mouse datasets are `RangedSummarizedExperiment` objects.

### Accessing Expression Data
* **Raw Counts**: `assays(rse)$counts`
* **Log-normalized Counts**: `assays(rse)$logcounts` (log2(CPM + 0.5) for genes/exons/jxns; log2(TPM + 0.5) for transcripts).

### Metadata and Analysis Results
The `rowData` and `colData` contain pre-computed analysis results and QC metrics.

**Feature Metadata (`rowData`):**
* `retained_after_feature_filtering`: Logical indicating if the feature passed expression thresholds.
* `DE_in_adult_brain_nicotine`: Logical indicating DE status for nicotine vs vehicle in adults.
* `DE_in_adult_brain_smoking`: Logical indicating DE status for smoking vs control in adults.
* `DE_in_pup_brain_nicotine`: Logical indicating DE status for nicotine vs vehicle in pups.
* `DE_in_pup_brain_smoking`: Logical indicating DE status for smoking vs control in pups.

**Sample Metadata (`colData`):**
* `Tissue`: "Brain" or "Blood".
* `Age`: "Adult" or "Pup".
* `Expt`: "Nicotine" or "Smoking".
* `Group`: "Experimental" or "Control".
* `retained_after_QC_sample_filtering`: Logical for QC pass.

## Working with Human Data

Human datasets are provided as `GRanges` objects containing differential expression statistics.

```R
# Download human adult brain smoking DE stats
de_human_adult <- myfiles[["EH8318"]]

# Convert to data frame for easier inspection
de_human_df <- as.data.frame(de_human_adult)

# Key columns: logFC, AveExpr, t, P.Value, adj.P.Val
head(de_human_df)
```

## Typical Workflow

1.  **Load Data**: Use `ExperimentHub` to pull the specific feature level of interest (e.g., `rse_gene`).
2.  **Filter Samples**: Subset the RSE object using `colData` to focus on specific conditions (e.g., only Pups in the Smoking experiment).
    ```R
    rse_pups <- rse_gene[, rse_gene$Age == "Pup" & rse_gene$Expt == "Smoking"]
    ```
3.  **Identify DEGs**: Use the pre-computed logical columns in `rowData` to quickly identify genes that were differentially expressed in the original study.
    ```R
    degs <- rse_gene[rowData(rse_gene)$DE_in_pup_brain_smoking, ]
    ```
4.  **Visualization**: Use the `logcounts` assay for heatmaps or boxplots of specific genes across groups.
5.  **Cross-Species Comparison**: Compare the `Symbol` or `EntrezID` of mouse DEGs against the human DE statistics provided in the human datasets.
