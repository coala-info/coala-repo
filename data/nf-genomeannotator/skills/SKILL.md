# genomeannotator

## Overview
nf-core/genomeannotator is a bioinformatics pipeline designed for the structural annotation of eukaryote genomes, with a specific focus on metazoans. It addresses the challenge of identifying gene structures by integrating multiple lines of evidence, including protein alignments, transcript data, and ab-initio predictions.

The pipeline processes a genome assembly by filtering contigs, masking repeats, and aligning external evidence to produce a final consensus gene set. Users receive standardized gene models and quality reports, such as MultiQC and BUSCO completeness scores, to evaluate the accuracy of the annotation.

## Data preparation
The pipeline requires a genome assembly and at least one source of evidence (proteins, transcripts, or RNA-seq).

*   **Genome Assembly**: A FASTA file provided via `--assembly`.
*   **RNA-seq Data**: A CSV samplesheet provided via `--rnaseq_samples`. It must contain a header and four columns: `sample`, `fastq_1`, `fastq_2`.
*   **Protein Evidence**: FASTA files for related species (`--proteins`) or targeted species-specific proteins (`--proteins_targeted`).
*   **Transcript Evidence**: FASTA files of transcripts or ESTs provided via `--transcripts`.
*   **Repeat Masking**: A FASTA file of known repeats (`--rm_lib`) or a taxonomic group name (`--rm_species`) to guide RepeatMasker.
*   **Reference Genomes**: A CSV samplesheet for lift-over annotations from related species provided via `--references`.

Minimal RNA-seq samplesheet example:
```csv
sample,fastq_1,fastq_2
control_1,control_1_R1.fastq.gz,control_1_R2.fastq.gz
```

## How to run
Run the pipeline using the `nextflow run` command. Specify the assembly, output directory, and a compute profile (e.g., `docker` or `singularity`).

```bash
nextflow run nf-core/genomeannotator \
  --assembly genome.fasta \
  --proteins proteins.fasta \
  --outdir ./results \
  -profile docker
```

To include RNA-seq evidence and enable specific sub-pipelines like PASA or EvidenceModeler:

```bash
nextflow run nf-core/genomeannotator \
  -r 1.0.0 \
  --assembly genome.fasta \
  --rnaseq_samples samplesheet.csv \
  --pasa \
  --evm \
  --outdir ./results \
  -profile singularity \
  -resume
```

## Outputs
Results are saved in the directory specified by the `--outdir` parameter.

*   **Consensus Models**: Final gene builds generated by EvidenceModeler if enabled.
*   **Gene Predictions**: Ab-initio models from AUGUSTUS and transcript-based models from PASA.
*   **Reports**: A MultiQC report summarizing alignment statistics and pipeline performance.
*   **Completeness**: BUSCO results evaluating the annotated gene set against specific taxonomic lineages.

For a detailed description of all output files, refer to the official [output documentation](https://nf-co.re/genomeannotator/output).
