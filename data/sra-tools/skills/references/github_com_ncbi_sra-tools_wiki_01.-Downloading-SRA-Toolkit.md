# NCBI SRA Toolkit

Below are the latest releases of various tools and release checksum file.

## SRA Toolkit

Compiled binaries/install scripts of March 25, 2026, version 3.4.1:

* [AlmaLinux 64 bit architecture](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/sratoolkit.3.4.1-alma_linux64.tar.gz) - non-sudo tar archive
* [Ubuntu Linux 64 bit architecture](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/sratoolkit.3.4.1-ubuntu64.tar.gz) - non-sudo tar archive
* [Cloud - apt-get install script](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/setup-apt.sh) - for Debian and Ubuntu - requires sudo permissions
* [Cloud - yum install script](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/setup-yum.sh) - for AlmaLinux - requires sudo permissions
* [MacOS x86 64 bit architecture](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/sratoolkit.3.4.1-mac-x86_64.tar.gz)
* [MacOS Arm64 bit architecture](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/sratoolkit.3.4.1-mac-arm64.tar.gz)
* [MS Windows 64 bit architecture](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/sratoolkit.3.4.1-win64.zip)
* [Docker image repository](https://hub.docker.com/r/ncbi/sra-tools)
* [md5 checksums](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/md5sum.txt)

---

## Magic-BLAST

Magic-BLAST is a tool for mapping large next-generation RNA or DNA sequencing runs against a whole genome or transcriptome

* Magic-BLAST executables for LINUX, MacOSX, and Windows as well as the source files are available on the [FTP site](https://ftp.ncbi.nlm.nih.gov/blast/executables/magicblast/LATEST)
* Read more about Magic BLAST on the [FTP site](https://ftp.ncbi.nlm.nih.gov/blast/executables/magicblast/README)

---

## Third Party Software

Builds of Third Party Software Tools with SRA support:

* HISAT2 version 2.2.2-ngs.3.4.1 - graph-based alignment of next generation sequencing reads to a population of genomes with direct support of SRA, built for:
  + [AlmaLinux 64 bit architecture](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/hisat2-2.2.2-64-ngs.3.4.1-linux.zip)

---

## Latest Source Code

* [NCBI VDB Software Development Kit](https://github.com/ncbi/ncbi-vdb) – March 25, 2026, version 3.4.1 release
* [NCBI SRA Toolkit](https://github.com/ncbi/sra-tools) – March 25, 2026, version 3.4.1 release

## File checksums

You may validate downloaded files with [md5 checksums](https://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/3.4.1/md5sum.txt) computed using ***md5sum -b***
