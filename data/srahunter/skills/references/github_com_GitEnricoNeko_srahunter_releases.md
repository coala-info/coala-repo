[0.0.4](/GitEnricoNeko/srahunter/releases/tag/0.0.4)

**Bug Fixes**

* Header Skipping: Resolved an issue where the header of SRA accession lists was improperly processed, leading to erroneous data handling.

**New Features and Improvements**

* datavzrd Enhancements: Implemented new functionalities in the datavzrd tool, enhancing its capabilities and performance.
* HTML Table Generation Toggle: Added an option to disable the generation of HTML tables. This feature helps speed up the processing of large datasets by reducing computational overhead.
* Version Flag: Integrated a --version flag to allow users to easily check the current version of the tool, improving user experience and support.
