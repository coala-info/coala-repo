COPYRIGHT AUTHOR NAME Clone - recursively copy Perl datatypes SYNOPSIS use Clone &#39;clone&#39;; my $data = { set =&gt; [ 1 .. 50 ], foo =&gt; { answer =&gt; 42, object =&gt; SomeObject-&gt;new, }, }; my $cloned_data = clone($data); $cloned_data-&gt;{foo}{answer} = 1; print $cloned_data-&gt;{foo}{answer}; # &#39;1&#39; print $data-&gt;{foo}{answer}; # &#39;42&#39; You can also add it to your class: package Foo; use parent &#39;Clone&#39;; sub new { bless {}, shift } package main; my $obj = Foo-&gt;new; my $copy = $obj-&gt;clone; DESCRIPTION This module provides a clone() method which makes recursive copies of nested hash, array, scalar and reference types, including tied variables and objects. clone() takes a scalar argument and duplicates it. To duplicate lists, arrays or hashes, pass them in by reference, e.g. my $copy = clone (\@array); # or my %copy = %{ clone (\%hash) }; SEE ALSO Storable &#39;s dclone() is a flexible solution for cloning variables, albeit slower for average-sized data structures. Simple and naive benchmarks show that Clone is faster for data structures with 3 or fewer levels, while dclone() can be faster for structures 4 or more levels deep.
