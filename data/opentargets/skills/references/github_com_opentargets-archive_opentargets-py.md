Include wrappers for all public methods, with query validation Tools for the most common calls (E.g. get data for a target gene symbol even if you do not know its Ensembl Gene Id) Supports automatic retrieval of paginated results with an iterator pattern Easily save query results as JSON, CSV or Excel file Handles Authentication Handles fair usage limits transparently Follows HTTP cache as set by the REST API Experimental HTTP2 support for better performance (beware the client library is in alpha) Support for advanced SSL and proxy configuration This client is supported for Python 3.5 and upper. Works on pythoon 2.7 on a best effort basis.
