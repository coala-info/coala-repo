# r-bcbiosinglecell

## Overview
`bcbiosinglecell` is an R package designed to facilitate the analysis of single-cell RNA-seq (scRNA-seq) data generated by the bcbio-nextgen pipeline. It provides a structured way to import bcbio outputs into R as a `bcbioSingleCell` object (which extends the `SingleCellExperiment` class), enabling seamless integration with Bioconductor workflows. The package focuses on automated quality control reporting, data normalization, and exploratory data analysis.

## Installation
To install the package from Bioconductor:
```R
if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("bcbiosinglecell")
```

## Key Workflows

### 1. Loading Data
The primary entry point is the `bcbioSingleCell()` constructor. Point it to the directory containing the bcbio-nextgen final output.

```R
library(bcbiosinglecell)

# Load the dataset
bcb <- bcbioSingleCell(
    uploadDir = "bcbio_output/final",
    sampleMetadataFile = "metadata.csv"
)

# Inspect the object
print(bcb)
```

### 2. Quality Control (QC)
The package provides functions to visualize common scRNA-seq metrics.

- **Plotting Metrics**: Use `plotMetrics()` to see a summary of reads, genes, and mitochondrial content.
- **Specific QC Plots**:
  - `plotGenesPerCell(bcb)`
  - `plotUMIsPerCell(bcb)`
  - `plotMitoRatio(bcb)`
  - `plotNovelty(bcb)` (log10 genes per UMI)

### 3. Filtering
Filter the object based on QC thresholds to remove low-quality cells.

```R
bcb_filtered <- filterCells(
    bcb,
    minUMIs = 500,
    minGenes = 200,
    maxMitoRatio = 0.10
)
```

### 4. Normalization and Dimensionality Reduction
The package supports standard Bioconductor methods (like `scran`) and provides wrappers for visualization.

- **Normalization**: `normalize(bcb)`
- **PCA**: `plotPCA(bcb)`
- **t-SNE/UMAP**: `plotTSNE(bcb)` or `plotUMAP(bcb)`

### 5. Marker Analysis and Visualization
- **Gene Expression**: `plotExpression(bcb, genes = c("GeneA", "GeneB"))`
- **Heatmaps**: `plotMarkerHeatmap(bcb)`

## Tips
- **Object Class**: The `bcbioSingleCell` object is compatible with any function that accepts a `SingleCellExperiment`.
- **Metadata**: Ensure your `metadata.csv` contains a `sampleID` column that matches the bcbio output folder names.
- **Reporting**: Use `prepareSingleCellReport()` to generate an automated R Markdown HTML report of the QC metrics.
