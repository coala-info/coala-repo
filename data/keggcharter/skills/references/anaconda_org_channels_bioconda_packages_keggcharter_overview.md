Both genomic potential (from either genomics or metagenomics) and gene expression quantification (from either (meta)transcriptomics or (meta)proteomics) can be represented. Genomic information is mapped in a binary exists or not rule, for each taxon available KEGGCharter will assign a color which will be included in the boxes corresponding to functions present in that taxon. Gene expression quantification is mapped as differential expression where quantification between the several columns specified is represented as a single row heatmap for each function present in the data.
