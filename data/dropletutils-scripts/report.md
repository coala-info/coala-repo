Usage: /usr/local/bin/dropletutils-read-10x-counts.R [options]


Options:
	-s SAMPLES, --samples=SAMPLES
		A comma-separated list containing one or more directory names, each

            added to the SCE object. By default, all columns are kept.

		Name of the metadata file column that corresponds to the unique identifier
            of each cell in the dataset (e.g. barcode). Must correspond to column names in
            the 10X directory.

		File name in which to store serialized SingleCellExperiment object.

	-i INPUT-OBJECT-FILE, --input-object-file=INPUT-OBJECT-FILE
		File name in which a serialized R SingleCellExperiment object can be found.

	-l LOWER, --lower=LOWER
		A numeric scalar specifying the lower bound on the total UMI count, at or below which all barcodes are assumed to correspond to empty droplets.

	-f FILTER-EMPTY, --filter-empty=FILTER-EMPTY
		Should barcodes estimated to have no cells be removed from the output object?

	-d FILTER-FDR, --filter-fdr=FILTER-FDR
		FDR filter for removal of barcodes with no cells
