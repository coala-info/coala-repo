#### Application Description

[Wally](https://github.com/tobiasrausch/wally) is a commmand-line application to plot aligned reads. This is a simple frontend web application for wally that allows plotting slices of the 1000 Genomes Oxford Nanopore Technologies (ONT) data hosted on the International Genome Sample Resource ([IGSR](https://www.internationalgenome.org/)) [FTP site](http://ftp.1000genomes.ebi.ac.uk/vol1/ftp/data_collections/1KG_ONT_VIENNA/). You can visualize both the CHM13 (T2T) and GRCh38 (hg38) alignments. All available data sets and samples can be found [here](https://github.com/gear-genomics/wallyapp/tree/main/datasets). For short-read data sets, the paired-end coloring follows [delly's](https://github.com/dellytools/delly) structural variant nomenclature and is described [here](https://github.com/tobiasrausch/wally#paired-end-view). Please note that many feature of [wally](https://github.com/tobiasrausch/wally) are not yet available in the frontend web application like split-alignment views, dotplots or match views. Please use the command-line application for this.
