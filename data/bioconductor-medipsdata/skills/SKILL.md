# bioconductor-medipsdata

## Overview

Use the Bioconductor R package **MEDIPSData** for: the package contains MeDIP seq data from 3 NSCLC samples and adjacent normal tissue (chr 20-22). All data has been aligned to human genome hg19.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("MEDIPSData")
```
