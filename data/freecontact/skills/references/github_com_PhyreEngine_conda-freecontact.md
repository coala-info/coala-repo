Prerequisites You will need an installation of conda . Your root conda installation must have the conda-build installed. Building You should be able to build this package by simply running ./build . A package will be built for each combination of BLAS library (of mkl and openblas ) and each supported CPU architecture. Patches This recipe removes some non-standard calls to tempfile in the configure script, as well disabling automatic detection of CPU flags. Some patching is also required to work with mkl .
