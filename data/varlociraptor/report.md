```text
varlociraptor-estimate 8.9.5
Perform estimations.

USAGE:
    varlociraptor estimate <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    alignment-properties    Estimate properties like insert size, maximum softclip length, and the PCR homopolymer
                            error model.
    contamination           Estimate contamination between samples (still experimental). Takes two samples, the one
                            that is presumably contaminated and the contaminating one, and calculates the posterior
                            distribution of the contamination fraction, which is printed into a TSV table. The most
                            likely fraction comes at the top. Uses a uniform prior distribution by default.
                            Optionally, e.g. a pathologist's estimate can be provided, such that the calculated
                            posterior distribution can be sharpened by the model with the given prior knowledge. For
                            now, it is always advisable to check the visual output generated by --output-plot.
    help                    Prints this message or the help of the given subcommand(s)
    mutational-burden       Estimate mutational burden. Takes Varlociraptor calls (must be annotated with e.g. VEP
                            but using ANN instead of CSQ) from STDIN, prints mutational burden estimate in Vega-lite
                            JSON format to STDOUT. It can be converted to an image via vega-lite-cli (see conda
                            package).
```
