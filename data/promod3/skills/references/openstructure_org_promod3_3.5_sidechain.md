Cite Us ProMod3 sidechain - Sidechain Modelling ¶ Tools and algorithms to model sidechains given backbone coordinates. The full module is heavily based on SCWRL4 [krivov2009] . The according paper describes the modelling of sidechains using two different rotamer models. A rigid model, as well as a flexible model. Both models are implemented in ProMod3 and can be applied in flexible ways. The following code fragment shows an example of a basic sidechain reconstruction algorithm using the functionality in the module. Note, that this code will crash as soon as you have structures containing all the weirdness the PDB throws at us.
