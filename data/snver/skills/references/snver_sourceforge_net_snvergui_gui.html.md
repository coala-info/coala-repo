SNVerGUI SNVerGUI A desktop tool for variant detection from next generation sequencing data SNVerGUI is a desktop tool for variant detection from next generation sequencing data. It is implemented on the top of SNVer, which is a statistical tool for calling common and rare variants in analysis of pool or individual next-generation sequencing data. SNVerGUI, an ease-of-use graphical user interface, allows users to run the entire variant calling pipeline after simply configuring several parameters. Moreover, SNVerGUI displays the results as a typical table view for customized filtering and sorting. It also automatically generates summarized report for further analysis. Under Construction...
