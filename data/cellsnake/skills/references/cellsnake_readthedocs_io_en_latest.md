# Welcome to **Cellsnake**’s Documentation![#](#welcome-to-cellsnake-s-documentation "Permalink to this headline")

**Cellsnake** is a command line tool designed for easy and scalable single-cell analysis. It can be used as a Snakemake workflow or as a standalone command line tool.

It is build on **Seurat4** but **Seurat5** version is coming soon!

Our main repository location: [sinanugur/cellsnake](https://github.com/sinanugur/cellsnake)

Our Snakemake workflow location: [sinanugur/scrna-workflow](https://github.com/sinanugur/scrna-workflow)
