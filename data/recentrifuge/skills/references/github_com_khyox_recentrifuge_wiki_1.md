### What?

With Recentrifuge, researchers can analyze results from [Centrifuge](http://www.ccb.jhu.edu/software/centrifuge/), [Centrifuger](https://github.com/mourisl/centrifuger), [Kraken](http://ccb.jhu.edu/software/kraken2/index.shtml), [LMAT](https://computation.llnl.gov/projects/livermore-metagenomics-analysis-toolkit), [CLARK](http://clark.cs.ucr.edu/), and other taxonomic classifiers using interactive charts with emphasis on the confidence level of the classifications. In addition to contamination-subtracted samples, Recentrifuge provides shared and exclusive taxa per sample, thus enabling robust contamination removal and comparative analysis in metagenomics.

### How?

Recentrifuge's novel approach combines robust statistics, arithmetic of scored taxonomic trees, and parallel computational algorithms. The software implements a robust method for the removal of negative-control and crossover taxa from the rest of samples.

### Is Recentrifuge for me?

Researchers use Recentrifuge to analyze very diverse metagenomic datasets. It is especially useful in the case of low microbial biomass metagenomic studies or when a more reliable detection of minority organisms is needed, like in clinical, environmental, and forensic analysis. Recentrifuge supports all the 48 taxonomic ranks of the [NCBI Taxonomy](https://www.ncbi.nlm.nih.gov/taxonomy), including several infraspecific levels such as strain or isolate. The package contains additional software to work with classifier's output and with large metagenomic databases.

### Installation and running instructions

* For installation of the software, please see the [installation page](/khyox/recentrifuge/wiki/Installation).
* For instructions about how to run the main Recentrifuge script:
  + if you have results from Centrifuge(r), please see [running Recentrifuge for Centrifuge](/khyox/recentrifuge/wiki/Running-recentrifuge-for-Centrifuge),
  + in case you have LMAT outputs, your choice is [running Recentrifuge for LMAT](/khyox/recentrifuge/wiki/Running-recentrifuge-for-LMAT),
  + if you have CLARK(S) results, please enter [running Recentrifuge for CLARK](/khyox/recentrifuge/wiki/Running-recentrifuge-for-CLARK),
  + in case you have data from Kraken, please check [running Recentrifuge for Kraken](/khyox/recentrifuge/wiki/Running-recentrifuge-for-Kraken).
  + if you use any other taxonomic classifier, please see [running Recentrifuge for a generic classifier](/khyox/recentrifuge/wiki/Running-recentrifuge-for-a-generic-classifier).

### Other useful scripts for related tasks with samples and databases

* To know how to extract specific reads from the samples attending to the classifier (Centrifuge) output, please see [running rextract](/khyox/recentrifuge/wiki/Using-rextract-to-extract-reads-from-the-samples).
* For instructions about how to extract entries from large metagenomic databases formatted as the NCBI BLAST nt in a fasta file, please see [using rextraccnt to extract entries from nt-like databases](/khyox/recentrifuge/wiki/Using-rextraccnt-to-extract-entries-from-nt-like-databases).
* To symmetrically split a fasta file into several files (e.g. for downstream parallel processing), please see [using refasplit to simmetrically split fasta files](/khyox/recentrifuge/wiki/Using-refasplit-to-split-fasta-files)

### Advanced topics

* **Migration guide**: if you are migrating from a version below v2.0.0 please take a look at [V2 migration guide](/khyox/recentrifuge/wiki/v2-migration-guide).
* **Release notes**: you can read about recent improvements and additions to Recentrifuge in [What's new](/khyox/recentrifuge/wiki/What%27s-new).
* For examples on how to use Recentrifuge **programmatically** in your Python code or notebooks, see the [Recentrifuge Colab notebook](https://colab.research.google.com/drive/1iEKi8If1Ak7p5TcKL76FqQvrd5-xCNWf?usp=sharing).
* For a comprehensive guide to the Recentrifuge **options and flags**, please check the [Recentrifuge command line](/khyox/recentrifuge/wiki/Recentrifuge-command-line).
* For details about **testing** the different parts of the package, please visit [testing and more](/khyox/recentrifuge/wiki/Testing-and-more).
* For an experimental feature related to **infraspecific** taxonomic ranks, please see the [advanced flag --strain](/khyox/recentrifuge/wiki/Recentrifuge-command-line#advanced).
* For other advanced issues related to Recentrifuge you may consult extra topics in [Troubleshooting](/khyox/recentrifuge/wiki/Troubleshooting).
