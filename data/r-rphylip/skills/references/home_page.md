## Rphylip: An R interface for PHYLIP

---

Rphylip is a nearly comprehensive R interface for Joseph Felsenstein's [PHYLIP](http://evolution.genetics.washington.edu/phylip.html) phylogeny methods program package. Rphylip passes user control of virtually all aspects of about 90% of PHYLIP programs to the R user, allowing PHYLIP programs to be seamlessly integrated into an R phylogenetics workflow.

Rphylip is now on [CRAN](http://cran.us.r-project.org/). Rphylip can thus be obtained from its [CRAN page](http://cran.r-project.org/web/packages/Rphylip/index.html) or by typing install.packages("Rphylip") at the R command prompt.

The latest development version of Rphylip source code can be obtained from [GitHub](http://github.com/liamrevell/Rphylip).

To use Rphylip one must (of course) first [install PHYLIP](http://evolution.genetics.washington.edu/phylip/getme.html). Any use of Rphylip in publication should include a [citation](http://evolution.genetics.washington.edu/phylip/faq.html#citation) to the PHYLIP program package.

Rphylip is the product of [Liam Revell](http://faculty.umb.edu/liam.revell/) and [Scott Chamberlain](http://scottchamberlain.info/). It is a new package, and thus should be considered highly *beta*. Please direct any questions about the package to its authors.
