Please contribute bug fixes or new features with a pull request to this repository. How to prepare a file for tabix Sort the file by chromosome, start, and end. Compress the file with bgzip . Index the compressed file with tabix . $ head -n5 example.bed | column -t chr19 53611131 53636172 ZNF415 chr10 72149121 72150375 CEP57L1P1 chr4 185009858 185139113 ENPP6 chrX 132669772 133119672 GPC3 chr6 134924279 134925376 FAM8A6P Sort it by chromosome, start, and end. Then, use bgzip to deflate the file into compressed blocks: $ sort -k1V -k2n -k3n example.bed | bgzip &gt; example.bed.gz Index the compressed file with tabix : $ tabix -s 1 -b 2 -e 3 example.bed.gz $ ls example.bed example.bed.gz example.bed.gz.tbi Alternative: Use subprocess from subprocess import Popen , PIPE def bgzip ( filename ): """Call bgzip to compress a file."""
