# bioconductor-pepxmltab

## Overview

Use the Bioconductor R package **pepXMLTab** for: The package tries to handle pepXML files generated from different softwares. The output will be a peptide-spectrum-matching tabular file. The package also provide function to filter the PSMs based on FDR.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("pepXMLTab")
```
