# Changelog

## Added

Modularized the CDST pipeline into a Python package under src/cdst/, enabling installation via pip install cdst-genome.

New CLI entry point cdst available after pip installation, providing the same subcommands (generate, matrix, mst, hc, run) as the original script.

## Changed

Restructured the original monolithic cdst.py into:

`core.py: core functions for MD5 hashing, matrix generation, MST and HC tree building.`
`cli.py: argparse-based command-line interface.`
`__init__.py: versioning and public API.`

Updated README.md to recommend pip installation as the primary method.

## Moved

The original stand-alone script (cdst.py) has been preserved for backward compatibility and is now located under:

`scripts/cdst.py`

This version remains directly runnable with Python for environments where pip/conda installation is not possible.

## Notes

Users are encouraged to migrate to the pip-installed package for long-term use, better reproducibility, and integration with conda/bioconda.

Stand-alone script remains available for quick prototyping or restricted environments.

**Full Changelog**: [v0.2.0...v0.2.1](https://github.com/l1-mh/CDST/compare/v0.2.0...v0.2.1)
