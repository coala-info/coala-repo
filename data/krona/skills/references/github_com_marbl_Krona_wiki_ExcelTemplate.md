##### The Krona Excel template allows Windows and Mac users to create Krona charts with Microsoft Excel.

#### Requirements

* Windows: Microsoft Office 97 or later
* Mac: Microsoft Office:Mac 2011 or later

#### Installing

The Krona.xltm file can simply be unzipped and opened. If desired, it can be placed in the Excel templates folder for easier access from within Excel; this location varies by Excel platform and version.

#### Creating a chart

1. Create a new workbook from the Krona template.
   * Depending on your version of Excel, templates may be accessed from the New Workbook window or directly from the File menu.
   * The Krona template will appear in "My templates".

* If there is a warning about macros, choose "Enable macros"
* Replace the sample data.
  + Name - The name of the highest level
  + Amount - This will be added to the wedges of the categories listed (or to the highest level if no categories are listed).
  + Score (optional) - If included, these values can be used to color wedges from red to green, with red representing the lowest wedge score and green representing the highest. Each score will be weighted by its amount and will contribute to the scores of all categories listed, so the final score for each wedge will be a weighted average.
  + Categories - Names of the wedges to add to, starting from the highest level. Any number of categories can be listed.
* Click "Create chart".

#### Saving the Workbook

* Save the Workbook as "Excel Macro-Enabled Workbook (.xlsm)".
