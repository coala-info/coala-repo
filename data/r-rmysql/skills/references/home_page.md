# Download MariaDB Server

MariaDB Server is one of the world’s most popular open source
relational databases and is available in the standard
repositories of all major Linux distributions.
Look for the package mariadb-server using the package
manager of your operating system. Alternatively you can use the
following resources:

We encourage you to try it out and see whether it will work in your application. You can offer feedback in the [form of bug reports](https://jira.mariadb.org), feature requests on JIRA and share use cases on [Zulip](https://mariadb.zulipchat.com/) or the [mailing lists](https://lists.mariadb.org/postorius/lists/developers.lists.mariadb.org/). Any will help improve future MariaDB Server versions!

## Reporting bugs

The [Reporting Bugs](https://mariadb.com/kb/en/reporting-bugs/) page on the Knowledge Base has details on how to report a bug.

The developers are generally happy to help with verifying bugs. If you need help, ask on [Zulip](https://mariadb.zulipchat.com/), or on the [maria-developers mailing list](https://lists.mariadb.org/postorius/lists/developers.lists.mariadb.org/).

If the bug is repeatable, it is very helpful if you create a test case for the bug for use with [mysql-test-run](https://mariadb.com/kb/en/mysql-test-run/). See [Debugging MariaDB with mysql-test-run](https://mariadb.com/kb/en/debugging-mariadb-with-mysql-test-run/) for more information.

For End of Life releases, MariaDB Foundation will not provide security updates, however **outside contributors are welcome to submit security and bug fixes and backports to no-longer maintained versions**.
