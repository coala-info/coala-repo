MAVIS is a python (requires >=3) command-line tool for the post-processing of structural variant calls.

The easiest way to install MAVIS is via [pip](https://pypi.python.org/pypi/mavis)

`pip install mavis`

Mavis can also be cloned from [github](https://github.com/bcgsc/mavis/tree/master)

`git clone https://github.com/bcgsc/mavis.git mavis
cd mavis
git checkout v2.0.0`

See the user manual for [obtaining the required reference input files](docs/latest/mavis_input.html#reference-input-files)
