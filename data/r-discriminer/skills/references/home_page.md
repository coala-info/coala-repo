## About

---

I'm a lecturer in the [College of Computing, Data Science, and Society (CDSS)](https://data.berkeley.edu/) affiliated with the [Department of Statistics](https://statistics.berkeley.edu/) at the [University of California, Berkeley](https://www.berkeley.edu/).

Among other things I ...

enjoy authoring teaching-learning materials for statistics and data science,

  love using graphical displays to understand data with visualization,

  cherish multivariate methods for exploring, analyzing, and visualizing
data in a context of multiple variables and high dimensionality,

  like helping researchers and scientists analyze their data,

  care about computational reproducibility topics,

  believe in and practice open education.

## Books

---

[![](images/cover-csl.png)](https://allmodelsarewrong.github.io/)

Introductory text about concepts of Statistical Learning, covering some of the common supervised as well as unsupervised methods (*work in progress*).

[![](images/cover-pca4ds.png)](https://pca4ds.github.io/)

This book aims to provide a comprehensive introduction to Principal Component Analysis (PCA) for Data Science.

---

[![](images/cover-plspm.png)](https://www.gastonsanchez.com/PLS_Path_Modeling_with_R.pdf)

This book provides a hands-on introduction to Partial Least Squares Path Modeling (PLS-PM) using the R package "plspm".

[![](images/cover-saga-of-pls.png)](https://leanpub.com/sagaofpls)

What we know today as Partial Least Squares (PLS) is the result of a long period of evolution, with a vast range of methods and techniques proposed since the late 1960s / early 1970s. This book narrates the story behind the origins, development, and evolution of PLS methods.

## Software

---

I'm a passionate R user, and in a not so distant past, I was an active developer and maintainer of several R packages.
All the code is available in my [github](https://github.com/gastonstat) repositories.

[![](images/software-plspm.png)](https://github.com/gastonstat/plspm)

[**plspm**](https://github.com/gastonstat/plspm) provides a toolkit exclusively dedicated to Partial Least Squares Path Modeling (PLS-PM) analysis.

[![](images/software-pathmox.png)](https://github.com/gastonstat/pathmox)

[**pathmox**](https://github.com/gastonstat/pathmox) is dedicated to the Pathmox approach for obtaining segmentation trees in PLS-PM analysis.

[![](images/software-plsdepot.png)](https://github.com/gastonstat/plsdepot)

[**plsdepot**](https://github.com/gastonstat/plsdepot) provides a set of tools for performing Partial Least Squares (PLS) analysis of one or two data tables.

[![](images/software-arcdiagram.png)](https://github.com/gastonstat/arcdiagram)

[**arcdiagram**](https://github.com/gastonstat/arcdiagram) is a minimalist package to help you plot pretty arc diagrams in R.

[![](images/software-colortools.png)](https://github.com/gastonstat/colortools)

[**colortools**](https://github.com/gastonstat/colortools) is designed to help users generate color schemes and color palettes.

[![](images/software-pathdiagram.png)](https://github.com/gastonstat/pathdiagram)

[**pathdiagram**](https://github.com/gastonstat/pathdiagram) provides simple functions to draw basic PLS path diagrams in R.

[![](images/software-cointoss.png)](https://github.com/gastonstat/cointoss)

[**cointoss**](https://github.com/gastonstat/cointoss) is a toy package with simple functions for simulating tossing a coin.

[![](images/software-dieroller.png)](https://github.com/gastonstat/dieroller)

[**dieroller**](https://github.com/gastonstat/dieroller) is a toy package with simple functions for simulating rolling a die.

[![](images/software-binomial.png)](https://github.com/gastonstat/binomial)

[**binomial**](https://github.com/gastonstat/binomial) is a toy package with simple functions for computing binomial probabilities.

[![](images/software-matrixkit.png)](https://github.com/gastonstat/matrixkit)

[**matrixkit**](https://github.com/gastonstat/matrixkit) is an R package that provides a first aid kit for some matrix operations commonly used in multivariate data analysis methods.

[![](images/software-turner.png)](https://github.com/gastonstat/turner)

[**turner**](https://github.com/gastonstat/turner) provides a set of handy functions to turn vectors (and lists of vectors) into other indexed data structures.

[![](images/software-tester.png)](https://github.com/gastonstat/tester)

[**tester**](https://github.com/gastonstat/tester) provides human readable functions to test characteristics of some common R objects.
