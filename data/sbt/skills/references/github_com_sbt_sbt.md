Apache-2.0 license sbt sbt is a build tool for Scala, Java, and more. For general documentation, see https://www.scala-sbt.org/ . sbt 2.x This is the 2.x series of sbt. The source code of sbt is split across several GitHub repositories, including this one. sbt/io hosts sbt.io module. sbt/zinc hosts Zinc, an incremental compiler for Scala. sbt/sbt , this repository hosts modules that implement the build tool.
