# bioconductor-metabocoreutils

## Overview

Use the Bioconductor R package **MetaboCoreUtils** for: The package provides also a set of adduct definitions and information on some commercially available internal standard mixes commonly used in MS experiments.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("MetaboCoreUtils")
```
