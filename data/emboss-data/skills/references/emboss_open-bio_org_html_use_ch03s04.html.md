3.4.&nbsp;Project Mailing Lists 3.4.&nbsp;Project Mailing Lists Prev &nbsp; Chapter&nbsp;3.&nbsp;Getting Started &nbsp; Next 3.4.&nbsp;Project Mailing Lists There are three EMBOSS mailing lists which you can join. These provide a forum for discussions about EMBOSS and its future development and are a valuable source of help. Subscription requests will be approved by a moderator. 3.4.1.&nbsp;User Mailing List emboss@emboss.open-bio.org This is an open list (anyone can join) for general announcements and discussions by end-users. To subscribe to this list, please visit: http://emboss.open-bio.org/mailman/listinfo/emboss 3.4.2.&nbsp;Developer Mailing List emboss-dev@emboss.open-bio.org This is a closed list for discussions by developers using EMBOSS. To subscribe to this list, please visit: http://emboss.open-bio.org/mailman/listinfo/emboss-dev If you wish to contribute to EMBOSS, please send the code and documentation to: emboss-submit@emboss.open-bio.org 3.4.3.&nbsp;Announcements Mailing List emboss-announce@emboss.open-bio.org This is a list for major announcements, mostly about new releases and updates. It has far fewer postings than the other groups and is suitable for system administrators who want to keep the software updated but who have little interest in other discussions. To subscribe to this list, please visit: http://emboss.open-bio.org/mailman/listinfo/emboss-announce 3.4.4.&nbsp;Mail Archives Archives of previous postings to the discussion mailing lists are available. 3.4.4.1.&nbsp;User Mailing List Archive http://emboss.open-bio.org/pipermail/emboss/ This is the archive of emboss@emboss.open-bio.org . 3.4.4.2.&nbsp;Developer Mailing List Archive http://emboss.open-bio.org/pipermail/emboss-dev/ This is the archive of emboss-dev@emboss.open-bio.org .
