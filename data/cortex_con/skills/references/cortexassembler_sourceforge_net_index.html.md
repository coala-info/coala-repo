CORTEX website Cortex &nbsp; Home About Us How to reference Cortex Publications Contact Us Welcome to Cortex software for genome assembly and variation analysis Cortex is an efficient and low-memory software framework for analysis of genomes using sequence data. There are two main executables, being developed in parallel streams: cortex_con (primary contact Mario Caccamo) is for consensus genome assembly, and cortex_var (primary contact Zamin Iqbal) is for variation and population assembly. cortex_con - for consensus assembly A beta release of cortex_con is now available (new paired-end module and color-space support coming up soon).
