# hint

## Overview

HINT is a computational framework designed for the comprehensive analysis of Hi-C data. It excels at two primary tasks: detecting large-scale structural variations (specifically inter-chromosomal translocations) and identifying fine-scale chromatin interactions (loops). It is particularly useful when working with Hi-C data from cancer genomes where chromosomal rearrangements can confound standard interaction calling.

## Command Line Usage

The `hint` tool is typically invoked through specific subcommands for different stages of the Hi-C analysis pipeline.

### Preprocessing and Quality Control
Use `hint pre` to process raw Hi-C data (BAM or HiC files) into a format suitable for downstream analysis.
- Ensure input files are sorted by read name if starting from BAM.
- Specify the resolution (e.g., 50kb for translocation detection, 5kb or 10kb for interaction calling).

### Translocation Detection
Use `hint tl` to identify chromosomal translocations.
- **Input**: Preprocessed Hi-C data.
- **Best Practice**: Use a coarser resolution (50kb or 100kb) to increase signal-to-noise ratio for large-scale rearrangements.
- **Output**: A list of predicted breakpoints and translocated segments.

### Chromatin Interaction Calling
Use `hint event` to detect significant chromatin interactions or loops.
- **Input**: Preprocessed Hi-C data and, optionally, translocation information to correct for copy number and rearrangement biases.
- **Best Practice**: Use high-resolution data (5kb) for accurate loop localization.
- **Tip**: If working with cancer samples, always provide the translocation map generated by `hint tl` to avoid false positives caused by structural variations.

## Expert Tips

- **Normalization**: HINT performs internal normalization; however, ensure your input data has sufficient depth (typically >300 million reads for mammalian genomes) for high-resolution interaction calling.
- **Memory Management**: Hi-C analysis is memory-intensive. When processing large datasets at high resolution, ensure the environment has at least 32GB-64GB of RAM available.
- **Genome Versions**: Always verify that the chromosome sizes and reference genome used during alignment match the parameters provided to HINT.



## Subcommands

| Command | Description |
|---------|-------------|
| hint cnv | prediction of copy number information, as well as segmentation from Hi-C. |
| hint pre | Preprocessing Hi-C data, alignment, create contact matrices, and normalization. |
| hint tl | interchromosomal translocations and breakpoints detection from Hi-C inter-chromosomal interaction matrices. |
