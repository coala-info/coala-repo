conda conda install -c conda-forge argopy # or PyPI pip install argopy argopy is continuously tested to work under most OS (Linux, Mac, Windows) and with python versions &gt;= 3.8 Usage # Import the main data fetcher: from argopy import DataFetcher # Define what you want to fetch... # a region: f = DataFetcher (). region ([ - 85 , - 45 , 10. , 20. , 0 , 10. ]) # floats: f = DataFetcher (). float ([ 6902746 , 6902747 , 6902757 , 6902766 ]) # or specific profiles: f = DataFetcher (). profile ( 6902746 , 34 ) # then fetch and get data as xarray datasets: ds = f . load (). data # or ds = f . to_xarray () # you can even plot some information: f . plot ( 'trajectory' ) They are many more usages and fine-tuning to allow you to access and manipulate Argo data: filters at fetch time (standard vs expert users, automatically select QC flags or data mode, ...) select data sources (erddap, ftp, local, argovis, ...) manipulate data (points, profiles, interpolations, binning, ...) visualisation (trajectories, topography, histograms, ...) tools for Quality Control (OWC, figures, ...) access meta-data and other Argo-related datasets (reference tables, deployment plans, topography, DOIs, ...) improve performances (caching, parallel data fetching) Just check out the documentation for more ! 🌿 Energy impact of argopy development The argopy team is concerned about the environmental impact of your favorite software development.
