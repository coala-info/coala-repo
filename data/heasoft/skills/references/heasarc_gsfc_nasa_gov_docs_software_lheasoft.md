HEASoft A Unified Release of the FTOOLS and X-ray Analysis Software Packages The current version of HEASoft is 6.36 (30 September 2025) Included Packages FTOOLS : General and mission-specific tools to manipulate FITS files. FITSIO : Core library responsible for reading and writing FITS files. FV : General FITS file browser/editor/plotter with a graphical user interface. Xspec : X-ray astronomical spectral data analysis. Xronos : X-ray astronomical timing data analysis. Ximage : X-ray astronomical imaging data analysis. XSTAR : Tool for calculating the physical conditions and emission spectra of photoionized gases. Download Users may download either the source code for the software and compile the tools themselves, or download precompiled executables for the most widely used computer platforms. The HEASoft Release Notes 6.36 give an overview of changes since HEASoft version 6.35.2. For a list of known issues in the latest release, visit the HEASoft Known Issues page.
