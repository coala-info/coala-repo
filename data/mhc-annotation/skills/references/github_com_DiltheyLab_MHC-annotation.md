There are several ways to install this dependency depending on your environment. Giving two examples: sudo apt-get install minimap2 conda install -c bioconda minimap2 You will also need the python packages biopython and bcbio-gff . You can install them in a conda environment with: conda install -c bioconda bcbio-gff conda install -c bioconda biopython Install Install this package with pip. pip install MHC-annotation Usage This tools assumes that you are trying to annotate a sequence that stems from the human MHC region (also known as: HLA-region). If you supply any other sequence, you will most likely receive no meaningful result. You can annotate a fasta file with the following command: mhca annotate &lt;your_sequence.fa&gt; &lt;output_folder&gt; You can check whether the transcript in the resulting gff file would produce meaningful products. mhca check_CDS &lt;your_sequence.fa&gt; &lt;output_folder/your_sequence.gff&gt; About Tools to annotate haplotypes of MHC with gene and transcript information Resources Readme License MIT license Uh oh!
