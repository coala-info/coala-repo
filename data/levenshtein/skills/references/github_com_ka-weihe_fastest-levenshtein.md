Measure the difference between two strings. $ npm i fastest-levenshtein Usage Node const { distance , closest } = require ( 'fastest-levenshtein' ) // Print levenshtein-distance between 'fast' and 'faster' console . log ( distance ( 'fast' , 'faster' ) ) //=&gt; 2 // Print string from array with lowest edit-distance to 'fast' console . log ( closest ( 'fast' , [ 'slow' , 'faster' , 'fastest' ] ) ) //=&gt; 'faster' Deno import { distance , closest } from 'https://deno.land/x/fastest_levenshtein/mod.ts' // Print levenshtein-distance between 'fast' and 'faster' console . log ( distance ( 'fast' , 'faster' ) ) //=&gt; 2 // Print string from array with lowest edit-distance to 'fast' console . log ( closest ( 'fast' , [ 'slow' , 'faster' , 'fastest' ] ) ) //=&gt; 'faster' Benchmark I generated 500 pairs of strings with length N.
