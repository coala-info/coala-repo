Welcome to *hybkit*, a toolkit for analysis of hyb-format chimeric
(hybrid) RNA sequence data defined with the Hyb software package by [*Travis et al. (Methods 2014)*](https://doi.org/10.1016/j.ymeth.2013.10.015).
This genomic data-type is generated from RNA proximity-ligation and ribonomics
techniques such as Crosslinking, Ligation, and
Sequencing of Hybrids (CLASH; [*Helwak et al. (Cell 2013)*](https://doi.org/10.1016/j.cell.2013.03.043)) and quick CLASH (qCLASH; [*Gay et al. (J. Virol. 2018)*](https://doi.org/10.1128/JVI.02138-17)).

This software is available via Github, at <http://www.github.com/RenneLab/hybkit> .

Full project documentation is available at [hybkit's ReadTheDocs](https://hybkit.readthedocs.io).

Project components:
:   1. hybkit toolkit of command-line utilities for manipulating,
       analyzing, and plotting hyb-format data.
    2. The hybkit python API, an extendable documented codebase
       for creation of custom analyses of hyb-format data.
    3. Integrated analysis of predicted secondary structure (fold) information for
       the API and command-line utilities.
