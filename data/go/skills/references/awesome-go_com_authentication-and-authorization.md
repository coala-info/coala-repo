# Authentication and Authorization

Libraries for implementing authentication and authorization.

* [authboss - Modular authentication system for the web. It tries to remove as much boilerplate and "hard things" as possible so that each time you start a new web project in Go, you can plug it in, configure it, and start building your app without having to build an authentication system each time.](/authentication-and-authorization/authboss-volatiletech-github/)
* [authgate - A lightweight OAuth 2.0 Authorization Server supporting Device Authorization Grant (RFC 8628), Authorization Code Flow with PKCE (RFC 6749 + RFC 7636), and Client Credentials Grant for machine-to-machine authentication.](/authentication-and-authorization/authgate-go-authgate-github/)
* [branca - branca token specification implementation for Golang 1.15+.](/authentication-and-authorization/branca-essentialkaos-github/)
* [casbin - Authorization library that supports access control models like ACL, RBAC, and ABAC.](/authentication-and-authorization/casbin-hsluoyz-github/)
* [cookiestxt - provides a parser of cookies.txt file format.](/authentication-and-authorization/cookiestxt-mengzhuo-github/)
* [go-githubauth - Utilities for GitHub authentication: generate and use GitHub application and installation tokens.](/authentication-and-authorization/go-githubauth-jferrl-github/)
* [go-guardian - Go-Guardian is a golang library that provides a simple, clean, and idiomatic way to create powerful modern API and web authentication that supports LDAP, Basic, Bearer token, and Certificate based authentication.](/authentication-and-authorization/go-guardian-shaj13-github/)
* [go-iam - Developer-first Identity and Access Management system with a simple UI.](/authentication-and-authorization/go-iam-melvinodsa-github/)
* [go-jose - Fairly complete implementation of the JOSE working group's JSON Web Token, JSON Web Signatures, and JSON Web Encryption specs.](/authentication-and-authorization/go-jose-go-jose-github/)
* [go-jwt - A JWT (JSON Web Token) library for Go.](/authentication-and-authorization/go-jwt-deatil-github/)
* [go-jwt - JWT authentication package providing access tokens and refresh tokens with fingerprinting, Redis storage, and automatic refresh capabilities.](/authentication-and-authorization/go-jwt-pardnchiu-github/)
* [goiabada - An open-source authentication and authorization server supporting OAuth2 and OpenID Connect.](/authentication-and-authorization/goiabada-leodip-github/)
* [gologin - chainable handlers for login with OAuth1 and OAuth2 authentication providers.](/authentication-and-authorization/gologin-dghubble-github/)
* [gorbac - provides a lightweight role-based access control (RBAC) implementation in Golang.](/authentication-and-authorization/gorbac-mikespook-github/)
* [gosession - This is quick session for net/http in GoLang. This package is perhaps the best implementation of the session mechanism, or at least it tries to become one.](/authentication-and-authorization/gosession-kwynto-github/)
* [goth - provides a simple, clean, and idiomatic way to use OAuth and OAuth2. Handles multiple providers out of the box.](/authentication-and-authorization/goth-markbates-github/)
* [jeff - Simple, flexible, secure, and idiomatic web session management with pluggable backends.](/authentication-and-authorization/jeff-abraithwaite-github/)
* [jwt - Lightweight JSON Web Token (JWT) library.](/authentication-and-authorization/jwt-pascaldekloe-github/)
* [jwt - Safe, simple, and fast JSON Web Tokens for Go.](/authentication-and-authorization/jwt-cristalhq-github/)
* [jwt-auth - JWT middleware for Golang http servers with many configuration options.](/authentication-and-authorization/jwt-auth-adam-hanna-github/)
* [jwt-go - A full featured implementation of JSON Web Tokens (JWT). This library supports the parsing and verification as well as the generation and signing of JWTs.](/authentication-and-authorization/jwt-golang-jwt-github/)
* [jwx - Go module implementing various JWx (JWA/JWE/JWK/JWS/JWT, otherwise known as JOSE) technologies.](/authentication-and-authorization/jwx-lestrrat-go-github/)
* [keto - Open Source (Go) implementation of "Zanzibar: Google's Consistent, Global Authorization System". Ships gRPC, REST APIs, newSQL, and an easy and granular permission language. Supports ACL, RBAC, and other access models.](/authentication-and-authorization/keto-ory-github/)
* [loginsrv - JWT login microservice with pluggable backends such as OAuth2 (Github), htpasswd, osiam.](/authentication-and-authorization/loginsrv-tarent-github/)
* [oauth2 - Successor of goauth2. Generic OAuth 2.0 package that comes with JWT, Google APIs, Compute Engine, and App Engine support.](/authentication-and-authorization/oauth2-golang-github/)
* [oidc - Easy to use OpenID Connect client and server library written for Go and certified by the OpenID Foundation.](/authentication-and-authorization/oidc-zitadel-github/)
* [openfga - Implementation of fine-grained authorization based on the "Zanzibar: Google's Consistent, Global Authorization System" paper. Backed by CNCF.](/authentication-and-authorization/openfga-openfga-github/)
* [osin - Golang OAuth2 server library.](/authentication-and-authorization/osin-openshift-github/)
* [otpgen - Library to generate TOTP/HOTP codes.](/authentication-and-authorization/otpgen-grijul-github/)
* [otpgo - Time-Based One-Time Password (TOTP) and HMAC-Based One-Time Password (HOTP) library for Go.](/authentication-and-authorization/otpgo-jltorresm-github/)
* [paseto - Golang implementation of Platform-Agnostic Security Tokens (PASETO).](/authentication-and-authorization/paseto-o1egl-github/)
* [permissions - Library for keeping track of users, login states, and permissions. Uses secure cookies and bcrypt.](/authentication-and-authorization/permissions-xyproto-github/)
* [scope - Easily Manage OAuth2 Scopes In Go.](/authentication-and-authorization/scope-sonicroshan-github/)
* [scs - Session Manager for HTTP servers.](/authentication-and-authorization/scs-alexedwards-github/)
* [securecookie - Efficient secure cookie encoding/decoding.](/authentication-and-authorization/securecookie-chmike-github/)
* [session - Go session management for web servers (including support for Google App Engine - GAE).](/authentication-and-authorization/session-icza-github/)
* [sessions - Dead simple, highly performant, highly customizable sessions service for go http servers.](/authentication-and-authorization/sessions-adam-hanna-github/)
* [sessionup - Simple, yet effective HTTP session management and identification package.](/authentication-and-authorization/sessionup-swithek-github/)
* [sjwt - Simple jwt generator and parser.](/authentication-and-authorization/sjwt-brianvoe-github/)
* [spicedb - A Zanzibar-inspired database that enables fine-grained authorization.](/authentication-and-authorization/spicedb-authzed-github/)
* [x509proxy - Library to handle X509 proxy certificates.](/authentication-and-authorization/x509proxy-vkuznet-github/)
