## Format conversion

*agtools* can perform the following graph file format conversions.

* FASTG to GFA
* ASQG to GFA
* GFA to GraphViz DOT
* GFA to ABySS DOT
* GFA to FASTA
* GFA to adjacency matrix

You can refer to the [CLI reference](../../cli/) for details on the commands you can use for these format conversions.

Please refer to the [file formats](../../fileformats/) section for further details on the different assembly graph file formats.
