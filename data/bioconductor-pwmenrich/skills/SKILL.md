# bioconductor-pwmenrich

## Overview

Use the Bioconductor R package **PWMEnrich** for: the package also implements high-level functions for PWM scanning and visualisation. The package does not perform "de novo" motif discovery, but is instead focused on using motifs that are either experimentally derived or computationally constructed by other tools.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("PWMEnrich")
```
