The initial parsing and indexing of a GTF file is a slow process, taking over a minute. But once that is done, the genes and transcripts can be retrieved in seconds. Exons are stored efficiently per gene with each transcript only having indices into these exon lists array.array are used as much as possible for speed and efficiency.
