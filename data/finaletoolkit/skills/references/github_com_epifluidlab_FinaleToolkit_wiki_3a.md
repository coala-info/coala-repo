Note

The relevant data used in this tutorial is accessible at: [![DOI](https://camo.githubusercontent.com/75764d327f9f7ba30b45feb7f0ce0c367b8fe65ab42803e38babfed1874aa430/68747470733a2f2f7a656e6f646f2e6f72672f62616467652f444f492f31302e353238312f7a656e6f646f2e31373033363333382e737667)](https://doi.org/10.5281/zenodo.17036338)

Warning

Make sure to download the relevant index file for each `.bam` (`.bam.bai` index), `.cram` (`.cram.crai` index), or `.gz` (`.tbi` index).

---

* `BH01.hg19.frag.bed.gz`: [DOWNLOAD](https://zenodo.org/records/14284133/files/BH01.hg19.frag.bed.gz?download=1)
* `BH01.hg19.frag.bed.gz.tbi`: [DOWNLOAD](https://zenodo.org/records/14284133/files/BH01.hg19.frag.bed.gz.tbi?download=1)
* `hg19.chrom.sizes`: [DOWNLOAD](https://zenodo.org/records/14284133/files/hg19.chrom.sizes?download=1)
* `hg19.2bit`: [DOWNLOAD](https://zenodo.org/records/14284133/files/hg19.2bit?download=1)
* `100kb.bins`: [DOWNLOAD](https://zenodo.org/records/14284133/files/100kb.bins?download=1)
* `hg19.gap.delfi.final.txt`: [DOWNLOAD](https://zenodo.org/records/14284133/files/hg19.gap.delfi.final.txt?download=1)
* `ENCFF001TDO.bed`: [DOWNLOAD](https://zenodo.org/records/14284133/files/ENCFF001TDO.bed?download=1)

---

We begin by generating the DELFI data using FinaleToolkit:

```
finaletoolkit delfi \
    BH01.hg19.frag.bed.gz \
    hg19.chrom.sizes \
    hg19.2bit \
    100kb.bins \
    -o FTK_DELFI.bed \
    -q 30 \
    -w 128 \
    -g hg19.gap.delfi.final.txt \
    -b ENCFF001TDO.bed \
```

We also generate the DELFI data by following the scripts [here](https://github.com/LudvigOlsen/delfi_scripts), and combined the raw ratios with the FinaleToolkit output in TSV format.

---

* `ft_vs_delfi.tsv`: [DOWNLOAD](https://github.com/epifluidlab/FinaleToolkit/blob/main/tests/data/ftk-tutorial/ft_vs_delfi.tsv)

---

We plot the results:

```
import numpy as np
import matplotlib.pyplot as plt

delfi = np.genfromtxt('ft_vs_delfi.tsv', skip_header=0, usecols=5)
ftk = np.genfromtxt('ft_vs_delfi.tsv', skip_header=0, usecols=8)

x = np.arange(len(delfi))

plt.figure(figsize=(15, 3))
plt.plot(x, ftk, color='orange', label='FinaleToolkit', linewidth=1)  # FinaleToolkit scores
plt.plot(x, delfi, color='blue', label='Cristiano et al. 2019', linewidth=1)  # Cristiano scores

# Styling
plt.xlabel('5Mb Genomic Window', fontsize=12)
plt.ylabel('DELFI Score', fontsize=12)
plt.legend(loc='upper left', fontsize=10)
plt.ylim(0.16, 0.24)  # Adjust y-axis limits based on the data range
plt.tight_layout()

# Show or save the plot
plt.show()
```
