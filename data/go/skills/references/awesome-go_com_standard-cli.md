# Standard CLI

Libraries for building standard or basic Command Line applications.

* [acmd - Simple, useful, and opinionated CLI package in Go.](/standard-cli/acmd-cristalhq-github/)
* [argparse - Command line argument parser inspired by Python's argparse module.](/standard-cli/argparse-akamensky-github/)
* [argv - Go library to split command line string as arguments array using the bash syntax.](/standard-cli/argv-cosiner-github/)
* [carapace - Command argument completion generator for spf13/cobra.](/standard-cli/carapace-rsteube-github/)
* [carapace-bin - Multi-shell multi-command argument completer.](/standard-cli/carapace-bin-rsteube-github/)
* [carapace-spec - Define simple completions using a spec file.](/standard-cli/carapace-spec-rsteube-github/)
* [climax - Alternative CLI with "human face", in spirit of Go command.](/standard-cli/climax-tucnak-github/)
* [clîr - A Simple and Clear CLI library. Dependency free.](/standard-cli/clir-leaanthony-github/)
* [cmd - Extends the standard flag package to support sub commands and more in idiomatic way.](/standard-cli/cmd-posener-github/)
* [cmdr - A POSIX/GNU style, getopt-like command-line UI Go library.](/standard-cli/cmdr-hedzr-github/)
* [cobra - Commander for modern Go CLI interactions.](/standard-cli/cobra-spf13-github/)
* [command-chain - A go library for configure and run command chains - such as pipelining in unix shells.](/standard-cli/go-command-chain-rainu-github/)
* [commandeer - Dev-friendly CLI apps: sets up flags, defaults, and usage based on struct fields and tags.](/standard-cli/commandeer-jaffee-github/)
* [complete - Write bash completions in Go + Go command bash completion.](/standard-cli/complete-posener-github/)
* [console Closed-loop application library for Cobra commands, with oh-my-posh prompts, and more.](/standard-cli/console-reeflective-github/)
* [Dnote - A simple command line notebook with multi-device sync.](/standard-cli/dnote-dnote-github/)
* [elvish - An expressive programming language and a versatile interactive shell.](/standard-cli/elvish-elves-github/)
* [env - Tag-based environment configuration for structs.](/standard-cli/env-codingconcepts-github/)
* [flaggy - A robust and idiomatic flags package with excellent subcommand support.](/standard-cli/flaggy-integrii-github/)
* [flagvar - A collection of flag argument types for Go's standard flag package.](/standard-cli/flagvar-sgreben-github/)
* [flash-flags - Ultra-fast, zero-dependency, POSIX-compliant flag parsing library that can be used as drop-in stdlib replacement with security hardening.](/standard-cli/flash-flags-agilira-github/)
* [getopt - An accurate Go getopt, validated against the GNU libc implementation.](/standard-cli/getopt-jon-codes-github/)
* [go-arg - Struct-based argument parsing in Go.](/standard-cli/go-arg-alexflint-github/)
* [go-flags - go command line option parser.](/standard-cli/go-flags-jessevdk-github/)
* [go-getoptions - Go option parser inspired by the flexibility of Perl’s GetOpt::Long.](/standard-cli/go-getoptions-davidgamba-github/)
* [go-readline-ny - A customizable line-editing library with Emacs keybindings, Unicode support, completion, and syntax highlighting. Used in NYAGOS shell.](/standard-cli/go-readline-ny-nyaosorg-github/)
* [gocmd - Go library for building command line applications.](/standard-cli/gocmd-devfacet-github/)
* [goopt - A declarative, struct-tag based CLI framework for Go, with a broad feature set such as hierarchical commands/flags, i18n, shell completion, and validation.](/standard-cli/goopt-napalu-github/)
* [hashicorp/cli - Go library for implementing command-line interfaces.](/standard-cli/cli-hashicorp-github/)
* [hiboot cli - cli application framework with auto configuration and dependency injection.](/standard-cli/hiboot-hidevopsio-github/)
* [job - JOB, make your short-term command as a long-term job.](/standard-cli/job-liujianping-github/)
* [kingpin - Command line and flag parser supporting sub commands (superseded by kong; see below).](/standard-cli/kingpin-alecthomas-github/)
* [liner - Go readline-like library for command-line interfaces.](/standard-cli/liner-peterh-github/)
* [mcli - A minimal but very powerful cli library for Go.](/standard-cli/mcli-jxskiss-github/)
* [mkideal/cli - Feature-rich and easy to use command-line package based on golang struct tags.](/standard-cli/cli-mkideal-github/)
* [mow.cli - Go library for building CLI applications with sophisticated flag and argument parsing and validation.](/standard-cli/mow-cli-jawher-github/)
* [ops - Unikernel Builder/Orchestrator.](/standard-cli/ops-nanovms-github/)
* [orpheus - CLI framework with security hardening, plugin storage system, and production observability features.](/standard-cli/orpheus-agilira-github/)
* [pflag - Drop-in replacement for Go's flag package, implementing POSIX/GNU-style --flags.](/standard-cli/pflag-spf13-github/)
* [readline - Shell library with modern and easy to use UI features.](/standard-cli/readline-reeflective-github/)
* [sflags - Struct based flags generator for flag, urfave/cli, pflag, cobra, kingpin, and other libraries.](/standard-cli/sflags-octago-github/)
* [structcli - Eliminate Cobra boilerplate: build powerful, feature-rich CLIs declaratively from Go structs.](/standard-cli/structcli-leodido-github/)
* [strumt - Library to create prompt chain.](/standard-cli/strumt-antham-github/)
* [subcmd - Another approach to parsing and running subcommands. Works alongside the standard flag package.](/standard-cli/subcmd-bobg-github/)
* [teris-io/cli - Simple and complete API for building command line interfaces in Go.](/standard-cli/cli-teris-io-github/)
* [urfave/cli - Simple, fast, and fun package for building command line apps in Go (formerly codegangsta/cli).](/standard-cli/cli-urfave-github/)
* [version - Collects and displays CLI version information in multiple formats along with upgrade notice.](/standard-cli/version-mszostok-github/)
* [wlog - Simple logging interface that supports cross-platform color and concurrency.](/standard-cli/wlog-dixonwille-github/)
* [wmenu - Easy to use menu structure for cli applications that prompt users to make choices.](/standard-cli/wmenu-dixonwille-github/)
