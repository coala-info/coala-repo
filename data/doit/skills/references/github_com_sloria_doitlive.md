It reads a file of shell commands and replays the commands in a fake terminal session as you type random characters. Get it now macOS with Homebrew : $ brew update $ brew install doitlive With pip: $ pip install doitlive Quickstart Create a file called session.sh . Fill it with bash commands. Run doitlive play session.sh . $ doitlive play session.sh Type like a madman.
