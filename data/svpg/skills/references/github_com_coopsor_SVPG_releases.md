[1.3.0](/coopsor/SVPG/releases/tag/v1.3.0)

1. Added support for graph-aware split alignment, enabling accurate detection of three classes of complex SVs, including tandem duplications, inversions, and translocations.
2. Introduced a PAN\_KNOWN annotation flag in the VCF output of pangenome-guided mode, allowing simultaneous reporting and differentiation of pangenome-supported common SVs and sample-specific novel SVs.
3. Implemented an optional realignment module (--realign). This module retrieves reads from graph-alignment challenging regions, assembles a local consensus sequence using abPOA, and realigns the consensus back to the reference using mappy to refine both breakpoints and variant sequences.
