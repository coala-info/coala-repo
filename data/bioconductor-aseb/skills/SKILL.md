# bioconductor-aseb

name: bioconductor-aseb
description: Predict lysine (K)-acetyl-transferase (KAT) specific acetylation sites using the Acetylation Set Enrichment Based (ASEB) method. Use this skill when analyzing protein sequences to identify which KAT families (e.g., CBP/P300, GCN5/PCAF) are likely responsible for acetylating specific lysine sites.

## Overview

The ASEB package implements a GSEA-like (Gene Set Enrichment Analysis) algorithm to predict KAT-specific acetylation sites. It compares a query peptide sequence (the lysine site plus 8 amino acids on each side) against a KAT-specific set of known sites and a background set of random sites. It calculates an Enrichment Score (ES) and a nominal p-value via permutation to determine the likelihood of acetylation by a specific KAT family.

## Core Workflow

### 1. Loading Data
ASEB works with FASTA files. You can either load them into R objects or provide file paths directly to the processing functions.

```r
library(ASEB)

# Load sequences into a SequenceInfo object
fasta_path <- system.file("extdata", "background_sites.fa", package="ASEB")
seq_info <- readSequence(fasta_path)
```

### 2. Predicting Acetylation Sites
Use `asebSites` for specific lysine peptides or `asebProteins` to scan entire protein sequences for all potential lysine sites.

```r
# Using R objects (SequenceInfo)
# background: 10,000 random sites + KAT-specific sites
# predefined: Known sites for a specific KAT family
# test: The sites you want to predict
results <- asebSites(backgroundSites, predefinedSites, testSites, permutationTimes=1000)

# Access results table
# site: ID, ES: Enrichment Score, p-value: Significance
print(results$results)
```

### 3. Visualizing Results
ASEB provides two main plotting functions to interpret the enrichment results.

*   **Enrichment Curve**: Shows the running-sum process for a specific site.
    ```r
    # curveInfo is generated by asebSites or asebProteins
    drawEScurve(results$curveInfo, max_p_value=0.05, min_es=0.1)
    ```
*   **Protein Statistics**: Visualizes ES and p-values for all lysine sites along a protein's length.
    ```r
    drawStat(results$curveInfo)
    ```

## Key Functions

*   `readSequence(file)`: Reads FASTA files into `SequenceInfo` objects.
*   `asebSites(background, predefined, test, ...)`: Predicts acetylation for specific 17-mer peptides (K in the middle).
*   `asebProteins(background, predefined, test, ...)`: Predicts acetylation for all lysines within full protein sequences.
*   `drawEScurve(...)`: Plots the GSEA-style running sum for individual sites.
*   `drawStat(...)`: Plots a summary of all sites on a protein (X-axis = position, Y-axis = ES/p-value).

## Usage Tips

*   **Input Format**: Query sequences should ideally be 17 amino acids long (the target Lysine with 8 residues flanking each side).
*   **KAT Families**: The package includes built-in sets for CBP/P300 (`predefined_sites.fa`) and GCN5/PCAF (`predefined_sites2.fa`).
*   **Performance**: For very large datasets, pass the file paths (strings) directly to `asebSites` or `asebProteins` instead of loading them with `readSequence` first to save memory.
*   **Interpretation**: Lower p-values indicate higher significance. If p-values are similar, the site with the higher Enrichment Score (ES) is more likely to be a true acetylation site.
