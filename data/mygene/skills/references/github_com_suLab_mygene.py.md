It's designed with simplicity and performance emphasized. mygene , is an easy-to-use Python wrapper to access MyGene.Info services. Requirements python &gt;=2.6 (including python3) requests (install using "pip install requests") Optional dependencies pandas (install using "pip install pandas") is required for returning a list of gene objects as DataFrame .
