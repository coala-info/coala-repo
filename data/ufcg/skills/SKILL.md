# ufcg

## Overview
UFCG is a specialized bioinformatics suite designed for fungal phylogenomics. It utilizes a curated database of 61 highly conserved marker genes across the kingdom Fungi to provide accurate taxonomic placement and evolutionary analysis. The pipeline automates the workflow from raw sequences to validated Newick trees, featuring a unique Gene Support Index (GSI) to measure branch reliability based on individual gene tree concordance.

## Core CLI Workflows

### 1. Database Initialization
Before running analysis, download the required HMM profiles and reference sequences.
```bash
ufcg download -t minimum
```

### 2. Profiling (Marker Extraction)
Extract core genes from fungal genome assemblies (.fasta) or RNA-seq reads (.fastq).

**From Genome Assemblies:**
```bash
ufcg profile -i /path/to/assemblies/ -o ./output_profiles/ --set PRO -t 16
```
*   `-i`: Input directory containing FASTA files.
*   `-o`: Output directory for `.ucg` (JSON-formatted profile) files.
*   `--set PRO`: Specifies the protein marker set.
*   `-m`: (Optional) Path to a TSV metadata file to annotate the profiles.

**From RNA-seq Reads:**
```bash
ufcg profile-rna -i /path/to/reads/ -o ./output_profiles/ -t 16
```

### 3. Phylogenetic Tree Reconstruction
Align extracted markers and infer a phylogenetic tree from a collection of `.ucg` files.
```bash
ufcg tree -i ./output_profiles/ -o ./phylogeny_results/ -n my_fungal_run -l taxon,strain -t 16
```
*   `-l`: Defines leaf labels. Options include `uid`, `acc`, `label`, `taxon`, `strain`, `taxonomy`.
*   `-a`: Alignment type (default: `nucleotide`; options: `protein`, `codon`, `codon12`).
*   `-p`: Specify tree-builder (default: `iqtree`; options: `fasttree`, `raxml`).

### 4. Label Pruning and Refinement
Update tree leaf labels without re-running the entire alignment/inference process.
```bash
ufcg prune -i ./phylogeny_results/my_fungal_run.trm -g UFCG -l taxon,taxonomy
```
*   `-i`: Path to the `.trm` (JSON metadata) file generated by the `tree` module.
*   `-g`: Target tree (use `UFCG` for the concatenated tree or a specific gene name like `RPB2`).

## Metadata Configuration
To ensure proper labeling, provide a TSV file with the following mandatory headers:
`Filename`, `Label`, `Accession`, `Taxon name`, `NCBI name`, `Strain name`, `Taxonomy`.

## Expert Tips & Troubleshooting
*   **Environment Variables**: If using AUGUSTUS, you must set the config path: `export AUGUSTUS_CONFIG_PATH=/path/to/augustus/config/`.
*   **Interactive Mode**: Use `ufcg profile -u` to launch an interactive wizard that guides you through parameter selection.
*   **GSI Values**: The `tree` module generates `concatenated_gsi_[N].nwk`. This tree replaces standard bootstrap values with the Gene Support Index, indicating how many individual gene trees support that specific branch.
*   **Intermediate Files**: `.ucg` files are JSON-formatted. You can manually inspect or edit them to check extracted sequences or metadata before building trees.
