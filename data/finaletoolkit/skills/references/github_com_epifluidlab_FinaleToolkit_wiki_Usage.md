Please see the pages listed below for tutorials for each of the fragmentomic features implemented by FinaleToolkit. Refer to our [project website](https://epifluidlab.github.io/FinaleToolkit/) for detailed API documentation and CLI man pages.

We have designed the tutorials below to be feasible for a decently performant PC.

* [`frag-length-bins`](https://github.com/epifluidlab/FinaleToolkit/wiki/Fragment-Length#frag-length-bins)
* [`frag-length-intervals`](https://github.com/epifluidlab/FinaleToolkit/wiki/Fragment-Length#frag-length-intervals)
* [`coverage`](https://github.com/epifluidlab/FinaleToolkit/wiki/Coverage#coverage)
* [`cleavage-profile`](https://github.com/epifluidlab/FinaleToolkit/wiki/Cleavage-Profile#cleavage-profile)
* [`end-motifs`](https://github.com/epifluidlab/FinaleToolkit/wiki/End-Motifs#end-motifs)
* [`interval-end-motifs`](https://github.com/epifluidlab/FinaleToolkit/wiki/End-Motifs#interval-end-motifs)
* [`wps`](https://github.com/epifluidlab/FinaleToolkit/wiki/Windowed-Protection-Score#wps)
* [`adjust-wps`](https://github.com/epifluidlab/FinaleToolkit/wiki/Windowed-Protection-Score#adjust-wps)
* [`mds`](https://github.com/epifluidlab/FinaleToolkit/wiki/Motif-Diversity-Score#mds)
* [`interval-mds`](https://github.com/epifluidlab/FinaleToolkit/wiki/Motif-Diversity-Score#interval-mds)
* [`gap-bed`](https://github.com/epifluidlab/FinaleToolkit/wiki/DELFI#gap-bed)
* [`delfi`](https://github.com/epifluidlab/FinaleToolkit/wiki/DELFI#delfi)
* [`filter-bam`](https://github.com/epifluidlab/FinaleToolkit/wiki/Miscellaneous-Utility-Functions#filter-bam)
* [`agg-bw`](https://github.com/epifluidlab/FinaleToolkit/wiki/Miscellaneous-Utility-Functions#agg-bw)
