Contents NAME SYNOPSIS DESCRIPTION FUNCTIONS &amp; METHODS NAME IPC::Run::Win32Process -- deliver nonstandard command lines via IPC::Run. SYNOPSIS use File::Spec (); use IPC::Run qw(run); use IPC::Run::Win32Process (); use Win32 (); $find_exe = File::Spec-&gt;catfile(Win32::GetFolderPath(Win32::CSIDL_SYSTEM), &#39;find.exe&#39;); run(IPC::Run::Win32Process-&gt;new($ENV{COMSPEC}, q{cmd.exe /c echo &quot;&quot;}), &#39;|&#39;, IPC::Run::Win32Process-&gt;new($find_exe, q{find_exe &quot;&quot;&quot;&quot;&quot;&quot;}), &#39;&gt;&#39;, \$out); DESCRIPTION This class facilitates executing Windows programs that don&#39;t use standard command line parsing rules . Notable programs having nonstandard rules include cmd.exe , cscript.exe , and Cygwin programs called from non-Cygwin programs. IPC::Run will use the two strings, verbatim, as the lpApplicationName and lpCommandLine arguments of CreateProcessA(). This furnishes unfiltered control over the child process command line.
