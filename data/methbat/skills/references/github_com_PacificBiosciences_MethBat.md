Key features include: Creation of a methylation profile for a collection of genomic regions (e.g., CpG islands) Creation of cohort / background profiles from a collection of methylation profiles Analysis of cohort profiles comparing sub-groups (e.g., case v. control, male v. female) Segmentation of the CpGs in a dataset into Methylated, Unmethylated, or ASM regions Cell type deconvolution from bulk methylation data using reference atlases Authors: Matt Holt , Chris Saunders Early release warning Please note that MethBat is still in early development. We are still tweaking the input / output file formats and making changes that can affect the behavior of the program.
