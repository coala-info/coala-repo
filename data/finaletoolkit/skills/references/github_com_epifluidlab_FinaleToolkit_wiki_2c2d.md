# Set up the environment

Start with an empty directory (we refer to this as the starting directory).
First, download the files linked above.

Make sure you have FinaleToolkit installed according to the [quick start guide](https://github.com/epifluidlab/FinaleToolkit/wiki/Quick-Start).
Finally, include the following bed file (`site.bed`):

```
12\t34442000\t34447000
```

Make sure that `\t` is replaced by actual tab characters.

# Generating Window Protection Score using the scripts provided by Snyder et al.

To generate these figures, we should first clone the original [WPS scripts](https://github.com/shendurelab/cfDNA) from ShendureLab:

```
git clone git@github.com:shendurelab/cfDNA.git
```

Please refer to their included README for details on setup and environment. Note that they use Python 2 and include a unique fork of Samtools.

After cloning this repository and downloading its dependencies, we can generate WPS data on our data of choice. From the starting directory, run the following:

```
cfDNA/samtools view -u -m 120 -M 180 BH01.hg19.mdups.12.asat.example.bam 12:34441000-34448000 | cfDNA/FilterUniqueBAM.py -p | cfDNA/extractReadStartsFromBAM2Wig.py -p -r 12:34442000-34447000 -w 120 -c COVERAGE.wig.gz -n WPS.wig.gz -s STARTS.wig.gz
```

As a result, you should see three new files in the starting directory:

* `COVERAGE.wig.gz`
* `WPS.wig.gz`
* `STARTS.wig.gz`

# Generating WPS using FinaleToolkit

Run the following:

```
finaletoolkit wps BH01.hg19.mdups.12.asat.example.bam site.bed -o results.bw
```

Now we have WPS from both the original scripts and FinaleToolkit

# Comparing and plotting results.

In a Jupyter notebook or Python script, we can open both files and compare them. First, import needed dependencies:

```
import gzip

import numpy as np
import pandas as pd
import scipy.stats as stats
import matplotlib.pyplot as plt
import matplotlib as mpl
import pyBigWig as pbw
```

Then we read the FinaleToolkit results:

```
with pbw.open('./results.bw', 'r') as bigwig:
    scores = bigwig.values('12', 34442000, 34447000)
    ft_wps = [('12', pos, pos+1, score) for pos, score in zip(range(34442000, 34447001), scores)]
```

Next, we read the Snyder et al. results:

```
shendure_wps = []
with gzip.open('./WPS.wig.gz', 'rt') as shendure_file:
    lines = shendure_file.readlines()
    header = lines[0].split()
    chrom = header[1].split('=')[1][3:]
    start = int(header[2].split('=')[1])
    step = int(header[3].split('=')[1])
    pos = start
    for line in lines[1:]:
        score = float(line)
        shendure_wps.append((chrom, pos, pos+1, score))
        pos += 1
```

We can turn both into numpy structure arrays (or pandas dataframes if you prefer):

```
t_wps = np.array(ft_wps, dtype=[('chrom', '<U16'), ('start', '<i8'), ('stop', '<i8'), ('score', '<f8')])
shendure_wps = np.array(shendure_wps, dtype=[('chrom', '<U16'), ('start', '<i8'), ('stop', '<i8'), ('score', '<f8')])
```

Now we can compare the two using `scipy.stats`.

```
pearson, _ = stats.pearsonr(shendure_wps['score'][:-1], ft_wps['score'])
spearman, _ = stats.spearmanr(shendure_wps['score'][:-1], ft_wps['score'])

pearson, spearman
```

```
# Regression line
ols = OLS(ft_wps['score'], shendure_wps['score'][:-1])
model = ols.fit()
regression = model.predict(shendure_wps['score'])
r2 = model.rsquared_adj
r2
```

We can also plot them against each other using `matplotlib`

```
# plotting comparison
fig, axes = plt.subplots(2, 2, sharex=True)
fig.set_dpi(300)
fig.set_size_inches(12, 5)
axes[0, 0].plot(ft_wps['start'], ft_wps['score'], 'C3', label="FinaleTools")
axes[0, 0].set_ylabel('FinaleTools')
axes[1, 0].plot(shendure_wps['start'], shendure_wps['score'], 'C0', label="Snyder et al. (2016)")
axes[1, 0].set_ylabel('Snyder et al. (2016)')

plt.setp(axes[1, 0].get_xticklabels(), rotation=20, ha="right",
         rotation_mode="anchor")
x_formatter = mpl.ticker.ScalarFormatter(useOffset=False, useMathText=False)
x_formatter.set_scientific(False)

fig.subplots_adjust(hspace=0)

axes[1, 0].xaxis.set_major_formatter(x_formatter)
axes[0, 0].grid()
axes[1, 0].grid()

gs = axes[0, 1].get_gridspec()
for ax in axes[:, 1]:
    ax.remove()

# for fig D
scatter_ax = fig.add_subplot(gs[:, 1])
scatter_ax.scatter(shendure_wps['score'][:-1], ft_wps['score'], s=40, alpha=0.1, linewidths=0)
scatter_ax.plot(shendure_wps['score'], regression, 'r-')
scatter_ax.set_xlabel('Snyder et al (2016)')
scatter_ax.set_ylabel('FinaleTools')
stats_string = f"$R^2={r2:.4f}$"

scatter_ax.text(-30, -80, stats_string)
axes[0, 0].set_title('(C)')
scatter_ax.set_title('(D)')
fig.savefig("wps_ft_vs_snyder.svg", format="svg")
```
