## Overview

The `pyqi` tool offers a Python API for interacting with Quantum Intermediate Representation (QIR). It's designed to facilitate the integration of QIR toolchains within Python-based workflows and to enable the execution of generated QIR. This includes capabilities for parsing and generating QIR, as well as running it with just-in-time compilation.

## Usage

`pyqi` is primarily used as a Python library. Its core functionality revolves around generating, parsing, and evaluating QIR.

### Generating QIR

The `pyqi` library allows for the programmatic generation of QIR. This is typically done by defining quantum circuits or operations within Python and then translating them into QIR.

### Parsing QIR

You can use `pyqi` to parse existing QIR code. This is useful for analyzing, validating, or manipulating QIR generated by other tools.

### Evaluating/Running QIR

`pyqi` provides infrastructure to execute generated QIR. This often involves just-in-time (JIT) compilation, allowing for immediate execution and testing of quantum programs represented in QIR.

### Integration with Python Frontends

A key use case for `pyqi` is its seamless integration into existing Python-based quantum computing frameworks or frontends. It acts as a bridge, enabling these Python environments to leverage the QIR toolchain.

## Expert Tips

*   **Leverage `qirlib` for advanced scenarios:** For more complex QIR manipulation and LLVM-based operations, consider the underlying Rust library `qirlib` which `pyqi` wraps.
*   **Understand QIR structure:** Familiarize yourself with the structure of QIR, which is based on LLVM IR, to effectively generate and parse it.
*   **Explore examples:** The `examples/` directory in the `pyqir` repository (as seen in `github_com_qir-alliance_pyqir.md`) is a valuable resource for understanding practical usage patterns.
