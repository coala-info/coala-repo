One can also add negspy/data/${ASSEMBLY}/chromSizes.txt which defines how the chromosomes should be ordered if presented in a linear fashion. There's no correct ordering, but for many assemblies convention dictates having the numbered chromosomes first, followed by X, Y, M and the extra contigs.
