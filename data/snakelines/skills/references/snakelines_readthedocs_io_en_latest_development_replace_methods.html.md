Replace default methods &#8212; SnakeLines 1.1.8 documentation Navigation index next | previous | SnakeLines 1.1.8 documentation &#187; Replace default methods ¶ Methods in each analysed step can be easily replaced with alternative tool or custom solution. All supported values are enumerated in the configuration value. For example, you may use spades or unicycler for assembly by changing value under method: assembly : assembler : # Method for joining reads method : spades # Supported values: spades, unicycler mode : standard # Supported values: standard, meta, plasmid, rna, iontorrent careful : True # Can not be combined with the meta mode. Tries to reduce number of mismatches and short indels, longer runtime You may use tool that is not yet implemented, or custom solution. In that case, you need to create new rule in the /rules/assembly/assembler/&lt;your_assembler&gt;.snake. Rule should contain the same set of outputs as the other assemblers as is defined in the dependency file (/src/dependency.yaml). Than you can easily switch analysis to your solution changing configuration file: assembly : assembler : # Method for joining reads method : &lt;your_assembler&gt; # Supported values: spades, unicycler, &lt;your_assembler&gt; &lt;param_name&gt; : &lt;its_value&gt; # Specific attributes for &lt;your_assembler&gt; Do not forget to change configuration of the tool according to your parameters as well.
