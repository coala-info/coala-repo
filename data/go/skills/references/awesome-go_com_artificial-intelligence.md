# Artificial Intelligence

Libraries for building programs that leverage AI.

* [ai - A Go toolkit for building AI agents and applications across multiple providers with unified LLM, embeddings, tool calling, and MCP integration.](/artificial-intelligence/ai-joakimcarlsson-github/)
* [chromem-go - Embeddable vector database for Go with Chroma-like interface and zero third-party dependencies. In-memory with optional persistence.](/artificial-intelligence/chromem-go-philippgille-github/)
* [fun - The simplest but powerful way to use large language models (LLMs) in Go.](/artificial-intelligence/fun-tozd-gitlab/)
* [goai - Go SDK for building AI applications. One SDK, 20+ providers. Inspired by Vercel AI SDK.](/artificial-intelligence/goai-zendev-sh-github/)
* [hotplex - AI Agent runtime engine with long-lived sessions for Claude Code, OpenCode, pi-mono and other CLI AI tools. Provides full-duplex streaming, multi-platform integrations, and secure sandbox.](/artificial-intelligence/hotplex-hrygo-github/)
* [langchaingo - LangChainGo is a framework for developing applications powered by language models.](/artificial-intelligence/langchaingo-tmc-github/)
* [langgraphgo - A Go library for building stateful, multi-actor applications with LLMs, built on the concept of LangGraph，with a lot of builtin Agent architectures.](/artificial-intelligence/langgraphgo-smallnest-github/)
* [LocalAI - Open Source OpenAI alternative, self-host AI models.](/artificial-intelligence/localai-mudler-github/)
* [Ollama - Run large language models locally.](/artificial-intelligence/ollama-jmorganca-github/)
* [OllamaFarm - Manage, load-balance, and failover packs of Ollamas.](/artificial-intelligence/ollamafarm-presbrey-github/)
