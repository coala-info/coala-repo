Third-party tools used into HORmon:

1. HORmon uses CentromereArchitect and StringDecomposer to annotate centromeric sequences on each clustering stage.

For more details please see:

https://github.com/ablab/HORmon/tree/centromere-architect

https://github.com/ablab/stringdecomposer

2. HORmon uses edlib library to calculate monomer alignment identities.

For more details about edlib library please refer to

https://github.com/Martinsos/edlib and to the following paper:

Martin Šošić, Mile Šikić; Edlib: a C/C ++ library for fast, exact sequence alignment using edit distance.

Bioinformatics 2017 btw753. doi: 10.1093/bioinformatics/btw753

3. HORmon uses Clustal Omega to calculate monomer consensus on each clustering stage.

For more details please refere to

Sievers F, Wilm A, Dineen DG, Gibson TJ, Karplus K, Li W, Lopez R, McWilliam H, Remmert M, Söding J, Thompson JD, Higgins DG (2011).

Fast, scalable generation of high-quality protein multiple sequence alignments using Clustal Omega.

Molecular Systems Biology 7:539 doi:10.1038/msb.2011.75
