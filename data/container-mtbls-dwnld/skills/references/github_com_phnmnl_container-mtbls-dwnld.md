Website http://workflow4metabolomics.org Git Repository https://github.com/phnmnl/container-mtbls-dwnld.git Installation For local individual installation: docker pull container-registry.phenomenal-h2020.eu/phnmnl/mtbls-dwnld Usage Instructions For direct docker usage: docker run container-registry.phenomenal-h2020.eu/phnmnl/mtbls-dwnld -h Networking considerations The networking needs to allow UDP connections with source port 33001 to the container for the fast Aspera download. This is the case for most commercial cloud providers, including Amazon AWS and Google GCP, but some local installations might have additional firewall rules in place. See also https://test-connect.asperasoft.com/ for more information. If these connections are not allowed, the fallback to the wget download via is still possible.
