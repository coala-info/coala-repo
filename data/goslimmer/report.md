### Original Help Text
```text
GOSlimmer converts a set of annotations from GO to a given GOslim version

Usage: 'java -jar GOSlimmer.jar OPTIONS'

Options:
-g, --go FILE_PATH	Path to the full Gene Ontology OBO or OWL file
-s, --slim FILE_PATH	Path to the GOslim OBO or OWL file
-a, --annotation FILE_PATH	Path to the tabular annotation file (GAF, BLAST2GO or 2-column table format
-o, --output FILE_PATH	Path to the output GOslim annotation file]
