# gfatools

## Overview

gfatools is a specialized suite of utilities designed for the efficient manipulation of sequence graphs. Developed primarily for the GFA and rGFA formats, it allows researchers to navigate complex assembly graphs by extracting local neighborhoods, converting graph-based representations into linear formats like FASTA or BED, and performing essential graph "cleaning" operations such as transitive reduction and bubble popping. It is particularly effective when dealing with pangenome graphs generated by tools like minigraph or assembly graphs from hifiasm.

## Common CLI Patterns

### Subgraph Extraction
To isolate a specific region of a graph for visualization or local analysis, use the `view` command.
- **Extract by segment and radius**:
  `gfatools view -l [SegmentName] -r [Radius] input.gfa > subgraph.gfa`
  *Note: The radius defines how many steps away from the target segment to include. gfatools uses BFS for this extraction.*

### Format Conversion
gfatools is the standard tool for linearizing graph data.
- **Convert GFA segments to FASTA**:
  `gfatools gfa2fa input.gfa > segments.fa`
- **Convert rGFA to stable FASTA**:
  `gfatools gfa2fa -s input.gfa > stable.fa`
- **Convert rGFA to BED**:
  `gfatools gfa2bed -m input.gfa > output.bed`
  *The `-m` flag is often used to handle multiple records or specific rGFA tags.*

### Graph Statistics and Validation
- **Get graph statistics**:
  `gfatools stat input.gfa`
  *Provides basic info including number of segments, arcs, and total sequence length.*
- **Check for multiple edges**:
  `gfatools chk input.gfa`

### Assembly Graph Cleaning
For assembly graphs, gfatools can simplify the topology:
- **Transitive reduction**:
  `gfatools asm -t input.gfa > reduced.gfa`
- **Bubble popping**:
  `gfatools asm -b input.gfa > clean.gfa`

## Expert Tips

- **rGFA Support**: When working with pangenome graphs (like those from minigraph), always check if the file is in rGFA format. Use the `-s` flag in `gfa2fa` to ensure you are getting the stable coordinate-based sequence rather than just individual segments.
- **BFS vs DFS**: Older versions of gfatools used DFS for subgraph extraction. Ensure you are using a recent version (v0.5+) to benefit from BFS-based extraction, which is more intuitive for biological radius searches.
- **PAF Integration**: If you have alignments in PAF format (specifically from hifiasm), you can convert them directly to GFA using the `paf2gfa` utility included in the repository.
- **Bubble Analysis**: Use the `bubble` command to output a list of all bubbles in the graph, which is useful for identifying variants or assembly artifacts.



## Subcommands

| Command | Description |
|---------|-------------|
| gfatools bubble | Extract bubbles from a GFA graph. |
| gfatools gfa2bed | Convert GFA to BED format |
| gfatools gfa2fa | Convert a GFA file to FASTA format |
| gfatools view | View and subset GFA graphs |
| gfatools_asm | Perform assembly operations on a GFA graph. |
| gfatools_blacklist | Identify and output regions from a GFA graph that are considered 'blacklisted'. |
| gfatools_stat | Print statistics about a GFA file. |
