FasTAN Reference FasTAN &lt;source:path&gt;[&lt;precursor] &lt;target:path&gt;[.1aln] &lt;precursor&gt; = .1gdb | &lt;fa_extn&gt; | &lt;1_extn&gt; &lt;fa_extn&gt; = (.fa|.fna|.fasta)[.gz] &lt;1_extn&gt; = any valid 1-code sequence file type FasTAN takes a FASTA or 1GDB as input and outputs the off-diagonal alignment delimiting tandem repeats and their period as a .1aln file. For details on a .1aln file and how to convert it to PAF or PSL format, see the repo FASTGA .
