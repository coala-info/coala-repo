# bioconductor-targetsearch

## Overview

Use the Bioconductor R package **TargetSearch** for: The package will automatically search those targets in the chromatograms resulting in a data matrix that can be used for further data analysis.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("TargetSearch")
```
