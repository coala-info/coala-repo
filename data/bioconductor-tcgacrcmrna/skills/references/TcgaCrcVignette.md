Introduction

The package contain data objects of primary colorectal cancer for a total of 450
mRNA gene expression profile

The ExpressionSet include samples information (phenoData) and probe (genes)

information The data were provided by TCGA consortium as level3 data [1].
data were generated by HiSeq and GenomeAnalyzer platforms.

> library(TCGAcrcmRNA)

2

Introduction

Data can be loaded as follow.

> data(TCGAmrna)
