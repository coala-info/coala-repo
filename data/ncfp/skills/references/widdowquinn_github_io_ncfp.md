README.md - ncfp | ncfp ncfp Program and package that retrieves nucleotide coding sequences from NCBI that correspond to a set of input protein sequences. View the Project on GitHub widdowquinn/ncfp README.md - ncfp This repository contains code for a script that identifies and writes the corresponding nucleotide sequences for each protein in an input multiple sequence file to be used, for example, in backthreading coding sequences onto protein alignments for phylogenetic analyses. ncfp uses the NCBI accession or UniProt gene name (as appropriate) to identify source nucleotide sequences in the NCBI databases, download them, and write them to a file. Quickstart: ncfp at the command-line Providing an input file of protein sequences as &lt;INPUT&gt;.fasta , and writing output to the directory &lt;OUTPUT&gt; , while specifying a user email to NCBI of &lt;EMAIL&gt; will generate two files: &lt;OUTPUT&gt;/ncfp_aa.fasta and &lt;OUTPUT&gt;/ncfp_nt.fasta . ncfp &lt;INPUT&gt;.fasta &lt;OUTPUT&gt; &lt;EMAIL&gt; The file &lt;OUTPUT&gt;/ncfp_aa.fasta contains the amino acid sequences for all input proteins for which a corresponding nucleotide coding sequence could be identified, in FASTA format. The file &lt;OUTPUT&gt;/ncfp_nt.fasta contains nucleotide coding sequences, where they could be found, for all the input proteins, in FASTA format. Any input protein sequences for which a corresponding nucleotide sequence cannot be recovered, for any reason, are placed in the file &lt;OUTPUT&gt;/skipped.fas .
