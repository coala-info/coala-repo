# Calling Consensus Reads - A Brief Overview

Reads with the same source molecule id are examined base-by-base to assess the likelihood of each base in the
source molecule. The likelihood model consists of three major steps:

1. [Adjusting the input base qualities](#adjusting-the-input-base-qualities)
2. [Computing the maximum posterior probability base quality](#computing-the-maximum-posterior-probability-base-quality)
3. [Adjusting the output consensus base quality](#adjusting-the-output-consensus-base-quality)

## Adjusting the input base qualities.

First, the base qualities are adjusted. The base qualities are assumed to represent the probability of a sequencing error (i.e. the sequencer observed the wrong base present on the cluster/flowcell/well). A fixed value is subtracted from the phred-scaled base qualities (ex. Q30 with a shift of 10 becomes Q20). Next, the base qualities are capped to a maximum phred-scaled value. These adjustments should only be used if there is a reason to believe the input base qualities are systematically over-estimated, otherwise this step should be ignored.

[![](https://camo.githubusercontent.com/3def3f78c8ee7a3951d0e5544934d7820de769f2810f6bfbd215a94036738b93/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f51272673706163653b3d2673706163653b6d696e28512d2673706163653b535f712c2673706163653b4d5f712929 "Q' = min(Q- S_q, M_q))")](https://www.codecogs.com/eqnedit.php?latex=Q'&space;=&space;min(Q-&space;S_q,&space;M_q)))

Next, the base qualities are converted to a probability (of error):

[![](https://camo.githubusercontent.com/c875fdaa86b1a390a8ae187290306b4c24c5d2130fd3e0d36c935ffadea9b239/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f505f7b51277d2673706163653b3d2673706163653b31305e7b282d51272f3130297d "P_{Q'} = 10^{(-Q'/10)}")](https://www.codecogs.com/eqnedit.php?latex=P_{Q'}&space;=&space;10^{(-Q'/10)})

Next a probability is calculated to encapsulate both the probability of a sequencing error and the probability of a base substitution between the time the source molecule ids were attached to the DNA and the time the base was sequenced. The resulting probability is the error rate of all processes from right after integrating the molecular tag through to the end of sequencing.

[![](https://camo.githubusercontent.com/48decf08e9986e51d45384f4ee87f76dd3a1dc13fae889f5628dad5aac62ab3a/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f505f7b51277d272673706163653b3d2673706163653b4572725f7b706f73747d2a28312d505f7b51277d292673706163653b26706c75733b2673706163653b28312d4572725f7b706f73747d292a505f7b51277d2673706163653b26706c75733b2673706163653b284572725f7b706f73747d2673706163653b2a2673706163653b505f7b51277d2673706163653b2a2673706163653b322f332929 "P_{Q'}' = Err_{post}*(1-P_{Q'}) + (1-Err_{post})*P_{Q'} + (Err_{post} * P_{Q'} * 2/3))")](https://www.codecogs.com/eqnedit.php?latex=P_{Q'}'&space;=&space;Err_{post}*(1-P_{Q'})&space;&plus;&space;(1-Err_{post})*P_{Q'}&space;&plus;&space;(Err_{post}&space;*&space;P_{Q'}&space;*&space;2/3)))

This latter formula computes the probability of seeing an error in the base sequence if there are two independent error processes. We sum three terms:

1. the probability of an error in trial one and no error in trial two: [![](https://camo.githubusercontent.com/40d26279bdcfc7067c963e85ba0a9521f540433bc1511b476b51bd1aee0f65ee/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f507228413d4572726f722c2673706163653b423d4e6f4572726f72292e "Pr(A=Error, B=NoError).")](https://www.codecogs.com/eqnedit.php?latex=Pr(A=Error,&space;B=NoError).)
2. the probability of no error in trial one and an error in trial two: [![](https://camo.githubusercontent.com/901cdf6b7d4073f39a95ca18ce05fa4e9566c6173a50f453a26521cb6f7b319f/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f507228413d4e6f4572726f722c2673706163653b423d4572726f72292e "Pr(A=NoError, B=Error).")](https://www.codecogs.com/eqnedit.php?latex=Pr(A=NoError,&space;B=Error).)
3. the probability of an error in both trials, but when the second trial does not reverse the error in first one, which for DNA (4 bases) would only occur 2/3 times: [![](https://camo.githubusercontent.com/3a99b94ad932be2738501743f28b0db9b87bc6b7e3362fa911a87869ee59f1c9/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f507228413d785c746f2673706163653b792c2673706163653b423d792673706163653b5c746f2673706163653b7a292673706163653b2a2673706163653b507228785c6e65712673706163653b7a2673706163653b7c2673706163653b785c6e65712673706163653b792c2673706163653b795c6e65712673706163653b7a2c2673706163653b5c6c6566742673706163653b5c7b2673706163653b782c2673706163653b792c2673706163653b7a2673706163653b5c72696768742673706163653b5c7d2673706163653b5c696e2673706163653b5c6c6566742673706163653b5c7b2673706163653b412c2673706163653b432c2673706163653b472c2673706163653b542673706163653b5c72696768742673706163653b5c7d29 "Pr(A=x\to y, B=y \to z) * Pr(x\neq z | x\neq y, y\neq z, \left \{ x, y, z \right \} \in \left \{ A, C, G, T \right \})")](https://www.codecogs.com/eqnedit.php?latex=Pr(A=x\to&space;y,&space;B=y&space;\to&space;z)&space;*&space;Pr(x\neq&space;z&space;|&space;x\neq&space;y,&space;y\neq&space;z,&space;\left&space;\{&space;x,&space;y,&space;z&space;\right&space;\}&space;\in&space;\left&space;\{&space;A,&space;C,&space;G,&space;T&space;\right&space;\}))

## Computing the maximum posterior probability base quality.

Second, a consensus sequence is called for all reads with the same source molecule id base-by-base. For a given base position in
