GraftM - How to get fast community profiles from metagenomes Overview GraftM is a meta-omic tool that identifies and classifies marker genes in short read datasets (metagenomes and metatranscriptomes), as well as assembled contigs, whole genomes and protein sequences. GraftM outputs a taxonomic/functional summary table, a krona plot, as well as various other run statistics. Both unaligned and aligned "hit" sequences are provided. GraftM is designed for speed and accuracy. It is able to find marker genes in a 200Mb of assembled metagenome in &lt;20 sec, and compares favourably with similar tools in accuracy benchmarking. Using GraftM GraftM was developed as a tool for the rapid analysis of short read metagenomic data. It was developed as a tool that could search and classify marker gene reads, without using pairwise sequence comparisons such as BLAST. The result is a flexible, easy-to-use tool that can be run on any gene: all in a fraction of the time required by other methods. We provide some GraftM packages, such as 16S rRNA, mcrA and pmoA, but for detailed instructions on how to create your own GraftM package, please visit the manual page.
