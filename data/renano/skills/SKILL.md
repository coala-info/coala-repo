# renano

## Overview
RENANO is a specialized tool for the lossless compression of FASTQ data generated by Oxford Nanopore Technologies. It leverages reference-based algorithms to significantly reduce file sizes by encoding base call sequences against a known genome. This skill provides the necessary command-line patterns for compressing data with or without a reference, as well as instructions for decompression and performance tuning.

## Installation
The recommended way to install RENANO is via Bioconda:
```bash
conda install -c bioconda renano
```

## Compression Workflows

### 1. Reference-Based Compression (Highest Efficiency)
Use this mode when both the compressor and decompressor have access to the same reference genome file.
```bash
renano -r reference.fasta input.fastq output.renano
```

### 2. Reference-Based Compression (Independent Decompression)
Use this mode if the decompressor will not have the reference genome. RENANO will include a compacted version of the reference within the compressed file.
```bash
renano -s reference.fasta input.fastq output.renano
```

### 3. Reference-Free Compression
Use this mode for general nanopore FASTQ compression when no reference is available.
```bash
renano input.fastq output.renano
```

## Decompression
To restore the original FASTQ file, use the `-d` flag. If the file was compressed using the `-r` flag, you must provide the same reference file during decompression.

**Standard Decompression:**
```bash
renano -d input.renano output.fastq
```

**Decompression with Reference:**
```bash
renano -d -r reference.fasta input.renano output.fastq
```

## Optimization and Best Practices

### Performance Tuning
*   **Multithreading:** Use the `-t` option to specify the number of threads. The default is 8. For high-throughput environments, match this to your available CPU cores.
    ```bash
    renano -t 16 input.fastq output.renano
    ```
*   **Context Length (`-k`):** Adjust the base call sequence context length (default 7, max 13). Increasing this can improve compression ratios at the cost of higher memory usage.
*   **DNA Neighborhood (`-l`):** Adjust the length of the DNA neighborhood sequence (default 6).

### Handling Metagenomic Data
For metagenomic or contaminated samples where a single reference is not obvious, it is recommended to construct a representative reference sequence using tools like Kraken2 to identify prevalent organisms before running RENANO.

### File Pathing
If installed via Conda, invoke the tool as `renano`. If compiled from source, use the path to the executable (e.g., `./renano`).
