K-mers are either encoded (k&lt;=32) or hashed ( k&lt;=64, using ntHash v1 ) into uint64 , and serialized in binary file with extension .unik . TaxIds can be assigned when counting k-mers from genome sequences, and LCA (Lowest Common Ancestor) is computed during set opertions including computing union, intersecton, set difference, unique and repeated k-mers. Related projects: kmers provides bit-packed k-mers methods for this tool. unik provides k-mer serialization methods for this tool. sketches provides generators/iterators for k-mer sketches ( Minimizer , Scaled MinHash , Closed Syncmers ). taxdump provides querying manipulations from NCBI Taxonomy taxdump files.
