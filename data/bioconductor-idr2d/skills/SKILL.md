# bioconductor-idr2d

## Overview

The `idr2d` package extends the Irreproducible Discovery Rate (IDR) framework to two-dimensional genomic data. While the original IDR method was designed for one-dimensional peaks (e.g., ChIP-seq), `idr2d` allows for the identification of reproducible interactions between genomic loci (anchors) generated by methods like Hi-C, ChIA-PET, and HiChIP. It also maintains support for 1D genomic data.

## Core Workflow

### 1. Data Preparation
Input data frames for both replicates must follow a specific structure where the **seventh column** contains the value used for ranking (e.g., FDR, p-value, or signal intensity).

```r
library(idr2d)

# Example for 1D data (ChIP-seq)
rep1_df <- idr2d:::chipseq$rep1_df
rep2_df <- idr2d:::chipseq$rep2_df

# Example for 2D data (ChIA-PET)
rep1_df_2d <- idr2d:::chiapet$rep1_df
rep2_df_2d <- idr2d:::chiapet$rep2_df
```

### 2. Estimating IDR
The package provides two primary functions depending on the data dimensionality. A critical parameter is `value_transformation`, which ensures that higher values represent higher significance.

*   **For 1D Data (Peaks):** Use `estimate_idr1d()`.
*   **For 2D Data (Interactions):** Use `estimate_idr2d()`.

```r
# For FDR/p-values, use "log_additive_inverse" (-log(x))
idr_results <- estimate_idr2d(rep1_df_2d, rep2_df_2d,
                              value_transformation = "log_additive_inverse")

# Access the results for the first replicate
rep1_idr_df <- idr_results$rep1_df
```

**Value Transformation Options:**
- `"log_additive_inverse"`: Recommended for p-values and FDRs (calculates `-log(x)`).
- `"log"`: Use when values are already scaled such that higher is better, but require log transformation.
- `"none"`: Use if values are already appropriately ranked.

### 3. Interpreting and Visualizing Results
Use the `summary` function to get a high-level overview of reproducible interactions at standard thresholds (IDR < 0.05 and IDR < 0.01).

```r
# Summary of reproducibility
summary(idr_results)

# Visualization functions
draw_idr_distribution_histogram(rep1_idr_df)
draw_rank_idr_scatterplot(rep1_idr_df)
draw_value_idr_scatterplot(rep1_idr_df)
```

## Key Functions Reference

- `estimate_idr1d(df1, df2, ...)`: Estimates IDR for 1D genomic features (peaks).
- `estimate_idr2d(df1, df2, ...)`: Estimates IDR for 2D genomic features (interactions).
- `draw_idr_distribution_histogram(idr_df)`: Plots the distribution of IDR values.
- `draw_rank_idr_scatterplot(idr_df)`: Plots the relationship between feature rank and IDR.
- `draw_value_idr_scatterplot(idr_df)`: Plots the relationship between the original value and IDR.
