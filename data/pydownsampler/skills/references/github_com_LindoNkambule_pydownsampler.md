Installation pydownsampler is available through pypi . To install, type: pip install pydownsampler Usage $ pydownsampler ( &lt; file &gt; ) [-d &lt; dcov &gt; ] $ pydownsampler ( &lt; file &gt; ) [-d &lt; dcov &gt; ] [-o &lt; output &gt; ] $ pydownsampler ( &lt; file &gt; ) [-c] $ pydownsampler [-h] | [--help] $ pydownsampler [-v] [--version] Arguments and Options Argument/ Option Description -h, --help Show help message to screen. -v, --version Show version. &lt;file&gt; Input BAM/CRAM/SAM file -d, --downcoverage The coverage you want to downsample to (Required argument) -o, --output Output filename prefix (Optional) -c, --coverage Check file coverage Examples # option 1 (default): $ pydownsampler input.bam -d 10 In the example above, the file ' input.bam ' will be downsampled to 10X coverage.
