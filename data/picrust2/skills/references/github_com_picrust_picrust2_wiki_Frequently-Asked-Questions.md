# Troubleshooting errors / submitting an issue

Please **read the FAQs below here before submitting an issue**.

If you are posting an issue or a question to the Google Group related to an error message that you have got, please:

* Check whether you get the same issue with the tutorial data. You can see instructions on downloading and running this [here](https://github.com/picrust/picrust2/wiki/Installation#test-installation-with-real-data).

If you still have an issue, please submit an issue including all of the information in the bug report template.

If you are able to run the tutorial data but are struggling with your own data, please:

* First try running all of the PICRUSt2 workflow steps separately, as described [here](https://github.com/picrust/picrust2/wiki/Workflow) (this will help to narrow down exactly where you are running into issues). Run all of these with the `--verbose` flag
* If this still doesn't work, please provide the full output in your terminal window saved as a text file as well as the files that you are trying to run (or some reduced files where you still see these issues if you do not want to send your full data)
