Search MetaboLights public studies [beta release]. Download, upload and validate MetaboLights submitted studies Create assay file templates. API Features Read and update ISA files with minimum pyhton package dependency. Read MetaboLights study metadata files Calculate hash value of each ISA metadata file or study ISA metadata folder Read ISA table files (s_ .txt, a_ .txt, m_*.txt) with Pagination support . Multi-column filters and sort options on ISA table files. Apply actions on ISA table files to manipulate them. Json serializable models with pydantic library. Installation The following command installs metabolights-utils from the Python Package Index. You will need Python 3.8+ and pip3 on your operating system. (If python3 is not installed) Download and install Python (If pip3 is not installed) Install pip3 Install metabolights-utils library cd &lt; directory to create a virtual environment &gt; # install metabolights-utils on new a virtual environment named mtbls-venv (you may change it) python3 -m venv mtbls-venv source mtbls-venv/bin/activate pip install --upgrade pip pip3 install -U metabolights-utils # test mtbls command mtbls --version CLI ( mtbls ) Usage After installation of metabolights-utils, mtbls command will be enabled to use or search MetaboLights studies.
