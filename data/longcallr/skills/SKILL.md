# longcallr

## Overview
longcallR is a specialized bioinformatics tool designed to handle the unique characteristics of long-read RNA-seq data from Oxford Nanopore (ONT) and PacBio platforms. It provides a unified workflow for calling SNPs, phasing reads into maternal and paternal haplotypes, and conducting downstream allele-specific analyses. It is particularly effective for resolving transcript-level variation that short-read sequencing cannot easily capture.

## Core Workflows

### 1. SNP Calling and Phasing
The primary command identifies variants and tags reads with haplotype information.

```bash
# Standard command structure
longcallR -b <input.bam> -f <ref.fa> -o <output_prefix> -t <threads> -p <preset>
```

**Platform Presets (`-p`):**
- `ont-cdna`: Nanopore cDNA reads.
- `ont-drna`: Nanopore direct RNA reads (disables strand bias filtering).
- `hifi-isoseq`: PacBio Iso-Seq reads.
- `hifi-masseq`: PacBio MAS-Seq reads (disables strand bias filtering).

**Expert Tip:** If read strandedness is uncertain or non-standard, use `--strand-bias false` to prevent the tool from over-filtering valid variants.

### 2. Direct Haplotagging
If you already possess a phased VCF file, you can skip the SNP calling/phasing step and tag your BAM directly.

```bash
longcallR -b <input.bam> -f <ref.fa> -o <output> --input-vcf <phased.vcf> --direct-haplotag
```

### 3. Allele-Specific Analysis
Once you have a phased BAM (generated by the steps above), use the subcommands for functional genomics.

**Allele-Specific Expression (ASE):**
Quantifies gene expression levels for each haplotype.
```bash
longcallR ase -a <annotation.gtf> -b <phased.bam> -o <output_prefix> -t <threads>
```

**Allele-Specific Junction (ASJ):**
Identifies differential splicing between haplotypes.
```bash
longcallR asj -a <annotation.gtf> -b <phased.bam> -f <ref.fa> -o <output_prefix> -t <threads>
```

## Alignment Best Practices
For optimal results with longcallR, align your raw reads using `minimap2` with these specific parameters:

| Data Type | Minimap2 Command |
| :--- | :--- |
| **PacBio MAS-Seq** | `minimap2 -ax splice:hq -uf ref.fa query.fa` |
| **PacBio Iso-Seq** | `minimap2 -ax splice:hq ref.fa query.fa` |
| **Nanopore dRNA** | `minimap2 -ax splice -uf -k14 ref.fa reads.fa` |
| **Nanopore cDNA** | `minimap2 -ax splice ref.fa reads.fa` |

*Note: Use `-uf` for single-stranded protocols to force alignment to a single transcript strand.*

## Visualization
To visualize allele-specific junctions in IGV, convert the TSV output to BED format using the provided utility script:
```bash
asj_to_bed.py <output.asj.tsv> [p_value_threshold] > output.asj.bed
```
