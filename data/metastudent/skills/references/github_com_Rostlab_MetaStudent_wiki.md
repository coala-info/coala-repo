Installation instructions from rostlab debian repository sudo apt-get install python-software-properties sudo apt-add-repository "deb http://rostlab.org/debian/ stable main contrib non-free" sudo apt-get update # ignore GPG error sudo apt-get install rostlab-debian-keyring # without verification sudo apt-get update sudo apt-get install metastudent Version unknown With this installation we are not getting a MetaStudent version number ( in any other case version number can be found in version.py on this path: /usr/lib/python2.7/dist-packages/metastudentPkg). Regarding data in MetaStudent, there are two data sets metastudent-data and metastuent-data-2. In the installation described above, metastudent-data-3 is required, which doesn't exist. 2. Installation from ubuntu package manager sudo apt-get install metastudent Version 1.0.11 With this installation we are getting version MetaStudent 1.0.11 (version number can be found in version.py on this path: /usr/lib/python2.7/dist-packages/metastudentPkg). Also regarding MetaStudent data, there are two data sets metastudent-data and metastuent-data-2, which are downloaded along with the installation. 3. Installation from GitHub Version 2.0.1 This version is under development and currently unstable. Prebuild JAR's are required to install this tool from source, which are not currently available.
