# nwkit mcmctree

[Jump to bottom](#wiki-pages-box)

Kenji Fukushima edited this page Mar 8, 2026
·
[5 revisions](/kfuku52/nwkit/wiki/nwkit-mcmctree/_history)

## Overview

**nwkit mcmctree** is a tool to prepare input tree files for **mcmctree** in the [**PAML**](http://web.mit.edu/6.891/www/lab/paml.html) package. This program allows introducing fossil constraints in a command-line interface for the divergence time estimation subsequently done by **mcmctree**. This function has been tested with **PAML 4.9** and supports:

* Lower bound
* Upper bound
* Pair bound
* Point estimate

Parameters such as tail probability can be specified. See [PAML MANUAL](http://abacus.gene.ucl.ac.uk/software/pamlDOC.pdf) for details.

For `--timetree point` and `--timetree ci`, tip labels are parsed in a species-aware manner. Accepted formats are `GENUS_SPECIES` and `GENUS_SPECIES_SUFFIX`. Fields after the species epithet are ignored when querying taxonomy and timetree.org. Repeated labels for the same species are allowed only when all such tips form a monophyletic clade in the input tree.

## 1. Introducing pre-selected individual constraints

### Command

You can use pipes to add multiple constraints. **--add\_header** should be specified only in the final step.

```
nwkit mcmctree \
--left_species Populus_trichocarpa \
--right_species Arabidopsis_thaliana \
--lower_bound 100 \
--upper_bound 120 \
--infile input.nwk \
| nwkit mcmctree \
--left_species Aquilegia_coerulea \
--right_species Arabidopsis_thaliana \
--lower_bound 130 \
--upper_bound 130 \
--add_header \
> output.nwk
```

### input.nwk

This is a rooted tree in the Newick format. The input tree can contain support values and branch lengths, which will be removed from the output to be compatible with **mcmctree**.

```
(Aquilegia_coerulea:0.23566,((Arabidopsis_thaliana:0.366634,Populus_trichocarpa:0.167905)100:0.0447481,Vitis_vinifera:0.139353)99:0.0680463)1:0;
```

### output.nwk

```
4 1
(Aquilegia_coerulea,((Arabidopsis_thaliana,Populus_trichocarpa)'B(100, 120, 0.025, 0.025)',Vitis_vinifera))'@130';
```

## 2. Automatically obtaining the divergence time estimates from timetree.org

[timetree.org](http://www.timetree.org/) offers the meta-analysis of divergence time estimation in the literature, and `nwkit mcmctree` can utilize their estimates as divergence time constraints. Internet connection is necessary for this function. Tip labels are expected to follow `GENUS_SPECIES` or `GENUS_SPECIES_SUFFIX`; suffixes are ignored for species-name parsing. If the same species appears multiple times, those tips must form a monophyletic clade. In the example below, many nodes receive the estimates from timetree.org, while a less-studied clade of *Gastrodia* is entirely unconstrained due to the lack of past study.

### Command

```
nwkit mcmctree --infile input.nwk --timetree ci --add_header --outfile output.nwk
```

### input.nwk

```
(Amborella_trichopoda:0.177631,(((((Ananas_comosus:0.161854,(Brachypodium_distachyon:0.0986098,Oryza_sativa:0.089494)100:0.240569)100:0.0583933,Musa_acuminata:0.19703)100:0.0327033,((Apostasia_shenzhenica:0.166953,(Dendrobium_catenatum:0.0882342,(Gastrodia_elata:0.0193713,((Gastrodia_fontinalis:0.00276105,Gastrodia_kuroshimensis:0.00177332)100:0.00158479,(Gastrodia_nipponica:0.000538266,Gastrodia_takeshimensis:0.000738236)100:0.00239979)100:0.0206825)100:0.13039)100:0.0684334)100:0.118601,Asparagus_officinalis:0.231787)100:0.0223242)100:0.0532105,Spirodela_polyrhiza:0.354852)100:0.0680591,(Aquilegia_coerulea:0.23566,((Arabidopsis_thaliana:0.366634,Populus_trichocarpa:0.167905)100:0.0447481,Vitis_vinifera:0.139353)100:0.0680463)100:0.0577504)1:0.177631)1:0;
```

### output.nwk

```
(Amborella_trichopoda,(((((Ananas_comosus,(Brachypodium_distachyon,Oryza_sativa)'B(41.51749, 62, 0.025, 0.025)')'B(94.0766, 117, 0.025, 0.025)',Musa_acuminata)'B(103, 119.5885, 0.025, 0.025)',((Apostasia_shenzhenica,(Dendrobium_catenatum,(Gastrodia_elata,((Gastrodia_fontinalis,Gastrodia_kuroshimensis),(Gastrodia_nipponica,Gastrodia_takeshimensis)))))'B(0, 0, 0.025, 0.025)',Asparagus_officinalis)'B(92.4749, 118.53631, 0.025, 0.025)')'B(121.73325, 136.2783, 0.025, 0.025)',Spirodela_polyrhiza)'B(133, 139.12, 0.025, 0.025)',(Aquilegia_coerulea,((Arabidopsis_thaliana,Populus_trichocarpa)'B(109.8224, 124.44, 0.025, 0.025)',Vitis_vinifera)'B(109.8224, 124.44, 0.025, 0.025)')'B(151.56, 170.05738, 0.025, 0.025)')'B(151.56, 170.05738, 0.025, 0.025)')'B(179.9364, 204.95, 0.025, 0.025)';
```
