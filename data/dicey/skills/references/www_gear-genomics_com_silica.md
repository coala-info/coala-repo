#### Application Description

Silica runs a virtual insilico PCR with a set of primers on a selected genome. Multiple primer pairs are allowed, off target products will be shown if present. All primer combinations are tested, provided sufficiend binding to the DNA and reasonable amplicon size.

#### Accepted Input

Primers can be provided in fasta format or uploaded as a fasta file (\*.fa).

#### Sample Data

The "Load Example" button loads an example primer set. Hit "Launch Analysis" to try the functionality.
