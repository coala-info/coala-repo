BiSCoT is a software that uses informations produced by a pre-existing assembly based on optical maps as input and improves the contiguity and the quality of the generated assembly. Its primary goal is to resolve 'negative gaps' created by the Bionano scaffolder. These gaps are created when the Bionano scaffolder detects an overlap between two contig maps. To resolve negative gaps, BiSCoT first determines if contigs share labels. If so ( Figure 1 case 1 ), BiSCoT merges contigs at the shared label. If they don't share labels ( Figure 1 case 2 ), BiSCoT launches a BLAT alignment to verify if contigs share a same genomic region. If a reliable alignment can be found, BiSCoT merges the contigs at the start of the alignment. Sometimes, contigs can be inserted into other contigs. This type of cases is not handled by the Bionano scaffolder which simply discards the inserted contig map. BiSCoT attempts to solve this problem by merging contigs if they share labels ( Figure 2 ).
