## `cleavage-profile`

### Let's calculate the cleavage profile (`cleavage-profile`) for the intervals 2kb to the left and right (`-l 2000`, `-r 2000`) of TSS sites across chromosome 22, using only fragments with a MAPQ of 30 or greater (`-q 30`) and length between 50bp and 350bp (`-min 50`, `-max 350`). We will specify the chromosome sizes (`--chrom_sizes`) from the hg19 genome. We will also specify the output (`-o`) as tss.hg19.2kb.22.bw, using one worker (`-w 1`) and verbose mode (`-v`).

```
finaletoolkit cleavage-profile \
    BH01.hg19.mdups.22.frag.gz \
    tss.hg19.22.bed \
    -o tss.hg19.2kb.22.bw \
    -c hg19.chrom.sizes \
    -l 2000 \
    -r 2000 \
    -q 30 \
    -min 50 \
    -max 350 \
    -w 1 \
    -v
```
