This module implements a large number of different color operations such as color space conversions, Delta E, and density to spectral. Requirements numpy NetworkX 2.0+ Python 2.7 or Python 3.5+ Installation The easiest way to install colormath is via pip/easy_install: $ pip install colormath The development dependencies are installed as follows: $ pip install 'colormath[development]' Documentation For documentation, see the project webpage at: http://python-colormath.readthedocs.org/ There are also a lot of useful examples under the examples directory within this directory.
