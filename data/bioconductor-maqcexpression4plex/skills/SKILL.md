# bioconductor-maqcexpression4plex

## Overview
The `maqcExpression4plex` package is a Bioconductor Experiment Data package. It contains sample expression data generated by NimbleGen Systems using a 4-plex array format. This dataset is part of the MicroArray Quality Control (MAQC) project and is primarily used for demonstrating or testing R packages designed for NimbleGen array processing, such as `oligo` or `pdInfoBuilder`.

## Loading the Data
The package does not contain a single pre-loaded `ExpressionSet` object in the global environment. Instead, it provides raw data files (XYS files) and metadata located within the package installation directory.

To use this data, you typically locate the system files and read them using the `oligo` package.

```r
# Load the package
library(maqcExpression4plex)

# Locate the directory containing the sample data
dataDir <- system.file("extdata", package="maqcExpression4plex")

# List the available XYS files (raw intensity files)
xysFiles <- list.files(dataDir, pattern="\\.xys$", full.names=TRUE)

# View the phenotype data file path
phenoFile <- file.path(dataDir, "phenoData.txt")
```

## Typical Workflow: Reading with oligo
The most common use case for this package is to provide input for the `oligo` package to demonstrate preprocessing.

```r
library(oligo)

# Get the list of XYS files
xysFiles <- list.files(system.file("extdata", package="maqcExpression4plex"),
                       pattern="\\.xys$", full.names=TRUE)

# Read the intensity data
# Note: This requires the appropriate annotation package (pd.hg18.60m.x3) to be installed
rawEvt <- read.xysfiles(xysFiles)

# Inspect the object
show(rawEvt)
```

## Data Structure
- **Platform**: NimbleGen 4-plex array.
- **Organism**: Human (HG18).
- **Content**: 24,000 genes.
- **Probe Design**: 3 60-mer probes per gene.
- **Metadata**: Phenotype information is stored in `extdata/phenoData.txt`.

## Tips
- This package is an "Experiment Data" package, meaning its primary purpose is to provide data for other software packages to process.
- Ensure you have the `oligo` package installed to perform any meaningful analysis on these files.
- If you need to inspect the sample metadata manually:
  ```r
  phenoData <- read.table(file.path(system.file("extdata", package="maqcExpression4plex"), "phenoData.txt"), header=TRUE, sep="\t")
  head(phenoData)
  ```
