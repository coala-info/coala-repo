# frc

## Overview

The `frc` skill provides a specialized workflow for managing object detection tasks using a Keras implementation of Faster R-CNN. It excels at simplifying the training process by using a unified text format for annotations, removing the need for complex XML or JSON parsing. This tool is best suited for legacy environments requiring Keras 2.0.3 and TensorFlow, offering straightforward scripts for training on the KITTI dataset, calculating Mean Average Precision (mAP), and running inference on single images or entire directories.

## Core CLI Patterns

### Training a Model
To begin training on a dataset formatted for this implementation:
`python train_frcnn_kitti.py`

### Running Inference (Prediction)
The prediction script is versatile and can handle individual files or batches:

- **Single Image**: `python test_frcnn_kitti.py -p /path/to/single_image.png`
- **Directory of Images**: `python test_frcnn_kitti.py -p /path/to/image_folder/`

### Performance Evaluation
To calculate the mAP (mean Average Precision) of your trained model:
`python measure_map.py`

## Data Preparation

This tool uses a specific, simplified annotation format. Every line in your detection label file should follow this structure:
`filepath,x1,y1,x2,y2,class_name`

**Example:**
`/data/images/0001.png,712.40,143.00,810.73,307.92,Pedestrian`
`/data/images/0002.png,599.41,156.40,629.75,189.25,Truck`

- **Note**: You do not need a separate class map file; the training program automatically generates the class statistics from this file.

## Expert Tips and Best Practices

- **Environment Constraint**: This implementation is strictly compatible with **Keras 2.0.3**. Using newer versions of Keras will result in errors during the model building or loading phase.
- **Model Storage**: Weights are saved in standard `.h5` format, making them "out of the box" compatible with other Keras utilities, provided the version constraint is met.
- **Input Recognition**: When using the `-p` flag for prediction, the tool automatically detects whether the path provided is a file or a directory.
- **Feature Extraction**: For advanced workflows, use `extract_featuremap.py` to visualize or analyze the intermediate representations generated by the Region Proposal Network (RPN).
