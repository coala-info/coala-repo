Note

The relevant data used in this tutorial is accessible at: [![DOI](https://camo.githubusercontent.com/75764d327f9f7ba30b45feb7f0ce0c367b8fe65ab42803e38babfed1874aa430/68747470733a2f2f7a656e6f646f2e6f72672f62616467652f444f492f31302e353238312f7a656e6f646f2e31373033363333382e737667)](https://doi.org/10.5281/zenodo.17036338)

Warning

Make sure to download the relevant index file for each `.bam` (`.bam.bai` index), `.cram` (`.cram.crai` index), or `.gz` (`.tbi` index).

---

* `BH01.hg19.frag.bed.gz`: [DOWNLOAD](https://zenodo.org/records/14284133/files/BH01.hg19.frag.bed.gz?download=1)
* `BH01.hg19.frag.bed.gz.tbi`: [DOWNLOAD](https://zenodo.org/records/14284133/files/BH01.hg19.frag.bed.gz.tbi?download=1)

---

We begin by running the following command:

```
finaletoolkit frag-length-bins \
   BH01.hg19.frag.bed.gz \
   -min 50 \
   -max 300 \
   --bin-size 1 \
   -q 30 \
   --output_file BH01_fraglengths.tsv
```

Then plot the figure:

```
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Load and preprocess data
frag_len = pd.read_csv('BH01_fraglengths.tsv', sep='\t')

# Create KDE plot
plt.figure(figsize=(4, 4), dpi=300)
kde = sns.kdeplot(
    data=frag_len,
    x='min',
    weights='count',
    color='#D97D72',
    linewidth=2,
    bw_adjust=0.1
)

# Scale y-axis density and adjust axis limits
kde_lines = kde.get_lines()[0]
kde_lines.set_ydata(kde_lines.get_ydata() * 100)
plt.xlim(50, 300)
plt.ylim(-0.1, 1.5)

# Customize ticks and labels
plt.xticks(range(50, 301, 50), fontsize=12)
plt.yticks([i * 0.5 for i in range(6)], fontsize=12)
plt.xlabel('Fragment sizes (bp)', fontsize=12)
plt.ylabel('Density', fontsize=12)

# Style plot
ax = plt.gca()
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)

# Finalize and display plot
plt.tight_layout()
plt.show()
```
