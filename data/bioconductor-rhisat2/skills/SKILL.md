# bioconductor-rhisat2

## Overview

Use the Bioconductor R package **Rhisat2** for: The package contains wrapper functions to create a genome index and to perform the read alignment to the generated index.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("Rhisat2")
```
