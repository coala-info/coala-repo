# Bot Building

Libraries for building and working with bots.

* [arikawa - A library and framework for the Discord API.](/bot-building/arikawa-diamondburned-github/)
* [bot - Zero-dependencies Telegram Bot library with additional UI components.](/bot-building/bot-go-telegram-github/)
* [echotron - An elegant and concurrent library for Telegram Bots in Go.](/bot-building/echotron-niconex-github/)
* [go-joe - A general-purpose bot library inspired by Hubot but written in Go.](https://joe-bot.net?utm_campaign=awesomego&utm_medium=referral&utm_source=awesomego)
* [go-sarah - Framework to build a bot for desired chat services including LINE, Slack, Gitter, and more.](/bot-building/go-sarah-oklahomer-github/)
* [go-tg - Generated from official docs Go client library for accessing Telegram Bot API, with batteries for building complex bots included.](/bot-building/go-tg-mr-linch-github/)
* [go-twitch-irc - Library to write bots for twitch.tv chat](/bot-building/go-twitch-irc-gempir-github/)
* [micha - Go Library for Telegram bot api.](/bot-building/micha-onrik-github/)
* [slack-bot - Ready to use Slack Bot for lazy developers: Custom commands, Jenkins, Jira, Bitbucket, Github...](/bot-building/slack-bot-innogames-github/)
* [slacker - Easy to use framework to create Slack bots.](/bot-building/slacker-slack-io-github/)
* [telebot - Telegram bot framework is written in Go.](/bot-building/telebot-tucnak-github/)
* [telego - Telegram Bot API library for Golang with full one-to-one API implementation.](/bot-building/telego-mymmrac-github/)
* [telegram-bot-api - Simple and clean Telegram bot client.](/bot-building/telegram-bot-api-go-telegram-bot-api-github/)
* [TG - Telegram Bot Framework for Go.](/bot-building/tg-enetx-github/)
* [wayback - A bot for Telegram, Mastodon, Slack, and other messaging platforms archives webpages.](/bot-building/wayback-wabarc-github/)
