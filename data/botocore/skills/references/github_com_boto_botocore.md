The botocore package is the foundation for the AWS CLI as well as boto3 . Botocore is maintained and published by Amazon Web Services . Notices On 2026-04-29, support for Python 3.9 will end for Botocore. This follows the Python Software Foundation end of support for the runtime which occurred on 2025-10-31. Getting Started Assuming that you have Python and virtualenv installed, set up your environment and install the required dependencies like this or you can install the library using pip : $ git clone https://github.com/boto/botocore.git $ cd botocore $ python -m venv .venv ... $ source .venv/bin/activate $ python -m pip install -r requirements.txt $ python -m pip install -e . $ pip install botocore Using Botocore After installing botocore Next, set up credentials (in e.g. ~/.aws/credentials ): [default] aws_access_key_id = YOUR_KEY aws_secret_access_key = YOUR_SECRET Then, set up a default region (in e.g. ~/.aws/config ): [default] region =us-east-1 Other credentials configuration method can be found here Then, from a Python interpreter: &gt; &gt;&gt; import botocore . session &gt; &gt;&gt; session = botocore . session . get_session () &gt; &gt;&gt; client = session . create_client ( 'ec2' ) &gt; &gt;&gt; print ( client . describe_instances ()) Getting Help We use GitHub issues for tracking bugs and feature requests and have limited bandwidth to address them.
