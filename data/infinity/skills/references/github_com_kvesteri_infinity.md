Can be compared to any object. Why? Python has float('inf') and float('-inf') . However these simply represent floating point infinity values. I wanted to create a class which can be compared to any comparable object. Writing float('inf') is clumsy compared to just inf pow(1, float('inf')) returns 1 whereas it should be undefined . In infinity this operation returns TypeError . When would you use Infinity ?
