Recent version history 10 Jan 2026: Fixed major-allele identification for multiallelic variants where the major allele is last; this bugfix has been backported. Variant-split now works properly with a mix of known/provisional REF alleles. --condition[-list] 'multiallelic' modifier implemented. 5 Jan: --glm permutation tests now support binary phenotypes (Firth regression required). 3 Jan 2026: --glm adaptive permutation test termination condition is now based on the original number of allele tests instead of variant tests (relevant when multiallelic variants are present). Fixed double-precision logistic/Firth regression's handling of RFMix2 local-covariate input; this bugfix has been backported. 31 Dec 2025: --glm permutation tests now work properly when the primary test isn't additive. Adaptive test now works properly when no variant needs the maximum allowed number of permutations, or chrX/chrY and other chromosome(s) are both present. 30 Dec: --glm permutation tests should now work properly when samples/covariates are different on chrX/chrY. --glm 'qt-residualize' no longer requires 'permute-qt-residuals' when used with a permutation test. Fixed --glm qt-residualize's handling of chrX/chrY when other chromosomes were also present; this bugfix (along with the 5 Dec --bcf tweak) have been backported. 29 Dec: Reorganized permutation-test output formatting under --perm-format . 27 Dec: Permutation tests implemented for --glm linear regression. 5 Dec: --bcf now tolerates irregular INFO Flag encoding (seen from e.g. minimac 4.1). 28 Nov: --import-overlong-var-ids implemented. --import-max-alleles is now enforced before the variant ID length check. .gen-import chromosome-filtering bug has been fixed. Bugfix and --import-max-alleles repositioning have been backported. 23 Nov: Fixed " --new-id-max-allele-len missing" + --pmerge[-list] / variant-split bug; this bugfix has been backported. 22 Nov: Fixed --import-dosage format=1 segfault that could occur when 'NA' appears before the first non-integer dosage; this bugfix has been backported. Implemented workaround for gcc 11+ compiler behavior which previously resulted in incorrect -flto builds; -flto is now turned on for non-Windows development builds. 18 Nov: --make-grm-sparse implemented. 15 Nov: Fixed handling of a few invalid --extract-if-info/--exclude-if-info and --keep-if/--remove-if expressions, and improved invalid-expression error messages. 11 Nov: --extract-if-info/--exclude-if-info and --keep-if/--remove-if now support a wider range of expressions (e.g. &amp;&amp;, ||, !, parentheses, backslash-escaping of special characters). --make-king-table now has an 'ibs' column set corresponding to Hamming distance. Fixed bug in 20 Sep - 26 Oct development builds where --exclude-palindromic-snps was incorrectly executed when "--{extract,exclude,extract-intersect} bed{0,1}", --select-sid-representatives, or --mendel was present in the same run. 26 Oct: Reworked logging for variant filters applied during import/merge/.pvar-load time. Fixed handling of --bcf + --vcf-require-gt. Fixed --haps error message when all variants have been filtered out. Bugfixes have been backported. 19 Oct: Fixed --glm bug that could result in spurious CORR_TOO_HIGH errors when only a single covariate, with variance &gt; 1, was present; this has been backported. 20 Sep: --exclude-palindromic-snps implemented. 15 Sep: --flip-subset implemented. Fixed ARM bug (i.e. only affected the macOS M1 precompiled binary) that could cause --extract/--exclude/--flip to segfault; this has been backported. 1 Sep: --zero-cluster is enabled again (it was inadvertently disabled in the 30 Aug build). 30 Aug: --flip implemented. 25 Aug: --fill-missing-with-ref implemented. 24 Aug: --zero-cluster implemented. --chr-set no-x/no-y/no-xy modifiers should now work as documented (i.e. PLINK 2 now consistently errors out when encountering the forbidden chromosome).
