For example: export BIOR_LITE_HOME=/usr/local/biotools/bior_scripts/4.3.1/bior_pipeline-4.3.1 Choose location of bgzip export BGZIP_DIR=/data5/bsi/bictools/src/htslib/1.4/bin Choose location of this biorAnnotateLite script export BIOR_ANNOTATE_LITE=/data5/bsi/BIOR/BiorAnnotateLite/1.0.0 Now, add all these to the path export PATH=$BIOR_LITE_HOME/bin:$BGZIP_DIR:$BIOR_ANNOTATE_LITE:$PATH Run a test which shows how to run the script against a particular VCF and Catalog-Drill file $BIOR_ANNOTATE_LITE/Test/test.sh OR, run it directly: bior_annotate_lite my.in.vcf catalogDrill.txt my.out.vcf.bgz my.log Now, try your own! #----------------------------------------------------------- #
