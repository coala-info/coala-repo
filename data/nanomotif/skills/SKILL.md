# nanomotif

## Overview
Nanomotif is a specialized bioinformatics tool designed to leverage the methylation information captured by Oxford Nanopore Technologies (ONT) sequencing. It allows researchers to move beyond simple base calling to understand the epigenetic landscape of microbial communities. By analyzing methylation pileups, the tool can identify the specific DNA sequences (motifs) targeted by Methyltransferases (MTases) and Restriction-Modification (RM) systems. This epigenetic "fingerprint" is then used to improve metagenomic binning, identify mis-binned contigs, and link mobile genetic elements to their hosts.

## Core Workflows

### 1. De Novo Motif Discovery
This is the primary entry point for most analyses. It identifies methylated motifs across an entire assembly.

```bash
nanomotif motif_discovery ASSEMBLY.fasta PILEUP.bed CONTIG_BIN.tsv -t 16 --out output_dir
```
*   **ASSEMBLY.fasta**: The genomic sequences.
*   **PILEUP.bed**: Methylation pileup generated by `modkit`.
*   **CONTIG_BIN.tsv**: A tab-separated file mapping contigs to bins.
*   **Outputs**: `motifs.tsv` (all found motifs), `motif-scored.tsv` (statistical scoring), and `bin-motifs.tsv` (motifs aggregated at the bin level).

### 2. Metagenomic Bin Decontamination
Use this to identify contigs within a bin that do not share the bin's consensus methylation profile, suggesting they are contaminants.

```bash
nanomotif detect_contamination \
    --assembly ASSEMBLY.fasta \
    --pileup PILEUP.bed \
    --bin_motifs bin-motifs.tsv \
    --contig_bins CONTIG_BIN.tsv \
    --out contamination_results
```
*   **Tip**: Add `--write_bins` and `--assembly_file` to automatically generate new, cleaned fasta files for your bins.

### 3. Associating Unbinned Contigs
Assign plasmids or other unbinned contigs to existing bins by training classifiers on the methylation patterns of established MAGs.

```bash
nanomotif include_contigs \
    --assembly ASSEMBLY.fasta \
    --pileup PILEUP.bed \
    --bin_motifs bin-motifs.tsv \
    --contig_bins CONTIG_BIN.tsv \
    --run_detect_contamination \
    --out inclusion_results
```

### 4. Linking Motifs to MTases
To associate identified motifs with specific genes in the genome:

1.  **Install dependencies**: `nanomotif MTase-linker install`
2.  **Run linkage**:
```bash
nanomotif MTase-linker run \
    --assembly ASSEMBLY.fasta \
    --contig_bin CONTIG_BIN.tsv \
    --bin_motifs bin-motifs.tsv \
    -d ML_dependencies \
    -o mtase_linker_out
```

## Expert Tips and Best Practices
*   **Input Preparation**: Ensure your methylation pileup is generated using `modkit pileup`. Nanomotif relies on the specific format and methylation calls provided by the Nanopore-native toolchain.
*   **Thread Management**: Motif discovery is computationally intensive. Always specify the `-t` flag to match your available CPU cores for significant speedups.
*   **Binning Context**: While `include_contigs` is powerful for finding hosts for plasmids, remember that methylation patterns can sometimes be shared across different species (MAGs). Treat these associations as putative and validate with other methods (e.g., coverage or GC content) if possible.
*   **Installation Check**: If you encounter issues, run `nanomotif check_installation` to execute a small test suite on a sample dataset to verify your environment.
