## Major Updates

* **Parallelized batch mode** for the `extract` module ([#25](https://github.com/endixk/ezaai/issues/25), [#32](https://github.com/endixk/ezaai/issues/32))
  + Allows running `extract` on a directory of FASTA files (`*.fa`, `*.fna`, `*.fasta`).
  + See the [tutorial](https://endixk.github.io/ezaai/tutorial#extract) or run `extract -h` for details.
* **[New website](https://endixk.github.io/ezaai)**
  + Now mobile-friendly and served over HTTPS (finally! 😅).

## Bug Fixes

* File deletion failures now raise a warning instead of stopping the process ([#21](https://github.com/endixk/ezaai/issues/21)).
* Added directory write-permission checks for outputs and temporary files ([#22](https://github.com/endixk/ezaai/issues/22)).
