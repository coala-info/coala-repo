# bioconductor-duplexdiscoverer

## Overview

The `DuplexDiscovereR` package provides a comprehensive workflow for analyzing chimeric reads generated by RNA proximity ligation experiments. It transforms raw split-read alignments into biologically meaningful RNA duplex groups (DGs). Key capabilities include chimeric read classification (filtering out splice junction contamination), overlap-based clustering, statistical ranking via p-values, and prediction of hybridization energies.

## Core Workflow

### 1. Automated Analysis
The fastest way to process data is using the wrapper function `runDuplexDiscoverer`.

```r
library(DuplexDiscovereR)

# Required: data (STAR/BEDPE/GInteractions), splice junctions (GRanges)
result <- runDuplexDiscoverer(
    data = RNADuplexesRawChimSTAR,
    junctions_gr = SampleSpliceJncGR,
    anno_gr = SampleGeneAnnoGR,      # Optional: for annotation
    df_counts = RNADuplexesGeneCounts, # Optional: for p-values
    fafile = "path/to/genome.fa",    # Optional: for RNAduplex energies
    lib_type = "SE",
    table_type = "STAR"
)

# Access results
dgs <- dd_get_duplex_groups(result)
reads <- dd_get_chimeric_reads(result)
```

### 2. Step-by-Step Custom Analysis
For finer control over filtering and clustering parameters:

**Preprocessing & Classification:**
```r
# Convert input to internal format
gi <- runDuplexDiscoPreproc(data = input_data, table_type = "STAR", return_gi = TRUE)

# Filter out known splice junctions and non-chimeric artifacts
gi <- getChimericJunctionTypes(gi)
gi <- getSpliceJunctionChimeras(gi, sj_gr = SampleSpliceJncGR)

# Keep only valid 2-arm duplexes
gi_filtered <- gi[gi$junction_type == "2arm" & gi$splicejnc == 0]
```

**Clustering & Collapsing:**
```r
# Cluster reads into Duplex Groups (DGs) based on spatial overlap
gi_clustered <- clusterDuplexGroups(gi_filtered)

# Collapse individual reads into consensus DGs
gi_dgs <- collapse_duplex_groups(gi_clustered)
```

### 3. Hybridization Energy Prediction
Requires `RNAduplex` (ViennaRNA) to be installed on the system path.

```r
# Predict base-pairing and energies for a GInteractions object
gi_with_hybrids <- getRNAHybrids(gi_dgs, fafile = "genome.fasta")
```

### 4. Comparing Samples
To find reproducible interactions across replicates:

```r
sample_list <- list(rep1 = gi_dgs1, rep2 = gi_dgs2)
comp_results <- compareMultipleInteractions(sample_list)

# Visualize overlaps with UpSetR
library(UpSetR)
upset(comp_results$dt_upset)
```

## Visualization and Export

**Gviz Integration:**
Use `DuplexTrack` to visualize interactions alongside gene models.

```r
library(Gviz)
dt <- DuplexTrack(gi = gi_dgs, gr_region = plot_range, 
                  annotation.column1 = "gene_name.A", 
                  annotation.column2 = "gene_name.B")
plotTracks(list(GeneRegionTrack(anno), dt))
```

**Exporting Data:**
```r
# To Data Frame/Table
df <- makeDfFromGi(gi_dgs)
write.table(df, "duplexes.tab")

# To SAM (for IGV visualization)
writeGiToSAMfile(gi_dgs, file_out = "output.sam", genome = "hg38")
```

## Tips for Success
- **Input Formats:** Supports STAR `Chimeric.out.junction`, `.bedpe`, and `GInteractions` objects.
- **Memory Management:** For very large datasets, use `collapseIdenticalReads()` before clustering to reduce the graph size and improve performance.
- **Custom Weights:** You can manually compute overlaps using `computeGISelfOverlaps()` and pass custom weights to `clusterDuplexGroups()` to refine how reads are grouped.
