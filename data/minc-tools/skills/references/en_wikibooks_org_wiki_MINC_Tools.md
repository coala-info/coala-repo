# Documentation for MINC Tools

[[edit](/w/index.php?title=MINC/Tools&veaction=edit&section=1 "Edit section: Documentation for MINC Tools") | [edit source](/w/index.php?title=MINC/Tools&action=edit&section=1 "Edit section's source code: Documentation for MINC Tools")]

There are a large number of add-on tools that have been written for MINC, this is an attempt to draw all of the documentation for these into one cohesive place. There is a lot of it out there! The tools are organised into rough sections, feel free to add your own tools in here and add a section if required. Note that the source code for all of these tools is available here: <http://packages.bic.mni.mcgill.ca/tgz> Some or at least most of these packages will also have binary versions available on <http://packages.bic.mni.mcgill.ca/> as well.

## Registration (alignment)

[[edit](/w/index.php?title=MINC/Tools&veaction=edit&section=2 "Edit section: Registration (alignment)") | [edit source](/w/index.php?title=MINC/Tools&action=edit&section=2 "Edit section's source code: Registration (alignment)")]

Registration is a process whereby two (more) images are aligned to each other. There are a number of methods by which you can do this but most of them involve some form of gradient descent and a matching algorithm.

[mni autoreg](/wiki/MINC/Tools/mni_autoreg "MINC/Tools/mni autoreg") - A registration framework written by Louis Collins

## Intensity Correction

[[edit](/w/index.php?title=MINC/Tools&veaction=edit&section=3 "Edit section: Intensity Correction") | [edit source](/w/index.php?title=MINC/Tools&action=edit&section=3 "Edit section's source code: Intensity Correction")]

Intensity correction in MRI typically refers to the correction of the B1 field effect, however there is also intensity correction that can occur between different scans of the same subject or indeed different subjects. There can also be intensity differences between slices of a single acquisition.

[iNormalize](/w/index.php?title=MINC/Tools/iNormalize&action=edit&redlink=1 "MINC/Tools/iNormalize (does not exist)") A global intensity correction package by Alex Zijdenbos

[N3](/wiki/MINC/Tools/N3 "MINC/Tools/N3") - An intensity nonuniformity correction package written by John G. Sled

## Geometric Distortion Correction

[[edit](/w/index.php?title=MINC/Tools&veaction=edit&section=4 "Edit section: Geometric Distortion Correction") | [edit source](/w/index.php?title=MINC/Tools&action=edit&section=4 "Edit section's source code: Geometric Distortion Correction")]

MRI scanners introduce geometric distortions caused by several factors.

[DistortionCorrection](/w/index.php?title=MINC/Tools/DistortionCorrection&action=edit&redlink=1 "MINC/Tools/DistortionCorrection (does not exist)") - a tool for correcting distortions in anatomical images caused by the nonlinearities in imaging gradients.

## C Libraries

[[edit](/w/index.php?title=MINC/Tools&veaction=edit&section=5 "Edit section: C Libraries") | [edit source](/w/index.php?title=MINC/Tools&action=edit&section=5 "Edit section's source code: C Libraries")]

These libraries have been written in order to extend the functionality of MINC, typically they add certain features to MINC such as a link to an image display library.

[bicpl](/w/index.php?title=MINC/Tools/bicpl&action=edit&redlink=1 "MINC/Tools/bicpl (does not exist)") - the BIC "programmers" library

[EBTKS](/w/index.php?title=MINC/Tools/EBTKS&action=edit&redlink=1 "MINC/Tools/EBTKS (does not exist)") - Everything but the kitchen sink

[bicInventor](/w/index.php?title=MINC/Tools/bicInventor&action=edit&redlink=1 "MINC/Tools/bicInventor (does not exist)") - a glue library that allows the use of Inventor/Coin in a MINC program

## Virtual Machine

[[edit](/w/index.php?title=MINC/Tools&veaction=edit&section=6 "Edit section: Virtual Machine") | [edit source](/w/index.php?title=MINC/Tools&action=edit&section=6 "Edit section's source code: Virtual Machine")]

A virtual machine, containing the MINC tools preinstalled and ready to use is available at <https://github.com/cobralab/MINC-VM>

## Interfaces to other Languages

[[edit](/w/index.php?title=MINC/Tools&veaction=edit&section=7 "Edit section: Interfaces to other Languages") | [edit source](/w/index.php?title=MINC/Tools&action=edit&section=7 "Edit section's source code: Interfaces to other Languages")]

[emma](/wiki/MINC/Tools/emma "MINC/Tools/emma") - an old, ageing and un-maintained interface to MATLAB (written by an Academy award winner!)
