EpiScanpy is the epigenomic extension of the very popular scRNA-seq analysis tool Scanpy ( Genome Biology , 2018) [Wolf18]. For more information on scanpy, read the following documentation . EpiScanpy paper is now accessible on Nature Communications . For more information on how to use the package, more information are available on the website .
