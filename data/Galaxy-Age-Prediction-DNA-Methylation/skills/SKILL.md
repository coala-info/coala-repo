# Age Prediction DNA Methylation

## Overview

This workflow implements a machine learning pipeline designed for age prediction using DNA methylation data. It facilitates the creation of epigenetic clocks by processing training and testing datasets through a structured scikit-learn framework within Galaxy.

The process begins with the **Pipeline Builder** to define the model architecture, followed by a **Hyperparameter Search** to optimize the estimator's performance. The workflow utilizes **Ensemble methods** to enhance prediction accuracy, allowing for robust statistical modeling of complex methylation patterns across the genome.

Final outputs include trained model objects, prediction results, and a suite of performance visualizations. Users can evaluate the model through actual vs. predicted curves and residual plots generated by [Plotly](https://plotly.com/), providing clear insights into the model's reliability and error distribution.

Developed in alignment with [Galaxy Training Network (GTN)](https://training.galaxyproject.org/) standards, this workflow is categorized under Statistics and is intended for researchers performing age-related epigenetic studies and bioinformatics-driven clinical research.

## How to run (Planemo)

### Local Galaxy (default engine)

```bash
planemo run age-prediction-dnam.ga job.yml --download_outputs --output_directory . --output_json output.json
```

Optional: `--galaxy_root /path/to/galaxy` speeds up spinning up a local Galaxy.

### Docker Galaxy

Append e.g. `--engine docker_galaxy --ignore_dependency_problems` (requires Docker).

### External Galaxy (usegalaxy.org / EU / …)

```bash
planemo run age-prediction-dnam.ga job.yml --engine external_galaxy --galaxy_url SERVER_URL --galaxy_user_key YOUR_API_KEY
```

Create a reusable profile: `planemo profile_create myprofile --galaxy_url ... --galaxy_user_key ... --engine external_galaxy` then `planemo run age-prediction-dnam.ga job.yml --profile myprofile`.

### Job file

- Generate a template: `planemo workflow_job_init age-prediction-dnam.ga -o job.yml`
- Lint: `planemo workflow_lint age-prediction-dnam.ga`

See **[Running Galaxy workflows](https://planemo.readthedocs.io/en/master/running.html)** in the Planemo documentation.
