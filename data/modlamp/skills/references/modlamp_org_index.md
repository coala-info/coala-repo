This is a Python package (for Python 3 since version 4.0.0) that was designed for working with peptides , proteins or any sequence of natural amino acids . It incorporates several modules, like sequence generation (module sequences ), sequence library analysis (module analysis ) or descriptor calculation (module descriptors ). For basic instructions on how to use the package, check the README or the example script .
