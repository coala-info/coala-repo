## Prerequisite:

## `gap-bed`

### Running DELFI usually requires a GAP track. This is a hard-coded function that supports the following genomes: `hg19`, `b37`, `human_g1k_v37`, `hg38`, `GRCh38`

```
finaletoolkit gap-bed human_g1k_v37 hg19.gap.bed
```

Our output should look something like this:

```
1	120697156	120747156	clone
1	120936695	121086695	contig
1	121485434	121535434	clone
1	121535434	124535434	centromere
1	124535434	142535434	heterochromatin
```

---

## `delfi`

### Let's calculate DELFI (`delfi`) over 100kb bins across chromosome 22, without merging the bins (`--no-merge-bins`), using only fragments with a MAPQ of 30 or greater (`-q 30`). We will use the ENCFF001TDO.bed blacklist file (`--blacklist-file`), with the hg19 gap track (`-g`). We will also output (`-o`) the result to BH01.22.delfi.bed, using one worker (`-w 1`) and verbose mode (`-v`).

```
finaletoolkit delfi \
    BH01.hg19.mdups.22.frag.gz \
    hg19.chrom.sizes \
    hg19.2bit \
    100kb.22.bins \
    --blacklist-file ENCFF001TDO.bed \
    -g hg19.gap.bed \
    --no-merge-bins \
    -q 30 \
    -o BH01.22.delfi.bed \
    -w 1 \
    -v
```

