# bioconductor-damefinder

## Overview

DAMEfinder is an R package designed to identify Differential Allele-Specific Methylation (DAME) regions. Allele-specific methylation (ASM) occurs when one parental allele is methylated differently than the other (e.g., genomic imprinting or X-inactivation). DAMEfinder identifies regions where this ASM status is gained or lost between experimental groups.

The package operates in two primary modes:
1.  **SNP-based (Exhaustive):** Requires a VCF of heterozygous SNPs. It links methylation status to specific alleles.
2.  **Tuple-based (Fast):** Uses `methtuple` output. It identifies ASM based on the co-occurrence of methylation states in pairs of CpG sites (tuples) without needing SNP information.

## SNP-based Workflow

Use this mode to detect sequence-dependent ASM linked to specific heterozygous SNPs.

### 1. Extract Methylation by Allele
Extract reads from BAM files and split them by SNP alleles.
```r
library(DAMEfinder)
library(BSgenome.Hsapiens.UCSC.hg19)

# Setup files
bam_files <- c("sample1.bam", "sample2.bam")
vcf_files <- c("sample1.vcf", "sample2.vcf")
sample_names <- c("NORM1", "CRC1")
reference_file <- DNAStringSet(BSgenome.Hsapiens.UCSC.hg19[[19]])

# Extract calls
snp.list <- extract_bams(bam_files, vcf_files, sample_names, reference_file, coverage = 2)
```

### 2. Summarize and Calculate ASM
Convert the list into a `RangedSummarizedExperiment` containing ASM scores (`der.ASM` or `z.ASM`).
```r
derASM <- calc_derivedasm(snp.list)
# Access scores
asm_scores <- assay(derASM, "der.ASM")
```

## Tuple-based Workflow

Use this mode for a faster, SNP-independent exploration of ASM.

### 1. Read Methtuple Data
Import `.tsv.gz` files generated by the external `methtuple` tool.
```r
tuple_files <- c("sample1.methtuple.tsv.gz", "sample2.methtuple.tsv.gz")
tuple_list <- read_tuples(tuple_files, sample_names)
```

### 2. Calculate ASM Score
Generate the ASM scores based on the log odds ratio of methylated/unmethylated pairs.
```r
ASM_mat <- calc_asm(tuple_list)
```

## Finding DAMEs

Once ASM scores are calculated (for either mode), use `find_dames` to detect differential regions.

```r
# Define design matrix
grp <- factor(c("CRC", "CRC", "NORM", "NORM"))
mod <- model.matrix(~grp)

# Find DAMEs using Simes method (default)
dames <- find_dames(derASM, mod)

# Find DAMEs using Empirical/Bumphunting method (for larger cohorts)
dames_emp <- find_dames(derASM, mod, pvalAssign = "empirical")
```

## Visualization

### Genomic Tracks
Visualize ASM scores and methylation levels across a specific region.
```r
# Add group info to colData for plotting
colData(derASM)$group <- grp
colData(derASM)$samples <- colnames(derASM)

# Define region
dame_region <- GRanges("19", IRanges(387966, 387983))

# Plot tracks
dame_track(dame = dame_region, derASM = derASM, plotSNP = TRUE)

# Plot group means
dame_track_mean(dame = dame_region, derASM = derASM)
```

### Methyl-circle Plots
Visualize the raw methylation state of reads overlapping a SNP or CpG.
```r
snp_pos <- GRanges("19", IRanges(267039, width = 1))

methyl_circle_plot(snp = snp_pos, 
                   vcfFile = "sample.vcf", 
                   bamFile = "sample.bam", 
                   refFile = reference_file)
```

### MDS Plot
Check for global sample clustering based on ASM profiles.
```r
methyl_MDS_plot(derASM, group = grp)
```

## Tips for Success
- **Filtering:** Always filter the `SummarizedExperiment` to remove sites with high `NA` counts across samples before running `find_dames` to improve power and reduce computation.
- **Smoothing:** `find_dames` includes a `smooth` parameter (default TRUE) which helps reduce noise in the t-statistics.
- **Memory:** For large genomes, process data per chromosome or use the `cores` parameter in `calc_derivedasm` for parallel processing.
