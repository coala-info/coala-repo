```
This directory includes external binaries used by the genome browser.

The contents should be copied into the cgi-bin directory, after the Genome
Browser CGIs have been compiled.

The following software is provided only for installation convenience, see these
webpages for more details and the respective licenses.

RNAplot, from the Vienna RNA package:
https://www.tbi.univie.ac.at/RNA/

dot_static from the Graphviz package:
http://www.graphviz.org/

MysqlDB for Python:
https://pypi.python.org/pypi/MySQL-python/1.2.3
```
