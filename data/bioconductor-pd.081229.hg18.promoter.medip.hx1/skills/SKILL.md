# bioconductor-pd.081229.hg18.promoter.medip.hx1

## Overview
The `pd.081229.hg18.promoter.medip.hx1` package is a specialized annotation dataset built using `pdInfoBuilder`. It provides the necessary infrastructure to map probes to genomic locations and sequences for the NimbleGen hg18 promoter Methylated DNA Immunoprecipitation (MeDIP) array. It is designed to work seamlessly with the `oligo` package for low-level analysis of oligonucleotide arrays.

## Usage in R

### Loading the Package
To initialize the annotation environment:
```r
library(pd.081229.hg18.promoter.medip.hx1)
```

### Integration with oligo
This package is primarily used as a backend for `oligo` functions. When reading raw data (CEL files), `oligo` will automatically look for this package if the array type matches.

```r
library(oligo)
# Assuming CEL files are in the current directory
raw_data <- read.celfiles(filenames = list.celfiles())
# The annotation slot will be set to "pd.081229.hg18.promoter.medip.hx1"
```

### Data Objects
- `pmSequence`: Contains sequences for Perfect Match probes.
- `bgSequence`: Contains sequences for background probes (if applicable).
- `mmSequence`: Contains sequences for Mismatch probes (if applicable).

## Workflow Tips
- **Memory Management**: These annotation objects can be large. Ensure your R session has sufficient memory when loading `pmSequence`.
- **Platform Specificity**: This package is specific to the `hg18` (NCBI36) build of the human genome. Ensure your downstream analysis (like peak calling or visualization) uses the correct genome version to avoid coordinate mismatches.
- **Feature IDs**: Use the `fid` column to link sequences back to the expression or intensity matrices generated by `oligo`.
