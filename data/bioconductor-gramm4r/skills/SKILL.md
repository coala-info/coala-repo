# bioconductor-gramm4r

## Overview

The `gramm4R` package implements the Generalized coRrelation analysis for Metabolome and Microbiome (GRaMM) strategy. It is designed to discover relationships between metabolomic and microbial datasets by accounting for data characteristics like sparsity and compositionality, adjusting for covariates, and capturing both linear and nonlinear associations using Maximum Information Coefficient (MIC) and Centered Log-Ratio (CLR) transformations.

## Installation

```R
if (!requireNamespace("BiocManager", quietly = TRUE)) install.packages("BiocManager")
BiocManager::install("gramm4R")
library(gramm4R)
```

## Core Workflow

### 1. Data Preprocessing
Use `preGramm` to handle missing values, normalize data, and apply CLR transformations. This is a critical first step for compositional microbiome data.

```R
data("metabolites")
data("microbes")
processed_data <- preGramm(metabolites, microbes)
# Access processed matrices via processed_data$x and processed_data$y
```

### 2. Association Testing
Use `naiveGramm` to calculate correlations. It automatically selects between linear regression and nonlinear (MIC) methods based on the data structure and can adjust for confounding variables.

```R
data("covariates")
results <- naiveGramm(metabolites, microbes, covariates)

# Interpretation:
# results$r    : Correlation coefficients
# results$p    : P-values
# results$type : Method used ("linear" or "nonlinear")
```

### 3. Visualization and Model Fitting
Use `nlfitGramm` to fit various regression models (linear, logarithmic, exponential, power, or S-curve) to the data pairs.

```R
# Fits models and saves the best-fitting plots (highest R-square) to a PDF
nlfitGramm(metabolites, microbes)
```

### 4. Integrated Analysis
The `Gramm` function executes the entire pipeline: preprocessing, association testing with covariates, and automated plotting of the top 10 correlation pairs.

```R
Gramm(metabolites, microbes, covariates)
# Output: Generates "R value top 10 pairs.pdf" in the working directory
```

## Supported Regression Models
When using `nlfitGramm`, the package evaluates the following models:
- `line2P`: y = a*x + b
- `line3P`: y = a*x^2 + b*x + c
- `log2P`: y = a*ln(x) + b
- `exp2P` / `exp3P`: Exponential fits
- `power2P` / `power3P`: Power law fits
- `S`: Sigmoidal fit (y = a / (1 + exp((x-b)/c)))

## Tips for Success
- **Data Format**: Ensure metabolites and microbes data frames have matching sample IDs (columns).
- **Covariates**: Always include relevant metadata (age, BMI, diet) in the `covariates` argument to reduce false positives in correlation analysis.
- **Output**: Check your working directory (`getwd()`) for PDF reports generated by `Gramm` and `nlfitGramm`.
