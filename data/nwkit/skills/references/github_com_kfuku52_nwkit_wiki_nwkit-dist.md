## Overview

`nwkit dist` calculates topological distances between two trees that have the same set of tip names.

## Examples

### Command

```
nwkit dist --infile input1.nwk --infile2 input2.nwk
```

### stdout

* `rf_dist`: Robinson-Foulds distance ([Robinson & Foulds, 1981, Math Biosci 53: 131-147](https://doi.org/10.1016/0025-5564%2881%2990043-2))
* `max_rf_dist`: The maximum Robinson-Foulds distance within the tree

```
rf_dist	max_rf_dist
4	54
```
