## Overview

`nwkit rescale` multiplies branch lengths by a given factor. This is useful for converting time units (e.g., from millions of years to years), normalizing tree height, or adjusting branch lengths for visualization.

* `--factor` specifies the multiplication factor (required)
* `--target` specifies which branches to rescale: `all`, `root`, `leaf`, or `intnode`

## Examples

### Rescale all branches

#### Command

```
nwkit rescale --infile input.nwk --target all --factor 0.5 --outfile output.nwk
```

#### input.nwk

```
((A:2,B:4):6,(C:8,D:10):12);
```

#### output.nwk

```
((A:1,B:2)1:3,(C:4,D:5)1:6)1:0;
```

All branch lengths are multiplied by 0.5.

### Rescale only leaf branches

#### Command

```
nwkit rescale --infile input.nwk --target leaf --factor 0.1 --outfile output.nwk
```

#### input.nwk

```
((A:2,B:4):6,(C:8,D:10):12);
```

#### output.nwk

```
((A:0.2,B:0.4)1:6,(C:0.8,D:1)1:12)1:0;
```

Only leaf branch lengths are multiplied by 0.1. Internal branch lengths remain unchanged.
