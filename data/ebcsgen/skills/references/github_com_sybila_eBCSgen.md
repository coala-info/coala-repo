The tool is deployed online as a part of BioDivine Galaxy toolset. For more information about the tool, see Wiki . To get started with the usage of Galaxy Platform and eBCSgen, we recommend this tutorial . Installation Conda Install eBCSgen from Bioconda with: # install eBCSgen in a new virtual environment to avoid dependency clashes conda create --name eBCSgen python=3.9 conda activate eBCSgen conda install --channel bioconda --channel conda-forge eBCSgen Docker You can also use prebuild docker image . Galaxy eBCSgen is also available as a series of Galaxy tools available in the main toolshed . Developer Documentation Setup Create your development environment using the provided script via conda to install all required dependencies.
