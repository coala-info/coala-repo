ASA³P is a fully featured bacterial assembly and analysis pipeline ( paper ). Features Can prepare the ASA³P project directory in a non-interactive way Can use project metadata from a JSON file Command line parameters (like --genus Escherichia ) will override JSON configuration(s) Usage start-asap -i READS_DIR -r REFERENCE -o OUTDIR [-p METADATA_JSON | -g GENUS -s SPECIES] See the documentation page.
