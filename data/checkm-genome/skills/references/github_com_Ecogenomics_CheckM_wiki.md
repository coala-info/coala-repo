# Genome Quality Commands

* [tree](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#tree) - place bins in the reference genome tree
* [tree\_qa](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#tree_qa) - assess phylogenetic markers found in each bin
* [lineage\_set](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#lineage_set) - infer lineage-specific marker sets for each bin
* [taxon\_list](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#taxon_list) - list available taxonomic-specific marker sets
* [taxon\_set](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#taxon_set) - infer taxonomic-specific marker set
* [analyze](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#analyze) - identify marker genes in bins
* [qa](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#qa) - assess bins for contamination and completeness

# Reported Statistics

* [qa](https://github.com/Ecogenomics/CheckM/wiki/Reported-Statistics#qa) - description of statistics reported by [qa](https://github.com/Ecogenomics/CheckM/wiki/Genome-Quality-Commands#qa) command

# Plots

* [gc\_plot](https://github.com/Ecogenomics/CheckM/wiki/Plots#gc_plot) - create GC histogram and delta-GC plot
* [coding\_plot](https://github.com/Ecogenomics/CheckM/wiki/Plots#coding_plot) - create coding density (CD) histogram and delta-CD plot
* [tetra\_plot](https://github.com/Ecogenomics/CheckM/wiki/Plots#tetra_plot) - create tetranucleotide distance (TD) histogram and delta-TD plot
* [dist\_plot](https://github.com/Ecogenomics/CheckM/wiki/Plots#dist_plot) - create image with GC, CD, and TD distribution plots together
* [nx\_plot](https://github.com/Ecogenomics/CheckM/wiki/Plots#nx_plot) - create Nx-plots
* [len\_hist](https://github.com/Ecogenomics/CheckM/wiki/Plots#len_hist) - sequence length histogram
* [marker\_plot](https://github.com/Ecogenomics/CheckM/wiki/Plots#marker_plot) - plot position of marker genes on sequences
* [gc\_bias\_plot](https://github.com/Ecogenomics/CheckM/wiki/Plots#gc_bias_plot) - plot bin coverage as a function of GC

# Bin Exploration and Modification

* [unique](https://github.com/Ecogenomics/CheckM/wiki/Bin-Exploration-and-Modification#unique) - ensure no sequences are assigned to multiple bins
* [merge](https://github.com/Ecogenomics/CheckM/wiki/Bin-Exploration-and-Modification#merge) - identify bins with complementary sets of marker genes
* [outliers](https://github.com/Ecogenomics/CheckM/wiki/Bin-Exploration-and-Modification#outliers) - identify outliers in bins relative to reference distributions
* [modify](https://github.com/Ecogenomics/CheckM/wiki/Bin-Exploration-and-Modification#modify) - modify sequences in a bin

# Utility Commands

* [unbinned](https://github.com/Ecogenomics/CheckM/wiki/Utility-Commands#unbinned) - identify unbinned sequences
* [coverage](https://github.com/Ecogenomics/CheckM/wiki/Utility-Commands#coverage) - calculate coverage of sequences
* [tetra](https://github.com/Ecogenomics/CheckM/wiki/Utility-Commands#tetra) - calculate tetranucleotide signature of sequences
* [profile](https://github.com/Ecogenomics/CheckM/wiki/Utility-Commands#profile) - calculate percentage of reads mapped to each bin
* [join\_tables](https://github.com/Ecogenomics/CheckM/wiki/Utility-Commands#join_tables) - join tab-separated value tables containing bin information
* [ssu\_finder](https://github.com/Ecogenomics/CheckM/wiki/Utility-Commands#ssu_finder) - identify SSU (16S/18S) rRNAs in sequences
