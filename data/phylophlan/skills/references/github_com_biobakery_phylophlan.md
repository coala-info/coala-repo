PhyloPhlAn is an accurate, rapid, and easy-to-use method for large-scale microbial genome characterization and phylogenetic analysis at multiple levels of resolution. PhyloPhlAn can assign both genomes and metagenome-assembled genomes (MAGs) to species-level genome bins (SGBs). PhyloPhlAn can reconstruct strain-level phylogenies using clade-specific maximally informative phylogenetic markers, and can also scale to very-large phylogenies comprising &gt;17,000 microbial species. Installation Bioconda You can install PhyloPhlAn using conda as follows: conda install -c bioconda phylophlan Repository You can clone the PhyloPhlAn repository from GitHub: git clone https://github.com/biobakery/phylophlan cd phylophlan python setup.py install Then remember to check that the Dependencies and Tools are in stalled and available in your system.
