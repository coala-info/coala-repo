These sequences can be used as input for a QC tool such as cutadapt. The sequences can be output in plain text format with a newline character as a separator between the sequences. Alternatively the sequences can be output in FASTA format. Documentation For documentation and manuals visit our github.io page . About ExtractAdaptersFastqc is part of BIOPET tool suite that is developed at LUMC by the SASC team . Each tool in the BIOPET tool suite is meant to offer a standalone function that can be used to perform a dedicate data analysis task or added as part of a pipeline, for example the SASC team's biowdl pipelines . All tools in the BIOPET tool suite are Free/Libre and Open Source Software.
