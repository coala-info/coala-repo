## Obtaining KEGG data

**1. Obtaining the tree structure from KEGG database.**

```
cd FunUniFrac/fununifrac/reproducibility
mkdir data/test_reproducibles
python generate_ko_hierarchy.py --outdir data/test_reproducibles
```

The resulting file is an edge list saved as `kegg\_ko\_edge\_df\_ko00001.txt' under data/test\_reproducibles.

**2. Obtaining pairwise distances between KOs.**

**2.1 Obtain raw KO data and sketch the KOs.**

We partitioned the raw data by grouping all genes (amino acid sequences) belonging to each particular KO together, thus obtaining 25,413 .faa files each containing the amino acid sequences belonging to the KO.

Location on Lab server: `/data/shared_data/cleaned_KEGG_FTP_2022_NOV/PartitionByKO/aa_per_KO`.

For each of these KOs, we create a sketch using `sourmash sketch`. First compile a .csv file that includes the path of all the .faa files mentioned above. Use scale 10 and kmer size 5.

```
bash reproducible-scripts/sketch_all_KO.sh
```

Output saved as `./data/test_reproducibles/KO_AA_sketch_scaled_10_k_5.sig.zip`.

**2.2 Compute pairwise distances among KOs**

Next we compute the pairwise distances among these 25,413 KO sketches using `sourmash compare`.

```
bash reproducible-scripts/build_pairwise_KO_distance_matrix.sh
```

Output saved as `./data/test_reproducibles/KO_scale_10_k_5_pw_dist.npy`
