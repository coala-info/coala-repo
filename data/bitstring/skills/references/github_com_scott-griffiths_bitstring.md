It has been actively maintained since 2006. The library has has now been downloaded over 100 million times! ✨ ✨ Note To see what been added, improved or fixed, and also to see what's coming in the next version, see the release notes . Overview Efficiently store and manipulate binary data in idiomatic Python. Create bitstrings from hex, octal, binary, files, formatted strings, bytes, integers and floats of different endiannesses. Powerful binary packing and unpacking functions. Bit-level slicing, joining, searching, replacing and more. Create and manipulate arrays of fixed-length bitstrings. Read from and interpret bitstrings as streams of binary data. Rich API - chances are that whatever you want to do there's a simple and elegant way of doing it. Open source software, released under the MIT licence. Documentation Extensive documentation for the bitstring library is available. Examples Installation $ pip install bitstring Creation &gt;&gt;&gt; from bitstring import Bits, BitArray, BitStream, pack &gt;&gt;&gt; a = BitArray( bin = ' 00101 ' ) &gt;&gt;&gt; b = Bits(a_file_object) &gt;&gt;&gt; c = BitArray( ' 0xff, 0b101, 0o65, uint6=22 ' ) &gt;&gt;&gt; d = pack( ' intle16, hex=a, 0b1 ' , 100 , a = ' 0x34f ' ) &gt;&gt;&gt; e = pack( ' &lt;16h ' , * range ( 16 )) Different interpretations, slicing and concatenation &gt;&gt;&gt; a = BitArray( ' 0x3348 ' ) &gt;&gt;&gt; a.hex, a.bin, a.uint, a.float, a.bytes ('3348', '0011001101001000', 13128, 0.2275390625, b'3H') &gt;&gt;&gt; a[ 10 : 3 : - 1 ].bin '0101100' &gt;&gt;&gt; ' 0b100 ' + 3 * a BitArray('0x866906690669, 0b000') Reading data sequentially &gt;&gt;&gt; b = BitStream( ' 0x160120f ' ) &gt;&gt;&gt; b.read( 12 ).hex '160' &gt;&gt;&gt; b.pos = 0 &gt;&gt;&gt; b.read( ' uint12 ' ) 352 &gt;&gt;&gt; b.readlist( ' uint12, bin3 ' ) [288, '111'] Searching, inserting and deleting &gt;&gt;&gt; c = BitArray( ' 0b00010010010010001111 ' ) # c.hex == '0x1248f' &gt;&gt;&gt; c.find( ' 0x48 ' ) (8,) &gt;&gt;&gt; c.replace( ' 0b001 ' , ' 0xabc ' ) &gt;&gt;&gt; c.insert( ' 0b0000 ' , pos = 3 ) &gt;&gt;&gt; del c[ 12 : 16 ] Arrays of fixed-length formats &gt;&gt;&gt; from bitstring import Array &gt;&gt;&gt; a = Array( ' uint7 ' , [ 9 , 100 , 3 , 1 ]) &gt;&gt;&gt; a.data BitArray('0x1390181') &gt;&gt;&gt; a[:: 2 ] *= 5 &gt;&gt;&gt; a Array('uint7', [45, 100, 15, 1]) Copyright (c) 2006 - 2025 Scott Griffiths About A Python module to help you manage your bits bitstring.readthedocs.io/en/stable/index.html Topics python binary-data bitarray bit-manipulation bitstring Resources Readme License MIT license Uh oh!
