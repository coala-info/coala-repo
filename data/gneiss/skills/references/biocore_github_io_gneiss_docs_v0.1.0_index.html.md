Introduction: Balances are an alternative technique to investigate relationships between proportions. Rather than focusing on proportions, log ratios or balances are investigated in analyses. This technique has many beneficial theoretical properties, while avoiding issues associated with compositionality in differential abundance, correlations and regression. More details on balances can be found here. Installation: To install this package, it is recommended to use conda.
