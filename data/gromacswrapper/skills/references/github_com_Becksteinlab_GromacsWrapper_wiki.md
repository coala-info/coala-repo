This allows for fairly seamless integration of the gromacs tools into python scripts. This is generally superior to shell scripts because of python’s better error handling and superior data structures. It also allows for modularization and code re-use. In addition, commands, warnings and errors are logged to a file so that there exists a complete history of what has been done.
