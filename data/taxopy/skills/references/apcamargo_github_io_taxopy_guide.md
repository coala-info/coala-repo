This can be achieved by downloading the set of files that store this data, known as taxdump , from an online source (NCBI by default) or by providing your own taxdump files. The TaxDb object stores the names, ranks, and parent-child relationships of all taxa, each represented by their respective TaxIds. For instance, TaxId 2 corresponds to the " Bacteria " taxon, which has the rank of superkingdom.
