# Continuous Integration

Tools for help with continuous integration.

* [abstruse - Abstruse is a distributed CI platform.](/continuous-integration/abstruse-bleenco-github/)
* [Bencher - A suite of continuous benchmarking tools designed to catch performance regressions in CI.](https://bencher.dev/?utm_campaign=awesomego&utm_medium=referral&utm_source=awesomego)
* [CDS - Enterprise-Grade CI/CD and DevOps Automation Open Source Platform.](/continuous-integration/cds-ovh-github/)
* [dot - A minimal, local first continuous integration system that uses Docker to run jobs concurrently in stages.](/continuous-integration/dot-opnlabs-github/)
* [drone - Drone is a Continuous Integration platform built on Docker, written in Go.](/continuous-integration/drone-drone-github/)
* [go-beautiful-html-coverage - A GitHub Action to track code coverage in your pull requests, with a beautiful HTML preview, for free.](/continuous-integration/go-beautiful-html-coverage-gha-common-github/)
* [go-fuzz-action - Use Go 1.18's built-in fuzz testing in GitHub Actions.](/continuous-integration/go-fuzz-action-jidicula-github/)
* [go-semver-release - Automate the semantic versioning of Git repositories.](/continuous-integration/go-semver-release-s0ders-github/)
* [go-test-coverage - A GitHub Action which reports issues when test coverage is below set threshold.](/continuous-integration/actions-marketplace-github/)
* [gomason - Test, Build, Sign, and Publish your go binaries from a clean workspace.](/continuous-integration/gomason-nikogura-github/)
* [gotestfmt - go test output for humans.](/continuous-integration/gotestfmt-gotesttools-github/)
* [goveralls - Go integration for Coveralls.io continuous code coverage tracking system.](/continuous-integration/goveralls-mattn-github/)
* [muffet - Fast website link checker in Go, see alternatives.](/continuous-integration/muffet-raviqqe-github/)
* [overalls - Multi-Package go project coverprofile for tools like goveralls.](/continuous-integration/overalls-go-playground-github/)
* [roveralls - Recursive coverage testing tool.](/continuous-integration/roveralls-lawrencewoodman-github/)
* [woodpecker - Woodpecker is a community fork of the Drone CI system.](/continuous-integration/woodpecker-woodpecker-ci-github/)
