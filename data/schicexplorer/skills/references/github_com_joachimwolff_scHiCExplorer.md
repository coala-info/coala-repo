Robust and efficient single-cell Hi-C clustering with approximate k-nearest neighbor graphs, Bioinformatics , btab394, https://doi.org/10.1093/bioinformatics/btab394 Availability The easiest way to install scHiCExplorer is using BioConda $ conda install schicexplorer -c bioconda -c conda-forge Install by cloning this repository You can install any one of the HiCExplorer branches on command line (linux/mac) by cloning this git repository: $ git clone https://github.com/joachimwolff/scHiCExplorer.git $ cd scHiCExplorer $ python setup.py install However, please take care all dependencies are installed, see the requirements.txt file.
