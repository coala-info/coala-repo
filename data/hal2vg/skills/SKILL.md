# hal2vg

## Overview

hal2vg is a specialized utility for bridging the gap between hierarchical alignments and sequence graphs. It transforms HAL files into formats like PackedGraph, which are optimized for the vg ecosystem. This is essential for pangenomics workflows where multiple genomes are aligned and need to be represented as a single graph structure for variant calling, mapping, or visualization.

## Usage Patterns

### Standard Conversion
The most common use case is converting a HAL file to a PackedGraph (.pg) for use in the vg toolkit.

```bash
hal2vg input.hal --inMemory --chop 32 --progress > output.pg
```

### Key Parameters
- `--inMemory`: Required for all but the smallest, trivial inputs to ensure performance and successful completion.
- `--chop 32`: Highly recommended. The vg toolkit is optimized for graphs where nodes are at most 32 bases long.
- `--progress`: Provides real-time feedback on the conversion process.

### Output Formats
While PackedGraph is the default, hal2vg supports other sequence graph formats from the libbdsg library:
- **PackedGraph** (Default): Memory-efficient, recommended for most vg workflows.
- **HashGraph**: Useful for smaller graphs or specific manipulation tasks.
- **ODGI**: Optimized for large-scale pangenome graph operations.

## Expert Tips and Best Practices

### Post-processing
After conversion, it is a best practice to sort the graph IDs. This improves performance for downstream vg operations.
```bash
vg ids --sort output.pg > output.sorted.pg
```

### Compatibility
Ensure your downstream tools are up to date. The output generated by hal2vg is generally intended for use with vg version 1.24.0 or greater.

### Algorithm Logic
The tool works by adding HAL sequences as threads to a "Pinch Graph." It extracts exact pairwise alignment blocks (no gaps or substitutions) and pinches them. For substitutions (SNPs), it uses a column iterator to align bases across the tree and pinch exact matches. Finally, it traces HAL sequences through the pinch graph to generate the final nodes and edges.

## Related Utilities
The hal2vg package includes several auxiliary tools for graph and HAL manipulation:
- `clip-vg`: For clipping variation graphs.
- `count-vg-hap-cov`: Calculates non-reference sequence components supported by a minimum number of haplotypes.
- `halMergeChroms`: Merges chromosomes within a HAL file.
- `halRemoveDupes`: Removes duplicate sequences.
- `filter-paf-deletions`: Filters deletions from PAF files.
