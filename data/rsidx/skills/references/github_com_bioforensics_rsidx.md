This package enables rapid search of large VCF files by rsID in the same way that tabix enables rapid search by genomic coordinates. In fact, the rsidx search uses tabix under the hood to search by genomic coordinates retrieved from the rsidx index. This index is simply an sqlite3 database containing a mapping of rsID values to genomic coordinates. Installation Installation with bioconda is recommended. conda install -c bioconda rsidx If you prefer installation with pip try the following. pip install git+https://github.com/bioforensics/rsidx NOTE : If you install rsidx with pip, you will need to install the program tabix on your own and ensure it is in your $PATH when running rsidx .
