# Software requirements

[Jump to bottom](#wiki-pages-box)

Ryan Wick edited this page Sep 10, 2025
·
[22 revisions](/rrwick/Trycycler/wiki/Software-requirements/_history)

Trycycler runs on macOS and Linux. It requires [Python](https://www.python.org/) 3.9 or later. It also uses a few Python packages ([Edlib](https://github.com/Martinsos/edlib/tree/master/bindings/python), [NumPy](http://www.numpy.org/) and [SciPy](https://www.scipy.org/)) but these should be taken care of by the [installation process](/rrwick/Trycycler/wiki/Installation).

There are two general ways you can go about installing Trycycler:

* Via conda. This has the advantage of taking care of external software dependencies for you, so if you want to go down this road, you can skip ahead to [Installing with conda](/rrwick/Trycycler/wiki/Installation#installing-with-conda) (i.e. you don't need to keep reading this page).
* Via [pip](https://pypi.org/project/pip/) and [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git). If you go down this road, then you'll need to ensure external dependencies are installed before you continue, so read on.

## External tool dependencies

Trycycler also requires a few external tools to run. Install these by following the instructions on their websites or by using a package manager such as [Homebrew](https://brew.sh/) or [APT](https://en.wikipedia.org/wiki/APT_%28software%29).

### Miniasm

[Miniasm](https://github.com/lh3/miniasm) is a very fast long-read assembly tool. [Trycycler subsample](/rrwick/Trycycler/wiki/Generating-assemblies#subsampling-reads) uses it to perform a quick-and-dirty genome assembly for the purposes of estimating genome size.

To verify that you have miniasm installed, try running this command:

```
miniasm
```

If you see its usage instructions printed to the terminal, it's installed!

### Minimap2

[Minimap2](https://github.com/lh3/minimap2) is a sequence alignment tool. It excels at many things, including aligning long reads to a reference.

To verify that you have minimap2 installed, try running this command:

```
minimap2 --help
```

If you see its usage instructions printed to the terminal, it's installed!

### Mash

[Mash](https://mash.readthedocs.io/en/latest/) is a tool for quickly estimating the similarity between sequences using *k*-mers.

To verify that you have Mash installed, try running this command:

```
mash --help
```

If you see its usage instructions printed to the terminal, it's installed!

### R with phylogenetics packages

[Trycycler cluster](/rrwick/Trycycler/wiki/Clustering-contigs) runs an [R](https://www.r-project.org/) script to generate a phylogenetic tree. The [ape](http://ape-package.ird.fr/) and [phangorn](https://github.com/KlausVigo/phangorn) packages are required.

To verify that you have these installed, try opening up an R terminal:

```
R
```

And then try to load the ape and phangorn packages:

```
library(ape)
library(phangorn)
```

If these load without error, then you're good to go! Otherwise, install them in R:

```
install.packages("ape")
install.packages("phangorn")
```

### MUSCLE

[MUSCLE](https://www.drive5.com/muscle/) is a tool for multiple sequence alignment. Trycycler was developed with [MUSCLE v3.8](https://drive5.com/muscle/downloads_v3.htm), and while Trycycler *might* work with MUSCLE v5 (released in late 2021), using the older version is strongly recommended ([read more here](/rrwick/Trycycler/wiki/FAQ-and-miscellaneous-tips#a-note-on-muscle-versions)).

To verify that you have MUSCLE installed, try running this command:

```
muscle
```

If you see its usage instructions printed to the terminal, it's installed!

### Assemblers

Aside from miniasm (see above), Trycycler does not itself require any particular genome assemblers, but you will need to use assemblers to [generate Trycycler's input assemblies](/rrwick/Trycycler/wiki/Generating-assemblies).

Which you use is up to you, but some of my favourites are:

* [Canu](https://github.com/marbl/canu)
* [Flye](https://github.com/fenderglass/Flye)
* [Miniasm+Minipolish](https://github.com/rrwick/Minipolish)
* [Raven](https://github.com/lbcb-sci/raven)

You might also want to install these tools to support other aspects of the assembly process:

* [Filtlong](https://github.com/rrwick/Filtlong) for read QC
* [any2fasta](https://github.com/tseemann/any2fasta) to convert GFA files to FASTA format

If you're using a PacBio sequencer, then you could also install the [SMRT Link](https://www.pacb.com/support/software-downloads/) software so you can use their [Microbial Assembly pipeline](https://www.youtube.com/watch?v=So5mgWGEDcQ).

### Phylogenetic tree viewer

Trycycler does not need a tree viewer to run, but you'll probably want one to help you interpret [Trycycler cluster](/rrwick/Trycycler/wiki/Clustering-contigs) results.

Anything that can view a [Newick tree](http://marvin.cs.uidaho.edu/Teaching/CS515/newickFormat.html) will do, such as one of the following:

* [Taxonium](https://taxonium.org)
* [FigTree](http://tree.bio.ed.ac.uk/software/figtree)
* [Dendroscope](http://dendroscope.org)
* [Archaeopteryx](https://sites.google.com/site/cmzmasek/home/software/archaeopteryx)
