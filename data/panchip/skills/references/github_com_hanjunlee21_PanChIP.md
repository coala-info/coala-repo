Prerequisites Python 3, BEDTools Installation Installation by PyPI pip3 install panchip Installation by Bioconda conda install bioconda::panchip Input PanChIP Analysis: a directory with only 6-column BED files PanChIP Filter: a 6-column BED file We recommend most PanChIP users to utilize BED files with constant non-zero fifth column values (e.g., 1, 500, 1000). Usage panchip [options] Commands: init Initialization of the PanChIP library analysis Analysis of a list peak sets filter Filtering library for quality control Run panchip &lt; command &gt; -h for help on a specific command.
