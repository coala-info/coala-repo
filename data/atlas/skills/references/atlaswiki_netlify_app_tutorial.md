# Chapter 3 Tutorial

These tutorials are meant to guide you through typical workflows when working with ancient DNA or other low-depth data.

For this, we assume that you have atlas installed and running on your local machine or on your HPC cluster account. A guide for installing atlas can be found [here](/getting_started#getting_started).

To ease notation, we also assume that you have set `atlas` as a global alias with:

```
alias atlas="/path/to/your/atlas/build/atlas"
```
