# Installation[¶](#installation "Permalink to this headline")

xPore requires [Python3](https://www.python.org) to run.

## PyPI installation (recommended)[¶](#pypi-installation-recommended "Permalink to this headline")

```
pip install xpore
```

## Installation from our GitHub repository[¶](#installation-from-our-github-repository "Permalink to this headline")

```
git clone https://github.com/GoekeLab/xpore.git
cd xpore
python setup.py install
```
