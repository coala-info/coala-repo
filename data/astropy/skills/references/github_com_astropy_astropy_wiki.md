L. Lim edited this page Nov 10, 2025 &middot; 78 revisions Welcome to the Astropy Project wiki! The Astropy Project is a community effort to develop a single core package for Astronomy in Python and foster interoperability between Python astronomy packages. More information about Astropy is available from our homepage , and you can find the documentation at https://docs.astropy.org/ . Note: There are a lot of outdated pages in this wiki and not all of them are listed here. When in doubt, please reach out to the Astropy Project for clarification. Thanks!
