SpatialLeiden integrates with the scverse by leveraging anndata but can also be used independently. Installation spatialleiden is available on PyPI and bioconda . # PyPI pip install spatialleiden # or conda conda install bioconda::spatialleiden For detailed installation instructions please refer to the documentation .
