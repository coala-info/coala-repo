# (Hybrid) Metagenomics workflow

## Overview

This [Common Workflow Language (CWL)](https://workflowhub.eu/workflows/367) workflow provides a comprehensive pipeline for hybrid metagenomic assembly and binning, integrating both short-read (Illumina) and long-read (Oxford Nanopore or PacBio) sequencing data. The process begins with rigorous quality control and contamination filtering using tools such as `fastp`, `fastplong`, and `hostile`, followed by taxonomic classification of reads via `Kraken2`.

Assembly is performed using `SPAdes` or `Flye`, depending on the input data types, and refined through polishing steps with `Medaka` and `PyPolCA` to ensure high consensus accuracy. The core workflow concludes with an assembly quality assessment using `QUAST`. Detailed setup instructions and tool definitions are maintained in the [UNLOCK GitLab repository](https://gitlab.com/m-unlock/cwl/) and the [official documentation](https://docs.m-unlock.nl/docs/workflows/setup.html).

The workflow includes several optional modules for downstream analysis:
*   **Binning and Refinement:** Supports `Metabat2`, `MaxBin2`, and `SemiBin`, with refinement via `Binette` and taxonomic assignment through `GTDB-Tk`.
*   **Metabolic Modeling:** Integration with the [Genome-scale metabolic models workflow](https://workflowhub.eu/workflows/372) for GEM generation (CarveMe) and species interaction analysis (SMETANA).

Additional specialized workflows for bioinformatics and metagenomics can be found on the [UNLOCK WorkflowHub project page](https://workflowhub.eu/projects/16/workflows?view=default).
