With the Bioconda channels configured the latest release version of the package can be installed via the regular conda install command: conda install bioconductor-sc3-scripts Test installation There is a test script included: bioconductor-sc3-scripts-post-install-tests.sh This downloads a well-known test 10X dataset and executes all of the scripts described below. Commands The available wrapped SC3 functions are described below. Each script has usage instructions available via --help, consult function documentation in SC3 for further details. Wrappers are currently written for SC3 version 1.8. sc3_prepare(): prepare a SingleCellExperiment for SC3 See ?sc3_prepare for argument meanings. sc3-sc3-prepare.R -i &lt;SingleCellExperiment object as .rds&gt; -f &lt;gene_filter&gt; -p &lt;pct_dropout_min&gt; \ -q &lt;pct_dropout_max&gt; -d &lt;d_region_min&gt; -e &lt;d_region_max&gt; -n &lt;svm_num_cells&gt; -m &lt;svm_max&gt; \ -t &lt;n_cores&gt; -s &lt;rand_seed&gt; -k &lt;kmeans_nstart&gt; -a &lt;kmeans_iter_max&gt; \ -o &lt;path to file where .rds output file will be stored&gt; sc3_estimate_k(): Calculate k size for SC3 clustering sc3-sc3-estimate-k.R -i &lt;SC3 prepared SingleCellExperiment object as .rds&gt; \ -t &lt;path to file where estimated k will be stored&gt; \ -o &lt;path to file where .rds output file will be stored&gt; sc3_calc_dists(): Calculate distances sc3-sc3-calc-dists.R -i &lt;SingleCellExperiment object from sc3_estimate_k() as .rds&gt; \ -o &lt;path to file where .rds output file will be stored&gt; sc3_calc_transfs(): Calculate transformations of the distance matrices sc3-sc3-calc-transfs.R -i &lt;SingleCellExperiment object from sc3_calc_dists() as .rds&gt; \ -o &lt;path to file where .rds output file will be stored&gt; sc3_kmeans(): Cluster transform matrix using k-means sc3-sc3-kmeans.R -i &lt;SingleCellExperiment object from sc3_calc_transfs() as .rds&gt; \ -k &lt;k values to try, comma-separated&gt; -o &lt;path to file where .rds output file will be stored&gt; sc3_calc_consens(): Calculate consensus clustering sc3-calc-consens.R -i &lt;SingleCellExperiment object from sc3_kmeans() as .rds&gt; \ -t &lt;path to file where text output file will be stored&gt; \ -o &lt;path to file where .rds output file will be stored&gt; About Wrapper scripts for the SC3 R package Resources Readme License Apache-2.0 license Uh oh!
