# Gene prediction

* Essential genes prediction with [DEG](http://tubic.tju.edu.cn/deg/)

# Operons prediction

* With [DOOR](http://161.117.81.224/DOOR2/index.php)
* With [OperonDB](http://operondb.cbcb.umd.edu/cgi-bin/operondb/operons.cgi)

# genomic island identification

* With [IslandViewer](http://www.pathogenomics.sfu.ca/islandviewer2/query.php)
* With [MobilomeFINDER](http://db-mml.sjtu.edu.cn/MobilomeFINDER/)

# Signal peptides prediction

* With [SignalP](http://www.cbs.dtu.dk/services/SignalP/)

# Functional annotation

* Fast functional annotation of novel sequences with [eggNOG mapper v2](https://github.com/eggnogdb/eggnog-mapper/wiki/eggNOG-mapper-v2)
* Functional annotation with [DAVID](https://david.ncifcrf.gov/)

# RNA prediction

* Ribosomal RNA prediction with [rnammer](http://www.cbs.dtu.dk/services/RNAmmer/)
* Transfer RNA prediction with [tRNAscan](http://lowelab.ucsc.edu/tRNAscan-SE/)

# Repeated sequence analysis

* Repeated sequence analysis with [RepeatMasker](http://www.repeatmasker.org/)
* Find tandem repeats with [TRF](http://tandem.bu.edu/trf/trf.html)

# CRISPR prediction

* Annotation of CRISPR-Cas systems with [CRISPRone](https://omics.informatics.indiana.edu/CRISPRone/index.php)
* [CRISPRs web server](https://crispr.i2bc.paris-saclay.fr/)

# Secondary metabolite biosynthetic gene clusters prediction

* With [antiSMASH](https://antismash.secondarymetabolites.org/#!/start)

# Prophage prediction

* Identification and annotation of prophage sequences within bacterial genomes and plasmids with [PHASTER](http://phaster.ca/)

# Average Nucleotide Identity (ANI) calculation

* [Kostas lab](http://enve-omics.ce.gatech.edu/ani/index)
* [ChunLab's online Average Nucleotide Identity (ANI) calculator](https://www.ezbiocloud.net/tools/ani)

# Protein sequence vs profile-HMM database

* Fast and sensitive homology searches with [The HMMER web server](https://www.ebi.ac.uk/Tools/hmmer/search/hmmscan)

# Multi-Locus Sequence Typing (MLST)

* With a [webserver](https://cge.cbs.dtu.dk//services/MLST/)
* With a [local tool](https://github.com/liaochenlanruo/MLST)
