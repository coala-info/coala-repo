Usage: Rscript  qualitymetrics_wrapper.R {args} 
 parameters: 
 	-h: display this help message and exit (optional).
 	dataMatrix_in {file}: set the input data matrix file (mandatory).

 	injectionOrderColName {val}: the name of the column containing injection order information (mandatory).
 	batchColName {val}: the name of the column containing batch information (mandatory).
 	sampleTypeColName {val}: the name of the column containing sample type information (mandatory).
 	sampleTypeTags {val}: the tags used inside the sample type column, defined as key/value pairs separated by commas (example: blank=blank,pool=pool,sample=sample) (mandatory).
 	figure {file}: set the output figure file (mandatory).
 	information {file}: set the output information file (mandatory).
