# bgen-cpp

## Overview
The `bgen-cpp` skill enables efficient manipulation of BGEN formatted genomic data. BGEN is the primary format used by large-scale biobanks (such as the UK Biobank) for storing imputed genetic variants. This skill focuses on the core C++ utilities provided in the reference implementation to index, filter, merge, and modify BGEN files without needing to convert them to heavier text-based formats like VCF.

## Core Utilities and CLI Patterns

### bgenix: Indexing and Subsetting
`bgenix` is the primary tool for retrieving specific variants or regions.

*   **Create an index**: Always index a BGEN file before attempting to subset it.
    ```bash
    bgenix -g data.bgen -index
    ```
*   **List variants**: View all variants contained in the file.
    ```bash
    bgenix -g data.bgen -list
    ```
*   **Extract by region**: Retrieve variants within a specific genomic range.
    ```bash
    bgenix -g data.bgen -range 01:1000000-2000000 > subset.bgen
    ```
*   **Extract by RSID**: Filter for specific variants using a list of IDs.
    ```bash
    bgenix -g data.bgen -rsid rs123 rs456 > filtered.bgen
    ```

### cat-bgen: Concatenation
Use `cat-bgen` to merge multiple BGEN files that share the same sample set (e.g., merging chromosome-specific files).

*   **Efficient merging**:
    ```bash
    cat-bgen -g chr1.bgen chr2.bgen chr3.bgen -og merged.bgen
    ```
*   **Note**: This tool performs a bitstream concatenation where possible, making it significantly faster than re-encoding.

### edit-bgen: Metadata Management
Use `edit-bgen` to modify the internal metadata of a BGEN file, such as the "free data" field or sample identifiers.

*   **Remove sample IDs**:
    ```bash
    edit-bgen -g data.bgen -remove-sample-ids
    ```

### bgen_to_vcf: Format Conversion
A common utility for interoperability with other bioinformatics pipelines.

*   **Convert to VCF**:
    ```bash
    bgen_to_vcf data.bgen > data.vcf
    ```

## Expert Tips and Best Practices

*   **C++11 Requirement**: If compiling from source, ensure your environment uses `gcc 4.7+` or `clang`. Set the compiler explicitly if needed: `CXX=/path/to/g++ ./waf configure`.
*   **Self-Contained Binaries**: The executables generated by the `waf` build system (found in `build/apps/` and `build/example/`) are generally self-contained and can be moved to any location in your `$PATH` without a formal `make install` step.
*   **UK Biobank Data**: When working with UK Biobank imputed data, `bgenix` is the standard tool for extracting specific genes or regions for downstream association testing.
*   **Performance**: For large-scale operations, prefer `cat-bgen` over manual merging, as it is optimized for the BGEN binary structure.
