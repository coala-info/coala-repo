AlignIO import rich from rich_msa import RichAlignment msa = Bio . AlignIO . read ( "tests/data/swissprot-halorhodopsin.muscle.afa" , "fasta" ) viewer = RichAlignment ( names = [ record . id for record in msa ], sequences = [ str ( record . seq ) for record in msa ], ) panel = rich . panel . Panel ( viewer , title = "swissprot-halorhodopsin.muscle.afa" ) rich . print ( panel ) You should get an output similar to the following picture, scaled to your terminal width: 🪛 Command-Line If you have the rich-msa library installed from PyPI, you can use it directly to view any alignment file, provided you have Biopython installed: $ python -m rich_msa -i tests/data/swissprot-halorhodopsin.muscle.afa Use the -f flag to change the file format from aligned FASTA (default) to any alignment format supported by Biopython. 💭 Feedback ⚠️ Issue Tracker Found a bug ?
