# mmult

## Overview
MMULT is a specialized toolkit designed for processing whole-genome bisulfite sequencing (WGBS) data across hundreds of samples. It excels at identifying methylation dynamics between biological groups and within single groups. By leveraging the Stan library for fast model fitting and entropy-based calculations for variability, it provides a computationally efficient workflow for discovering differentially methylated cytosines (DMCs) and regions (DMRs) in large cohorts.

## Submodule Usage and Best Practices

### 1. BBGLM (Beta-Binomial Generalized Linear Model)
Use this module to resolve DNA methylation dynamics between multiple biological groups.

*   **Input Format**: Requires methylation BED files (typically generated by `MCALL` in MOABS).
*   **Group Specification**: Replicates within a group are comma-separated; multiple groups are defined by repeating the `-m` flag.
*   **Memory Management**: For large datasets, use `-c` to specify chromosomes and `-l` to set the split length (default 20M) to process data in smaller batches.
*   **Command Pattern**:
    ```bash
    bbglm -m g1_r1.bed,g1_r2.bed -m g2_r1.bed -m g3_r1.bed,g3_r2.bed -o results.txt
    ```
*   **Key Parameters**:
    *   `--qval`: Q-value threshold for DMCs (default: 0.05).
    *   `--nominaldiff`: Minimum methylation difference (default: 0.2).
    *   `--mindmc`: Minimum DMCs required to define a DMR (default: 3).

### 2. CpGCDIFEnrich
Use this module to consolidate methylation differences from individual comparisons using KL-divergence.

*   **Input Format**: Requires comparison files generated by `MCOMP` in MOABS.
*   **Logic**: It compares sample Credible Methylation Differences (CDIFs) against the background distribution of the whole genome.
*   **Command Pattern**:
    ```bash
    cpgcdifenrich -c Comp1VsNormal -c Comp2VsNormal -o consolidated_results.txt
    ```
*   **Key Parameters**:
    *   `--kldthr`: KL-divergence threshold (default: 0.67957).
    *   `--maxzerocdif`: Maximum allowed percentage of zero CDIFs for a DMC (default: 0.05).

### 3. VMCVMRNME
Use this module to detect Variable Methylated CpGs (VMCs) and Regions (VMRs) within a single biological group.

*   **Application**: Ideal for subtype detection in large-scale samples.
*   **Metric**: Uses normalized entropy to identify sites with low randomness and high variation.

## Expert Tips
*   **Threading**: Both `bbglm` and `cpgcdifenrich` support multi-threading. Adjust `-t` (read/num threads) and `-b` (batch threads) based on your CPU availability to significantly decrease runtime.
*   **Chromosome Encoding**: When using `-c`, you can encode the size (e.g., `-c chr1:248956422`) to help the tool calculate batch splits accurately.
*   **Minimum Depth**: Always consider the `-d` (mindepth) parameter. While the default is 1, increasing this to 5 or 10 for high-coverage WGBS data improves the reliability of GLM fits.



## Subcommands

| Command | Description |
|---------|-------------|
| mmult_bbglm | Analyze methylation data using a generalized linear model. |
| mmult_cpgcdifenrich | Calculate CpG methylation difference enrichment |
| mmult_vmcvmrnme | Detects VMCs (Variable Methylation Calls) and VMRs (Variable Methylation Regions) from methylation BED files. |
