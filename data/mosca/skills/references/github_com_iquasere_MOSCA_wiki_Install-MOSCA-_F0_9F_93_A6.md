# Install MOSCA 📦

[Jump to bottom](#wiki-pages-box)

João Sequeira edited this page May 17, 2023
·
[2 revisions](/iquasere/MOSCA/wiki/Install-MOSCA-%F0%9F%93%A6/_history)

MOSCA should be installed from Bioconda. Alternatively, it can be compiled from the source code.

## Prerequisites 📋

Before you can install and run MOSCA, you need some [Anaconda](https://www.anaconda.com/download) distribution installed, and [mamba](https://anaconda.org/conda-forge/mamba). As an example, Miniconda and mamba can be installed in Linux with these commands:

```
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
conda install -c conda-forge mamba
```

## Installing the Pipeline 💾

### From Bioconda

To install MOSCA, simply run:

```
mamba install -c conda-forge -c bioconda mosca
```

### From source code

Obtain the latest release (suggested) by checking the last version at the [releases page](https://github.com/iquasere/MOSCA/releases) and running

```
wget https://github.com/iquasere/MOSCA/archive/refs/tags/{VERSION}.zip
unzip {VERSION}.zip
```

or clone the repository to your local machine:

```
git clone https://github.com/iquasere/MOSCA.git
```

If you downloaded the latest release:

```
bash MOSCA-{VERSION}/workflow/envs/install.bash
```

If you cloned the repo:

```
bash MOSCA/workflow/envs/install.bash
```

Finally, activate MOSCA's environment

```
conda activate mosca
```

## Running the Pipeline 🏃‍♂️

To check if MOSCA was properly installed, run:

```
mosca -v
```

To run MOSCA, use the following command:

```
mosca -c config.json
```

Where `config.json` must point to a MOSCA configuration file. Such file can be obtained at [MOSGUITO](https://iquasere.github.io/MOSGUITO/), and information on all parameters is available at the [respective page](https://github.com/iquasere/MOSCA/wiki/Configure-and-run-MOSCA-%F0%9F%9B%A0).

That's it! You're now ready to use MOSCA to analyze your meta-omics data. 🔍
