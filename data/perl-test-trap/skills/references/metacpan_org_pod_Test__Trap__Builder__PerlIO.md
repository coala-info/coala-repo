Version 0.3.5 DESCRIPTION This module provides a capture strategy perlio , based on PerlIO::scalar, for the trap&#39;s output layers. Note that you may specify different strategies for each output layer on the trap. See also Test::Trap (:stdout and :stderr) and Test::Trap::Builder (output_layer). CAVEATS These layers use in-memory files, and so will not (indeed cannot) trap output from forked-off processes -- including system() calls. Threads? No idea. It might even work correctly.
