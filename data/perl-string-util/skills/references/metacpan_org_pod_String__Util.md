INSTALLATION cpanm String::Util USAGE No functions are exported by default, they must be specified: use String::Util qw(trim eqq contains) alternately you can use :all to export all of the functions use String::Util qw(:all) FUNCTIONS collapse($string) collapse() collapses all whitespace in the string down to single spaces. Also removes all leading and trailing whitespace. Undefined input results in undefined output. $var = collapse(&quot; Hello world! &quot;); # &quot;Hello world!&quot; hascontent($scalar), nocontent($scalar) hascontent() returns true if the given argument is defined and contains something besides whitespace. An undefined value returns false. An empty string returns false. A value containing nothing but whitespace (spaces, tabs, carriage returns, newlines, backspace) returns false. A string containing any other characters (including zero) returns true. nocontent() returns the negation of hascontent() . $var = hascontent(&quot;&quot;); # False $var = hascontent(&quot; &quot;); # False $var = hascontent(&quot;a&quot;); # True $var = nocontent(&quot;&quot;); # True $var = nocontent(&quot;a&quot;); # False trim($string), ltrim($string), rtrim($string) Returns the string with all leading and trailing whitespace removed. $var = trim(&quot; my string &quot;); # &quot;my string&quot; ltrim() trims leading whitespace only. rtrim() trims trailing whitespace only. nospace($string) Removes all whitespace characters from the given string. This includes spaces between words. $var = nospace(&quot; Hello World! &quot;); # &quot;HelloWorld!&quot; htmlesc($string) Formats a string for literal output in HTML. An undefined value is returned as an empty string. htmlesc() is very similar to CGI.pm&#39;s escapeHTML. However, there are a few differences. htmlesc() changes an undefined value to an empty string, whereas escapeHTML() returns undefs as undefs. jsquote($string) Escapes and quotes a string for use in JavaScript. Escapes single quotes and surrounds the string in single quotes. Returns the modified string. unquote($string) If the given string starts and ends with quotes, removes them. Recognizes single quotes and double quotes. The value must begin and end with same type of quotes or nothing is done to the value. Undef input results in undef output. Some examples and what they return: unquote(q|&#39;Hendrix&#39;|); # Hendrix unquote(q|&quot;Hendrix&quot;|); # Hendrix unquote(q|Hendrix|); # Hendrix unquote(q|&quot;Hendrix&#39;|); # &quot;Hendrix&#39; unquote(q|O&#39;Sullivan|); # O&#39;Sullivan option: braces If the braces option is true, surrounding braces such as [] and {} are also removed. Some examples: unquote(q|[Janis]|, braces=&gt;1); # Janis unquote(q|{Janis}|, braces=&gt;1); # Janis unquote(q|(Janis)|, braces=&gt;1); # Janis repeat($string, $count) Returns the given string repeated the given number of times. The following command outputs &quot;Fred&quot; three times: print repeat(&#39;Fred&#39;, 3), &quot;\n&quot;; Note that repeat() was created a long time based on a misunderstanding of how the perl operator &#39;x&#39; works. The following command using x would perform exactly the same as the above command. print &#39;Fred&#39; x 3, &quot;\n&quot;; Use whichever you prefer. eqq($scalar1, $scalar2) Returns true if the two given values are equal. Also returns true if both are undef . If only one is undef , or if they are both defined but different, returns false. Here are some examples and what they return. $var = eqq(&#39;x&#39;, &#39;x&#39;); # True $var = eqq(&#39;x&#39;, undef); # False $var = eqq(undef, undef); # True neqq($scalar1, $scalar2) The opposite of neqq , returns true if the two values are *not* the same. Here are some examples and what they return. $var = neqq(&#39;x&#39;, &#39;x&#39;); # False $var = neqq(&#39;x&#39;, undef); # True $var = neqq(undef, undef); # False ords($string) Returns the given string represented as the ascii value of each character. $var = ords(&#39;Hendrix&#39;); # {72}{101}{110}{100}{114}{105}{120} options convert_spaces=&gt;[true|false] If convert_spaces is true (which is the default) then spaces are converted to their matching ord values. So, for example, this code: $var = ords(&#39;a b&#39;, convert_spaces=&gt;1); # {97}{32}{98} This code returns the same thing: $var = ords(&#39;a b&#39;); # {97}{32}{98} If convert_spaces is false, then spaces are just returned as spaces. So this code: ords(&#39;a b&#39;, convert_spaces=&gt;0); # {97} {98} alpha_nums If the alpha_nums option is false, then characters 0-9, a-z, and A-Z are not converted. For example, this code: $var = ords(&#39;a=b&#39;, alpha_nums=&gt;0); # a{61}b deords($string) Takes the output from ords() and returns the string that original created that output. $var = deords(&#39;{72}{101}{110}{100}{114}{105}{120}&#39;); # &#39;Hendrix&#39; contains($string, $substring) Checks if the string contains substring $var = contains(&quot;Hello world&quot;, &quot;Hello&quot;); # true $var = contains(&quot;Hello world&quot;, &quot;llo wor&quot;); # true $var = contains(&quot;Hello world&quot;, &quot;&quot;); # true $var = contains(&quot;Hello world&quot;, &quot;QQQ&quot;); # false $var = contains(undef, &quot;QQQ&quot;); # false $var = contains(&quot;Hello world&quot;, undef); # false # Also works with grep @arr = grep { contains(&quot;cat&quot;) } @input; startswith($string, $substring) Checks if the string starts with the characters in substring $var = startwith(&quot;Hello world&quot;, &quot;Hello&quot;); # true $var = startwith(&quot;Hello world&quot;, &quot;H&quot;); # true $var = startwith(&quot;Hello world&quot;, &quot;&quot;); # true $var = startwith(&quot;Hello world&quot;, &quot;Q&quot;); # false $var = startwith(undef, &quot;Q&quot;); # false $var = startwith(&quot;Hello world&quot;, undef); # false # Also works with grep @arr = grep { startswith(&quot;X&quot;) } @input; endswith($string, $substring) Checks if the string ends with the characters in substring $var = endswith(&quot;Hello world&quot;, &quot;world&quot;); # true $var = endswith(&quot;Hello world&quot;, &quot;d&quot;); # true $var = endswith(&quot;Hello world&quot;, &quot;&quot;); # true $var = endswith(&quot;Hello world&quot;, &quot;QQQ&quot;); # false $var = endswith(undef, &quot;QQQ&quot;); # false $var = endswith(&quot;Hello world&quot;, undef); # false # Also works with grep @arr = grep { endswith(&quot;z&quot;) } @input; crunchlines($string) Compacts contiguous newlines into single newlines.
