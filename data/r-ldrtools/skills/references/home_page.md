## LDRTools: Tools for Linear Dimension Reduction

Linear dimension reduction subspaces can be uniquely defined using orthogonal projection matrices. This package provides tools to compute distances between such subspaces and to compute the average subspace. For details see Liski, E.Nordhausen K., Oja H., Ruiz-Gazen A. (2016) Combining Linear Dimension Reduction Subspaces <[doi:10.1007/978-81-322-3643-6\_7](https://doi.org/10.1007/978-81-322-3643-6_7)>.
