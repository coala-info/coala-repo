# HTML Tags in Widget Palette

[Jump to bottom](#wiki-pages-box)

aerwin edited this page Jul 1, 2012
·
[6 revisions](/maqetta/maqetta/wiki/HTML-Tags-in-Widget-Palette/_history)

This page contains information about the HTML5 tags available in the Widgets Palette of Maqetta.

The table below contains an alphabetical list of all of HTML5 tags, and information about whether they are to be included in the Widget Palette of Maqetta. Tags flagged with an asterisk (\*) are new in HTML5.

Generally speaking, the HTML tags included in the palette are either containers or UI controls.

| Tag | Description | Include in Palette? | Comments |
| --- | --- | --- | --- |
| [<!--...-->](http://www.w3schools.com/html5/tag_comment.asp) | Defines a comment | No |  |
| [<!DOCTYPE>](http://www.w3schools.com/html5/tag_doctype.asp) | Defines the document type | No |  |
| **[<a>](http://www.w3schools.com/html5/tag_a.asp)** | **Defines a hyperlink** | **Yes** |  |
| [<abbr>](http://www.w3schools.com/html5/tag_abbr.asp) | Defines an abbreviation | No |  |
| ~~[<acronym>](http://www.w3schools.com/html5/tag_acronym.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| [<address>](http://www.w3schools.com/html5/tag_address.asp) | Defines contact information for the author/owner of a document/article | No |  |
| ~~[<applet>](http://www.w3schools.com/html5/tag_applet.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| [<area>](http://www.w3schools.com/html5/tag_area.asp) | Defines an area inside an image-map | No |  |
| [<article>](http://www.w3schools.com/html5/tag_article.asp)\* | Defines an article | No | Potential future enhancement for HTML5 layout elements ([#1590](https://github.com/maqetta/maqetta/issues/1590)) |
| [<aside>](http://www.w3schools.com/html5/tag_aside.asp)\* | Defines content aside from the page content | No | Potential future enhancement for HTML5 layout elements ([#1590](https://github.com/maqetta/maqetta/issues/1590)) |
| **[<audio>](http://www.w3schools.com/html5/tag_audio.asp)\*** | **Defines sound content** | **Yes** |  |
| [<b>](http://www.w3schools.com/html5/tag_b.asp) | Defines bold text | No | Use Rich Text Editor instead |
| [<base>](http://www.w3schools.com/html5/tag_base.asp) | Specifies the base URL/target for all relative URLs in a document | No |  |
| ~~[<basefont>](http://www.w3schools.com/html5/tag_basefont.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| [<bdi>](http://www.w3schools.com/html5/tag_bdi.asp)\* | Isolates a part of text that might be formatted in a different direction from other text outside it | No | Potential future enhancement for HTML5 layout elements ([#1590](https://github.com/maqetta/maqetta/issues/1590)) |
| [<bdo>](http://www.w3schools.com/html5/tag_bdo.asp) | Overrides the current text direction | No |  |
| ~~[<big>](http://www.w3schools.com/html5/tag_big.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| [<blockquote>](http://www.w3schools.com/html5/tag_blockquote.asp) | Defines a section that is quoted from another source | No | Use Rich Text Editor instead |
| [<body>](http://www.w3schools.com/html5/tag_body.asp) | Defines the document's body | No |  |
| **[<br>](http://www.w3schools.com/html5/tag_br.asp)** | **Defines a single line break** | **Yes** |  |
| **[<button>](http://www.w3schools.com/html5/tag_button.asp)** | **Defines a clickable button** | **Yes** |  |
| [<canvas>](http://www.w3schools.com/html5/tag_canvas.asp)\* | Used to draw graphics, on the fly, via scripting (usually JavaScript) | No | Potential future enhancement for adding <canvas> ([#1591](https://github.com/maqetta/maqetta/issues/1591)) |
| [<caption>](http://www.w3schools.com/html5/tag_caption.asp) | Defines a table caption | No | Table-related tags which go inside <table> are handled by Maqetta's table editing capabilities |
| ~~[<center>](http://www.w3schools.com/html5/tag_center.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| [<cite>](http://www.w3schools.com/html5/tag_cite.asp) | Defines the title of a work | No |  |
| [<code>](http://www.w3schools.com/html5/tag_phrase_elements.asp) | Defines a piece of computer code | No | Use Rich Text Editor instead |
| [<col>](http://www.w3schools.com/html5/tag_col.asp) | Specifies column properties for each column within a <colgroup> element | No | Table-related tags which go inside <table> are handled by Maqetta's table editing capabilities |
| [<colgroup>](http://www.w3schools.com/html5/tag_colgroup.asp) | Specifies a group of one or more columns in a table for formatting | No | Table-related tags which go inside <table> are handled by Maqetta's editing capabilities |
| [<command>](http://www.w3schools.com/html5/tag_command.asp) | Defines a command button that a user can invoke | No |  |
| [<datalist>](http://www.w3schools.com/html5/tag_datalist.asp)\* | Specifies a list of pre-defined options for input controls | No | Potential future enhancement for HTML5 UI widgets ([#1591](https://github.com/maqetta/maqetta/issues/1591)) |
| [<dd >](http://www.w3schools.com/html5/tag_dd.asp) | Defines a description of an item in a definition list | No |  |
| [<del>](http://www.w3schools.com/html5/tag_del.asp) | Defines a text that has been deleted from a document | No |  |
| [<details>](http://www.w3schools.com/html5/tag_details.asp)\* | Defines additional details that the user can view or hide | No | Potential future enhancement for HTML5 layout elements ([#1590](https://github.com/maqetta/maqetta/issues/1590)) |
| [<dfn>](http://www.w3schools.com/html5/tag_phrase_elements.asp) | Defines a definition term | No |  |
| ~~[<dir>](http://www.w3schools.com/html5/tag_dir.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| **[<div>](http://www.w3schools.com/html5/tag_div.asp)** | **Defines a section in a document** | **Yes** |  |
| [<dl>](http://www.w3schools.com/html5/tag_dl.asp) | Defines a definition list | No |  |
| [<dt>](http://www.w3schools.com/html5/tag_dt.asp) | Defines a term (an item) in a definition list | No |  |
| [<em>](http://www.w3schools.com/html5/tag_phrase_elements.asp) | Defines emphasized text | No |  |
| **[<embed>](http://www.w3schools.com/html5/tag_embed.asp)\*** | **Defines a container for an external application or interactive content (a plug-in)** | **Yes** |  |
| **[<fieldset>](http://www.w3schools.com/html5/tag_fieldset.asp)** | **Groups related elements in a form** | **Yes** |  |
| [<figcaption>](http://www.w3schools.com/html5/tag_figcaption.asp)\* | Defines a caption for a <figure> element | No | Potential future enhancement for HTML5 layout elements ([#1590](https://github.com/maqetta/maqetta/issues/1590)) |
| [<figure>](http://www.w3schools.com/html5/tag_figure.asp)\* | Specifies self-contained content | No | Potential future enhancement for HTML5 layout elements ([#1590](https://github.com/maqetta/maqetta/issues/1590)) |
| ~~[<font>](http://www.w3schools.com/html5/tag_font.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| [<footer>](http://www.w3schools.com/html5/tag_footer.asp)\* | Defines a footer for a document or section | No | Potential future enhancement for HTML5 layout elements ([#1590](https://github.com/maqetta/maqetta/issues/1590)) |
| **[<form>](http://www.w3schools.com/html5/tag_form.asp)** | **Defines an HTML form for user input** | **Yes** |  |
| ~~[<frame>](http://www.w3schools.com/html5/tag_frame.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| ~~[<frameset>](http://www.w3schools.com/html5/tag_frameset.asp)~~ | ~~Not supported in HTML5~~ | No |  |
| **[<h1>](http://www.w3schools.com/html5/tag_hn.asp)** | **Defines HTML heading 1** | **Yes** |  |
| **[<h2>](http://www.w3schools.com/html5/tag_hn.asp)** | **Defines HTML heading 2** | **Yes** |  |
| **[<h3>](http://www.w3schools.com/html5/tag_hn.asp)** | **Defines HTML heading 3** | **Yes** |  |

| [<h5>](http://www.w3schools.com/html5/tag_hn.asp) | Defines HTML heading 5 | No |  |
