# Gene Expression Analysis Workflow in Complex Microbiomes

## Overview

This [Common Workflow Language](https://workflowhub.eu/workflows/1955) (CWL) workflow provides a comprehensive pipeline for gene expression profiling in complex microbiomes. By integrating metagenomic and metatranscriptomic NGS short-read data from the same samples, the workflow enables data-driven analysis through three primary stages: metagenomic assembly, read mapping, and functional annotation.

The workflow follows a structured execution path:
*   **Metagenomic Assembly:** Utilizes `Megahit` for contig assembly, `Prodigal` for protein sequence prediction, and `SeqKit` for statistical assessment.
*   **Read Mapping:** Employs `BWA MEM` to map both metagenomic and metatranscriptomic reads back to the assembled contigs, with results processed via `SAMtools`.
*   **Annotation:** Performs rRNA contamination screening using `BLASTN`, protein database searching via `DIAMOND` (against Swiss-Prot), and generates a final GTF file for downstream quantification.

Developed during [DBCLS BioHackathon 2025](https://github.com/dbcls/bh25/), this pipeline is based on methodologies detailed in [Mameda et al. (2025)](https://doi.org/10.3390/microorganisms13050995). The source code and associated scripts are maintained in the [RyoMameda/workflow_cwl](https://github.com/RyoMameda/workflow_cwl) repository.
