For a maintained IMC file parser implementation please refer to readimc . Background An IMC file conversion tool that aims to convert IMC raw data files (.mcd, .txt) into an intermediary ome.tiff, containing all the relevant metadata. Further it contains tools to generate simpler TIFF files that can be directly be used as input files for e.g. CellProfiller, Ilastik, Fiji etc. Documentation is available at https://bodenmillergroup.github.io/imctools Requirements This package requires Python 3.7 or later. Using virtual environments is strongly recommended.
