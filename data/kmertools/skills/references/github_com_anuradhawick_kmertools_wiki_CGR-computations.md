# Chaos Game Representation (CGR) computations

## Overview

To get started run the command `kmertools comp cgr --help` and you will see the following help output.

```
Generates Chaos Game Representations

Usage: kmertools comp cgr [OPTIONS] --input <INPUT> --output <OUTPUT>

Options:
  -i, --input <INPUT>        Input file path
  -o, --output <OUTPUT>      Output vectors path
  -c, --counts               Disable normalisation and output raw counts (only with k-mer mode)
  -k, --k-size <K_SIZE>      Set k-mer size or default to full sequence CGR
  -v, --vec-size <VEC_SIZE>  Set vector size (output will be a square matrix with N=vecsize)
  -t, --threads <THREADS>    Thread count for computations 0=auto [default: 0]
  -h, --help                 Print help
```

## Notes

* Default operation computes the CGR of the entire sequences and outputs the results.
* K-mers are computed by retaining the lexicographically minumum k-mer and its reverse compliment. This is only performed if a k-mer size is specified.
* Counts are normalised by the total number of valid k-mers of the sequence in k-mer based mode.
* Invalid k-mers are non ACGT letters k-mers (ACCN, AWCT, etc). Such windows are ignored automatically in k-mer based mode. However, invalid bases are not allowed when CGR computations are done sequence wide.

Kmertools uses the following cordinate system, starting at the origin `(vec-size/2, vec-size/2)`

```
-------------
|  C  | G   |
-------------
|  A  | T/U |
-------------
```

## Options

### K-size

K-mer sizes between 3 and 7 are allowed. Note that larger k-mers results in very large outputs. The shape of the resulting vector will be;

* For even k sizes

$$|N| = \frac{4^k + 4^{\frac{k}{2}}}{2}$$

* For odd k sizes

$$|N| = \frac{4^k}{2}$$

### Counts (only in k-mer mode)

This option disables normalisation and returns the array of k-mer counts. This is useful if you do not wish to normalise data but wants to do some other analysis.

### Vec-size

This is the output matrix size. This is automatically computed if nothing is provided. In default mode, `--vec-size 1` is assumed and an array of positions in a $\ 1 \times 1$ matrix. The results would be as follows.

```
(0.75,0.75) (0.875,0.875) ...
```

Each tuple corresponds to a base position in the matrix.

Simillarly in the case of `k-mer` based mode. You can expect an output as follows.

```
(0.5,0.5,0.014492) (0.5,8.5,0.014492) ...
```

Each entry corresponds to the position and normalised count. If you requested `--counts`, the output would be as follows.

```
(0.5,0.5,1) (0.5,8.5,1) ...
```

## Example application

### CGR Embedding of HIV virus

Download [`NC_001802.1`](https://www.ncbi.nlm.nih.gov/nuccore/NC_001802.1?report=fasta) from NCBI. This is provided [here](./examples/cgr/NC_001802.1.fasta).

```
kmertools comp cgr -i examples/cgr/NC_001802.1.fasta -o examples/cgr/NC_001802.1.cgr
```

Load from python as follows ([script](./examples/cgr/cgr.1.py)).

```
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

with open("NC_001802.1.cgr") as cgr:
    data = list(map(eval, cgr.read().split()))
    df = pd.DataFrame(data, columns=["X", "Y"])

    plt.figure(figsize=(10, 10))
    plt.title("NC_001802 CGR Plot")
    sns.scatterplot(df, x="X", y="Y", alpha=0.5)
    plt.savefig("NC_001802.1.cgr.png", dpi=150)
    plt.show()
```

The scatter plot would be as follows.

![](./examples/cgr/NC_001802.1.cgr.png)

### K-mer based CGR Embedding of HIV virus

You can use a command similar to below for k-mer based CGR computation.

```
kmertools comp cgr -i examples/cgr/NC_001802.1.fasta -o examples/cgr/NC_001802.1.k4.cgr -k 4
```

Load from python as follows ([script](./examples/cgr/cgr.2.py)).

```
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np

with open("NC_001802.1.k4.cgr") as cgr:
    data = list(map(eval, cgr.read().split()))
    df = pd.DataFrame(data, columns=["X", "Y", "Alpha"])

    # Take the floor of X and Y values
    df["X"] = np.floor(df["X"]).astype(int)
    df["Y"] = np.floor(df["Y"]).astype(int)

    heatmap_data = df.pivot(columns=["Y"], index=["X"], values="Alpha").fillna(0)

    plt.figure(figsize=(10, 10))
    plt.title("NC_001802 CGR 4-mer Plot")
    sns.heatmap(heatmap_data, cmap="grey")
    plt.savefig("NC_001802.1.k4.cgr.png", dpi=150)
    plt.show()
```

The scatter plot would be as follows.

![](./examples/cgr/NC_001802.1.k4.cgr.png)
