Consan: pairwise structural RNA alignment Consan: pairwise structural RNA alignment Description: Pairwise RNA structural alignment, both unconstrained and constrained on alignment pins. Source Code: The Consan package contains source and minimal compile instructions. Consan was written in ANSI C and tested on Mandrake Linux. consan-1.2.tar.gz This is the Consan code only package (without datasets) as a single gzip'ed tar ball (692 KB). consan-1.2_all.tar.gz This is the complete Consan package (with datasets, described below) as a single gzip'ed tar ball (1.9 MB). Version History v1.0 Initial version. v1.1 Improved documentation. v1.2 Fixed bug which caused problems with long path names to dpswalign .
