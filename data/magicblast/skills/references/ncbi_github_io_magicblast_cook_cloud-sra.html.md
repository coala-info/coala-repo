Starting with version 1.6.0 Magic-BLAST downloads SRA data from the cloud providers rather than NCBI servers. Benefits include uninterrupted downloads, faster download speeds, and huge bandwidth for parallel downloads to multiple ruining processes or cloud instances. For more information see SRA in the Cloud . To take advantage of the increased download bandwidth when running Magic-BLAST in the cloud, please configure SRA downloads, using vdb-config -i program from SRA Toolkit .
