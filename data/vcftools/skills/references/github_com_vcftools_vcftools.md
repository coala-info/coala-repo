Credits Adam Auton (Binary Executable) Petr Danecek (Perl Module) Anthony Marcketta (Binary Executable) Building VCFtools General help about the building process's configuration step can be acquired via: ./configure --help Build from Release Tarball ./configure make make install You may need sudo permissions to run make install . Build from GitHub git clone https://github.com/vcftools/vcftools.git cd vcftools ./autogen.sh ./configure make make install You many need sudo permissions to run make install . Documentation Documentation and usage examples can be found here: https://vcftools.github.io/examples.html A manual page is also available. If prefix is set to /usr or if MANPATH points to $prefix/share/man , you can access the manual page via: man vcftools Getting Help The best way to get help regarding VCFtools is to use the GitHub Issues page.
