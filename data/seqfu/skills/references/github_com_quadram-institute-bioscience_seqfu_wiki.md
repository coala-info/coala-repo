# 𝕤𝕖𝕢𝕗𝕦

A collection of Sequence FASTX Utilities, partly shipped with this repository and partly coming from external sources.

They have been originally built these principles:

* Reading both FASTQ and FASTA sequences with the same parser
* Parsing *both* name *and* comments from sequence headers (i.e. `>Seq_name length=1200`)
* Supporting .gz input files, and possibly other compression formats
* Supporting streams (standard input / standard output)
* Native support for Illumina Paired-End libraries when needed

New scripts also adopt `BioX::Seq`.

## Provided Tools

* **[seqfu](wiki/seqfu-main)** - core utility

### Legacy tools (pre v. 0.7.0)

* [fu-cat](wiki/fu-cat), concatenate FASTX files
* [fu-grep](wiki/fu-grep), extract sequences by DNA pattern, by name or comment
* [fu-len](wiki/fu-len), filter sequences by size
* [fu-count](wiki/fu-count), count sequences
* [fu-rename](wiki/fu-rename), rename sequences with a prefix
* [fu-sort](wiki/fu-sort), sort sequences by size
* [pe-cat](wiki/pe-cat), concatenate paired-end files (error tolerant, can be used to repair broken PE)
* [pe-len](wiki/pe-len), filter paired end sets by length
* [pe-grep](wiki/pe-grep), filter paired end sets
* [pe-ren](wiki/pe-ren), rename FASTQ files using barcodes or Illumina paired ends
* [n50](wiki/n50), calculate N50, number of sequences, minimum, maximum and total length
* [interleafq](wiki/interleafq), interleave and deinterleave paired sequences

🧬 *[SeqFU](https://github.com/quadram-institute-bioscience/seqfu/wiki)* - a collection of tools to parse and manipulate FASTA and FASTQ files, supporting compressed input
