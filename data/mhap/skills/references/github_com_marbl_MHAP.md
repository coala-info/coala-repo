This is the development branch, please use the latest tagged . Build You must have a recent JDK and Apache Maven available. To checkout and build run: git clone https://github.com/marbl/MHAP.git cd MHAP mvn install Maven executables vary by system, it could also be maven/mvn/mv32, depending on your installation.
