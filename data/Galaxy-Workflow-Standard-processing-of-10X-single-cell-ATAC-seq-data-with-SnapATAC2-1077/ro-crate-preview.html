Research Object Crate for Workflow - Standard processing of 10X single cell ATAC-seq data with SnapATAC2

Original URL: https://workflowhub.eu/workflows/1077/ro_crate?version=1

Workflow for Single-cell ATAC-seq standard processing with SnapATAC2. This workflow takes a fragment file as input and performs the standard steps of scATAC-seq analysis: filtering, dimension reduction, embedding and visualization of marker genes with SnapATAC2. Finally, the clusters are manually annotated with the help of marker genes. In an alternative step, the fragment file can also be generated from a BAM file. * newer Version: Updated SnapATAC2 version from 2.5.3 to 2.6.4
