# epimetheus-py

## Overview

`epimetheus-py` is a specialized bioinformatics tool designed to convert modification pileup files (such as those generated by `modkit`) into motif-specific methylation data. While basecallers provide position-specific modification calls, `epimetheus-py` aggregates this data to report on the methylation state of specific DNA motifs (e.g., GATC). It supports high-performance lookups using compressed formats and provides both contig-level summaries and read-level quality assessments.

## Core CLI Workflows

### 1. Optimization with BGZF
Before running analysis, always compress your pileup files. This enables fast random access via `hts-lib`, speeding up subsequent methylation lookups by approximately 6x.

```bash
# Compress a bed pileup to BGZF format
epimetheus bgzip compress --input pileup.bed --output pileup.bed.gz
```

### 2. Contig-Level Methylation Analysis
Use this to find the median or weighted-mean methylation degree of specific motifs across entire contigs.

```bash
epimetheus methylation-pattern contig \
    --pileup pileup.bed.gz \
    --assembly assembly.fasta \
    --output results.tsv \
    --motifs 'GATC_a_1' 'CCWGG_m_2' \
    --threads 8 \
    --output-type median
```

**Key Parameters:**
- `--motifs`: Follow the format `<sequence>_<mod_type>_<position>`.
  - `a` = 6mA, `m` = 5mC, `h` = 5hmC.
  - Position is 0-indexed within the motif.
- `--output-type`: Choose `median` (default), `weighted-mean`, or `raw` (all positions).
- `--min-valid-read-coverage`: Minimum depth required to include a site in the calculation (default: 3).

### 3. Read-Level Methylation Analysis
Use this to extract the basecaller's confidence (0-255) for specific motif modifications on individual reads.

```bash
epimetheus methylation-pattern read-bam \
    --bam indexed_reads.bam \
    --assembly assembly.fasta \
    --output read_level_results.tsv \
    --motifs 'GATC_a_1'
```

## Expert Tips and Best Practices

- **Memory Management**: When processing large metagenomes, adjust `--batch-size` (default: 1000). Lower this value if you encounter RAM limitations, or increase it to improve throughput on high-memory systems.
- **Motif Discovery**: `epimetheus-py` requires you to know your motifs beforehand. If motifs are unknown, use a tool like `nanomotif` first to identify candidate sequences.
- **Filtering Quality**: Use the `--min-valid-cov-to-diff-fraction` parameter (default: 0.8) to filter out positions where a high proportion of reads show conflicting mapping or basecalling information, ensuring higher confidence in the reported methylation values.
- **Contig Filtering**: If you only need data for specific contigs, use the `bgzip decompress` command with the `--contigs` or `--contigs-file` flag to extract a subset of the pileup.
