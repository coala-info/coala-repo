Details: Please read the Readme file before using this program! You can also download the Perl version of SNAP and process your data on your local machine. Input Paste alignment [Sample Input] or upload alignment file Options In addition to summary table, show XYPLOT of the cumulative behavior of the average synonymous and non-synonymous substitutions as you move across the coding region.
