## Needed Files

The data used for this figure were accessed from [FinaleDB](http://finaledb.research.cchmc.org/) and [![DOI:10.5281/zenodo.3928545](https://camo.githubusercontent.com/86adbe76d57b7648cda6b407224ad9945e8eee1805d0ca3a7f0c74f3ec01d6c6/68747470733a2f2f7a656e6f646f2e6f72672f62616467652f444f492f31302e353238312f7a656e6f646f2e363931343830362e737667)](https://doi.org/10.5281/zenodo.3928545). Please refer to [this table](https://github.com/epifluidlab/FinaleToolkit/wiki/Figure-3c-and-d-sample-table) for a full list of entries, and assume it is called `entries.tsv`.

In addition to the Fragment files, end-motifs and MDS, we will also need a 2bit file, which can be obtained from [![DOI](https://camo.githubusercontent.com/86fbf69640185963585c4b0194e719a8bc8cf3cb5f712d90f9bec627ab9179e0/68747470733a2f2f7a656e6f646f2e6f72672f62616467652f444f492f31302e353238312f7a656e6f646f2e31343238343133332e737667)](https://doi.org/10.5281/zenodo.14284133).

This page assumes the following directory structure:

```
.
├── data/
│   ├── liver/
│   │   └── MAL####.hg19.frag.gz
│   ├── finaledb/
│   │   └── EE######.hg19.frag.gz
│   ├── hg19.2bit
│   └── samples.tsv
└── out
```

Note

These files use Broad style chromosome names (sometimes referred to as `b37` reference genome) that omit the `chr` prefix found in UCSC `hg19` files.

Next write a python script, Jupyter notebook, or open a python interpreter. First, we need import all dependencies

```
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import matplotlib as mpl
from finaletoolkit.frag import end_motifs, gen_kmers
```

Let's open our list of samples

```
sample_df = pd.read_csv(
    "data/samples.tsv",
    sep='\t',
)
```

Now we can iterate over each of these samples and calculate end-motifs and mds for both using FinaleToolkit's python API.

```
# setting up data structures to store results
kmers = gen_kmers(4, 'ACGT')
results = {
    'file': [],
    'disease': [],
    'mds': []
}
for kmer in kmers:
    results[kmer]=[]

# finding end-motifs and MDS values for each file
for file, disease in sample_df.itertuples(index=False):
    file_motifs = end_motifs(file, "data/hg19.2bit", workers=NTASKS) # set NTASKS according to your available CPU cores

    results['file'].append(file)
    results['disease'].append(disease)
    results['mds'].append(file_motifs.motif_diversity_score()) # appending mds numbers
    motif_freqs = file_motifs.frequencies()  # a dictionary of 4mers and their associated frequencies
    for kmer, freq in motif_freqs.items():
        results[kmer].append(freq)
    results_df = pd.DataFrame.from_dict(results)
```

Lastly, we can generate figures from this data

```
healthy_mds = results_df .loc[end_motifs_tsv['disease']=='Healthy', 'mds']
hcc_mds = results_df .loc[end_motifs_tsv['disease']=='Liver cancer', 'mds']
mds_series = [healthy_mds, hcc_mds]

# Get motif freqs of non-HCC and HCC patients for six motifs of interest
# for HCC
motifs = ['CCCA', 'CCAG', 'CCTG', 'TAAA', 'AAAA', 'TTTT']

# motif-freqs are stored in DataFrames in a list. For each motif there
# is a DataFrame for Non-HCC and a DataFrame for HCC

motif_freq_list = []
for motif in motifs:
    non_hcc = results_df.loc[(end_motifs_tsv['disease']=='Healthy'), motif]
    hcc = results_df.loc[end_motifs_tsv['disease']=='Liver cancer', motif]
    motif_freq_list.append(non_hcc*100)
    motif_freq_list.append(hcc*100)
motif_freq_list[1]

# box plot positions, colors, x tick labels
colors = []
positions = []
labels = []
ticks = []
tick_labels = []
for i in range(len(motifs)):
    positions.append(3*i+1)
    positions.append(3*i+2)
    ticks.append(3*i+1.5)
    tick_labels.append(motifs[i])
    colors.append('C0')
    colors.append('C3')
    labels.append('Non-HCC')
    labels.append('HCC')

# plotting
fig, axes = plt.subplots(1, 2)
fig.set_dpi(300)
fig.set_size_inches(10, 5)

# Box plot for HCC motifs
motif_box_plot = axes[0].boxplot(
        motif_freq_list,
        labels=labels,
        positions=positions,
        patch_artist=True,
)
for patch, color in zip(motif_box_plot['boxes'], colors):
        patch.set_facecolor(color)
axes[0].plot([0, 18], [100/256, 100/256], 'g--')
axes[0].set_xticks(ticks, tick_labels)
# FIXME: add legend

# Box plot for MDS
mds_box_plot = axes[1].boxplot(
        mds_series,
)

# axes[1].set_xticks([1, 2, 3], ['Healthy', 'HBV', 'HCC'])
axes[1].set_xticks([1, 2], ['Healthy', 'HCC'])
axes[0].set_title('(C)')
axes[1].set_title('(D)')
axes[0].title.set_size(20)
axes[1].title.set_size(20)
axes[0].set_yscale('log')
y_formatter = mpl.ticker.PercentFormatter()
axes[0].yaxis.set_major_formatter(y_formatter)
axes[0].set_yticks([0.5, 1, 2, 3])

# legend
colors = {'Healthy':'C0', 'HCC':'C3'}
labels = list(colors.keys())
handles = [plt.Rectangle((0,0),1,1, color=colors[label]) for label in labels]
axes[0].legend(handles, labels)
fig.savefig("hcc_vs_healthly.svg", format="svg")
```
