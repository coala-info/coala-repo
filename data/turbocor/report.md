### Original Help Text
```text
turbocor-compute 
Compute entries of a thresholded correlation matrix. Output to an HDF5 file.

USAGE:
    turbocor compute [FLAGS] [OPTIONS] <input-features.h5> <output-correlation.h5> [TEMPFILE]

FLAGS:
    -h, --help         Prints help information
        --transpose    Dataset matrix should be transposed
    -V, --version      Prints version information

OPTIONS:
        --dataset <DATASET>    Name of dataset in input HDF5 file. (Default: X)
        --lower-bound <C>      Suppress abs correlations below this number. (Default: 0.8)
    -t, --threads <THREADS>    Number of threads to use. (By default all cpu cores are used.)

ARGS:
    <input-features.h5>        Feature matrix in HDF5 format.
    <output-correlation.h5>    Output sparse COO correlation matrix in HDF5 format.
    <TEMPFILE>                 Path to temporary file to use.
```
