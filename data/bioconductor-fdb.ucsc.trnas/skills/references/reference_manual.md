FDb.UCSC.tRNAs

Annotation package for FeatureDb object(s)

Description

This package loads one or more FeatureDb objects. Such FeatureDb objects are an R interface to
prefabricated databases contained by this package.

The names of any objects exposed by this package indicate the origin and resources exposed. So
for example TxDb.Hsapiens.UCSC.hg19.knownGene would be a TranscriptDb object, of Homo
Sapiens data from UCSC build hg19 based on the knownGene Track.

Note

This data package was made from resources at UCSC on 2012-09-10 13:15:01 -0700 (Mon, 10 Sep
2012) and based on the hg19 genome based on the tRNAs table
