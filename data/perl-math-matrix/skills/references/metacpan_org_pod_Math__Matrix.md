All methods return new objects, so, for example, $X-&gt;add($Y) does not modify $X . $X -&gt; add($Y); # $X not modified; output is lost $X = $X -&gt; add($Y); # this works Some operators are overloaded (see &quot;OVERLOADING&quot; ) and allow the operand to be modified directly. $X = $X + $Y; # this works $X += $Y; # so does this METHODS Constructors new() Creates a new object from the input arguments and returns it. If a single input argument is given, and that argument is a reference to array whose first element is itself a reference to an array, it is assumed that the argument contains the whole matrix, like this: $x = Math::Matrix-&gt;new([[1, 2, 3], [4, 5, 6]]); # 2-by-3 matrix $x = Math::Matrix-&gt;new([[1, 2, 3]]); # 1-by-3 matrix $x = Math::Matrix-&gt;new([[1], [2], [3]]); # 3-by-1 matrix If a single input argument is given, and that argument is not a reference to an array, a 1-by-1 matrix is returned. $x = Math::Matrix-&gt;new(1); # 1-by-1 matrix Note that all the folling cases result in an empty matrix: $x = Math::Matrix-&gt;new([[], [], []]); $x = Math::Matrix-&gt;new([[]]); $x = Math::Matrix-&gt;new([]); If &quot;new()&quot; is called as an instance method with no input arguments, a zero filled matrix with identical dimensions is returned: $b = $a-&gt;new(); # $b is a zero matrix with the size of $a Each row must contain the same number of elements. new_from_sub() Creates a new matrix object by doing a subroutine call to create each element. $sub = sub { ... }; $x = Math::Matrix -&gt; new_from_sub($sub); # 1-by-1 $x = Math::Matrix -&gt; new_from_sub($sub, $m); # $m-by-$m $x = Math::Matrix -&gt; new_from_sub($sub, $m, $n); # $m-by-$n The subroutine is called in scalar context with two input arguments, the row and column indices of the element to be created. Note that no checks are performed on the output of the subroutine. Example 1, a 4-by-4 identity matrix can be created with $sub = sub { $_[0] == $_[1] ? 1 : 0 }; $x = Math::Matrix -&gt; new_from_sub($sub, 4); Example 2, the code $x = Math::Matrix -&gt; new_from_sub(sub { 2**$_[1] }, 1, 11);
