Release notes are here . We recommend running WiGiTS with Oncoanalyser , a Nextflow implementation of the WiGiTS pipeline. A detailed list of the resource files used to run the WiGiTs pipeline is here . Download links to the resource files can be found on the Oncoanalyser page . The pipeline can be run on a targeted panel eg TSO500 - see detailed information here .
