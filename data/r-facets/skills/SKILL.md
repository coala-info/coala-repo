# r-facets

## Overview
The `facets` (Fraction and Allele-Specific Copy Number Estimate from Tumor/normal Sequencing) package provides a comprehensive pipeline for analyzing high-throughput DNA sequencing data. It implements a joint segmentation of total and allele-specific signals to identify copy number alterations and estimate tumor purity (cellular fraction) and ploidy.

## Installation
To install the package and its required GC content data dependency:
```R
remotes::install_github("veseshan/pctGCdata")
remotes::install_github("mskcc/facets", build_vignettes = TRUE)
```

## Core Workflow

### 1. Data Input
The package requires a SNP matrix file (typically generated by the `snp-pileup` C++ tool provided with the source code).
```R
library(facets)
set.seed(1234)
rcmat <- readSnpMatrix(filename)
```

### 2. Preprocessing
`preProcSample` filters SNPs based on coverage and mapping quality, and calculates log-ratio and log-odds-ratio.
```R
# ndepth: minimum depth in normal sample
# het.thresh: threshold for calling a SNP heterozygous
xx <- preProcSample(rcmat, ndepth = 35, het.thresh = 0.25, snp.nbhd = 250)
```

### 3. Segmentation
`procSample` performs the segmentation. The `cval` parameter controls the segmentation sensitivity (higher values lead to fewer segments).
```R
# cval: critical value for segmentation (e.g., 150 for WES, 300 for deeper sequencing)
oo <- procSample(xx, cval = 300)
```

### 4. Copy Number Estimation
`emcncf` uses an Expectation-Maximization algorithm to estimate the cellular fraction and allele-specific copy numbers.
```R
fit <- emcncf(oo)

# Access results
fit$purity    # Estimated tumor purity
fit$ploidy    # Estimated ploidy
head(fit$cncf) # Segment-level copy number calls (tcn: total, lcn: lesser)
```

### 5. Visualization
```R
# Plot the log-ratio and allele-log-odds-ratio with segments
plotSample(x = oo, emfit = fit)

# Diagnostic plot for the EM fit
logRlogORsummary(fit)
```

## Key Functions and Parameters
- `readSnpMatrix()`: Reads the output from snp-pileup.
- `preProcSample()`: Handles GC bias correction and SNP filtering.
- `procSample()`: Estimates the diploid log-ratio (`dipLogR`) and segments the genome.
- `emcncf()`: The primary caller for purity, ploidy, and allele-specific states (TCN/LCN).
- `dipLogR`: The log-ratio level corresponding to the diploid state. If `oo$flags` indicates issues (e.g., "mafR not sufficiently small"), the `dipLogR` might need manual inspection.
