The purpose of the Legionella pneumophila SBT scheme is to provide a rapid and easily comparable method for the epidemiological typing of clinical and environmental isolates of Legionella pneumophila in outbreak investigations. Install conda install -c conda-forge -c bioconda -c defaults legsta Input The any2fasta tool is used to convert input files to FASTA for feeding to isPcr . It can accept FASTA, Genbank, EMBL, GFF, and many other formats . The files may also be compressed with gzip, bzip2 or zip. Output Output is a TSV file (or CSV if --csv is used).
