## Update to v0.3.5

This release includes new features and improvements for enhanced functionality and user experience:

* Support for RefSeq ID Versions: KOnezumi-AID now accepts RefSeq ID versions.
* Enhanced Output Presentation: Output has been refined for clarity and sophistication, making results easier to interpret.
* Batch Processing: Added support for batch processing to facilitate large-scale projects like CRISPR screening.
* Version Command: Introduced a version command to easily check the current version.
* macOS ARM64 Compatibility: Now fully supports OSX-ARM64 environments.
