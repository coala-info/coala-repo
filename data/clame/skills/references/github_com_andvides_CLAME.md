It immplements a fm-index search algorithm for nucleotide sequence alignment. Then it uses strongly connected component strategy to bin sequences with similar DNA composition. Installation To download and install the SDSL - Succinct Data Structure Library, follow the instructions on the web page. https://github.com/simongog/sdsl-lite To download CLAME source codes git clone https://github.com/andvides/CLAME.git cd CLAME/ make Type ./clame -h to show the help CLAME Versions Type make or make all: To Compile CLAME. Type make debug: To compile a debug version of CLAME. Usage ./clame -b 70 -multiFasta test/Bancomini.fna -nt 4 -output bminiBins -print Output files bminiBins_*.fasta: Output fasta file for all the bins reported bminiBins.binning: All bins reported bmini.fm9: FM-index output bmini.index: First colum contains the origal name for each read, the second column the index used by CLAME bminiBins.links: Histogram links by number of reads bminiBins.result: Adjacency list for the overlap detected by each read Authors Benavides A(1), Alzate JF (2),(3) and Cabarcas F (1),(2) Grupo Sistemic, Departamento de Ingeniería Electrónica, Facultad de Ingenieria, Universidad de Antioquia.
