## Overview
nf-core/cellpainting is designed for the automated analysis of image-based profiling experiments, most notably the Cell Painting assay. It addresses the need for scalable and reproducible processing of large-scale microscopy datasets by orchestrating image analysis software in a high-performance computing environment.

The pipeline transforms raw multi-channel TIFF images into structured morphological data. Users provide image paths and experimental metadata, and the workflow generates processed results suitable for downstream statistical analysis along with comprehensive quality control reports.

## Data preparation
Input is defined via a CSV samplesheet specified with the `--input` parameter. The samplesheet must contain metadata describing the experimental origin and location of every image file.

Required columns in the samplesheet:
- `channel`: Alphanumeric identifier for the image channel (e.g., DNA, ER, RNA).
- `path`: Path to the `.tif` or `.tiff` image file.
- `source`, `batch`, `plate`, `well`, `site`, `row`, `col`: Metadata fields describing the experiment structure.

Example `samplesheet.csv`:
```csv
channel,path,source,batch,plate,well,site,row,col
DNA,/path/to/dna.tif,source1,batch1,plate1,A01,1,1,1
ER,/path/to/er.tif,source1,batch1,plate1,A01,1,1,1
```

## How to run
Run the pipeline using the standard Nextflow command. You must specify a profile (e.g., `docker`, `singularity`) and the output directory.

```bash
nextflow run nf-core/cellpainting \
    -r 1.0.0 \
    -profile <docker/singularity/conda> \
    --input samplesheet.csv \
    --outdir <RESULTS_DIR>
```

Key parameters include:
- `--cellprofiler_mode`: Set to `analysis` (default), `illumination`, or `assaydevelopment`.
- `--cellprofiler_analysis_cppipe`: Path to a custom CellProfiler analysis pipeline file.
- `--cellprofiler_illumination_cppipe`: Path to a custom illumination correction pipeline file.
- `-resume`: Use this Nextflow flag to restart a run from the last successful step if it was interrupted.

## Outputs
Results are saved to the directory specified by the `--outdir` parameter. The primary deliverables include extracted morphological features and profiling data generated by CellProfiler.

Users should first examine the MultiQC report for an overview of the run statistics and quality metrics. Detailed descriptions of all output files and their formats can be found in the [official output documentation](https://nf-co.re/cellpainting/output).
