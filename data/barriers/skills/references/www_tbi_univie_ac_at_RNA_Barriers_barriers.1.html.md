DESCRIPTION barriers 1.8.1 Compute local minima and energy barriers of a landscape This program reads an energy sorted list of conformations of a landscape, and computes local minima and energy barriers of the landscape. For RNA secondary structures, suitable input is produced by RNAsubopt. For each local minimum found it prints to stdout, the conformation of the minimum, its energy, the number of the &quot;parent&quot;&minus;minimum it merges with, and the height of the energy barrier. Additional information on each minimum, such as saddle point conformation and basin sizes can be included via options. A PostScript drawing of the resulting tree is written to &quot;tree.ps&quot; in the current directory. &minus;h , &minus;&minus;help Print help and exit &minus;&minus;detailed&minus;help Print help, including all details and hidden options, and exit &minus;&minus;full&minus;help Print help, including hidden options, and exit &minus;V , &minus;&minus;version Print version and exit General Options: Command line options which alter the general behavior of this program. &minus;v , &minus;&minus;verbose Be verbose, i.e. print more information. (default=off) &minus;q , &minus;&minus;quiet Be quiet (also inhibit PS tree output). (default=off) Graphs and Move Sets: The graph and move set options specify the types of states/conformations in the system as well as their neighborhood. Some graphs provide different neighborhood settings. The type of landscape may also be set by the input file using the fourth word on first line. E.g.: AUCGUGAGCUGUGUAGCUUAGCUAGCUAGC &minus;610 100 :: RNA&minus;noLP for a landscape of canonical RNA structures. &minus;G , &minus;&minus;graph = Graph Define type of the graph, i.e. configuration space. (default=&lsquo;RNA&rsquo;) The configuration space (graph) determines the neighborhood of the system and can be any of the following types: &rsquo;RNA&rsquo;: RNA secondary structures &rsquo;RNA&minus;noLP&rsquo;: canonical RNA structures &rsquo;RNA2&rsquo;: RNA secondary structures &rsquo;RNA2&minus;noLP&rsquo;: canonical RNA structures, no lonely pairs &rsquo;Q2&rsquo;: Spin Glass (strings of &quot;+&minus;&quot;) &rsquo;Qa,ALPHA&rsquo;: a&minus;letter Hamming graph, e.g. Q3 strings of &quot;ABC&quot;. ALPHAbet is optional, default is &rsquo;ABC...&rsquo; &rsquo;T&rsquo;: Phylogenetic Treesin bracket notation, e.g. &quot;((1)(((2)(3))(5))(4))&quot; &rsquo;P&rsquo;: Permutations as comma separated list, e.g. 5,1,4,2,3 &rsquo;X&rsquo;: Exchange Moves on balances +/&minus; strings &rsquo;?&rsquo;: General graph; adjacency list in file &minus;M , &minus;&minus;moves = STRING Select the move&minus;set for generating neighbors of a configuration (if Graph allows several different ones). The available move sets heavily depend on the choice of the graph. For RNA structures, the available move sets are: &rsquo;[no]Shift&rsquo;: (Dis&minus;)allow shift moves. (default: noShift) &rsquo;ligand&rsquo;: RNA&minus;ligand/protein interaction For permutation graphs, the possible move sets are: &rsquo;T&rsquo;: for Transpositions (default) &rsquo;C&rsquo;: for Canonical Transpositions &rsquo;R&rsquo;: for Reversal moves. Spin Glass graphs provide the two move sets: &rsquo;p&rsquo;: point mutations, i.e. single spin flip (default), and &rsquo;c&rsquo;: flip 2nd half. Trees always use NNI (Nearest Neighbour Interchange). &minus;&minus;poset = INT Input is a poset from n objective functions. (default=&lsquo;0&rsquo;) Barrier Tree Computation: &minus;c , &minus;&minus;connected Restrict the output to the connected component. (default=off) Depending on the input data, the corresponding barrier tree(s) might not be connected, i.e. the internal representation actually is a barrier forest. Enabling this option will restrict the output to the one connected component the ground state belongs to. In case there are multiple ground states that belong to different trees, the first (according to the input order) will be selected. Note, that restricting the rate computation to the connected component also ensures that the rate matrix is ergodic. &minus;&minus;bsize Print the size of of each basin in output. (default=off) &minus;&minus;ssize Print saddle component sizes. (default=off) &minus;&minus;max = INT Compute only the lowest &lt;num&gt; local minima. (default=&lsquo;100&rsquo;) &minus;&minus;minh= &lt;delta&gt; Print only minima with energy barrier greater than delta. (default=&lsquo;0.000001&rsquo;) &minus;&minus;saddle Print the saddle point conformations in output. (default=off) Transition Rate Computation: &minus;&minus;rates Compute rates between macro states (basins). (default=off) Two output files are written, &quot;rates.out&quot; containing the rates in ASCII format and &quot;rates.bin&quot; in binary form. Both output files can be readily used in conjunction with the treekin program. &minus;&minus;rates&minus;text&minus;file= &lt;filename&gt; Specify file name for the rates output. (default=&lsquo;rates.out&rsquo;) &minus;&minus;rates&minus;binary&minus;file= &lt;filename&gt; Specify file name for the binary rates output. (default=&lsquo;rates.bin&rsquo;) &minus;&minus;noLP&minus;rate = DOUBLE Scale rates of double moves for RNA&minus;noLP (e.g. 0.01). (default=&lsquo;1.0&rsquo;) &minus;&minus;microrates Compute microscopic rates between connected states. (default=off) &minus;T , &minus;&minus;temp = DOUBLE Set temperature in degree centigrade. (default=&lsquo;37.0&rsquo;) This temperature is used to compute Boltzmann fators in conjunction with &minus;&minus;rates option. Miscellaneous Options: &minus;P , &minus;&minus;path= &lt;l1&gt;=&lt;l2&gt; Backtrack an optimal path between local minimum l2 and l1. The option can be specified multiple times and each path will be written to a corresponding file &quot;path.l1.l2.txt&quot;. &minus;&minus;mapstruc= &lt;filename&gt; Map conformations to minima in the tree. Once the barrier tree (and optionally rates) are constructed , read stuctures from &quot;filename&quot; and map them to their corresponding minima (macro states). Output is written to file &quot;mapstruc.out&quot; in table format with the following columns: 1: gradient_min 2: index_in_subopt 3: energy 4: basin 5: truebasin 6: gradient_basin 7: true_grad_basin The last number is probably what you want, true basins are those that fulfill the minh criterium. &minus;&minus;mapstruc&minus;output= &lt;file&gt; Specify the filename for structure mapping output.
