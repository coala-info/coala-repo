Research Object Crate for VVV2_align_SE

Original URL: https://workflowhub.eu/workflows/517/ro_crate?version=1

SINGLE-END workflow. Align reads on fasta reference/assembly using bwa mem, get a consensus, variants, mutation explanations. IMPORTANT: * For "bcftools call" consensus step, the --ploidy file is in "Données partagées" (Shared Data) and must be imported in your history to use the worflow by providing this file (tells bcftools to consider haploid variant calling). * SELECT the mot ADAPTED VADR MODEL for annotation (see vadr parameters).
