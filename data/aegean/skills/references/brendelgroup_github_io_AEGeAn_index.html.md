## Integrated genome analysis toolkit

The AEGeAn Toolkit is designed for the **A**nalysis and **E**valuation of **Ge**nome **An**notations. The toolkit includes a variety of analysis programs as well as a C library whose API provides access to AEGeAn's core functions and data structures. AEGeAn is free to use and is released under an unrestrictive open source [license](https://github.com/BrendelGroup/AEGeAn/blob/master/LICENSE "AEGeAn license").

### Documentation

Preliminary documentation for the AEGeAn Toolkit, including download and installation instructions, API documentation, and program guides is available at [http://aegean.readthedocs.io](http://aegean.readthedocs.io "Documentation for AEGeAn").
