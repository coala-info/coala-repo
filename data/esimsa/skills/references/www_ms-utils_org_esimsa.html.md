About esimsa esimsa deconvolutes a peak list based only on mass differences between isotopic peaks. esimsa handles overlapping and resolved isotopic distributions but does not model the isotopic distributions themselves. Originally, esimsa was reposited here . Although this site is no longer maintained, it contains more information on esimsa. Download esimsa here . (Right-click and choose "Save Lin k As...".) Compiling esimsa Compile esimsa with: gcc -o esimsa esimsa.c -lm Using esimsa Usage: esimsa &lt; peaklist &gt; &lt; maximum charge &gt; &lt; output &gt; where &lt; peaklist &gt; is an undeconvoluted peak list from a mass spectrum, the &lt; maximum charge &gt; the maximum charge to be considered and &lt; output &gt; the file to which the deconvoluted mass list is written. magnus.palmblad@gmail.com
