Usage antaRNA can be used in several ways. This includes python object management or a direct command line call of the program. The program differentiates between two modi: MFE (dot-bracket structure optimization) and DP (dot plot structure optimization). Each of the modi has specific options and individual means of controlling its input. However, both modi also share options and input. Therefore a commandline call of the programm constitutes as: $python antaRNA.py [COMMON INPUT AND OPTIONS] [MODUS{MFE,DP}] [MODUS INPUT AND OPTIONS] See in Sources for detailed description of the explicit usage...
