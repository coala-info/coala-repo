qw/make_executable split_like_shell/; unshift @ARGV, split_like_shell($ENV{PROGRAM_OPTS}); write_script_to(&#39;Build&#39;); make_executable(&#39;Build&#39;); DESCRIPTION This module provides various portable helper functions for module building modules. FUNCTIONS make_executable($filename) This makes a perl script executable. split_like_shell($string) This function splits a string the same way as the local platform does. detildefy($path) This function substitutes a tilde at the start of a path with the users homedir in an appropriate manner. man1_pagename($filename, $ext = $Config{man1ext}) Returns the man page filename for a script. man3_pagename($filename, $basedir = &#39;lib&#39;, $ext = $Config{man3ext}) Returns the man page filename for a Perl library.
