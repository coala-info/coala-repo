Warning

Make sure to download the relevant index file for each `.bam` (`.bam.bai` index), `.cram` (`.cram.crai` index), or `.gz` (`.tbi` index).

## `mds`

### Let's retrieve the 5' end 5-mer motif diversity score (`mds`) from our [`end-motifs`](https://github.com/epifluidlab/FinaleToolkit/wiki/End-Motifs#end-motifs) results.

```
finaletoolkit mds BH01.22.em.tsv
```

#### Our output should be:

```
0.9439263367500222
```

## `interval-mds`

### Let's retrieve the 5' end 4-mer motif diversity score across the intervals (`interval-mds`) from our [`interval-end-motifs`](https://github.com/epifluidlab/FinaleToolkit/wiki/End-Motifs#interval-end-motifs) results.

```
finaletoolkit interval-mds BH01.100kb.22.em.tsv BH01.100kb.22.mds.tsv
```

#### Our TSV output should look something like this:

```
22	29800000	29900000	.	0.9383096384112906
22	29900000	30000000	.	0.9368196445448467
22	30000000	30100000	.	0.9440284013873861
22	30100000	30200000	.	0.9214779113674006
22	30200000	30300000	.	0.9434638292355728
22	30300000	30400000	.	0.9493896730236999
```
