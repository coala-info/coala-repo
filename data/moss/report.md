### Original Help Text
```text
Multiple Sample Somatic variant caller

Usage:
  moss [options] -r <reference> -b <normal BAM> -b <tumor BAM>
       -n <germline VCF> -v <candidate VCF> -o <output VCF>

Compulsary arguments:
  -b, --bam <BAM>        original BAM files for one sample,
                         can be addressed multiple times
  -R, --realigned <BAM>  realigned BAM files for one sample,
                         can be addressed multiple times
  -r, --ref <FASTA>      reference FASTA file
  -o, --output <OUT>     output VCF file
  -n, --normal <NORMAL>  normal sample's germline VCF result
  -v, --vcf <VCF>        tumor samples' somatic VCF result
  -l, --loci <LOCI>      candidate loci VCF files generated by python script

General options
  -h, --help             show this help message and exit

Other options:
  -t, --tau <TAU>        optional threshold for somatic score, default is 0
  -m, --mu <MU>          set somatic mutation prior, 1-5x10^(-m), default is 1-5e-4
  -B, --min-base-qual <MINBASE>
                         minimum threshold for base quality, default is 13
  -M, --min-mapping-qual <MINMAP>
                         minimum threshold for mapping quality, default is 30
  -d, --max_dep <depth>  max depth of the dataset, default is 500
  -g, --grid-size <grid> number of grid points in [0, 1] for approximate integration,
                         default is 20, must be non-negative
  --dry                  set to dry run
  --filter-total          set to filter variants with total tumor depth < 150
  --filter-vaf            set to filter variants with VAF in any samples < 0.1
  --ignore0              ignore samples with all reads same as the normal allele
  --pileup               output pileup, only effective in dry-run mode
```
