# Vignette of the a4Reporting package

#### 2025-10-29

# 1 Introduction

This document explains the functionalities available in the **a4Reporting** package.

This package contains utility functions to enhance reporting of data for the Automated Affymetrix Array Analysis suite of packages.

# 2 Create an annotation table

The `annotationTable` function enhances the reporting of table of genes or gene ontology term.

# 3 Top table of classification objects

A few utility functions to enhances the reporting of top tables for `glmnet`, `lognet`, `elnet`, `pam` and `pam` classification confusion are available.
