## gottcha_gottcha.pl

### Tool Description
Genomic Origin Through Taxonomic CHAllenge (GOTTCHA) is an annotation-independent and signature-based metagenomic taxonomic profiling tool that has significantly smaller FDR than other profiling tools. This Perl script is a wrapper to run the GOTTCHA profiling tool with pre-computed signature databases. The procedure includes 3 major steps: split-trimming the input data, mapping reads to a GOTTCHA database using BWA, profiling/filtering the result.
