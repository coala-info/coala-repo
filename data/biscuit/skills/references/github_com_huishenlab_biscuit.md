It was written to perform alignment, DNA methylation and mutation calling, and allele specific methylation from bisulfite sequencing data. Publication If you use BISCUIT, kindly cite : Wanding Zhou, Benjamin K Johnson, Jacob Morrison, Ian Beddows, James Eapen, Efrat Katsman, Ayush Semwal, Walid Abi Habib, Lyong Heo, Peter W Laird, Benjamin P Berman, Timothy J Triche, Hui Shen, BISCUIT: an efficient, standards-compliant tool suite for simultaneous genetic and epigenetic inference in bulk and single-cell studies, Nucleic Acids Research, Volume 52, Issue 6, 12 April 2024, Page e32, https://doi.org/10.1093/nar/gkae097 Download and Install Instructions for downloading and installing BISCUIT can be found in the User Guide Download and Install page. All releases of BISCUIT are available on GitHub . Notable Dependencies When building BISCUIT, the following libraries are needed: zlib pthreads ncurses libdeflate (optional, but greatly improves the speed of htslib ) curl It is also useful to have dupsifter installed for marking duplicate reads during the alignment phase. For BASH scripts found in the scripts/ directory, you will need the following dependencies in your PATH environment variable: QC.sh BISCUIT GNU awk samtools (for v1.6.1 and earlier) bedtools (for v1.6.1 and earlier) build_biscuit_QC_assets.pl perl bgzip (for v1.7.0 and later) tabix (for v1.7.0 and later) flip_pbat_strands.sh GNU awk samtools Usage and Documentation A User Guide has been created to provide useful documentation regarding the usage of BISCUIT. The guide can be found at: https://huishenlab.github.io/biscuit/ . Issues Any issues with BISCUIT can be submitted on the Issues page: https://github.com/huishenlab/biscuit/issues .
