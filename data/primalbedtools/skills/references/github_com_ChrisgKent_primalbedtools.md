It provides several functions for common BED file operations including coordinate remapping, sorting, updating, and amplicon generation. Functions are wrapped in a CLI for ease of use. Installation Install primalbedtools using pip: pip install primalbedtools or conda: conda install bioconda::primalbedtools Basic Usage primalbedtools &lt; command &gt; [options] Commands remap Remap BED file coordinates from one reference to another using a multiple sequence alignment. primalbedtools remap --bed &lt; bed_file &gt; --msa &lt; msa_file &gt; --from_id &lt; source_id &gt; --to_id &lt; target_id &gt; Arguments: --bed : Input BED file (required) --msa : Multiple sequence alignment file (required) --from_id : Source sequence ID to remap from (required) --to_id : Target sequence ID to remap to (required) Example: primalbedtools remap --bed primers.bed --msa alignment.fasta --from_id MN908947.3 --to_id BA.2 sort Sort BED file by chromosome, amplicon number, and primer direction. primalbedtools sort &lt; bed_file &gt; Arguments: bed : Input BED file Example: primalbedtools sort primers.bed &gt; primers.sorted.bed update Update primer names to v2 format ( prefix_number_DIRECTION_index ). primalbedtools update &lt; bed_file &gt; Arguments: bed : Input BED file Example: primalbedtools update primers.v1.bed &gt; primers.v2.bed amplicon Generate amplicon information from primer pairs. primalbedtools amplicon &lt; bed_file &gt; [--primertrim] Arguments: bed : Input BED file -t, --primertrim : Generate primer-trimmed amplicon information Example: primalbedtools amplicon primers.bed &gt; amplicons.txt primalbedtools amplicon primers.bed --primertrim &gt; trimmed_amplicons.txt merge Merge primers with the same properties (chromosome, amplicon number, direction). primalbedtools merge &lt; bed_file &gt; Arguments: bed : Input BED file Example: primalbedtools merge primers.bed &gt; primers.merged.bed fasta Convert BED file to FASTA format. primalbedtools fasta &lt; bed_file &gt; Arguments: bed : Input BED file Example: primalbedtools fasta primers.bed &gt; primers.fasta validate_bedfile Validate a BED file for internal consistency (correct primer pairings, etc.). primalbedtools validate_bedfile &lt; bed_file &gt; Arguments: bed : Input BED file Example: primalbedtools validate_bedfile primers.bed validate Validate a BED file against a reference genome. primalbedtools validate &lt; bed_file &gt; &lt; fasta_file &gt; Arguments: bed : Input BED file fasta : Reference FASTA file Example: primalbedtools validate primers.bed reference.fasta downgrade Downgrade a BED file from v2 to v1 primer name format. primalbedtools downgrade &lt; bed_file &gt; [--merge-alts] Arguments: bed : Input BED file --merge-alts : Merge alternative primers (removes _alt suffixes) Example: # Downgrade with alternative primers primalbedtools downgrade primers.v2.bed &gt; primers.v1.bed # Downgrade without alternative primers primalbedtools downgrade primers.v2.bed --merge-alts &gt; primers.v1.merged.bed About Tools for working with primerbed files chrisgkent.github.io/primalbedtools/ Resources Readme License MIT license Uh oh!
