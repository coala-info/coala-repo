Install The recommended method for script installation is via a Bioconda recipe called bioconda-scater-scripts. With the Bioconda channels configured the latest release version of the package can be installed via the regular conda install command: conda install scater-scripts Test installation There is a test script included: bioconductor-scater-scripts-post-install-tests.sh This downloads a well-known test 10X dataset and executes all of the scripts described below. Commands Currently wrapped Scater functions are described below. Each script has usage insructions available via --help, consult function documentation in Scater for further details. read10XResults() scater-read-10x-results.R -d &lt;10X data directory&gt; -o &lt;output SingleCellExperiment in .rds format&gt; calculateCPM() scater-calculate-cpm.R -i &lt;input SingleCellExperiment in .rds format&gt; -s &lt;size_factors&gt; -o &lt;output SingleCellExperiment in .rds format&gt; -t &lt;output matrix in .csv format&gt; Matrix filtering scater-filter.R -i &lt;input SingleCellExperiment in .rds format&gt; -s &lt;cell QC metric 1&gt;,&lt;cell QC metric 2&gt;,... -l &lt;Lower limit to metric 1&gt;,&lt;lower limit for metric 2&gt; -t &lt;feature QC metric 1&gt;,... -m &lt;feature QC metric lower limit 1&gt; -o &lt;filtered SingleCellExperiment in .rds format&gt; -u &lt;output matrix showing filtered cells by metric&gt; -v &lt;output matrix showing filtered features by metric&gt; normalize() scater-normalize.R -i &lt;input SingleCellExperiment in .rds format&gt; -e &lt;exprs_values&gt; -l &lt;return_log&gt; -f &lt;log_exprs_offset&gt; -c &lt;centre_size_factors&gt; -r &lt;return_norm_as_exprs&gt; -o &lt;output SingleCellExperiment in .rds format&gt; calculateQCMetrics() scater-calculate-qc-metrics.R -i &lt;input SingleCellExperiment in .rds format&gt; -e &lt;exprs_values&gt; -f &lt;feature_controls&gt; -c &lt;cell_controls&gt; -n &lt;nmads&gt; -p &lt;pct_feature_controls_threshold&gt; -o &lt;output SingleCellExperiment in .rds format&gt; isOutlier() scater-is-outlier.R -m &lt;metrics file&gt; -n &lt;nmads&gt; -t &lt;type&gt; -l &lt;log&gt; -d &lt;min.diff&gt; -o &lt;outliers file&gt; Accessory scripts In addition to the function wrappers above the following accessory scripts are provided: scater-extract-qc-metric.R This script extracts a single column of QC metric data, for example for use with the outlier detection script described above: scater-extract-qc-metric.R -i &lt;input SingleCellExperiment in .rds format&gt; -o &lt;output file&gt; -m &lt;metric name&gt; Output is a two-column csv file with &lt;cell name&gt;,&lt;metric value&gt; per line.
