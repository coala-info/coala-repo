# Blockchain

Tools for building blockchains.

* [cometbft - A distributed, Byzantine fault-tolerant, deterministic state machine replication engine. It is a fork of Tendermint Core and implements the Tendermint consensus algorithm.](/blockchain/cometbft-cometbft-github/)
* [cosmos-sdk - A Framework for Building Public Blockchains in the Cosmos Ecosystem.](/blockchain/cosmos-sdk-cosmos-github/)
* [gno - A comprehensive smart contract suite built with Golang and Gnolang, a deterministic, purpose-built Go variant for blockchains.](/blockchain/gno-gnolang-github/)
* [go-ethereum - Official Go implementation of the Ethereum protocol.](/blockchain/go-ethereum-ethereum-github/)
* [gosemble - A Go-based framework for building Polkadot/Substrate-compatible runtimes.](/blockchain/gosemble-limechain-github/)
* [gossamer - A Go implementation of the Polkadot Host.](/blockchain/gossamer-chainsafe-github/)
* [kubo - An IPFS implementation in Go. It provides content-addressable storage which can be used for decentralized storage in DApps. It is based on the IPFS protocol.](/blockchain/kubo-ipfs-github/)
* [lnd - A complete implementation of a Lightning Network node.](/blockchain/lnd-lightningnetwork-github/)
* [nview - Local monitoring tool for a Cardano Node. It's a TUI (terminal user interface) designed to fit most screens.](/blockchain/nview-blinklabs-io-github/)
* [solana-go - Go library to interface with Solana JSON RPC and WebSocket interfaces.](/blockchain/solana-go-gagliardetto-github/)
* [tendermint - High-performance middleware for transforming a state machine written in any programming language into a Byzantine Fault Tolerant replicated state machine using the Tendermint consensus and blockchain protocols.](/blockchain/tendermint-tendermint-github/)
* [tronlib - A comprehensive, production-ready Go SDK for interacting with the TRON blockchain with TRC20 token support.](/blockchain/tronlib-kslamph-github/)
