## Fixes

* [#148](https://github.com/kcleal/dysgu/issues/148) Migrate to superintervals >= 0.3
* [#135](https://github.com/kcleal/dysgu/issues/135) Partial fix for issue where ALT and REF bases are reported as the same. This occurred at complex SV sites in a polyploid genome. More events now undergo re-mapping resulting in better annotation of these loci and a reduction in the problem. A Warning is also now triggered if this issue occurs, and the SV will be annotated with symbolic notation to prevent issues with downstream tools.
