About COSAP COSAP is an easy yet comprehensive pipeline creation tool for NGS data. It provides reproducibility and aims to give deeper insight about the powers and limitations of the current tools by allowing users to compare results of different pipelines. A typical variant calling pipeline consists of Read Trimming, Read Mapping, Duplicate Removal&amp;Base Calibration, Variant Calling and Variant Annotation steps. COSAP provides several tool options for each of these steps. Usage The documentation for this project has moved. Please visit docs.cosap.bio for the latest information.
