## Overview

Polymutt is a bioinformatics tool designed for variant calling and, specifically, for the detection of de novo mutations within family pedigrees. It leverages a likelihood-based framework to analyze genetic data from multiple individuals in a family, enabling the identification of novel genetic variations that arise in offspring.

## Usage Instructions

Polymutt is a command-line tool. The primary function involves specifying input files and family structure.

### Core Functionality

The core of Polymutt's usage revolves around providing it with variant calls (e.g., VCF files) and defining the relationships within the family being analyzed.

### Input Files

*   **Variant Calls:** Polymutt typically operates on existing variant calls. You will need to provide these, often in VCF format.
*   **Family Pedigree:** A crucial input is the definition of family relationships. This is usually specified through a dedicated file or command-line arguments that map individuals to their parents.

### Common CLI Patterns

While specific command structures can vary based on the exact version and desired output, a general pattern involves:

```bash
polymutt --vcf <input.vcf> --ped <family.ped> --output <output_prefix> [options]
```

*   `--vcf`: Path to the VCF file containing variant calls.
*   `--ped`: Path to the pedigree file defining family relationships.
*   `--output`: A prefix for the output files generated by Polymutt.

### Expert Tips

*   **Pedigree File Format:** Ensure your pedigree file (`.ped`) strictly adheres to the expected format. This typically includes columns for family ID, individual ID, father ID, mother ID, sex, and affection status. Incorrect formatting is a common source of errors.
*   **De Novo Mutation Detection:** For de novo mutation detection, ensure that the VCF file contains genotypes for all relevant family members (parents and offspring). Polymutt's likelihood model relies on this complete genotype information.
*   **Output Interpretation:** Polymutt generates several output files. Pay close attention to files that specifically flag potential de novo mutations, often including likelihood scores or p-values to help prioritize findings.
*   **Parameter Tuning:** Explore available command-line options for fine-tuning the variant calling and de novo mutation detection process. This might include parameters related to filtering, allele frequency thresholds, or specific statistical models. Consult the tool's documentation for a comprehensive list.
