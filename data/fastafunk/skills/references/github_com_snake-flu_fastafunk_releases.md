[v0.1.0](/snake-flu/fastafunk/releases/tag/v0.1.0)

This release represents some major changes:

* metadata columns are now interpreted in a case sensitive way, instead of changing them
* new `low-memory` mode for many funks, which aim to stream metadata and fasta instead of reading into memory
* tests added and changed
* added `shuffle`, `drop_columns`, `filter_column` options
