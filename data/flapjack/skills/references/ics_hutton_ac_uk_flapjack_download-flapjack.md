View the release notes to see what’s new. Flapjack uses the BSD 2-Clause License . Windows and macOS installers are digitally signed by The James Hutton Institute (checksums can be viewed here ). You may need to right-click the installer on macOS and select Open to override Gatekeeper restrictions. Windows (64 bit) Linux (64 bit) macOS Alternatively, if you use Bioconda , then Flapjack can be installed with: conda install flapjack Our installers are built using the multi-platform installer builder install4j . Follow the instructions provided by the graphical installer to install Flapjack. Note that Flapjack should not require administrator privileges to install or run, but if you have any problems, don’t hesitate to contact us at flapjack@hutton.ac.uk . Flapjack is primarily developed on Windows 10, and gets additional regular testing on macOS and CentOS. Archived versions You can download a limited selection of older copies of Flapjack from our archive . If you do need to return to an older version (for whatever reason), please let us know why.
