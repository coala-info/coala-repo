## Overview

`nwkit drop` removes information on specified nodes and/or branches.

* `--name yes` drops node labels
* `--support yes` drops support values
* `--length yes` drops branch lengths

... of all, root, internal, or terminal branches/nodes depending on the `--target` option.

The `--fill` option can replace the dropped values with a specified string instead of removing them entirely.

### 2. Drop leaf names with --fill

The `--fill` option replaces the dropped values with a specified string.

#### Command

```
nwkit drop --infile input.nwk --target leaf --name yes --fill unknown --outfile output.nwk
```

#### input.nwk

```
((A:1,B:1)AB:1,(C:1,D:1)CD:1)root;
```

#### output.nwk

```
((unknown:1,unknown:1)AB:1,(unknown:1,unknown:1)CD:1)root:0;
```

All leaf names are replaced with `unknown`. Internal node names (`AB`, `CD`, `root`) are preserved.
