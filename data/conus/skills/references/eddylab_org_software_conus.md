## software for the exploration of lightweight stochastic context free grammars

### Description:

Software developed for exploring the consequences of different
single sequence SCFG designs in predicting RNA secondary structure.

### Source Code:

<conus-1.0.tar.gz>

This is the full CONUS package (without datasets) as a
gzip'ed tar ball (382 KB).

This package contains source, guides, and GNU configure script.
CONUS was written in ANSI C and tested on Mandrake Linux.

[See below for a download which includes the datasets.]
