The project exposes a JavaScript API for OS X integration, such as displaying native OS X 10.9 notifications. The MacGap project is extremely lightweight and nimble; a blank application is about 980KB. Features: tiny compiled app sizes Mac App Store compatible access to many Mac OS X-specific features For JavaScript developers.
