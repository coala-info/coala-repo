# Welcome to SeqinR: Biological Sequences in R project!

The seqinR package for the R environment contains utilities to retrieve and analyse biological sequences.

![](SeqinRGenericCover.png)

Download the last under development version of seqinR from
[here](http://r-forge.r-project.org/R/?group_id=220).

Before reading the full manual, a short introduction to seqinr is available [here](./src/mainmatter/introduction.pdf)

Version 3.1-5 of the seqinR manual is available [here](seqinr_3_1-5.pdf).

Reading FASTA files (help from the very last version of
the manual) is [here](./src/mainmatter/getseqflat.pdf).

FAQ are [here](./src/appendix/FAQ.pdf).

The list of available genetic codes is [here](./src/appendix/gencodes.pdf).

Last release notes are [here](./src/appendix/releasenotes.pdf).

You can browse the seqinr source repository
[here](https://r-forge.r-project.org/scm/viewvc.php/?root=seqinr).

The **project summary page** is [here](http://r-forge.r-project.org/projects/seqinr/).
