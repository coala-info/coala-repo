# fastaptamer

## Overview
FASTAptamer is a specialized toolkit designed to process and analyze the large datasets generated by high-throughput sequencing of biocombinatorial libraries. It streamlines the transition from raw sequence data to actionable insights by identifying unique sequences, quantifying their prevalence, and tracking how specific motifs or families evolve through successive rounds of selection. This skill provides the necessary command patterns to execute these core bioinformatic workflows.

## Installation
The toolkit is available via Bioconda. It is recommended to install it within a dedicated conda environment:
```bash
conda install -c bioconda fastaptamer
```

## Core CLI Workflows

### 1. Sequence Counting and Normalization
The first step in any analysis is collapsing raw reads into a list of unique sequences with their associated counts.
- **Command**: `fastaptamer_count`
- **Usage**: Processes a FASTA file to generate a `.count` file containing unique sequences, their abundance, and normalized RPM (reads per million).
- **Best Practice**: Ensure input files are quality-filtered and adapters are trimmed before running this command.

### 2. Comparing Populations
To identify sequences that are increasing in frequency between rounds, use the comparison tool.
- **Command**: `fastaptamer_compare`
- **Usage**: Takes two `.count` files (e.g., Round 5 and Round 10) and generates a file showing the ratio of abundance for each sequence.
- **Tip**: This is essential for identifying "winners" in a selection that may not be the most abundant but have the highest rate of enrichment.

### 3. Sequence Clustering
Group sequences into families based on sequence similarity to identify conserved structural or functional motifs.
- **Command**: `fastaptamer_cluster`
- **Usage**: Clusters sequences based on a user-defined Levenshtein edit distance (typically a distance of 1 to 3).
- **Expert Tip**: Clustering is computationally intensive. For very large datasets, consider filtering for a minimum abundance threshold before clustering to reduce the search space.

### 4. Enrichment Analysis
Calculate the fold-enrichment of sequences across multiple rounds or different selection conditions.
- **Command**: `fastaptamer_enrich`
- **Usage**: Compares multiple populations to track the trajectory of specific sequences across the entire selection timeline.

### 5. Motif Searching
Search for specific nucleotide patterns or degenerate motifs within a population.
- **Command**: `fastaptamer_search`
- **Usage**: Allows for degenerate string matching to find all sequences containing a specific consensus motif.

## Expert Tips for High-Throughput Selections
- **Data Volume**: When working with millions of reads, always use the `.count` files generated by `fastaptamer_count` for downstream steps rather than raw FASTA files to save time and memory.
- **Normalization**: Always rely on the normalized RPM values provided by the tool when comparing different sequencing runs, as raw counts are biased by the total depth of the library.
- **Edit Distance**: When clustering, start with an edit distance of 1. Increasing the distance significantly increases processing time and may group unrelated sequences if the random region is short.
