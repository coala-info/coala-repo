# r-sendmailr

## Overview
The `sendmailr` package provides a simple, portable SMTP client for R with minimal dependencies. It is designed for sending emails, including attachments and inline HTML content, without requiring a local mail server if external SMTP details are provided.

## Installation
To install the stable version from CRAN:
```r
install.packages("sendmailR")
```

## Core Functions

### sendmail()
The primary function for dispatching emails.
- `from`: Sender email address.
- `to`: Recipient email address(es).
- `subject`: Email subject line.
- `msg`: The body of the email. Can be a character vector or a list of `mime_part` objects.
- `control`: A list containing SMTP server settings (e.g., `list(smtpServer="smtp.example.com")`).
- `engine`: Set to `"curl"` for authentication and SSL/TLS support.
- `engineopts`: A list containing `username` and `password` when using the curl engine.

### mime_part()
Used to create multi-part messages, such as adding attachments.
- For files: `mime_part("path/to/file.csv", name = "data.csv")`
- For R objects: `mime_part(iris)` (automatically serializes the object).

## Common Workflows

### Basic Email
```r
library(sendmailR)

from <- "<sender@example.com>"
to <- "<recipient@example.com>"
subject <- "Notification from R"
body <- "The analysis is complete."

sendmail(from, to, subject, body, 
         control = list(smtpServer = "your.smtp.server"))
```

### Email with Attachments
To send an email with a file attachment, wrap the body and the file in a `list()`.
```r
attachment <- mime_part("/path/to/report.pdf", name = "Monthly_Report.pdf")
body <- list("Please find the attached report.", attachment)

sendmail(from, to, "Report Delivery", body, 
         control = list(smtpServer = "your.smtp.server"))
```

### Authenticated SMTP (Gmail/Outlook)
Requires the `curl` package. Use the `engine = "curl"` argument.
```r
sendmail(
  from = "you@gmail.com",
  to = "someone@example.com",
  subject = "Secure Mail",
  msg = "Sent via SSL/TLS",
  engine = "curl",
  engineopts = list(username = "you@gmail.com", password = "your_app_password"),
  control = list(smtpServer = "smtp://smtp.gmail.com:587", verbose = TRUE)
)
```

## Tips
- **Vectorized Recipients**: The `to` argument can be a character vector to send to multiple addresses.
- **HTML Reports**: Use `mime_part` with appropriate headers to send inline HTML generated by `knitr` or `rmarkdown`.
- **Debugging**: Set `verbose = TRUE` inside the `control` list to see the SMTP transaction logs.
