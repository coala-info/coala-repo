These can be made by several aligners, including minimap2 -c , wfmash , or lastz --format=paf:wfmash . With alignments in aln.paf (or aln.paf.gz if gzip-compressed), we would convert it into a CHAIN format file using this call: paf2chain -i aln.paf &gt; aln.chain info paf2chain performs the reverse operation of chain2paf .
