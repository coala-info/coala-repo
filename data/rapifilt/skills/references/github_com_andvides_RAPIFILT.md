It trims lowquality bases and removes short sequences. Installing cd RAPIFILT_v1.1 make export PATH=$PATH:$(pwd) Running DATMA rapifilt -fastq bmini.fq -l 20 -r 20 -o bmini_out Output RAPIFILT generates the following files: bmini_out.fastq: Filtered sequences. bmini_out_bad.fastq: Removed sequences. bmini_out_stat.txt: Statistics per base.
