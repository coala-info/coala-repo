# bioconductor-ocplus

## Overview

Use the Bioconductor R package **OCplus** for: The package includes tools both for planned experiments (for sample size assessment) and for already collected data (identification of differentially expressed genes).

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("OCplus")
```
