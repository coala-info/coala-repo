# GetoptLong: Parsing Command-Line Arguments and Simple Variable Interpolation

This is a command-line argument parser which wraps the 
    powerful Perl module Getopt::Long and with some adaptations for easier use
	in R. It also provides a simple way for variable interpolation in R.</p>
