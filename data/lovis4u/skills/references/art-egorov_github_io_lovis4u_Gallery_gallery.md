Sample data &amp; Configuration files The necessary sample data as well as adjustable tool configuration files are provided by lovis4u at the post-install step with the command: lovis4u --data This copies the lovis4u_data folder to your working directory. Also important to describe the set of editable configuration files with parameters already adjusted for different tasks and page layout: A4p1 - adjusted for one-column portrait A4 page layout. Figure width - 90mm. A4p2 - adjusted for two-column portrait A4 page layout. Figure width - 190mm. A4L - adjusted for landscape A4 page layout. Figure width - 240mm. standard - flexible width length, scale - 0.05 mm per nucleotide with minimal width - 80mm.
