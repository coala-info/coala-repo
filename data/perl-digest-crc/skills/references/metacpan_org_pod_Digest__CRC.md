AUTHOR COPYRIGHT NAME Digest::CRC - Generic CRC functions SYNOPSIS # Functional style use Digest::CRC qw(crc64 crc32 crc16 crcccitt crc crc8 crcopenpgparmor); $crc = crc64(&quot;123456789&quot;); $crc = crc32(&quot;123456789&quot;); $crc = crc16(&quot;123456789&quot;); $crc = crcccitt(&quot;123456789&quot;); $crc = crc8(&quot;123456789&quot;); $crc = crcopenpgparmor(&quot;123456789&quot;); $crc = crc($input,$width,$init,$xorout,$refout,$poly,$refin,$cont); # add data to existing $crc = crc32(&quot;ABCD&quot;, $crc); # OO style use Digest::CRC; $ctx = Digest::CRC-&gt;new(type=&gt;&quot;crc16&quot;); $ctx = Digest::CRC-&gt;new(width=&gt;16, init=&gt;0x2345, xorout=&gt;0x0000, refout=&gt;1, poly=&gt;0x8005, refin=&gt;1, cont=&gt;1); $ctx-&gt;add($data); $ctx-&gt;addfile(*FILE); $digest = $ctx-&gt;digest; $digest = $ctx-&gt;hexdigest; $digest = $ctx-&gt;b64digest; DESCRIPTION The Digest::CRC module calculates CRC sums of all sorts. It contains wrapper functions with the correct parameters for CRC-CCITT, CRC-16, CRC-32 and CRC-64, as well as the CRC used in OpenPGP&#39;s ASCII-armored checksum.
