The 'specifier use' table depicts the frequency each specifier is used to decode a given tRNA family. * The 'SPEC' column displays the given T-box riboswitch specifier, the 'AA' column shows the corresponding amino acid - tRNA family, while the 'FREQ' column is the frequency observed within the T-box riboswitch annotation database. ** If you found this table useful, please cite our BioRXiv publication on this work: Jorge A. Marchand, Merrick D. Pierson Smela, Thomas H. H. Jordan, Kamesh Narasimhan, George M. Church. "TBDB – A database of structurally annotated T-box riboswitch:tRNA pairs". 2020. BioRXiv.
