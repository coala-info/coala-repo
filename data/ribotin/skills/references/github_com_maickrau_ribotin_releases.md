[Latest](/maickrau/ribotin/releases/latest)

-Distal and proximal junction sequences which border the rDNA arrays are included in the output morphgraph
-ribotin-verkko and ribotin-hifiasm now output the morphs near the borders of the rDNA tangles as well
-Morph sequences are polished with ONT reads
-New tangle consensus algorithm handles more structurally divergent tangles
-New parameter `--verbose` prints more information
-Experimental parameter `--extra-phasing` to do morph phasing with ONT r10 reads
-Bugfixes

Known issues (affects this release and all previous releases):
-Morph consensus sequences can be inaccurate in microsatellite and homopolymer repeats.

[Version 1.4](/maickrau/ribotin/releases/tag/v1.4)

Changes:
-Input hifi and ONT read files can be in `.bam` format
-ONT loops which contribute to each morph are listed in output file `raw_loops.fa` and their alignments to the morphs are in output file `raw_loop_to_morphs_alignments.bam`
-Temporary files are moved to a temporary folder so they don't clutter the output folder as much
-Bugfixes

Known issues (affects this release and all previous releases):
-Morph consensus sequences can be inaccurate in microsatellite and homopolymer repeats.
-ribotin-verkko and ribotin-hifiasm modes handle morphs which were fully assembled by verkko/hifiasm badly. This issue does not affect ribotin-ref.
-Low copy count structurally divergent morphs might be missed. Reducing estimated morph size may in some cases help with this.

[ribotin.tar.gz](https://github.com/user-attachments/files/18780237/ribotin.tar.gz)
