| Key features  * Designed for eQTL analysis of large datasets. * Performs testing for all or only local transcript-SNP pair.s * Ultra-fast, no loss of precision. * Equally fast for models with covariates. * Supports   + Linear additive and ANOVA models. Supports testing for the effect of genotype-covariate interaction.   + Covariates to account for sex, population structure, surrogate variables, etc.   + Correlated and heteroskedastic errors.   + Correction for multiple testing using FDR [![external link](external.png)](http://en.wikipedia.org/wiki/False_discovery_rate).   + Separate p-value thresholds and FDR control for local and distant eQTLs ([more info](runit.html#cis)). * Convenient R package at CRAN Repository [![external link](external.png)](http://cran.r-project.org/web/packages/MatrixEQTL/index.html).  **Performance comparison:**  | Method | No covar. | 10 covar. |  | | --- | --- | --- | --- | |  |  |  |  | | --- | --- | --- | --- | | Matrix eQTL, Matlab[![external link](external.png)](http://www.mathworks.com/products/matlab/) | 11.8 | 11.8 | **minutes** | | Matrix eQTL, Rev R[![external link](external.png)](http://www.revolutionanalytics.com/) | 14.6 | 14.6 | **minutes** | | Matrix eQTL, R+GOTO[![external link](external.png)](http://prs.ism.ac.jp/~nakama/SurviveGotoBLAS2/binary/windows/x64/) | 19.4 | 19.4 | **minutes** | | Plink [![external link](external.png)](http://pngu.mgh.harvard.edu/~purcell/plink/) | 9.4 | 583.3 | days | | Merlin [![external link](external.png)](http://www.sph.umich.edu/csg/abecasis/Merlin/index.html) | 19.6 | 20.0 | days | | R/qtl [![external link](external.png)](http://www.rqtl.org/) | 1.0 | 4.7 | days | | snpMatrix [![external link](external.png)](http://www.bioconductor.org/packages/2.3/bioc/html/snpMatrix.html) | 3.2 | 5.1 | days | | eMap [![external link](external.png)](http://www.bios.unc.edu/~wsun/software.htm) | 17.8 | N/A | days | | FastMap [![external link](external.png)](http://comptox.unc.edu/resources.html) | 10.3 | N/A | hours |    **Info:** [Details of the testing procedure](PerformanceTesting.html).     **Fact:** [Matrix eQTL results match those by other software](CompareResults.html).
