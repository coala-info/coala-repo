Welcome to the MetaSBT wiki!

MetaSBT is a scalable framework for automatically indexing microbial genomes and accurately characterizing metagenome-assembled genomes with Sequence Bloom Trees.

This is the content of the wiki:

* [Getting started](https://github.com/cumbof/MetaSBT/wiki/Getting-started)
  + [Installing `MetaSBT`](https://github.com/cumbof/MetaSBT/wiki/Getting-started#installing-metasbt)
  + [Dependencies](https://github.com/cumbof/MetaSBT/wiki/Getting-started#dependencies)
* [Available features](https://github.com/cumbof/MetaSBT/wiki/Available-features)
  + [`db`: list and retrieve public databases](https://github.com/cumbof/MetaSBT/wiki/Available-features#1-db-list-and-retrieve-public-databases)
  + [`index`: create a baseline with reference genomes](https://github.com/cumbof/MetaSBT/wiki/Available-features#2-index-create-a-baseline-with-reference-genomes)
  + [`kraken`: build a custom Kraken database](https://github.com/cumbof/MetaSBT/wiki/Available-features#3-kraken-build-a-custom-Kraken-database)
  + [`pack`: pack a database into a compressed tarball](https://github.com/cumbof/MetaSBT/wiki/Available-features#4-pack-pack-a-database-into-a-compressed-tarball)
  + [`profile`: characterize genomes and metagenome-assembled genomes](https://github.com/cumbof/MetaSBT/wiki/Available-features#5-profile-characterize-genomes-and-metagenome-assembled-genomes)
  + [`sketch`: sketch genomes into bloom filters](https://github.com/cumbof/MetaSBT/wiki/Available-features#6-sketch-sketch-genomes-into-bloom-filters)
  + [`summarize`: summarize the content of a database](https://github.com/cumbof/MetaSBT/wiki/Available-features#7-summarize-summarize-the-content-of-a-database)
  + [`test`: run unit tests](https://github.com/cumbof/MetaSBT/wiki/Available-features#8-test-run-unit-tests)
  + [`unpack`: install a database](https://github.com/cumbof/MetaSBT/wiki/Available-features#9-unpack-install-a-database)
  + [`update`: update a database with new genomes](https://github.com/cumbof/MetaSBT/wiki/Available-features#10-update-update-a-database-with-new-genomes)
* [Retrieving genomes from NCBI GenBank](https://github.com/cumbof/MetaSBT/wiki/Retrieving-genomes-from-NCBI-GenBank)
