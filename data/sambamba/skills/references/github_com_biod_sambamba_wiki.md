About Sambamba is a set of command-line tools for working with SAM/BAM file formats which are used for storing next generation sequencing (NGS) datasets. A number of functions are production ready, see sambamba tool documentation Author The author of the project is Artem Tarasov a.k.a. lomereiter. In case you have any questions or suggestions, you can either create an issue on Github or contact me via e-mail lomereiter at gmail dot com . Acknowledgements This started as a Google Summer of Code project, and I wish to thank Open Bioinformatics Foundation, and all Bio* communities, for the initial project idea and their continuous support.
