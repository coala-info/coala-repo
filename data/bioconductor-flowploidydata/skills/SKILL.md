# bioconductor-flowploidydata

## Overview

The `flowPloidyData` package provides a collection of flow cytometry data files in LMD format, generated by a Beckman-Coulter Gallios flow cytometer. These files represent plant leaf tissues co-chopped with internal standards (like soybean or tomato). The dataset includes a variety of data qualities to help users understand how to handle both high-quality and problematic flow cytometry results within the `flowPloidy` ecosystem.

## Loading and Using Data

The primary purpose of this package is to provide the file paths to the raw data for use in `flowPloidy` or `flowCore` analysis.

### Accessing File Paths

Use the `flowPloidyFiles()` function to retrieve a character vector of the full paths to the included LMD files.

```r
library(flowPloidyData)

# Get all file paths
files <- flowPloidyFiles()

# View the first few file paths
head(files)
```

### Typical Workflows

#### 1. Batch Processing with flowPloidy
The most common use case is passing these files directly to the `batchFlowHist` function from the `flowPloidy` package to create flow histogram objects.

```r
library(flowPloidy)
library(flowPloidyData)

# Process all example files
# Note: flowPloidy must be installed to use batchFlowHist
fh_list <- batchFlowHist(flowPloidyFiles())
```

#### 2. Single File Inspection with flowCore
You can also use the paths to read individual files using the standard Bioconductor `flowCore` infrastructure.

```r
library(flowCore)
library(flowPloidyData)

# Read the first example file
fcs_data <- read.FCS(flowPloidyFiles()[1])
```

## Tips for Analysis

- **Peak Identification**: In these samples, you should expect to see a G1 peak for the sample and a G1 peak for the standard. G2 peaks may also be present at approximately double the intensity.
- **Data Quality**: This package intentionally includes "bad" data. If a file fails to produce a clean histogram, it may be one of the low-quality examples intended for troubleshooting practice.
- **Standards**: Common standards used in these files include soybean and tomato.
