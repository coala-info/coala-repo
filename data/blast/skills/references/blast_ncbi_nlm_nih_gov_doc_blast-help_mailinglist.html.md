# Mailing List[¶](#mailing-list "Link to this heading")

NCBI has created a mailing list for BLAST news.This list is intended for distribution of short messages
announcing updates and new features and to give advance notice about upcoming changes in the NCBI BLAST
service.

All postings to this list will come from the NCBI BLAST Help desk.

If you want to subscribe to this list please go to <https://www.ncbi.nlm.nih.gov/mailman/listinfo/blast-announce>

Under the heading **Subscribing to ‘Blast-announce’** fill in your e-mail address and choose a password for your account. The password will be
used for accessing the BLAST-Announce archives when they become available. You will need to verify this password and
then click the **Subscribe** button.

Once you subscribe to the mailing list you will be able to receive BLAST news via e-mail. If you have any questions
please [Write to the Help Desk](https://support.nlm.nih.gov/knowledgebase/category/?id=CAT-01239)

See the Blast-announce archives [here](https://www.ncbi.nlm.nih.gov/mailman/pipermail/blast-announce/)
