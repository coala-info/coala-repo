Integrate an hard fork from cameo (v0.13.6) named cameobrs to add features. Installation conda install -c bioconda straindesign Usage Define the best combination of genes deletion to optimize a target python -m straindesign simulate-deletion \ [input files] --input-model-file &lt; SBML file &gt; --input-pathway-file &lt; SBML file &gt; --input-medium-file &lt; CSV/TSV file &gt; [parameters] --biomass-rxn-id &lt; id reaction, str &gt; --target-rxn-id &lt; id reaction, str &gt; --substrate-rxn-id &lt; id reaction, str &gt; [output file] --output-file &lt; CSV/TSV file &gt; Delete genes in a model python -m straindesign reduce-model \ [input files] --input-model-file &lt; SBML file &gt; --input-straindesign-file &lt; CSV file &gt; and/or --input-gene-str &lt; id gene, str &gt; [parameters] --parameter-strategy-str &lt; yield-max, gene-max, gene-min &gt; [output file] --output-file-sbml &lt; SBML file &gt; You can provide a list of genes to delete in the model or the file produced by the command simulate-deletion . If this file is provided, the combination of genes is choosen among three strategies: yield-max: genes are sorted by the best yield gene-max: the combination of the maximum number of genes gene-min: the combination of the minimum number of genes Produce a pareto plot python -m straindesign analyzing-model \ [input files] --input-model-file &lt; SBML file &gt; --input-medium-file &lt; CSV/TSV file &gt; --input-pathway-file &lt; SBML file &gt; [parameters] --biomass-rxn-id &lt; id reaction, str &gt; --target-rxn-id &lt; id reaction, str &gt; --substrate-rxn-id &lt; id reaction, str &gt; [output file] --output-pareto-png &lt; PNG file &gt; You can provide an heterologous pathway to implement the metabolic pathway producing the targeted compound represented by the target-rxn-id , the reaction which produces this compound. The substrate-rxn-id argument lets you to choose the main carbon source.
