```text
usage: arborator [-h] --profile PROFILE --metadata METADATA [--config CONFIG]
                 --outdir OUTDIR [--partition_col PARTITION_COL]
                 [--id_col ID_COL] [--outlier_thresh OUTLIER_THRESH]
                 [--min_members MIN_MEMBERS] [--only_report_labeled_columns]
                 [--count_missing] [--missing_thresh MISSING_THRESH]
                 [--distm DISTM] [--skip_qc] [--thresholds THRESHOLDS]
                 [--delimiter DELIMITER] [--method METHOD]
                 [--tree_distances {patristic,cophenetic}] [--force]
                 [--sort_matrix] [--n_threads N_THREADS] [--version]

Arborator, an aggregate tool for producing summary reports of genetic distances within groups v. 1.2.2

options:
  -h, --help            show this help message and exit
  --profile PROFILE, -p PROFILE
                        Allelic profiles (default: None)
  --metadata METADATA, -r METADATA
                        Matched metadata for samples in the allele profile
                        (default: None)
  --config CONFIG, -c CONFIG
                        Configuration json (default: None)
  --outdir OUTDIR, -o OUTDIR
                        Result output files (default: None)
  --partition_col PARTITION_COL, -a PARTITION_COL
                        Metadata column name for aggregating samples (default:
                        None)
  --id_col ID_COL, -i ID_COL
                        Sample identifier column (default: None)
  --outlier_thresh OUTLIER_THRESH
                        Threshold to flag outlier comparisons within a group
                        (default: 100)
  --min_members MIN_MEMBERS, -m MIN_MEMBERS
                        Minimum number of members to perform clustering
                        (default: 2)
  --only_report_labeled_columns
                        Only report labeled columns (default: False)
  --count_missing, -n   UNUSED: Count missing as differences (default: False)
  --missing_thresh MISSING_THRESH
                        UNUSED: Maximum percentage of missing data allowed per
                        locus (0 - 1) (default: None)
  --distm DISTM         UNUSED: Distance method raw hamming or scaled
                        difference [hamming, scaled] (default: None)
  --skip_qc, -s         UNUSED: Skip QA/QC steps (default: False)
  --thresholds THRESHOLDS, -t THRESHOLDS
                        thresholds delimited by , (default: 100)
  --delimiter DELIMITER, -d DELIMITER
                        UNUSED: delimiter desired for nomenclature code
                        (default: None)
  --method METHOD, -e METHOD
                        cluster method [single, complete, average] (default:
                        average)
  --tree_distances {patristic,cophenetic}
                        Defines how distances in distance matrices are
                        interpretted by GAS and represented in the output tree
                        (Newick file). Use "patristic" to interpret distances
                        in the matrix as sum of branch lengths between
                        clusters or leaves, and "cophenetic" to interpret
                        distances in the matrix as the minimum distance two
                        clusters or leaves need to be in order to be grouped
                        into the same cluster. (default: patristic)
  --force, -f           Overwrite existing directory (default: False)
  --sort_matrix         Sorts the samples in the distance matrix generated by
                        GAS. The order of sample rarely has an effect on the
                        assigned cluster labels and sorting them ensures the
                        same inputs always generate the same outputs.
                        (default: False)
  --n_threads N_THREADS
                        CPU Threads to use (default: 1)
  --version, -V         show program's version number and exit
```
