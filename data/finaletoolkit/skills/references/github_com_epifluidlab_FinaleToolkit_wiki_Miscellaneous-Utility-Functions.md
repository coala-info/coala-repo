## `filter-file`

### When using a BED/BAM/CRAM file multiple times for FinaleToolkit functions, it can be helpful to pre-filter the file. Let's filter our BAM file to only contain fragments of length 50bp to 350bp (`-min 50`, `-max 350bp`) and MAPQ 30 or greater (`-q 30`). We will output (`-o`) the result to BH01.hg19.mdups.22.filtered.bam, using one worker (`-w 1`) and verbose mode (`-v`).

```
# You need to have samtools installed in your environment to be able to run this successfully.
finaletoolkit filter-file \
    BH01.hg19.mdups.22.bam \
    -q 30 \
    -min 50 \
    -max 350 \
    -o BH01.hg19.mdups.22.filtered.bam \
    -w 1 \
    -v
```

## `agg-bw`

### Let's say that we want to combine by taking the mean (`--mean`) of the cleavage-profile interval data that we generated in [`cleavage-profile`](https://github.com/epifluidlab/FinaleToolkit/wiki/Cleavage-Profile#cleavage-profile), and output (`-o`) to tss.wig.

```
finaletoolkit agg-bw \
    tss.hg19.2kb.22.bw \
    tss.hg19.intervals.22.bed \
    -o tss.wig \
    --mean \
    -v
```

#### Visualizing this output WIG file should look something like this. Because of the fair amount of noise in the data, this visualization has a running median filter of size 50.

```
import numpy as np
import matplotlib.pyplot as plt

data = np.genfromtxt("tss.wig", skip_header=1)
x = np.linspace(-2000, 1999, 4000)

# Apply moving average
window_size = 50  # Adjust window size for smoothing
smoothed_data = np.convolve(data, np.ones(window_size)/window_size, mode='valid')

# Create a single subplot
fig, axes = plt.subplots(figsize=(8, 4), dpi=300)

# Set the x-ticks and labels
xticks = [-2000, -1000, 0, 1000, 2000]
xtick_labels = ['-2kb', '-1kb', '0', '1kb', '2kb']

# Plot the smoothed data
axes.plot(x[(window_size-1)//2 : -(window_size-1)//2], smoothed_data, color="#C19DB9", linewidth=1.5)
axes.axvline(0, color="black", linewidth=0.8)
axes.set_xlabel("Distance to TSS motif (bp)", fontsize=12)
axes.set_ylabel("Average Cleavage Proportion", fontsize=12)
axes.set_xticks(xticks)
axes.set_xticklabels(xtick_labels, fontsize=12)

# Hide top and right spines
axes.spines['top'].set_visible(False)
axes.spines['right'].set_visible(False)

# Adjust layout
plt.tight_layout()

# Show the plot
plt.show()
```
