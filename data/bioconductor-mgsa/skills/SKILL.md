# bioconductor-mgsa

## Overview

Use the Bioconductor R package **mgsa** for: The package mgsa implements MGSA and tools to use MGSA together with the Gene Ontology.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("mgsa")
```
