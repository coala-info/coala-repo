Assembly

Mon, 01 Jan 0001 00:00:00 +0000
https://edgardomortiz.github.io/captus.docs/assembly/
Captus&rsquo; first module is called captus\_assembly but can also be executed as simply captus and it aims to create marker alignments across samples starting from the raw sequencing reads of each sample (or alternatively from previously cleaned reads or even previously assembled reads).
To accomplish this the module has four commands: clean, assemble, extract, and align which are tipically run in that order:
1. clean This command will perform adaptor trimming (plus poly-A trimming if you are cleaning RNAseq reads) followed by quality trimming using bbduk.
