This repo will be archived and go to read-only mode. Introduction run_dbcan is the standalone version of the dbCAN3 annotation tool for automated CAZyme annotation. This tool, known as run_dbcan , incorporates HMMER, Diamond, and dbCAN_sub for annotating CAZyme families, and integrates Cazyme Gene Clusters (CGCs) and substrate predictions. For usage discussions, visit our issue tracker . To learn more, read the dbcan doc . If you're interested in contributing, whether through issues or pull requests, please review our contribution guide.
