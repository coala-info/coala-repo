Available options: usage: pMMRCalculator.py [-h] [-i &lt;INPUT FILES PREFIX&gt;] [-o &lt;OUTPUT FILE&gt;] [-s &lt;INPUT FILES SUFFIX&gt;] [-v] [-j] Calculate the pairwise mismatch rate of genotyped between all individuals in the input eigenstrat dataset. Available options: -h, --help show this help message and exit -i &lt;INPUT FILES PREFIX&gt;, --Input &lt;INPUT FILES PREFIX&gt; The desired input file prefix. Input files are assumed to be &lt;INPUT PREFIX&gt;.geno, &lt;INPUT PREFIX&gt;.snp and &lt;INPUT PREFIX&gt;.ind . -o &lt;OUTPUT FILE&gt;, --Output &lt;OUTPUT FILE&gt; The desired output file name. Omit to print to stdout. -s &lt;INPUT FILES SUFFIX&gt;, --Suffix &lt;INPUT FILES SUFFIX&gt; The desired input file suffix. Input files are assumed to be &lt;INPUT PREFIX&gt;.geno&lt;INPUT SUFFIX&gt;, &lt;INPUT PREFIX&gt;.snp&lt;INPUT SUFFIX&gt; and &lt;INPUT PREFIX&gt;.ind&lt;INPUT SUFFIX&gt; . -v, --version Print the version of the script and exit. -j, --json Create additional json formatted output file named &lt;OUTPUT FILE&gt;.json . [Default: 'pmmrcalculator_output.json'] Usage example: pMMRCalculator.py -i test.input -o test.output.txt Example output: Ind1 Ind2 nSNPs nMismatch pMismatch Indiviudal1 Individual2 585385 137736.0 0.23529 Indiviudal1 Individual3 585199 140524.5 0.24013 Indiviudal1 Individual4 584162 140474.0 0.24047 Indiviudal2 Individual3 583507 140741.5 0.24120 Indiviudal2 Individual4 586613 141040.0 0.24043 Indiviudal3 Individual4 586005 140373.0 0.23954 The output will include a header and five coloumns. These columns contain the ID of the two individuals in each pairwise comparison, the number of overlapping SNPs between the two individuals, the cumulative mismatch proportions across all SNPs, and the pairwise mismatch rate between the individuals.
