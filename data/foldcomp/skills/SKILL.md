# foldcomp

## Overview

Foldcomp is a specialized compression library and format designed to handle the massive volume of data generated by protein structure prediction efforts. By encoding structures using torsion angles rather than raw 3D coordinates, it achieves a compression ratio of approximately 13 bytes per residue—roughly an order of magnitude smaller than traditional formats. This skill provides the necessary commands and patterns to perform high-speed batch compression, manage structural databases, and programmatically access structural data via Python.

## Command Line Interface (CLI) Patterns

### Compression and Decompression
Foldcomp supports both single-file and batch processing of directories or tarballs.

*   **Compress a single file:**
    `foldcomp compress <input.pdb|input.cif> [<output.fcz>]`
*   **Batch compress a directory to a database:**
    `foldcomp compress -t <threads> -d <input_directory> <output_db_name>`
*   **Decompress a database to a directory:**
    `foldcomp decompress -t <threads> <input_db> <output_directory>`
*   **Decompress a specific subset from a database:**
    `foldcomp decompress --id-list <ids.txt> <input_db> <output_directory>`

### Data Extraction and Validation
Quickly retrieve metadata without full decompression.

*   **Extract pLDDT scores:**
    `foldcomp extract --plddt <input.fcz> <output.fasta>`
*   **Extract amino acid sequences:**
    `foldcomp extract --amino-acid <input.fcz> <output.fasta>`
*   **Verify file integrity:**
    `foldcomp check <input.fcz>`
*   **Calculate RMSD between two structures:**
    `foldcomp rmsd <struct1.pdb> <struct2.pdb>`

## Python API Usage

The Python interface is the preferred method for downloading prebuilt databases and integrating structural data into data science workflows.

### Downloading Prebuilt Databases
Foldcomp provides a helper to fetch major datasets (e.g., AlphaFoldDB, ESMAtlas).
```python
import foldcomp
# Example: Download AlphaFoldDB Swiss-Prot
foldcomp.setup('afdb_swissprot_v4')
```

### Programmatic Data Access
You can iterate through databases or extract specific geometric data directly into Python dictionaries.
```python
import foldcomp

# Accessing specific structural data
with open("protein.fcz", "rb") as f:
    fcz_data = f.read()
    # Returns dict with phi, psi, omega, residues, coordinates, etc.
    data = foldcomp.get_data(fcz_data)

# Iterating through a database
with foldcomp.open("path/to/database") as db:
    for name, pdb_content in db:
        print(f"Processing {name}")
```

## Expert Tips and Best Practices

*   **Thread Optimization:** When performing batch compression on large directories, always use the `-t` flag to match your available CPU cores for significant speedups.
*   **Database Format:** Use the `-d` (database) or `-z` (tar) flags during compression to avoid creating millions of small files on your filesystem, which can degrade performance.
*   **Single Chain Limitation:** Note that Foldcomp currently only supports the compression of single-chain PDB files.
*   **pLDDT Precision:** Use the `-p` or `--plddt-digits` flag (1-4) during extraction to control the precision of the output scores.
*   **Memory Efficiency:** When working with millions of entries, use the Python `foldcomp.open()` context manager to stream structures rather than loading entire datasets into memory.
