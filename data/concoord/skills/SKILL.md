# concoord

## Overview
CONCOORD is a specialized tool for sampling the conformational space of proteins. It operates on the principle that internal constraints and geometric barriers restrict protein dynamics to a limited number of collective degrees of freedom. By converting a starting PDB structure into a set of interatomic distance bounds, CONCOORD can rapidly generate thousands of alternative structures that satisfy these constraints. This makes it a powerful tool for predicting protein function related to large-scale motions and for generating structural ensembles for docking or NMR studies.

## Command Line Usage
The software consists of two primary programs that must be run sequentially.

### 1. Generating Distance Constraints (`dist`)
The `dist` program analyzes the starting structure to identify covalent bonds and non-bonded interactions, assigning geometric freedom based on interaction strength.

```bash
dist -p input.pdb -o constraints.dat
```
* `-p`: Input PDB file.
* `-o`: Output data file containing distance bounds.

### 2. Generating Conformations (`disco`)
The `disco` program uses the bounds generated by `dist` to produce new structures via iterative coordinate correction.

```bash
disco -p constraints.dat -n 50 -o output.pdb
```
* `-p`: Input data file (from `dist`).
* `-n`: Number of structures to generate.
* `-o`: Output filename.

## Expert Options and Best Practices
* **Robust Generation**: Use the `-dyn` flag in `disco` for dynamic adaptation of tolerances. This is recommended for complex structures where standard convergence might fail.
* **Structure Quality**: Use the `-reg` option in `disco` to apply structure regularization to the generated conformations, ensuring better geometry.
* **NMR Ensembles**: Use the `-on` flag to write output in NMR-style PDB format (multiple models separated by `MODEL` and `ENDMDL` tags in a single file).
* **Collision Avoidance**: Use `-rc` and `-l` to provide finer control over de-bumping during the structure generation process.
* **Convergence Control**: Use the `-con` option in `disco` to adjust convergence checks if the default settings are too strict or too loose for your specific protein.
* **Environment Setup**: If installed via source, ensure you source the `CONCOORDRC` file to add the binaries to your PATH and define the location of library files.
