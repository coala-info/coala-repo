# pangbank-api

## Overview
This skill provides guidance for managing the PanGBank database, which stores pangenome collections generated by PPanGGOLiN. It focuses on using the `pangbank_db` command-line interface for administrative tasks—such as importing new data releases, listing existing collections, and managing database schemas—as well as configuring the FastAPI-based REST server for programmatic exploration of pangenome data.

## Environment Configuration
The tool relies on specific environment variables to locate the database and associated data files. Always ensure these are set before running CLI commands.

- `PANGBANK_DB_PATH`: The absolute path to your SQLite database file (e.g., `/path/to/database.sqlite`).
- `PANGBANK_DATA_DIR`: The root directory containing pangenome data and Mash files.

```bash
export PANGBANK_DB_PATH="/path/to/database.sqlite"
export PANGBANK_DATA_DIR="/path/to/pangenome_data/"
```

## Database Management (pangbank_db)
The `pangbank_db` utility is the primary tool for data ingestion and maintenance.

### Adding a Collection Release
To import a new set of pangenomes, you must provide a JSON configuration file.
```bash
pangbank_db add-collection-release collection_release.json
```

**Expert Tip: Path Handling in JSON**
The tool distinguishes between relative and absolute paths within the JSON configuration:
- **Relative Paths**: `pangenomes_directory` and `mash_sketch` must be relative to the `PANGBANK_DATA_DIR`.
- **Absolute Paths**: `taxonomy.file`, `genome_sources[].file`, and `genome_metadata_sources[].file` must be full absolute paths.

### Maintenance Commands
- **List Collections**: View all pangenome collections currently in the database.
  ```bash
  pangbank_db list-collection
  ```
- **Delete a Release**: Remove a specific version of a collection.
  ```bash
  pangbank_db delete-collection <collection_name> --release-version <version>
  ```

## API Server Deployment
If you need to serve the pangenome data via a RESTful interface, use the FastAPI integration.

1. **Installation**: Ensure the API dependencies are installed: `pip install pangbank-api[fastapi]`.
2. **Execution**:
   ```bash
   fastapi dev pangbank_api/main.py
   ```

## Schema Migrations
PanGBank uses Alembic for database versioning. If the database schema needs updating or initialization:

- **Apply Migrations**: Bring the database to the latest version.
  ```bash
  alembic upgrade head
  ```
- **Rollback**: Revert the last change if an error occurs.
  ```bash
  alembic downgrade -1
  ```
