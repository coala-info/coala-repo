# Legacy syntax (panacus v0.3)

[Jump to bottom](#wiki-pages-box)

Peter Heringer edited this page Nov 11, 2025
·
[1 revision](/codialab/panacus/wiki/Legacy-syntax-%28panacus-v0.3%29/_history)

Panacus v0.3 used a different YAML syntax where each analysis contained a graph parameter naming the GFA file to use.

```
- !Hist
  graph: ../simple_files/simple_graphs/t_groups.gfa
- !Hist
  name: testing this
  count_type: Bp
  graph: ../simple_files/simple_graphs/t_group2.gfa
  display: false
```
