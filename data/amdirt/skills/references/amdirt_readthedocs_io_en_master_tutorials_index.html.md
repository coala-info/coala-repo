# Tutorials[](#tutorials "Link to this heading")

Introductory tutorials for sub-commands can be found below in text and/or video format.

All videos can be also found on the [SPAAM Community Youtube channel](https://www.youtube.com/%40spaam-community).

All tutorials assume you have already installed amdirt following the installation instructions ([Install](../README.html)).

Note that tutorials currently exist for user-facing commands, sub-commands such as validate and autofill are generally executed by users. For these please see the respective ([How Tos](../how_to/index.html)) page.
