You should have given your experiment the same name in MinKNOW . You should have given your metadata file this name, and put it in your workspace metadata folder ( &lt;path/to/your/workspace&gt;/metadata/&lt;expt_name&gt;.csv|xlsx ). The dashboard will open in a browser window on your computer. Made with Material for MkDocs
