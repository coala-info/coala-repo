# banner

## Overview

BANNER is a machine learning tool designed to interpret and classify microbiome sketches generated by HULK. It utilizes a Random Forest Classifier to provide rapid, sketch-based classification of metagenomic data. By training on a matrix of labeled sketches, BANNER can identify patterns in microbiome composition and predict the status of unknown samples, either from static files or via a live data stream.

## Command Line Usage

BANNER follows a simple two-step workflow: training a model and using that model for prediction.

### Training a Model
To create a classifier, you must provide a matrix of labeled HULK sketches.

```bash
banner train -m hulk-banner-matrix.csv -o banner.rfc
```
*   `-m`: Path to the input matrix (CSV format) containing labeled HULK sketches.
*   `-o`: The filename for the resulting Random Forest Classifier model.

### Predicting Labels
Once a model is trained, you can predict labels for new samples. BANNER is designed to work seamlessly with HULK's streaming output.

**Streaming Prediction:**
```bash
hulk sketch -f mystery-sample.fastq --stream -p 8 | banner predict -m banner.rfc
```

**Static Prediction:**
```bash
banner predict -m banner.rfc -i mystery-sample.sketch
```

## Expert Tips and Best Practices

*   **Binary Classification:** Currently, BANNER is optimized for and primarily supports 2-label classification (e.g., "Healthy" vs. "Diseased"). Ensure your training matrix reflects this binary structure for best results.
*   **Integration with HULK:** BANNER is most effective when used as a downstream consumer of `hulk sketch`. Using the `--stream` flag in HULK allows for real-time classification of sequencing data as it is processed.
*   **Matrix Preparation:** The input matrix for training should be a CSV where rows represent samples and columns represent the sketch features, with a final column for the label.
*   **Environment Requirements:** BANNER requires a Python 3.6+ environment with `scikit-learn`, `numpy`, `scipy`, and `pandas` installed. It is recommended to install via Bioconda to ensure all dependencies are correctly resolved.



## Subcommands

| Command | Description |
|---------|-------------|
| banner predict | Collects sketches from STDIN and classifies them using a RFC |
| banner train | Train takes a banner-matrix file from hulk and uses it to train a Random Forest Classifier |
