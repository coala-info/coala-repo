# bioconductor-lintind

## Overview
LinTInd (Lineage Tracing by Indel) is an R package designed to process and analyze data from synthetic lineage tracing systems. It focuses on identifying unique "scars" (indels) generated by CRISPR/Cas9 at defined target sites. The package provides tools for data preprocessing, indel frequency calculation, cell-to-cell similarity scoring based on shared mutations, and phylogenetic tree reconstruction to infer cellular relationships.

## Core Workflow

### 1. Data Input and Preprocessing
LinTInd typically starts with a data frame containing read sequences or pre-aligned indel information.

```r
library(LinTInd)

# Read your alignment data (typically requires columns: ID, Sequence/Indel info)
# Example: Generating a target sequence reference
target <- "GGTGTGCGTTAGGGCAATTCCTGG" 
```

### 2. Indel Extraction and Annotation
The package identifies insertions and deletions relative to the target site.

```r
# Extract indels from sequences
# strings: vector of sequences; target: reference sequence
indels <- IndelExtract(strings, target)

# Formating for downstream analysis
# This creates a standardized matrix of indels per cell/read
indel_info <- IndelForm(indels)
```

### 3. Lineage Reconstruction
LinTInd uses the distribution of indels to calculate similarity between cells and build lineage trees.

```r
# Calculate the distance matrix based on shared indels
dist_matrix <- IndelDist(indel_info)

# Generate a phylogenetic tree (hclust or neighbor-joining)
tree <- IndelTree(dist_matrix)

# Plot the tree with indel metadata
PlotTree(tree, indel_info)
```

### 4. Visualization and Statistics
Analyze the diversity and frequency of the generated scars.

```r
# Plot the distribution of indel types and lengths
PlotIndel(indel_info)

# Calculate the Shannon entropy to evaluate barcode diversity
diversity <- BarcodeEntropy(indel_info)
```

## Key Functions
- `IndelExtract()`: Identifies mutations by comparing query sequences to the reference target.
- `IndelForm()`: Converts raw indel strings into a structured format suitable for tree building.
- `IndelDist()`: Computes a distance matrix. It accounts for the likelihood of independent identical mutations (homoplasy).
- `IndelTree()`: Wraps tree construction methods to produce a phylo object.
- `PlotTree()`: Visualizes the reconstructed lineage alongside the indel patterns for each cell.

## Tips for Success
- **Target Sequence**: Ensure the `target` sequence provided to `IndelExtract` matches the exact expected sequence between the primers used in your library prep.
- **Filtering**: Before tree construction, filter out cells with very few reads or targets with no indels to reduce noise in the lineage reconstruction.
- **Large Datasets**: For single-cell datasets with thousands of cells, use the `IndelDist` function carefully as the distance matrix calculation is $O(N^2)$.
