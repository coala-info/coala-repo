PyScanFCS interactively extracts the fluctuating fluorescence signal from such measurements and applies a multiple-tau algorithm. The obtained correlation curves can be evaluated using PyCorrFit . Getting started Installation Installers for PyScanFCS are available at the release page . Alternatively, install with pip install pyscanfcs . Documentation A detailed documentation including an explanation of the graphical user interface and available model functions is available as a PDF file .
