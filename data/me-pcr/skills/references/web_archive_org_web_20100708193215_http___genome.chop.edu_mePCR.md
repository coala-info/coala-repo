TIMESTAMPS The Wayback Machine - https://web.archive.org/web/20100708193215/http://genome.chop.edu/mePCR/ &nbsp; me-PCR _ A refined ultrafast algorithm for identifying sequence-defined genomic elements Multithreaded Electronic PCR Last updated 2008/02/18 This is the home of me-PCR, descended from NCBI's e-PCR. The manual for the latest stable version is here . NCBI's e-PCR has improved greatly over the years, and in general, you should use e-PCR rather than me-PCR. The me-PCR package includes: Manual Source code Prior to 1.0.6, packages contained ready-to-use binaries for AIX, OS X, Solaris, Linux, and Windows. I can create packages for these platforms upon request.
