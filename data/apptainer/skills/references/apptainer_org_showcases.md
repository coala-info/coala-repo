## Showcases

Learn some knowledge from those who have their hands on the wheel with Apptainer. Learn their ways, techniques, or even get some enlightenment about where to use Apptainer.

### [Apptainer 1.1.0 Has Been Released](https://youtu.be/lZENN9LVqY8)

A discussion of the new features and updated security posture of the groundbreaking Apptainer 1.1.0 release.

Ctrl IQ Weekly Webinar Series

### [Singularity: Containers for Science, Reproducibility, and HPC](https://youtu.be/lZENN9LVqY8)

This talk explores how Singularity combines software packaging models with minimalistic containers to create very lightweight application bundles.

2017 HPC Advisory Council Stanford Conference

### [Container Education Series: Building Apptainer Containers](https://youtu.be/DmrkxzW_F1Q)

A deep dive starting generally and venturing into the obscure and dark corners of container building with Apptainer.

Ctrl IQ Weekly Webinar Series

### [Building Containers with Singularity](https://www.youtube.com/watch?v=nrgO3Q8-6hQ)

Learn how to use Singularity to create containers both in interactive mode and using definition files to automate the build.

Daniel Persson

### [Introduction to Singularity: Containers for Scientific and High-Performance Computing](https://www.youtube.com/watch?v=vEjLuX0ClN0)

In this webinar, we provide an overview of Singularity and how you might incorporate the use of containers in your own research. We also show you how to access and use some of the containerized applications that we make available to users on the XSEDE systems like Comet and Expanse at SDSC.

San Diego Supercomputer Center

### [Singularity: Containers for High-Performance Computing](https://www.youtube.com/watch?v=a2Yh0t1Azd0)

This session will provide an overview of the main features and benefits of Singularity, as well as a demonstration of how to run Singularity on WestGrid and Compute Canada systems, how to access filesystems from the container, and how to convert Docker containers to Singularity applications.

Westgrid

### [Run NVIDIA GPU Cloud Containers in Singularity](https://www.youtube.com/watch?v=iOLVqqHQsBU)

This video covers installing and configuring Singularity, pulling and saving NGC containers and running NGC containers in an HPC environment.

NVIDIA Developer

### [Singularity Introduction](https://www.youtube.com/watch?v=29NLgM9fnh4)

Michael Bauer provides and introduction into the HPC container runtime Singularity.

ISC High Performance

### [Containers Using Singularity on HPC](https://www.youtube.com/watch?v=2Aap7pstFog)

In this video from the 2018 Swiss HPC Conference, Abhinav Thota, from Indiana University presents: Containers Using Singularity on HPC.

2018 Swiss HPC Conference

### [Easy-Build Singularity Containers](https://www.youtube.com/watch?v=34SFF5Ow7Og)

In this hands on tutorial we will build 2 containers: a simple one to give users an understanding of how the set of scripts are working and how we can build for example Debian or CentOS containers without much knowledge of the background mechanisms. In the second, more advanced part, we show how to open up the containers and give users a chance to install their bespoke pipelines and use the power of the containers to run that on virtually any suitable platform.

EUM '21

### [Singularity in High Performance Computing (HPC)](https://www.youtube.com/watch?v=z-RtotX0i_0)

In this webinar, we cover the basics of using Singularity and best practices of using it on Compute Canada HPC systems. We then discuss the common use cases and our experience with Singularity, and review the recent developments and changes in the newest versions of Singularity 3. The webinar is intended for everyone with basic skills in the Linux command line.

Westgrid

### [Singularity: Simple, Secure Containers for HPC](https://www.youtube.com/watch?v=cWncCvFM2ak)

Singularity is the most widely used container solution in high-performance computing (HPC). Instead of a layered filesystem, a Singularity container is stored in a single file. This simplifies the container management lifecycle and facilitates features such as image signing and verification to produce trusted containers.

Computational Biomedicine

### [Benchmarking MPI Applications in Singularity Containers on Traditional HPC and Cloud Infrastructures](https://www.youtube.com/watch?v=3Q-IS6MFgb4)

In this presentation, Benchmarking over traditional HPC is presented.

HPC-CH

### [Software Containers with Singularity](https://www.youtube.com/watch?v=xty42A05Wg0)

An overview of software containers and how to use Singularity to create and run containers for research and high-performance computing tasks.

USC Advanced Research Computing
