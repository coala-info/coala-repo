# bioconductor-pairedgsea

## Overview

Use the Bioconductor R package **pairedGSEA** for: The package allows you to store intermediate results for further investiation, if desired. pairedGSEA comes with a wrapper function for running an Over-Representation Analysis (ORA) and functionalities for plotting the results.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("pairedGSEA")
```
