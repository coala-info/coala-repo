Improved handling of rare telomeres with non-exact matches to canonical telomere sequence within bona fide telomere sequence. Previous versions did not count these gaps as telomere, current version does. 2. Improved sub/telomere boundary identification by incorporating a scanning window which calculates the telomere density along the read and ensures that sub/telomere boundaries are not prematurely called when there is a "gap" consisting of 1 basepair mismatched telomeric sequence at the subtelomeric boundary. Deprecated support for R9, only recommend R10 sequencing for telomere measurement from now on as this is the chemistry ONT will continue to support and produces better data anyhow. 4.
