Status: Open. #819 In SnapKit/SnapKit; · Purool opened on Sep 11, 2024 ## SnapKit 3.0 Migration Guide SnapKit 3.0 has many breaking changes from earlier versions. You can read the [notes on releases](https://github.com/SnapKit/SnapKit/releases) for more details. 1. Upgrade SnapKit installation 2. Find all &lt;code&gt;snp_&lt;/code&gt; and replace with &lt;code&gt;snp.&lt;/code&gt; 3. Find all &lt;code&gt;equalTo(UIEdgeInsets)&lt;/code&gt; and ensure any right or bottom inset values are no longer inverted 4. Build and fix deprecated APIs 5. Test every screen of UI to ensure visual compatibility If you find any issues with migration please open an issue. ### Known Quirks 1. &lt;code&gt;make.center.equalTo(0)&lt;/code&gt; now positions a view at 0,0 in it&#x27;s superview rather than the center, use &lt;code&gt;make.center.equalToSuperview&lt;/code&gt; instead. 2. &lt;code&gt;updateConstraints&lt;/code&gt; will throw a fatal error if it generates a *new* constraint that did not exist before. Instead use &lt;code&gt;makeConstraints&lt;/code&gt; to add additional constraints.
