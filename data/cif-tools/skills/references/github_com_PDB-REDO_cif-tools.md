Requirements The tools are based on libcif++ and the code is written in modern C++ so you need a compiler capable of handling C++17 code. Building Make sure you install libcif++ first before building. You will also need to install libcfp . After that, building should be as easy as typing: git clone https://github.com/PDB-REDO/cif-tools.git cd cif-tools mkdir build cd build cmake .. -DCMAKE_BUILD_TYPE=Release cmake --build . cmake --install .
