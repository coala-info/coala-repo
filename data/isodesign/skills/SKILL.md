# isodesign

## Overview
IsoDesign is a specialized scientific tool for identifying the most effective labeling strategies in 13C-fluxomics. It helps researchers balance the trade-off between high flux resolution (at reaction, pathway, and network levels) and the financial cost of labeled substrates. The tool supports both isotopic stationary and non-stationary experiment designs and leverages parallel computing for performance.

## Installation and Environment Setup

### Recommended Conda Installation
Install IsoDesign and its core dependencies using the bioconda and conda-forge channels:
```bash
conda install isodesign -c conda-forge -c bioconda
```

### Pip Installation and R Dependencies
If using pip, a secondary step is required to ensure the underlying R-based simulation engine (`influx_si`) functions correctly:
1. Install the Python package:
   ```bash
   pip install isodesign
   ```
2. Install the mandatory R dependencies:
   ```bash
   influx_s --install_rdep
   ```

**Note**: Ensure the system has Python 3.10+ and R 3.4.0+ installed prior to setup.

## Usage Patterns

### Launching the Interface
The primary way to interact with IsoDesign's features is through its graphical user interface:
```bash
isodesign
```

### Core Workflow Logic
When using IsoDesign to design an experiment, follow this procedural logic:
1. **Substrate Definition**: Define all possible isotopic forms for each substrate involved in the experiment.
2. **Cost Analysis**: Input substrate prices to allow the tool to optimize for "resolution per dollar."
3. **Scoring Selection**: Choose scoring criteria based on the biological question (e.g., focusing on specific reaction resolutions vs. global network resolution).
4. **Optimization**: Combine multiple criteria to find the balance between experimental precision and budget constraints.
5. **Decision Support**: Use the visual representations generated by the tool to select the final label input strategy.

## Expert Tips
- **Parallel Computing**: For complex networks or non-stationary designs, ensure the environment allows for parallel execution to significantly reduce computation time.
- **R Dependency Failures**: If simulations fail to run, re-run `influx_s --install_rdep` to verify that all R libraries required for the flux calculations are present.
- **Library Usage**: Beyond the GUI, IsoDesign can be imported as a Python library for integration into custom metabolic modeling pipelines.
