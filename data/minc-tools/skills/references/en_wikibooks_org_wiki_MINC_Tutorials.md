**MINC in a few less than hard steps**

The following set of tutorials are broken into three sections, the first involves all issues with installing MINC. The second running the MINC command line tools and the third concerns using an example pipeline for automated analysis. These tutorials assume that you are using a linux-based system
an a "sh" variant shell such as bash, but should be reasonably easy to adapt to other settings.

## Installing and configuring MINC

[[edit](/w/index.php?title=MINC/Tutorials&veaction=edit&section=1 "Edit section: Installing and configuring MINC") | [edit source](/w/index.php?title=MINC/Tutorials&action=edit&section=1 "Edit section's source code: Installing and configuring MINC")]

The first step is to install the MINC library itself, this can be done by either compiling from source or by using a binary installer if you happen to be working on a platform that is currently supported. So you can pick either one of the first two tutorials here before progressing to the 3rd and 4th.

[SourceInstall](/wiki/MINC/Tutorials/SourceInstall "MINC/Tutorials/SourceInstall") - Building the MINC tools from source

[BinaryInstalls](/wiki/MINC/Tutorials/BinaryInstalls "MINC/Tutorials/BinaryInstalls") - How to install a binary MINC package

[InstallingPerl](/wiki/MINC/Tutorials/InstallingPerl "MINC/Tutorials/InstallingPerl") - How to install the required perl modules

[AddingPaths](/wiki/MINC/Tutorials/AddingPaths "MINC/Tutorials/AddingPaths") - Finishing off the installation
