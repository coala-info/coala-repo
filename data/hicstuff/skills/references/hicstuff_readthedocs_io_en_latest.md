# Welcome to hicstuff’s documentation![¶](#welcome-to-hicstuff-s-documentation "Permalink to this headline")

hicstuff is a python library to generate and manipulate Hi-C matrices. It aims to be easy to use and install.
You can follow hicstuff’s development on [Github](https://github.com/koszullab/hicstuff) .
