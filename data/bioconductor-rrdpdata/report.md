## rRDPData

### Package Description
The package provides the data for the RDP Classifier 2.14 released in August 2023. It contains the latest bacterial and archaeal taxonomy training set No. 19 as described in Wang Q, Cole JR. 2024. Updated RDP taxonomy and RDP Classifier for more accurate taxonomic classification. Microbiol Resour Announc 0:e01063-23.
