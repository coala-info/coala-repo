# nwkit draw

[Jump to bottom](#wiki-pages-box)

## Overview

`nwkit draw` renders a phylogenetic tree to `pdf`, `png`, or `svg`.

By default, when leaf labels follow `GENUS_SPECIES` or `GENUS_SPECIES_SUFFIX`, internal nodes can be annotated as speciation nodes or duplication nodes. Speciation nodes are shown in blue and duplication nodes in red.

Support values attached to internal branches are drawn on the corresponding horizontal branches.
Tip labels can also be colored from a trait table.

## Basic example

### Command

```
nwkit draw -i tree.nwk -o tree.svg
```

### Output

![Example output](nwkit_draw_00.png)

## Species-aware node annotation

### Command

```
nwkit draw -i tree.nwk -o tree.svg \
  --species_overlap_node_plot auto
```

In the default `auto` mode, node markers are shown only when all tip labels can be parsed as species labels.

## Custom species parsing

If your labels do not follow `GENUS_SPECIES[_...]`, provide `--species_regex`.

### Command

```
nwkit draw -i tree.nwk -o tree.svg \
  --species_regex '^([A-Za-z]+)\.([A-Za-z]+)\|'
```

When the regex contains capture groups, non-empty captured groups are joined with underscores.

## Disabling node markers

### Command

```
nwkit draw -i tree.nwk -o tree.svg \
  --species_overlap_node_plot no
```

## Coloring tip labels with a trait table

When `--trait` and `--group_by` are supplied, leaf labels are colored by group and the groups are added to the legend.

### Command

```
nwkit draw -i tree.nwk -o tree.svg \
  --trait traits.tsv \
  --group_by habitat
```

The trait table must contain a `leaf_name` column and the selected grouping column.

### Example input (`tree.nwk`)

```
((A:1,B:1):1,(C:1,D:1):1);
```

### Example input (`traits.tsv`)

| leaf\_name | habitat |
| --- | --- |
| A | forest |
| B | forest |
| C | marine |

### Example output (`tree.svg` / `tree.png`)

![Trait-colored example output](nwkit_draw_trait_01.png)

## Filtering support labels

### Command

```
nwkit draw -i tree.nwk -o tree.svg \
  --species_overlap_node_plot no \
  --support_min 90
```

## Other options

* `--image_format auto|pdf|png|svg`: infer the output format from `--outfile` or set it explicitly
* `--ladderize yes|no`: ladderize the tree before drawing
* `--species_overlap_node_plot yes|no|auto`: control speciation/duplication node markers
* `--trait PATH`: optional TSV used for tip-label coloring
* `--group_by COLUMN`: trait column used for coloring tip labels
* `--trait_palette NAME`: matplotlib colormap used for trait-group colors
* `--support_labels yes|no`: enable or disable support labels
* `--support_min FLOAT`: show only support labels at or above the threshold
