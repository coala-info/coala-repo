NAME Mozilla::CA - Mozilla&#39;s CA cert bundle in PEM format SYNOPSIS use IO::Socket::SSL; use Mozilla::CA; my $host = &quot;www.paypal.com&quot;; my $client = IO::Socket::SSL-&gt;new( PeerHost =&gt; &quot;$host:443&quot;, SSL_verify_mode =&gt; 0x02, SSL_ca_file =&gt; Mozilla::CA::SSL_ca_file(), ) || die &quot;Can&#39;t connect: $@&quot;; $client-&gt;verify_hostname($host, &quot;http&quot;) || die &quot;hostname verification failure&quot;; DESCRIPTION Mozilla::CA provides a copy of Mozilla&#39;s bundle of Certificate Authority certificates in a form that can be consumed by modules and libraries based on OpenSSL. The module provide a single function: SSL_ca_file() Returns the absolute path to the Mozilla&#39;s CA cert bundle PEM file.
