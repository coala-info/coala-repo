hapmap370k-package

Example HapMap data from Illumina 370k BeadChips

Description

Example HapMap dataset from the Illumina 370k platform. Arrays were processed at CIDR using
samples obtained from the International HapMap Consortium This package is only intended for use
in the demonstration of BioConductor tools.
