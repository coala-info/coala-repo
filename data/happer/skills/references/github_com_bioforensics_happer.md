Given a reference sequence and haplotype alleles annotated in BED format, happer will mutate the reference and produce sequences to match the specified haplotypes. Installation To install: pip3 install happer To make sure the package installed correctly: pip3 install pytest py.test --pyargs happer happer requires Python version 3. Usage Reference sequences must be provided in Fasta format, and haplotype alleles must be specified in BED format as follows. Alleles corresponding to different haplotypes at the locus are separated by a | character, so for example a diploid individual should have 2 | -separated alleles annotated at each locus, while a tetraploid would have 4 alleles.
