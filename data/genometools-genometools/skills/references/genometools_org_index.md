It is based on a C library named &ldquo;libgenometools&rdquo; which consists of several modules. If you are interested in gene prediction, have a look at GenomeThreader . Platforms GenomeTools has been designed to run on every POSIX compliant UNIX system &mdash; for example Linux or macOS. A reduced Windows version is also available.
