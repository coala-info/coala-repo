Queries the sequence data in query_seq against the SBWT index sbwt and its LCS array lcs using index::query_sbwt . Then, derandomizes the resulting k -bounded matching statistics vector using derandomize::derandomize_ms_vec and translates the matching statistics to a character representation of the alignment using translate::translate_ms_vec . Returns a vector containing the character representation of the alignment. Panics if the query file is not readable or if it’s not a valid FASTX file. § Output format See the documentation for translate . § Example use kbo::build; use kbo::matches; use kbo::BuildOpts; use kbo::MatchOpts; let reference: Vec&lt;Vec&lt;u8&gt;&gt; = vec!
