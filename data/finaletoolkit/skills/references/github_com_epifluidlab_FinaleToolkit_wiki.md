Computational approaches can extract fragmentation features from cfDNA whole genome sequencing (WGS) data. However, various established cfDNA fragmentation features (ex. Windowed Protection Score (WPS), Motif Diversity Score (MDS), DELFI) have source code which is often absent, poorly documented, ad hoc, and unmaintained. FinaleToolkit is a fast and comprehensive toolkit that provides a user-friendly interface to extract various cfDNA fragmentation features from fragmentation data.
