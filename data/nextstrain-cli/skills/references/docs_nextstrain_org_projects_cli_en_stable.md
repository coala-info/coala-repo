If you’re just getting started with Nextstrain, please see our general documentation instead. The Nextstrain CLI is a program called nextstrain . It aims to provide a consistent way to run and visualize pathogen builds and access Nextstrain components like Augur and Auspice computing platforms, such as Docker runtime , Conda runtime , Singularity runtime , and AWS Batch runtime . Note If you’re unfamiliar with Nextstrain builds, you may want to follow our Running a phylogenetic workflow tutorial first and then come back here.
