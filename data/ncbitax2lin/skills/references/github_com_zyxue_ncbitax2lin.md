The output file can be overwritten with --output option. FAQ Q : I have a large number of sequences with their corresponding accession numbers from NCBI, how to get their lineages? A : First, you need to map accession numbers (GI is deprecated) to tax IDs based on nucl_*accession2taxid.gz files from ftp://ftp.ncbi.nih.gov/pub/taxonomy/accession2taxid/. Secondly, you can trace a sequence's whole lineage based on its tax ID. The tax-id-to-lineage mapping is what NCBItax2lin can generate for you. If you have any question about this project, please feel free to create a new issue . Note on taxdump.tar.gz.md5 It appears that NCBI periodically regenerates taxdump.tar.gz and taxdump.tar.gz.md5 even when its content is still the same.
