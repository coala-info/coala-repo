From Source Recommended to use conda to manage the environment from the provided environment.yml file. git clone https://github.com/CFIA-NCFAD/gfflu.git cd gfflu conda env create -f environment.yml conda activate gfflu Annotation gfflu outputs a SnpEff compatible GFF with the same features identified as the Influenza Virus Sequence Annotation Tool .
