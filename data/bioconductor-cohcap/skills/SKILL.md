# bioconductor-cohcap

## Overview

COHCAP (City of Hope CpG Island Analysis Pipeline) is a comprehensive Bioconductor package designed for the analysis of DNA methylation data. It provides a structured workflow to move from raw methylation beta values to biological insights by identifying differentially methylated CpG sites and islands, and correlating these changes with gene expression.

The package is particularly effective for:
- **Illumina Arrays**: Native support for 27k and 450k; custom support for EPIC.
- **Targeted BS-Seq**: Specialized preprocessing for bisulfite sequencing data.
- **Integrative Analysis**: Linking methylation status to transcriptomic data.

## Core Workflow

The standard COHCAP pipeline follows a sequential four-step process.

### 1. Data Annotation and Normalization
Prepare the methylation data for analysis. For Illumina 450k/27k, COHCAP uses built-in UCSC annotations.

```r
library(COHCAP)
# beta.file: path to text file with beta values
# project.name: string for output naming
# project.folder: directory for output files
beta.table <- COHCAP.annotate(beta.file, project.name, project.folder, platform="450k-UCSC")
```

### 2. CpG Site Statistics
Calculate statistics for individual CpG sites and filter for differential methylation.

```r
# sample.file: path to sample description file (groups/replicates)
# ref: the name of the reference group in your sample file
filtered.sites <- COHCAP.site(sample.file, beta.table, project.name, project.folder, ref="control_group")
```
*Outputs:* Filtered sites are saved in the "CpG Site" folder; .wig files for genome browsers are generated by default.

### 3. CpG Island Analysis
Aggregate site-level data to identify differentially methylated regions (islands).

```r
# Uses the filtered sites from the previous step
island.list <- COHCAP.avg.by.island(sample.file, filtered.sites, beta.table, project.name, project.folder, ref="control_group")
```
*Note:* Use `COHCAP.avg.by.site` if you prefer statistics based on site-level significance within islands rather than averaging beta values first.

### 4. Integration with Gene Expression
Identify genes where methylation and expression are inversely correlated.

```r
# expression.file: path to normalized expression data
COHCAP.integrate.avg.by.island(island.list, project.name, project.folder, expression.file, sample.file)
```

## Specialized Workflows

### EPIC Array Analysis
For EPIC arrays, you must provide custom CpG island mapping. Use the "custom" platform setting in `COHCAP.annotate` and provide the appropriate annotation template.

### Targeted BS-Seq
Bisulfite sequencing data requires a preprocessing step to format the data and create a custom annotation file optimized for the specific targeted regions.

```r
# This step is computationally intensive and creates the necessary input files
COHCAP.BSSeq.preprocess(project.name, project.folder, ...)
```

## Key Tips and Interpretation

- **Project Structure**: COHCAP automatically creates subfolders ("Raw Data", "CpG Site", "CpG Island", "Integrate"). Check "Raw Data" for full statistics before filtering.
- **Sample File**: Ensure your sample description file matches the column names in your beta value file.
- **Reference Group**: Always explicitly define the `ref` parameter to ensure the direction of fold-change (delta-beta) is biologically meaningful (e.g., "Treatment" vs "Control").
- **Visualization**: The `Average-by-Island` workflow automatically generates box-plots for significant islands and scatter plots for expression correlations.
