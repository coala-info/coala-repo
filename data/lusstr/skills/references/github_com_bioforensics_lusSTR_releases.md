[lusSTR version 0.7](/bioforensics/lusSTR/releases/tag/0.7)

* Added LUS+ as possible data type for applying filters and creating files for use in EuroForMix ([#57](https://github.com/bioforensics/lusSTR/pull/57))
* Split Kintelligence SNP data into 10 bins ordered by marker read counts for use in EuroForMix ([#58](https://github.com/bioforensics/lusSTR/pull/58))
* Added ability to accommodate Kintelligence data analyzed using UAS version 2.5 ([#59](https://github.com/bioforensics/lusSTR/pull/59) and [#60](https://github.com/bioforensics/lusSTR/pull/60))
  o Different file format
  o Differing SNP rsIDs
  o All SNPs reported on forward strand
* Create two sets of marker plots for STR data to mimic an STR electropherogram ([#61](https://github.com/bioforensics/lusSTR/pull/61))
  o The y-axis (read counts) scaled for each individual marker
  o The y-axis (read counts) scaled equally for all markers to the marker with the highest read counts
* Small bug fix in applying the detection threshold ([#56](https://github.com/bioforensics/lusSTR/pull/56))
