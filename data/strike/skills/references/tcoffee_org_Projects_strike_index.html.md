STRIKE (Single sTRucture Induced Evaluation) is a program to evaluate protein multiple sequence alignments using a single protein structure. DOWNLOAD Download the package here INSTALLATION Typing 'make' in the command line will produce the executable. The Program STRIKE uses the STRIKE matrix to evaluate an alignment. This matrix gives a score for a contact similar to the Blosum matrix which gives a score to each nucleotide match. Contacts from the given structure are projected onto the alignment and each projected contact is scored. The result is a single score for the whole alignment. The higher the score, the better is the agreement between alignmetn and structure. DOCUMENTATION A detailed description is given in the incorporated Readme.
