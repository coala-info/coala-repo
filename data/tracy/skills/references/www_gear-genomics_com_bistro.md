##### How to use

1. Paste DNA sequences in the **Input** tab, one sequence per line.
2. Ensure all sequences have the **same length** and contain only `A`, `C`, `G`, or `T` (case-insensitive).
3. Click **Launch Analysis** to see the stacked per-position base frequencies in the **Results** tab.

###### Example input

```
GATAGCTCAG
TGCAATGGTC
TTATACCAGG
CAGCAGTTTA
ACTCTGAGGC
ACAGGACTCT
```

Tip: Use the **Load Example** button in the Input tab to autofill these sequences.
