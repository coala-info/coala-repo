Specifically, it will: Compared unassembled or assembled metagenomic sequences to a protein family database Calculate metagenome functional abundance and diversity Compare metagenomes using a variety of statistical and ecological tools Identify protein families that differentiate metagenomes using robust statistical tests Shotmap can be run on a multicore computer or can optionally interface with an SGE-configured computing cluste (i.e., a cloud). Shotmap can also optionally manage the information and data associated with this workflow in a relational database. Quickstart The fastest way to get shotmap working on your data is to first build a search database (which only needs to be done one time): perl $SHOTMAP_LOCAL/scripts/build_shotmap_searchdb.pl -r=&lt;/directory/path/to/reference/database&gt; -d=&lt;/directory/path/to/search/database&gt; And then execute ShotMAP: perl $SHOTMAP_LOCAL/scripts/shotmap.pl -i=&lt;/directory/path/to/metagenomes&gt; -d=&lt;/directory/path/to/search/database&gt; -o=&lt;/directory/path/to/result/database/&gt; And then compare the results (if more than one sample): perl $SHOTMAP_LOCAL/scripts/compare_shotmap_results.pl -i=&lt;/directory/path/to/result/database/&gt; -m=&lt;/path/to/optional/metadata/file&gt; -o=&lt;/directory/path/to/new/result/directory/&gt; Documentation Table of Contents Workflow Synopsis Overview on How to Run ShotMAP Dependencies Installation Guide Building a ShotMAP Search Database Metadata Files Workflow Output Script Documentation install.pl test_shotmap.pl build_shotmap_searchdb.pl shotmap.pl compare_shotmap_samples.pl test_config_file.pl Advanced Documentation configuration files remote (cloud) users relational database users Change Log v1.1 - 2016.04.26 update to indexed database file names changes to shotmap.R to handle variation in R version error handline in shotmap.R for qvalue calculation bug fixes in installer script to work on more general systems addition of scripts/run_tests.pl change in rapsearch options to reduce disc footpring About A Shotgun Metagenome Annotation Pipeline Resources Readme License GPL-2.0 license Uh oh!
