Installation Using Docker See details First you must have Docker installed and running. Secondly have look at the availabe rapgreen biocontainers at quay.io . Then: # get the chosen rapgreen container version docker pull quay.io/biocontainers/rapgreen:1.0--hdfd78af_0 # use an rapgreen docker run quay.io/biocontainers/rapgreen:1.0--hdfd78af_0 rapgreen --help Using Singularity See details First you must have Singularity installed and running. Secondly have look at the availabe rapgreen biocontainers at quay.io . Then: # get the chosen rapgreen container version singularity pull docker://quay.io/biocontainers/rapgreen:1.0--hdfd78af_0 # run the container singularity run rapgreen:1.0--hdfd78af_0 You are now in the container. You can use an RapGreen.
