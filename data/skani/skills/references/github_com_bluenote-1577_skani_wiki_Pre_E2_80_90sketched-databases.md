# Pre-sketched databases for skani

Pre-sketched databases for skani are available for download. These databases can be searched against with < 30 GB of RAM in seconds with `skani search`.

### Sketched GTDB-R226 database (> 140,000 bacteria/archaea species representative genomes)

* ~30 GB compressed and 50 GB uncompressed.

#### Parameters:

* Default: `-c 125, -m 1000`

Download: <http://faust.compbio.cs.cmu.edu/skani-files/skani_gtdb_r226-v0.3.tar.gz>

```
wget http://faust.compbio.cs.cmu.edu/skani-files/skani_gtdb_r226-v0.3.tar.gz
tar -zxvf skani_gtdb_r226-v0.3.tar.gz

skani search my_genome.fa -d skani_gtdb_r226-v0.3 -o results.tsv
```
