# cdskit accession2fasta

[Jump to bottom](#wiki-pages-box)

Kenji Fukushima edited this page Mar 6, 2023
·
[2 revisions](/kfuku52/cdskit/wiki/cdskit-accession2fasta/_history)

`cdskit accession2fasta` retrieves fasta sequences from a list of [GenBank](https://www.ncbi.nlm.nih.gov/genbank/) accessions.

### Example

#### Command

```
cdskit accession2fasta --accession_file input.txt -o output.fasta
```

#### input.txt

```
FM211860.1
FM211829.1
FM211877.1
FM211842.1
FM211850.1
```
