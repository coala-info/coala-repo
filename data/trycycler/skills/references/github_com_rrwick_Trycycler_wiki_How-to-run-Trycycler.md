# How to run Trycycler

[Jump to bottom](#wiki-pages-box)

Ryan Wick edited this page Nov 17, 2020
·
[10 revisions](/rrwick/Trycycler/wiki/How-to-run-Trycycler/_history)

Running Trycycler is a multi-step process involving five separate Trycycler commands. It's not all automated into a single command because there are some judgement calls and/or human interventions to be done between the steps.

For a brief overview, check out the [Quick start](/rrwick/Trycycler/wiki/Quick-start) page. For more detail:

* Step 1: [Generating assemblies](/rrwick/Trycycler/wiki/Generating-assemblies)
* Step 2: [Clustering contigs](/rrwick/Trycycler/wiki/Clustering-contigs)
* Step 3: [Reconciling contigs](/rrwick/Trycycler/wiki/Reconciling-contigs)
* Step 4: [Multiple sequence alignment](/rrwick/Trycycler/wiki/Multiple-sequence-alignment)
* Step 5: [Partitioning reads](/rrwick/Trycycler/wiki/Partitioning-reads)
* Step 6: [Generating a consensus](/rrwick/Trycycler/wiki/Generating-a-consensus)
* Step 7: [Polishing after Trycycler](/rrwick/Trycycler/wiki/Polishing-after-Trycycler)
