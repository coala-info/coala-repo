# crispr_studio

## Overview
CRISPRStudio is a specialized tool for the comparative visualization of CRISPR arrays. It automates the process of identifying homologous spacers across different bacterial or archaeal isolates and represents them as color-coded diamonds in an SVG format. This allows for rapid identification of shared evolutionary history, spacer acquisition events, and array conservation. It is particularly useful for researchers performing comparative genomics on CRISPR-Cas systems.

## Core Workflow
The tool requires a GFF3 file generated by **CRISPRDetect** as its primary input.

1.  **Input Preparation**: Ensure you have a `.gff` file from CRISPRDetect.
2.  **Execution**: Run the Python script with the `-i` flag.
3.  **Output**: The tool generates an SVG file that can be opened in vector graphics editors (e.g., Inkscape, Adobe Illustrator).

## Command Line Usage

### Basic Command
```bash
python CRISPR_Studio_1.0.py -i input_file.gff
```

### Common Parameters
| Flag | Description | Usage Tip |
| :--- | :--- | :--- |
| `-i` | **Input GFF3** | Must be the output from CRISPRDetect. |
| `-s` | **Sorting** | Options: `CRISPRDetect` (default order), `DistMatrix` (similarity-based), or `File` (custom list). |
| `-l` | **Subset List** | Provide a `.txt` file with specific isolate names to filter the visualization. |
| `-c` | **Cut-off** | Adjust the score cut-off for spacer pairing (default is 2). |
| `-r` | **Reproducible Colors** | Use this when adding new sequences to an existing dataset to maintain previous color assignments. |

### Visualization Filters
*   **Highlight Shared Spacers**: Use `-gU` to gray out unique spacers, leaving only those present in multiple arrays colored.
*   **Highlight Unique Spacers**: Use `-gS` to gray out similar spacers, emphasizing unique acquisition events.
*   **Show Spacer Length**: Use `-n` to display the nucleotide length over the diamonds (recommended for data validation rather than final figures).

## Expert Tips
*   **Data Correction**: If the script stops due to spacer length warnings (spacers >1.5x or <1.5x average size), you can bypass this check using the `-f` flag, or provide a corrected fasta file after the flag: `-f corrected.fasta`.
*   **Isolate Naming**: When using the `-l` (subset) or `-s File` (sorting) options, the isolate names in your text file must match the names in the CRISPRDetect GFF file exactly.
*   **Environment**: CRISPRStudio is optimized for Python 3.6. Compatibility issues may arise with Python 3.7+ due to `scikit-bio` dependencies. If installation fails, ensure `pandas`, `scipy`, and `numpy` are managed by `pip install scikit-bio` directly.
