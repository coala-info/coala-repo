# tbl2asn-forever

## Overview
The `tbl2asn-forever` package provides a stable, long-term version of the NCBI `tbl2asn` utility. It is designed to bundle sequence data, annotation features, and metadata into a single structured file (.sqn) suitable for GenBank submission. This tool is essential for bioinformaticians who need to convert raw genomic data and manual annotations into the specific ASN.1 format required by NCBI's processing pipeline.

## Command Line Usage
The core command is `tbl2asn`. It typically processes all files in a directory that share a common base name.

### Basic Submission Generation
To generate a submission file from a FASTA file (`genome.fsa`) and a feature table (`genome.tbl`):
```bash
tbl2asn -i genome.fsa -t template.sbt -a s -V vb
```
*   `-i`: Input FASTA file.
*   `-t`: Submission template file (generated via NCBI's web tool).
*   `-a`: File type (e.g., `s` for set).
*   `-V`: Verification level (`vb` performs full validation).

### Bulk Processing
If you have multiple genomes in a directory, `tbl2asn` can process them in batch:
```bash
tbl2asn -p ./data_dir -t template.sbt -M n -Z
```
*   `-p`: Path to the directory containing `.fsa` and `.tbl` files.
*   `-M`: Genomic code (e.g., `n` for normal).
*   `-Z`: Discrepancy report generation (crucial for identifying annotation errors).

## Best Practices
*   **File Naming**: Ensure your FASTA file and feature table have the same prefix (e.g., `contig1.fsa` and `contig1.tbl`). The tool matches them automatically.
*   **Template Files**: Always use a fresh `.sbt` template file from the NCBI Submission Template Tool to ensure contact information and bioproject metadata are current.
*   **Validation**: Always check the resulting `.val` files. These contain errors and warnings from the NCBI validator. A submission with "Error" level messages will likely be rejected.
*   **Discrepancy Report**: Review the `.dr` file generated by the `-Z` flag. It highlights inconsistencies like missing stop codons or mismatched locus tags that the standard validation might miss.
