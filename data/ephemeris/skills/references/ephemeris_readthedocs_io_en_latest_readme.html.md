# Introduction[¶](#introduction "Link to this heading")

[![https://readthedocs.org/projects/pip/badge/?version=latest](https://readthedocs.org/projects/pip/badge/?version=latest)](https://ephemeris.readthedocs.org)
[![https://badge.fury.io/py/ephemeris.svg](https://badge.fury.io/py/ephemeris.svg)](https://pypi.python.org/pypi/ephemeris/)

Ephemeris is a small Python 3 library and set of scripts for managing the
bootstrapping of [Galaxy](http://galaxyproject.org/) plugins - tools, index data, and workflows.

* Free software: Academic Free License version 3.0
* Documentation: <https://ephemeris.readthedocs.org>.
* Code: <https://github.com/galaxyproject/ephemeris>
