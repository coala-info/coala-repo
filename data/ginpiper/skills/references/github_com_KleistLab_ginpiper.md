Dependencies ginpiper uses the following packages: ggplot2 R0 scales Installation ginpiper can be installed in R using devtools : install.packages("devtools") library(devtools) devtools::install_github("KleistLab/ginpiper") Reference [1] Smith, M. R. and Trofimova, M., et al. (2021).
