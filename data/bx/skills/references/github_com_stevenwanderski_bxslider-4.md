Installation Step 1: Link required files First and most important, the jQuery library needs to be included (no need to download - link directly from Google). Next, download the package from this site and link the bxSlider CSS file (for the theme) and the bxSlider Javascript file. &lt;!-- jQuery library (served from Google) --&gt; &lt; script src =" https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js " &gt; &lt;/ script &gt; &lt;!-- bxSlider Javascript file --&gt; &lt; script src =" /js/jquery.bxslider.min.js " &gt; &lt;/ script &gt; &lt;!-- bxSlider CSS file --&gt; &lt; link href =" /lib/jquery.bxslider.css " rel =" stylesheet " /&gt; Or, if you prefer, you can get the bxSlider's resources from the CDN : &lt; script src =" https://cdnjs.cloudflare.com/ajax/libs/bxslider/4.2.17/jquery.bxslider.min.js " &gt; &lt;/ script &gt; &lt; link href =" https://cdnjs.cloudflare.com/ajax/libs/bxslider/4.2.17/jquery.bxslider.min.css " rel =" stylesheet " /&gt; Step 2: Create HTML markup Create a &lt;ul class="bxslider"&gt; element, with a &lt;li&gt; for each slide. Slides can contain images, video, or any other HTML content! &lt; ul class =" bxslider " &gt; &lt; li &gt; &lt; img src =" /images/pic1.jpg " /&gt; &lt;/ li &gt; &lt; li &gt; &lt; img src =" /images/pic2.jpg " /&gt; &lt;/ li &gt; &lt; li &gt; &lt; img src =" /images/pic3.jpg " /&gt; &lt;/ li &gt; &lt; li &gt; &lt; img src =" /images/pic4.jpg " /&gt; &lt;/ li &gt; &lt;/ ul &gt; Step 3: Call the bxSlider Call .bxSlider() on &lt;ul class="bxslider"&gt; .
