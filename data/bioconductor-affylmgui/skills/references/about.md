# About affylmGUI

affylmGUI is a Graphical User Interface for analysis of
Affymetrix microarray data, using Gordon Smyth's [limma](http://bioinf.wehi.edu.au/limma/) package (Linear
Modes for MicroArray data). limma and affylmGUI are both [R](http://www.r-project.org/) packages. The [limma](http://bioinf.wehi.edu.au/limma/) package offers R
users a command-line interface to state-of-the-art techniques for
linear modelling of microarray data and for identifying
differentially expressed genes. The affylmGUI package, while not as
powerful as limma to the expert user, offers a simple
point-and-click interface to many of the commonly-used limma and
affy functions. To use the affylmGUI package, you need to have [R 1.8.1](http://www.r-project.org/) or later, Tcl/Tk 8.3 or
later ([ActiveTcl for
Windows/Linux](http://www.activestate.com/Products/ActiveTcl/) or [Fink
Tcl/Tk for MacOSX (X11)](http://fink.sourceforge.net/pdb/search.php?s=tcltk)) and the [limma](http://bioinf.wehi.edu.au/limma), affylmGUI, tkrplot
R packages. The affyPLM (Bioconductor), R2HTML (CRAN) and xtable
(CRAN) packages are recommended. The [limma](http://www.bioconductor.org/repository/devel/package/html/limma.html)
and [affyPLM](http://www.bioconductor.org/repository/devel/package/html/affyPLM.html)
packages are available on [Bioconductor](http://www.bioconductor.org/), the affylmGUI package is available [here](http://bioinf.wehi.edu.au/affylmGUI/index.html), and
the other packages are available from [CRAN](http://cran.r-project.org/). affylmGUI has been
succesfully tested on Windows 2000, Windows XP, and Red Hat Linux,
and on Mac OSX with X11.
