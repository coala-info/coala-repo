This library is used as the scaffolding for TBProfiler . Installation Conda conda install -c bioconda pathogen-profiler Manual The library is written in pure python and has no dependancies to install: git clone https://github.com/jodyphelan/pathogen-profiler.git python setup.py install At runtime, some of the functions will need tools involved with trimming, mapping and variant calling. Please install the following tools: trimmomatic, bwa, minimap2, bowtie2, samtools, bcftools and parallel .
