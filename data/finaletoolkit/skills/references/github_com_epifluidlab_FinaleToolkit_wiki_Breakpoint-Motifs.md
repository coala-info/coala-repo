## `breakpoint-motifs`

Let's retrieve the 6 bp sequence context (`-k 6`) surrounding the 5' breakpoint of each fragment (`breakpoint-motifs`) from fragments that have a MAPQ score greater than 30 (`-q 30`). We will output (`-o`) the TSV to `BH01.22.bp.tsv`, using one worker (`-w 1`) and verbose mode (`-v`).

```
finaletoolkit breakpoint-motifs BH01.hg19.mdups.22.bam hg19.2bit \
  -q 30 \
  -k 6 \
  -o BH01.22.bp.tsv \
  -w 1 \
  -v
```

The output should look like this:


## `interval-breakpoint-motifs`

FinaleToolkit introduces the ability to calculate breakpoint motifs over intervals instead of genome-wide.

Let's retrieve the 6 bp sequence context (`-k 6`) surrounding the 5' breakpoint of each fragment within fixed genomic intervals (`interval-breakpoint-motifs`) from fragments that have a MAPQ score greater than 30 (`-q 30`). We will output (`-o`) the TSV to `BH01.hg19.bp.5mb.tsv`, with sixteen workers (`-w 16`) and verbose mode (`-v`).

```
finaletoolkit interval-breakpoint-motifs BH01.hg19.mdups.22.bam hg19.2bit 100kb.22.bins \
  -q 30 \
  -k 6 \
  -o BH01.hg19.bp.tsv \
  -w 16 \
  -v
```

The output file should look like the following:

```
contig	start	stop	name	count	AAAAAA	AAAAAC	AAAAAG	AAAAAT	AAAACA	...
22	0	100000	.	0	NaN	NaN	NaN	NaN	NaN	...
22	100000	200000	.	0	NaN	NaN	NaN	NaN	NaN	...
...
22	27900000	28000000	.	75216	0.000665	0.000133	0.000120	0.000133	0.000439	...
...
```

Note that `NaN` appears under motifs for regions where coverage is 0.
