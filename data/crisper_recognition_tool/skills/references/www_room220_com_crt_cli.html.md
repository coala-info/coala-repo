CRT - Command Line Interface Command Line Interface (CLI) To execute CRT from the command line: &nbsp; java -cp CRT1.2-CLI.jar crt [options] inputFile [outputFile] &nbsp;&nbsp;&nbsp; java -cp CRT1.2-CLI.jar -- executes a java program called crt , which is the main class defined in the file CRT1.2-CLI.jar ;&nbsp; -cp (class path) &nbsp;&nbsp;&nbsp; [options] -- optional; specifies input for the crt program; if no options are given, default values are used; see options below &nbsp;&nbsp;&nbsp; inputFile -- required; must be a file in FASTA format &nbsp;&nbsp;&nbsp; outputFile -- optional; file to which results are to be written;&nbsp; if no file is given, results will be written a file named "a.out" For example &nbsp;&nbsp;&nbsp; java -cp CRT1.2-CLI.jar crt ecoli.fna &nbsp;&nbsp;&nbsp; java -cp CRT1.2-CLI.jar crt ecoli.fna ecoli.out &nbsp;&nbsp;&nbsp; java -cp CRT1.2-CLI.jar crt -minNR 3 -minRL 21 ecoli.fna OPTIONS &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; -minNR&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; minimum number of repeats a CRISPR must contain; default 3 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; -minRL&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; minimum length of a CRISPR's repeated region;&nbsp; default 19 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; -maxRL&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; maximum length of a CRISPR's repeated region;&nbsp; default 38 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; -minSL&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; minimum length of a CRISPR's non-repeated region (or spacer region);&nbsp; default 19 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; -maxSL&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; maximum length of a CRISPR's non-repeated region (or spacer region);&nbsp; default 48 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; -screen&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; print results to the screen, instead of a file; default 0; (range: 0-1) &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; -searchWL&nbsp;&nbsp;&nbsp; length of search window used to discover CRISPRs; (range: 6-9) Simplier Execution of CRT using aliases If operating within a Unix environment, execution of CRT can be made simplier using aliases.&nbsp; Aliases allow the shortening of long commands.&nbsp; In this case, we can substitute "java -cp CRT1.2-CLI.jar crt" with "crt".&nbsp; To do this, type the command listed below at the prompt. Bash shell syntax &nbsp;&nbsp;&nbsp; alias crt="java -cp CRT1.2-CLI.jar crt" &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; C shell (csh) or tcsh shell syntax &nbsp;&nbsp;&nbsp; alias crt "java -cp CRT1.2-CLI.jar crt" This allows the program to be executed as follows. &nbsp;&nbsp;&nbsp; crt ecoli.fna &nbsp;&nbsp;&nbsp; crt ecoli.fna ecoli.out &nbsp;&nbsp;&nbsp; crt -minNR 3 -minRL 21 ecoli.fna To make an alias permanent, place it in your shell's corresponding .rc file. For csh this is .cshrc and for bash, this is .bashrc.&nbsp; (You may have to type 'source .bashrc' or restart the shell before the change takes effect.)&nbsp; Click here for more details.
