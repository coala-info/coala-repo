limmaGUI Package Vignette

James Wettenhall

April 1, 2004

limmaGUI is a Graphical User Interface for doing cDNA analysis with the limma
package (LInear Models for MicroArray Data). For Affymetrix analysis with the limma
package, see affylmGUI.

The limmaGUI Documentation can be accessed online or by installing the limmaGUI

package locally. Then at the R prompt you can type

> if (interactive())
+

browseURL("http://bioinf.wehi.edu.au/limmaGUI/Doc/")

or alternatively

> if (interactive())
+ {
+
+
+ }

library(limmaGUI)
limmaGUIhelp()

or alternatively

> if (interactive())
help.start()
+

and follow the links the limmaGUI package help.
