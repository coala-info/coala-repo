# r-learnbayes

name: r-learnbayes
description: Bayesian statistical inference using the LearnBayes R package. Use this skill when performing Bayesian analysis, including summarizing posterior distributions, implementing MCMC algorithms (Metropolis-Hastings, Gibbs sampling), conducting Bayesian regression, or using hierarchical models.

## Overview

The `LearnBayes` package is designed to help users learn and apply the basic tenets of Bayesian statistical inference. It provides functions for summarizing one and two-parameter posterior distributions, predictive distributions, and more complex models via Markov Chain Monte Carlo (MCMC) methods. It is particularly useful for educational purposes and for implementing custom posterior functions.

## Installation

To install the package from CRAN:

```R
install.packages("LearnBayes")
library(LearnBayes)
```

## Core Workflows

### 1. Summarizing Posterior Distributions
For standard conjugate priors or simple discrete distributions:
- `discint()`: Finds the shortest probability interval for a discrete distribution.
- `histprior()`: Constructs a prior density from a histogram.
- `normal.select()`: Finds the mean and standard deviation of a normal prior given two quantiles.
- `beta.select()`: Finds the shape parameters of a beta prior given two quantiles.

### 2. Posterior Simulation via MCMC
When the posterior does not have a standard form, use simulation:
- `laplace()`: Computes the posterior mode and the associated variance-covariance matrix (Laplace approximation).
- `rwmetropolis()`: Implements a random walk Metropolis algorithm. Requires a user-defined function for the log-posterior.
- `indepmetropolis()`: Implements an independence Metropolis-Hastings algorithm.
- `gibbs()`: Illustrates Gibbs sampling for specific models.

### 3. Bayesian Regression and Hierarchical Models
- `blinreg()`: Samples from the posterior of a linear regression model with a noninformative prior.
- `bayes.probit()`: Samples from the posterior of a binary probit regression model.
- `robust.bayes()`: Performs Bayesian inference for a regression model with a t-error distribution.
- `hierarchical.poisson()`: Fits a hierarchical Poisson model.

### 4. Model Comparison and Prediction
- `pbetap()`: Predictive distribution for a binomial observation with a beta prior.
- `pbetat()`: Bayesian test of a hypothesis about a proportion.
- `mnormt.test()`: Bayesian test of a hypothesis about a normal mean.
- `bfexch()`: Computes the Bayes factor for testing exchangeability of means.

## Tips
- Use `coda` package integration (via `mcmc` objects) to check for convergence of the chains generated by `LearnBayes`.
- The `triplot()` function is excellent for visualizing the Prior, Likelihood, and Posterior on a single graph for one-parameter models.
- For binomial data, `beta.select()` is the standard way to elicit prior hyperparameters from experts.
