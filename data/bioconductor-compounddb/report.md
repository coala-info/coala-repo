# bioconductor-compounddb Bioconductor Package Report

## CompoundDb

### Package Description
The package provides also a backend for Bioconductor's Spectra package and allows thus to match experimetal MS/MS spectra against MS/MS spectra in the database. Databases can be stored in SQLite format and are thus portable.
