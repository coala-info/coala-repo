Contributing - dbi.perl.org Toggle navigation About DBI Documentation Contributing Support Contributing Database programming with Perl - how to contribute The dbi-users Mailing List The dbi-users mailing list is the heart of the DBI user community. There are typically over two thousand users subscribed, ranging from novices to experienced users and driver developers. Messages are on-topic and the list is widely regarded as being of very high quality. It's a great way to learn about the DBI and, as you learn from the list and your own experience, you can help others along their path. Changes and Enhancements If you'd like the DBI to do something new, or different, a good way to make that happen is to do it yourself and send me a patch (a pull request or diff) to the source code that shows the changes. Information about how to do that can be found here , (including an important note about talking to me first to avoid duplicated or wasted effort).
