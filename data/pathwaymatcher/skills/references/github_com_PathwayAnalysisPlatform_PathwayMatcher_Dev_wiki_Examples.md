# Genetic Variants

Single-nucleotide polymorphisms (SNP) represented with the RsId identifiers defined in [dbSNP](https://www.ncbi.nlm.nih.gov/projects/SNP/)[[1]](#references) were obtained from the [GIANT (Genetic Investigation of ANthropometric Traits)](http://portals.broadinstitute.org/collaboration/giant/index.php/Main_Page) consortium. BMI Data association results for Models Adjusting for Smoking Status as obtained [here](http://portals.broadinstitute.org/collaboration/giant/index.php/GIANT_consortium_data_files) were downloaded and matched to pathways using **PathwayMatcher**. The results allows categorizing variants at the exome wide scale according to possible effects in pathways.

![](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/raw/master/docs/wiki/mhs.gif)
*Manhattan plot of the association results with the markers matched to a specific top level pathway in blue.*

It is therefore possible to inspect association results focusing on specific top-level pathways, related to major biological functions.
![](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/raw/master/docs/wiki/qqs.gif)
*QQ plot of the association results with markers segregated according to possible effect on top level pathways in blue.*

The multiple levels of the pathway database allow zooming in on specific functions and reactions.

![](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/raw/master/docs/wiki/mh.png)
*Manhattan plot of the association results with markers possibly affecting the **Signal Transduction** pathway in blue and within this top level pathway, markers possibly affecting **cAMP signalling** in red.*

* The illustrations can be reproduced using the *BMI* dataset below. Other examples are available as obtained from [DisGeNET](http://www.disgenet.org/web/DisGeNET/menu/home)[[1]](#references), using the [curated variant-disease association list](http://www.disgenet.org/web/DisGeNET/menu/downloads#vdascurated):
  + [BMI](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/blob/master/resources/input/GeneticVariants/RsId/CysticFibrosis.txt)
  + [Cystic fibrosis](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/blob/master/resources/input/GeneticVariants/RsId/CysticFibrosis.txt)
  + [Diabetes](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/blob/master/resources/input/GeneticVariants/RsId/Diabetes.txt)
  + [Diabetes in the Youth](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/blob/master/resources/input/GeneticVariants/RsId/DiabetesInYouth.txt)
  + [Huntington disease](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/blob/master/resources/input/GeneticVariants/RsId/HuntingtonsDisease.txt)
  + [Hypoglycemia](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/blob/master/resources/input/GeneticVariants/RsId/Hypoglycemia.txt)
  + [Ulcerative colitis](https://github.com/PathwayAnalysisPlatform/PathwayMatcher/blob/master/resources/input/GeneticVariants/RsId/UlcerativeColitis.txt)
