# bioconductor-generegionscan Bioconductor Package Report

## GeneRegionScan

### Package Description
The package includes a number of functions used for plotting these probe level data as a function of location along sequences of mRNA-strands. This can be used for analysis of variable splicing, and is especially well suited for use with exon-array data.
