### New features

* Support custom contig separator characters in pairix index by [@nvictus](https://github.com/nvictus) in [#398](https://github.com/open2c/cooler/pull/398)

### Performance improvements and bug fixes

* Make annotate faster by using indexing instead of merge by [@Phlya](https://github.com/Phlya) in [#353](https://github.com/open2c/cooler/pull/353)
* Fix input text file handling on certain pandas versions by [@thomas-reimonn](https://github.com/thomas-reimonn) in [#384](https://github.com/open2c/cooler/pull/384)
* Reduce memory consumption of very high-resolution merges by [@nvictus](https://github.com/nvictus) in [#408](https://github.com/open2c/cooler/pull/408)

### Maintenance

* Add windows and macos platforms to CI by [@nvictus](https://github.com/nvictus) in [#383](https://github.com/open2c/cooler/pull/383)
* Add type annotations by [@GarrettNg](https://github.com/GarrettNg) in [#387](https://github.com/open2c/cooler/pull/387)
