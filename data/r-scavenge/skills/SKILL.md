# r-scavenge

## Overview
SCAVENGE (Single Cell Analysis of Variant Enrichment through Network propagation of GEnomic data) is an R package designed to overcome sparsity in single-cell epigenomic data. It identifies cells most relevant to a phenotype by co-localizing genetic variants (fine-mapped GWAS posterior probabilities) with single-cell profiles and then using network propagation (random walk) to assign relevance scores across the cell-to-cell similarity network.

## Installation
To install the package from CRAN:
```R
install.packages("SCAVENGE")
```
To install the development version from GitHub:
```R
if(!require("remotes")) install.packages("remotes")
remotes::install_github("sankaranlab/SCAVENGE")
library(SCAVENGE)
```

## Core Workflow

### 1. Data Preparation
SCAVENGE requires two primary inputs:
- **scATAC-seq Data**: A peak-by-cell count matrix.
- **Fine-mapped GWAS Data**: Posterior probabilities for variants associated with a trait.

### 2. Calculate Seed Scores
The first step is to calculate the initial enrichment (seed scores) for each cell based on the overlap between fine-mapped variants and the cell's open chromatin regions.
```R
# Example: Calculate seed scores
# trait_variants: data frame with chr, pos, and posterior probability (PP)
# peak_metadata: data frame with peak coordinates
seed_scores <- get_seed_scores(atac_matrix, trait_variants, peak_metadata)
```

### 3. Construct Cell-to-Cell Network
Build a k-nearest neighbor (kNN) graph based on the high-dimensional features of the single cells (e.g., using Latent Semantic Indexing or PCA components).
```R
# Create mutual nearest neighbor graph
knn_graph <- get_knn_graph(pca_coordinates, k = 30)
```

### 4. Network Propagation
Propagate the seed scores across the network to assign a "SCAVENGE score" to every cell, representing its transitive association with the phenotype-enriched cells.
```R
# Perform random walk with restart
scavenge_scores <- random_walk(seed_scores, knn_graph, alpha = 0.05)
```

### 5. Statistical Significance
Compare the observed scores against a null distribution (generated by permuting seed scores or using background cell sets) to identify significantly enriched cell populations.

## Key Functions
- `get_seed_scores()`: Maps fine-mapped variants to peaks and calculates per-cell initial enrichment.
- `get_knn_graph()`: Constructs the adjacency matrix for the cell-to-cell network.
- `random_walk()`: Implements the network propagation algorithm.
- `cap_seed()`: Utility to cap high seed scores to prevent single-cell outliers from dominating the propagation.

## Tips for Success
- **Fine-mapping is critical**: Using raw GWAS summary statistics is not recommended as LD can obscure causal cell types. Always use fine-mapped posterior probabilities.
- **scATAC-seq only**: Currently, SCAVENGE is optimized for scATAC-seq. It is not yet recommended for scRNA-seq data.
- **Network Quality**: The quality of the cell-to-cell network (often derived from LSI or Harmony-integrated coordinates) significantly impacts the propagation results.
