# CUT'N'RUN data processing workflow

## Overview

This [Common Workflow Language](https://workflowhub.eu/workflows/1765) (CWL) workflow provides a standardized pipeline for processing bulk CUT'N'RUN data. It transforms raw sequencing reads into genome-wide accessibility tracks (bigWig) and identified peaks, supporting epigenomics research through automated sequence read processing and quality control.

The workflow follows a multi-stage execution path:
*   **Preprocessing and Alignment:** Raw reads are trimmed using Trim Galore! and aligned to the reference genome via Bowtie 2.
*   **Filtering and Normalization:** Aligned reads are merged and filtered, followed by the calculation of spike-in counts to facilitate downstream normalization.
*   **Signal Generation and Peak Calling:** The pipeline utilizes SAMtools to generate coverage tracks and MACS2 to identify regions of enrichment (peaks).

Final outputs include comprehensive quality control metrics and a summary report generated by the `chip_qc` and `create_summary_qc_report` steps. This ensures that users can evaluate the technical success of the CUT'N'RUN experiment alongside the biological results.

| fastq2 |  | array | List of fastq files containing the second mate of raw reads in case of paired end  (also see parameter "fastq1"). | is_paired_end |  | boolean | If paired end data is used set to true, else set to false.  |
| adapter1 |  | string? | Adapter sequence for first reads.  If not specified (set to "null"), trim_galore will try to autodetect whether ...\n - Illumina universal adapter (AGATCGGAAGAGC)\n - Nextera adapter (CTGTCTCTTATA)\n - Illumina Small RNA 3-prime Adapter (TGGAATTCTCGG)\n ... was used.\n You can directly choose one of the above configurations  by setting the string to "illumina", "nextera", or "small_rna".  Or you specify the adaptor string manually (e.g. "AGATCGGAAGAGC").  |
| adapter2 |  | string? | Adapter sequence for second reads (only relevant for paired end data).  If it is not specified (set to "null"), trim_galore will try to autodetect whether ...\n - Illumina universal adapter (AGATCGGAAGAGC)\n - Nextera adapter (CTGTCTCTTATA)\n - Illumina Small RNA 3-prime Adapter (TGGAATTCTCGG)\n ... was used.\n You can directly choose one of the above configurations  by setting the string to "illumina", "nextera", or "small_rna". | ignoreForNormalization |  | string? | List of space-delimited chromosome names that shall be ignored  when calculating the scaling factor.  Specify as space-delimited string.  Default: "chrX chrY chrM" Specify as space-delimited string.  Default: "chrX chrY chrM"  |
