The bioconda channel is a Conda channel providing bioinformatics related packages for Linux and macOS , supporting both x86_64 and aarch64/arm64 architectures. This repository hosts the corresponding recipes. User guide Please visit https://bioconda.github.io for details. Developer guide Please visit the new docs at https://bioconda.github.io/contributor/index.html for details. Nightly build status The nightly uploader jobs build any recipes that exist on master but were not successfully uploaded to the bioconda channel .
