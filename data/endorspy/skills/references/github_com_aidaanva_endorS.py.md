Usage usage: python endorS.py [-h] [--version] -r [ &lt; samplesfile &gt; .stats] -qF [ &lt; samplesfile &gt; .stats] -dedup [ &lt; samplesfile &gt; .stats] Author Aida Andrades Valtueña (aida.andrades[at]gmail.com) Description and options endorS.py calculates percent on target (aka Endogenous DNA) from samtools flagstat files and print to screen. The percent on target reported will be different depending on the combination of samtools flagstat provided. This program also calculates clonality (aka ' cluster factor ' ) and percent duplicates when the flagstat file after duplicate removal is provided Use --output flag to write results to a file optional arguments: -h, --help show this help message and exit --raw [ &lt; samplefile &gt; .stats], -r [ &lt; samplefile &gt; .stats] output of samtools flagstat in a txt file, assumes no quality filtering nor duplicate removal performed --qualityfiltered [ &lt; samplefile &gt; .stats], -q [ &lt; samplefile &gt; .stats] output of samtools flagstat in a txt file, assumes some form of quality or length filtering has been performed, must be provided with at least one of the options -r or -dedup --deduplicated [ &lt; samplefile &gt; .stats], -d [ &lt; samplefile &gt; .stats] output of samtools flagstat in a txt file, whereby duplicate removal has been performed on the input reads -v, --version show program ' s version number and exit --output [OUTPUT], -o [OUTPUT] specify a file format for an output file. Options: &lt;json&gt; for a MultiQC json output. Default: none --name [NAME], -n [NAME] specify name for the output file.
