Generating arguments for the 'voronota query-balls' This page allows to easily generate a set of arguments for the 'voronota query-balls' command. in: out: Chain: Residue number: Residue name: Atom serial: Atom name: Tags in: Tags out: Adjunct in: = to Adjunct in: = to Adjunct out: = to Generate query arguments Some tips: Hover over any input field to see a tool tip with an example of valid input. Any input field can be left empty. Both single values and lists (separated with commas, spaces or semicolons) are accepted in the 'Chain-Residue-Atom' input fields (first five rows of input fields). Integer intervals, denoted using ':' symbol, are accepted in 'Residue number' and 'Atom serial' input fields. a : b means an inclusive interval [ a , b ]. Multiple tags can be specified in magenta input fields using logical operators '|' and '&'.
