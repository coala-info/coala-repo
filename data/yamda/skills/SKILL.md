# yamda

## Overview

YAMDA (Yet Another Motif Discovery Algorithm) is a GPU-accelerated software package designed for the rapid identification of DNA motifs. By utilizing the PyTorch framework, it achieves significant speedups over traditional Expectation-Maximization (EM) based tools. It is optimized for processing large-scale genomic data, such as peaks from ChIP-seq or Digital Genomic Footprinting (DGF) experiments, and supports both GPU and CPU execution.

## CLI Usage and Best Practices

The primary entry point for motif discovery is the `run_em.py` script.

### Core Command Pattern
```bash
python run_em.py -i <input.fa> -j <control.fa> -oc <output_directory> [options]
```

### Key Arguments
- `-i`: Path to the positive FASTA file (e.g., masked genomic sequences under peaks).
- `-j`: Path to the negative/control FASTA file (e.g., shuffled sequences).
- `-oc`: Name of the output directory where results and discovered motifs will be stored.
- `-r`: Enable searching on both the forward and reverse strands.
- `-e`: Execute the EM-based discovery refinement.

### Preprocessing Sequences
Before running discovery, it is often necessary to clean FASTA files of low-complexity or "annoying" sequences that can lead to false positives or software errors.
```bash
python erase_annoying_sequences.py -i <raw_input.fa> -o <cleaned_output.fa>
```

### Expert Tips
1. **GPU Acceleration**: Ensure PyTorch is correctly configured with CUDA support to leverage the "thousandfold speedup" mentioned in the documentation. Without a GPU, YAMDA will fall back to CPU, which is functional but significantly slower.
2. **Masking**: Use masked FASTA files (where repeats are N-masked) to prevent the discovery of repetitive elements instead of biological motifs.
3. **Control Sets**: For the best results in ChIP-seq analysis, use a control set (`-j`) generated by shuffling the letters of your input sequences. The `fasta-shuffle-letters` tool from the MEME suite is a recommended companion for this task.
4. **Environment**: YAMDA is optimized for Python 3.6. If using a modern environment, ensure `pyfaidx` and `torch` versions are compatible with the legacy requirements if errors occur.
