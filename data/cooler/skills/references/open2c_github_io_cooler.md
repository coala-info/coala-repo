*Cooler* is a support library for a storage format, also called cooler, used to store genomic interaction data of any size, such as [Hi-C](https://en.wikipedia.org/wiki/Chromosome_conformation_capture) contact matrices.

Simple, sparse matrix [data model](https://cooler.readthedocs.io/en/latest/datamodel.html)

Fast, portable and efficient storage, powered by [HDF5](https://www.hdfgroup.org/solutions/hdf5/)

Extensive command line interface, powered by [Click](https://click.palletsprojects.com/)

Flexible Python API, powered by [NumPy](http://www.numpy.org/), [Pandas](https://pandas.pydata.org/) and [h5py](http://docs.h5py.org/en/stable/)

Multiscale visualization, powered by [HiGlass](https://higlass.io)
