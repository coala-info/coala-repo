Read our preprint on bioRxiv. 1. Demo Output example by paplot. http://genomon-project.github.io/paplot/ 2. Documentation English: http://paplot-doc.readthedocs.org/en/latest/ Japanese: http://paplot-doc.readthedocs.org/ja/latest/ 3. Dependency python &gt;= 2.7 Required web browser for viewing result file. (Tested in the following) Firefox 48 Google Chrome 44 Internet Explorer 11 4. Install git clone https://github.com/Genomon-Project/paplot.git cd paplot python setup.py build install --user 5. Run For example, (using sample data) cd {paplot install directory} # QC Report paplot qc example/qc_brush/data.csv ./tmp demo # Chromosomal Aberration Report paplot ca example/ca_option/data.csv ./tmp demo # Mutation Matrix Report paplot mutation example/mutation_option/data.csv ./tmp demo # Mutational Signature Report paplot signature "example/signature_stack/data*.json" ./tmp demo # pmsignature Report (see https://github.com/friend1ws/pmsignature) paplot pmsignature "example/pmsignature_stack/data*.json" ./tmp demo 6.
