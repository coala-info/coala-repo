## Overview

`nwkit skim` identifies clades with shared trait values and samples a specified number of leaves from each clade.

## 1. Sample one leaf per clade with the same trait value

### Command

```
nwkit skim \
	--infile input.nwk \
	--outfile output.nwk \
	--trait trait.tsv \
	--retain_per_clade 1 \
	--group_by 'photosynthesis_pathway' \
	--prioritize_non_missing 'yes' \
	--filter_by 'busco_completeness' \
	--filter_mode 'descending' \
	--output_groupfile 'yes'
```

### input.nwk

```
(((Rhizophora_mangle,Rhizophora_stylosa),(Euphorbia_lathyris,Euphorbia_maculata)),((Oryza_australiensis,Oryza_meridionalis,Oryza_rufipogon),((Stipagrostis_hirtigluma,(Aristida_congesta,Aristida_purpurea)),((Arundinella_fluviatilis,Arundinella_hirta),(Panicum_bisulcatum,Panicum_coloratum,Panicum_hallii)))));
```

## 2. Sample one leaf per minimal clade with mixed trait values (contrastive clade)

### Command

```
nwkit skim \
	--infile input.nwk \
	--outfile output.nwk \
	--trait trait.tsv \
	--retain_per_clade 1 \
	--group_by 'photosynthesis_pathway' \
	--prioritize_non_missing 'yes' \
	--filter_by 'busco_completeness' \
	--filter_mode 'descending' \
	--only_contrastive_clades 'yes' \ # Return only contrastive clades
	--output_groupfile 'yes'
```

### output.nwk

```
((Euphorbia_lathyris:1,Euphorbia_maculata:1)1:2,(Panicum_bisulcatum:1,Panicum_coloratum:1,Panicum_hallii:1)1:4)1:0;
```
