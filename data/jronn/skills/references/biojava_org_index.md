GitHub project Protein structures BioJava provides an API to maintain local installations of the PDB, load and manipulate structures, perform standard analysis such as sequence and structure alignments and visualize them in 3D. Biological sequences BioJava supports reading and writing popular sequence file formats, translating DNA sequences into proteins and other common bioinformatics routines.
