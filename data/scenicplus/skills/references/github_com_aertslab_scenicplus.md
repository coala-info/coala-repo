Installing To install SCENIC+ (in a Linux environment): We highly recommend to install SCENIC+ in a new conda environment. $ conda create --name scenicplus python=3.11 -y $ conda activate scenicplus $ git clone https://github.com/aertslab/scenicplus $ cd scenicplus $ pip install .
