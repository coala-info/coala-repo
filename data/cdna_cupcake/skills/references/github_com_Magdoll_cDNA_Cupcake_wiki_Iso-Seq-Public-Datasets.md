## Human & Mouse - Single-Cell Iso-Seq

#### (1) Post-natal mouse brain

| Key | Detail |
| --- | --- |
| Species | Mouse, postnatal brain |
| Sequencing | 10X --> Sequel II |
| Analysis | Iso-Seq & [R code](https://github.com/noush-joglekar/scisorseqr) |
| Data | [GEO:GSE158450](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE158450) |

Joglekar et al., "[A spatially resolved brain region- and cell type-specific isoform atlas of the postnatal mouse brain](https://www.nature.com/articles/s41467-020-20343-5)", Nature Communications (2021)

#### (2) Mouse C2C12 cells (ENCODE project)

| Key | Detail |
| --- | --- |
| Species | Mouse, C2C12 cells |
| Sequencing | Parse(SplitSeq) --> Sequel II |
| Analysis | Iso-Seq & [R code](https://github.com/noush-joglekar/scisorseqr) |
| Data | [GEO:GSE168776](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE168776) |

Rebboah et al., "[Mapping and modeling the genomic basis of differential RNA isoform expression at single-cell resolution with LR-Split-seq](https://www.biorxiv.org/content/10.1101/2021.04.26.441522v1)", biorxiv (2021)

## Human - Whole Transcriptome

#### (1) HGSVC Human Transcriptome Samples (1000genomes)

| Key | Detail |
| --- | --- |
| Species | 12 HGSVC ([Human Genome Structural Variation Consortium, Phase 2](https://www.internationalgenome.org/data-portal/data-collection/hgsvc2)) samples; Human |
| Sequencing | Sequel II |
| Analysis | Iso-Seq |
| Data | [1000GenomesFTP](http://ftp.1000genomes.ebi.ac.uk/vol1/ftp/data_collections/HGSVC2/working/20210728_PacBio_Iso-Seq_JAX/) |

#### (2) ENCODE Consortium, various cell lines

| Key | Detail |
| --- | --- |
| Species | Human, various cell lines |
| Sequencing | Sequel I & II |
| Analysis | Iso-Seq + [TALON](https://github.com/mortazavilab/TALON) |
| Data | [ENCODE portal](https://www.encodeproject.org/matrix/?type=Experiment&status=released&assay_slims=Transcription&assay_title=long+read+RNA-seq) |

Wyman et al., "[A technology-agnostic long-read analysis pipeline for transcriptome discovery and quantification](https://www.biorxiv.org/content/10.1101/672931v1)", bioRxiv (2019)

#### (3) Human fetal tissue; HAP1 cells, 2021

| Key | Detail |
| --- | --- |
| Species | XpressRef Universal Total RNA (mixed adult and fetal tissues); HAP1 cells |
| Sequencing | Sequel II, RNA-Seq |
| Analysis | Iso-Seq |
| Data | [GEO:GSE160383](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE160383) |

Troskie et al., "[Long-read cDNA sequencing identifies functional pseudogenes in the human transcriptome](https://genomebiology.biomedcentral.com/articles/10.1186/s13059-021-02369-0)", Genome Biology (2021)

#### (4) Universal Human Reference RNA (UHRR), 2021 Release

| Key | Detail |
| --- | --- |
| Species | Human, UHRR |
| Sequencing | Sequel II |
| Analysis | Iso-Seq |
| Data | [UHRRisoseq2021](https://downloads.pacbcloud.com/public/dataset/UHRRisoseq2021/) |
| Browser | [UCSC Genome Browser Track](https://genome.ucsc.edu/s/Magdoll/2021_UHRR) |

This is PacBio's internal UHRR data release.

An associated publication that used the public UHRR dataset: Kuo et al., "[Illuminating the dark side of the human transcriptome with long read transcript sequencing](https://bmcgenomics.biomedcentral.com/articles/10.1186/s12864-020-07123-7)", BMC Genomics (2020)

#### (5) Alzheimer's Brain, 2020 Release

| Key | Detail |
| --- | --- |
| Species | Human, Alzheimer whole brain |
| Sequencing | Sequel II |
| Analysis | Iso-Seq |
| Data | [AlzIsoSeq](https://downloads.pacbcloud.com/public/dataset/Alzheimer2019_IsoSeq/) |
| Browser | [UCSC Genome Browser Track](https://genome.ucsc.edu/s/Magdoll/2019_Alzheimer8M) |

This is PacBio's internal Alzheimer data release. [PacBio blogpost here](https://www.pacb.com/blog/data-release-alzheimer-brain-isoform-sequencing-iso-seq-dataset/)

## Plant & Animal

#### (1) Arabidopsis thaliana

| Key | Detail |
| --- | --- |
| Species | A. thaliana, various tissues |
| Sequencing | Sequel I |
| Analysis | Iso-Seq & [TAMA](https://github.com/GenomeRIK/tama) |
| Data | SRA PRJNA755474 |

Reference: Zhang et al. "[A high resolution single molecule sequencing-based Arabidopsis transcriptome using novel methods of Iso-seq analysis](https://www.biorxiv.org/content/10.1101/2021.09.02.458763v1?ct=)", biorxiv (2021)

#### (2) Atlantic salmon

| Key | Detail |
| --- | --- |
| Species | Atlantic salmon (Salmo salar) |
| Sequencing | Sequel II |
| Analysis | Iso-Seq & SQANTI |
| Data | [TSA:GIYK00000000](https://www.ncbi.nlm.nih.gov/nuccore/GIYK00000000) |

Ramberg et al., "[A de novo Full-Length mRNA Transcriptome Generated From Hybrid-Corrected PacBio Long-Reads Improves the Transcript Annotation and Identifies Thousands of Novel Splice Variants in Atlantic Salmon](https://www.frontiersin.org/articles/10.3389/fgene.2021.656334/full)", Frontiers in Genetics (2021)

#### (1) Yeast

| Key | Detail |
| --- | --- |
| Species | S. cerevisiae, S288C and CEN.PK |
| Sequencing | Sequel II |
| Analysis | Iso-Seq & custom code |
| Data | [SRA PRJNA58809](https://www.ncbi.nlm.nih.gov/sra/?term=PRJNA588095), genome[S288C](https://www.ncbi.nlm.nih.gov/assembly/GCA_010356925.1), genome[CEN.PK](https://www.ncbi.nlm.nih.gov/assembly/GCA_010356935.1) |

Fiddes et al., "[Long read transcript sequencing of S. cerevisiae reveals complex transcriptional dynamics and improves analysis of scRNA-Seq](https://www.dropbox.com/s/o4qey3830wm70lu/2020BoG_Fiddes_IsoSeq_yeast.pdf?dl=0)", Biology of Genomes poster (2020)
