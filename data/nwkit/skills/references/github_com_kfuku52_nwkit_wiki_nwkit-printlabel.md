## Overview

`nwkit printlabel` searches and prints node labels. The search range can be limited with the `--target` option.

## Examples

### Command

```
nwkit printlabel --infile input.nwk --pattern Populus_trichocarpa_.*
```

### Printing sister node labels

In the above example, `--pattern Populus_trichocarpa_.*` identified all *Populus trichocarpa* genes. By adding `--sister yes` to the above command, `nwkit printlabel` prints the sisters of the *Populus trichocarpa* genes.

