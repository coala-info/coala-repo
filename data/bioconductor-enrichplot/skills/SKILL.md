# bioconductor-enrichplot

## Overview
The `enrichplot` package provides a suite of visualization methods to help interpret functional enrichment results. It supports various output objects from the `clusterProfiler` ecosystem. The package excels at showing the relationships between genes and biological pathways, as well as the overlap between different enriched terms.

## Core Workflow
1. **Input**: An `enrichResult` or `gseaResult` object (typically generated by `clusterProfiler`, `DOSE`, `meshes`, or `ReactomePA`).
2. **Visualization**: Choose a function based on the aspect of the data you wish to highlight (e.g., significance, gene-concept networks, or pathway overlap).
3. **Customization**: Most functions support standard `ggplot2` parameters and specific arguments like `showCategory` to control the number of terms displayed.

## Key Visualization Functions

### 1. Basic Enrichment Plots
- `barplot(obj, showCategory=20)`: Standard bar chart of enriched terms ranked by p-value or count.
- `dotplot(obj, showCategory=20)`: Similar to barplot but encodes gene ratio as size and p-value as color.

### 2. Gene-Concept Network
- `cnetplot(obj, foldChange=NULL)`: Displays the linkages between genes and biological concepts (e.g., GO terms). Useful for identifying genes that belong to multiple pathways.
- Use `circular = TRUE` for a circular layout.

### 3. Enrichment Map
- `emapplot(pairwise_termsim(obj))`: Visualizes the overlap between enriched terms as a network. Terms with high similarity are clustered together. 
- **Note**: You must run `pairwise_termsim()` on your result object before calling `emapplot`.

### 4. Heatmap-like Functional Classification
- `heatplot(obj, foldChange=NULL)`: Displays gene-concept relationships as a heatmap. Useful if `cnetplot` becomes too cluttered.

### 5. GSEA Specific Plots
- `gseaplot2(obj, geneSetID, title = "")`: Provides the classic GSEA running score plot with the ranking metric and hit list.
- `ridgeplot(obj)`: Visualizes the distributions of fold changes for each enriched category.

### 6. Overlap Analysis
- `upsetplot(obj)`: Uses the UpSet layout to show the complex overlap between different gene sets.

## Tips for Effective Visualization
- **Category Selection**: Use `showCategory` to limit the plot to the top N most significant terms to avoid overcrowding.
- **Gene Labels**: In `cnetplot`, use `cex_label_gene` and `cex_label_category` to adjust text sizes.
- **Color Scales**: Since these are `ggplot2` objects, you can append `+ scale_fill_gradient()` or other scales to customize colors.
- **Interactive Plots**: Use `plotly` or `ggplotly()` on these outputs if interactive exploration of gene IDs is required.
