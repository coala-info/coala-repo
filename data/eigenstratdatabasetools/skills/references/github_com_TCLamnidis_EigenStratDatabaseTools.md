These can be installed using pip: pip install sh argparse pandas Eigenstrat Database Tools A tool to check two different EingenStrat databses for shared individuals, and extract or remove individuals from an EigenStrat database. usage: eigenstrat_database_tools.py (-i &lt;Input file prefix&gt;) (-c &lt;input ind file&gt; | -R | -E) [-L &lt;SAMPLE LIST&gt; | -S Ind [-S Ind2]] [-o &lt;OUTPUT FILE PREFIX&gt;] A tool to check two different EingenStrat databses for shared individuals, and extract or remove individuals from an EigenStrat database. Available options: -h, --help show this help message and exit -g &lt;GENO FILE NAME&gt;, --genoFn &lt;GENO FILE NAME&gt; The path to the input geno file. -s &lt;SNP FILE NAME&gt;, --snpFn &lt;SNP FILE NAME&gt; The path to the input snp file. -i &lt;IND FILE NAME&gt;, --indFn &lt;IND FILE NAME&gt; The path to the input ind file. -o &lt;OUTPUT FILES PREFIX&gt;, --Output &lt;OUTPUT FILES PREFIX&gt; The desired output file prefix. Three output files are created, &lt;OUTPUT FILES PREFIX&gt;.geno , &lt;OUTPUT FILES PREFIX&gt;.snp and &lt;OUTPUT FILES PREFIX&gt;.ind . -C &lt;INPUT FILE&gt;, --Check &lt;INPUT FILE&gt; Check the -i .ind file and the second .ind file for duplicate individuals. Population assignment and/or individual sex are not checked, only individual names. Names are case sensitive. -E, --Extract Extract the selected individuals from the EigenStrat database, creating a new set of EigenStrat files with the selected individuals only. -R, --Remove Remove the selected individuals from the EigenStrat database, creating a new set of EigenStrat files without the selected individuals. -L &lt;INDIVIDUAL LIST FILE&gt;, --SampleList &lt;INDIVIDUAL LIST FILE&gt; A list of samples to be Extracted (-E) or Removed (-R) from the database. Can be a list of individual names (1 per line), or the output of duplicate check (-C). Required with -E or -R. Mutually exclusive with -S. -S &lt;INDIVIDUAL&gt;, --Sample &lt;INDIVIDUAL&gt; A samples to be Extracted (-E) or Removed (-R) from the database. Can be called multiple times. Required with -E or -R. Mutually exclusive with -L. -v, --version Print the version and exit. Eigenstrat SNP Coverage A tool to calculate the numbered of covered and total reads for each individual in an EigenStrat dataset. usage: eigenstrat_snp_coverage.py (-i &lt;Input file prefix&gt;) [-s &lt;Input file suffix&gt;] [-o &lt;Output filepath&gt;] [-j | --json] A tool to check two different EingenStrat databses for shared individuals, and extract or remove individuals from an EigenStrat database. Available options: -h, --help show this help message and exit -g &lt;GENO FILE NAME&gt;, --genoFn &lt;GENO FILE NAME&gt; The path to the input geno file. -s &lt;SNP FILE NAME&gt;, --snpFn &lt;SNP FILE NAME&gt; The path to the input snp file. -i &lt;IND FILE NAME&gt;, --indFn &lt;IND FILE NAME&gt; The path to the input ind file. -o &lt;OUTPUT FILEPATH&gt;, --Output &lt;OUTPUT FILEPATH&gt; The filepath where the output table should be saved. Omit to print to stdout. -j &lt;JSON OUTPUT FILEPATH&gt;, --json &lt;JSON OUTPUT FILEPATH&gt; Create additional json formatted output file named &lt;JSON OUTPUT FILEPATH&gt; . -v, --version Print the version and exit. Rename Snps A tool to rename the SNPs in an EigenStrat snp file based on a reference EigenStrat or plink snp file. Uses the genetic coordinate of SNPs to determine identity. usage: rename_snps.py (-i &lt;INPUT .SNP FILE&gt;) (-n &lt;NAMED SNP FILE&gt;) [-f &lt;NAMED FILE FORMAT&gt;] Rename the SNPs in an eigenstrat snp file based on a reference eigenstrat or plink snp file. Uses the genetic coordinate of SNPs to determine identity. Available options: -h, --help show this help message and exit -i &lt;INPUT .SNP FILE&gt;, --Input &lt;INPUT .SNP FILE&gt; The input snp file. -n &lt;NAMED SNP FILE&gt;, --Name_file &lt;NAMED SNP FILE&gt; An Eigenstrat or plink formatted snp file with the desired SNP names. -f &lt;NAMED FILE FORMAT&gt;, --Format &lt;NAMED FILE FORMAT&gt; The format of the desired snp name file. Can be either 'EIGENSTRAT' or 'PLINK' [default: EIGENSTRAT] -v, --version Print the version and exit.
