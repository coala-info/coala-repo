MS-GF+ Documentation MS-GF+ Documentation Contacts PNNL Proteomics [proteomics@pnnl.gov] Sangtae Kim [sangtae.kim (at) gmail.com] Summary MS-GF+ is an MS/MS database search tool that is sensitive (it identifies more peptides than other database search tools and as many peptides as spectral library search tools) and universal (works well for diverse types of spectra, different configurations of MS instruments and different experimental protocols). MS-GF+ supports the HUPO PSI standard input (mzML) and output (mzIdentML).
