# Client Libraries[¶](#client-libraries "Permanent link")

SolveBio has official libraries for different programming languages. All the libraries are open source and welcome contributions from the community.

## Python[¶](#python "Permanent link")

[![PyPI](https://img.shields.io/pypi/v/solvebio.svg)](https://pypi.python.org/pypi/solvebio)

SolveBio for Python is the recommended way to interact with the SolveBio API.

SolveBio for Python is available through [pip](https://pip.pypa.io/en/stable/):

```
pip install --upgrade solvebio
```

The source is available on [GitHub](https://github.com/solvebio/solvebio-python), and the package can be found on [PyPi](https://pypi.python.org/pypi/solvebio).

## R[¶](#r "Permanent link")

[![CRAN_Status_Badge](http://www.r-pkg.org/badges/version/solvebio)](https://cran.r-project.org/package%3Dsolvebio)

SolveBio for R supports a subset of the SolveBio API. You can use it to query datasets and easily pull data into R for further analysis and visualization. A number of [examples](https://github.com/solvebio/solvebio-r/tree/master/examples) are also available.

SolveBio for R is available to anyone with a working [R environment](https://www.r-project.org/):

```
install.packages("solvebio")
```

The source is available on [GitHub](https://github.com/solvebio/solvebio-r), and the package can be found on [CRAN](https://cran.r-project.org/web/packages/solvebio/index.html).

## JavaScript (Node.js)[¶](#javascript-nodejs "Permanent link")

[![npm](https://img.shields.io/npm/v/solvebio.svg)](https://www.npmjs.com/package/solvebio)

SolveBio maintains a lightweight JavaScript library. It can be installed via [npm](https://www.npmjs.com/):

```
npm install solvebio
```

For web-apps, you can install it with [bower](https://bower.io/):

```
bower install solvebio
```

The source is available on [GitHub](https://github.com/solvebio/solvebio-js), and the package can be found on [npm](https://www.npmjs.com/package/solvebio).

## Ruby[¶](#ruby "Permanent link")

[![Gem](https://img.shields.io/gem/v/solvebio.svg)](https://rubygems.org/gems/solvebio)

Ruby client deprecated

As of July 28th, 2017 the SolveBio Ruby client is no longer supported. Many of the endpoints in use have been deprecated. Please contact SolveBio with any questions.

SolveBio for Ruby is available as a [gem](https://rubygems.org/gems/solvebio):

```
sudo gem install solvebio
```

If you use bundler, you can use this line:

```
gem 'solvebio', :git => 'https://github.com/solvebio/solvebio-ruby'
```

The source is available on [GitHub](https://github.com/solvebio/solvebio-ruby), and the package can be found on [RubyGems](https://rubygems.org/gems/solvebio).
