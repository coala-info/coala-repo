Reference If you use slaMEM , please cite: Fernandes, Francisco and Ana T. Freitas. slaMEM: efficient retrieval of maximal exact matches using a sampled LCP array . Bioinformatics 30.4 (2014): 464-471. Manual Install make Usage ./slaMEM ( &lt; options &gt; ) &lt; reference_file &gt; &lt; query_file(s) &gt; Options: mem : find MEMs: any number of occurrences in both ref and query (default) mam : find MAMs: unique in ref but any number in query l : minimum match length (default=20) o : output file name (default="*-mems.txt") b : process both forward and reverse strands n : discard 'N' characters in the sequences m : minimum sequence size (e.g. to ignore small scaffolds) r : load only the reference(s) whose name(s) contain(s) this string Extra: v : generate MEMs map image from this MEMs file Example: ./slaMEM -b -l 10 ./ref.fna ./query.fna ./slaMEM -v ./ref-mems.txt ./ref.fna ./query.fna About Finding Maximal Exact Matches (MEMs) using a Sampled LCP Array Resources Readme License GPL-3.0 license Uh oh!
