# terminus

## Overview
Terminus is a specialized tool designed to address the uncertainty inherent in transcript-level abundance estimation. While tools like Salmon provide estimates for individual isoforms, the underlying sequence similarity often makes these estimates unstable. Terminus analyzes these estimates and collapses transcripts into groups where the total abundance can be confidently and robustly determined. This provides a data-driven resolution for RNA-seq analysis that sits between the often too-granular transcript level and the sometimes too-broad gene level.

## Installation and Setup
Terminus is built using Rust. To use it, ensure you have the Rust toolchain installed.

```bash
# Build from source
cargo build --release

# The executable will be located at:
# ./target/release/terminus
```

## Core Commands and Usage
Terminus operates through two primary sub-commands: `group` and `collapse`.

### 1. Grouping Transcripts
The `group` command identifies which transcripts should be bundled together based on the uncertainty in their abundance estimates.

```bash
terminus group [OPTIONS] --input <salmon_dir> --output <output_dir>
```
*   **--input**: Path to the directory containing Salmon quantification results (specifically looking for the `aux_info` and `quant.sf` files).
*   **--output**: Directory where the grouping information will be stored.
*   **--threshold**: (Optional) Set the uncertainty threshold for grouping.

### 2. Collapsing Abundances
The `collapse` command uses the groups generated in the previous step to produce a new set of abundance estimates at the group level.

```bash
terminus collapse [OPTIONS] --input <salmon_dir> --groups <group_file> --output <output_file>
```
*   **--groups**: The mapping file generated by the `group` command.
*   **--output**: The resulting file containing the collapsed, robust abundance estimates.

## Best Practices
*   **Data-Driven Resolution**: Use Terminus when standard transcript-level analysis shows high variance or when gene-level analysis masks important isoform-specific signals.
*   **Salmon Integration**: Ensure Salmon was run with the `--writeGibbsSamples` or `--writeBootstrap` flags, as Terminus utilizes the posterior samples to estimate uncertainty.
*   **Downstream Analysis**: The output of the `collapse` command can be used in differential expression tools (like DESeq2 or Sleuth) by treating the "groups" as the primary features of interest.



## Subcommands

| Command | Description |
|---------|-------------|
| terminus collapse | analyze a collection of per-sample groups, and produce a consensus grouping. |
| terminus group | perform per-sample grouping of transcripts; required prior to consensus collapse. |
