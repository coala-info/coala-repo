# Welcome to the `commec` wiki!

![commec logo](https://camo.githubusercontent.com/cb4efb1adfff858d5844ac1e1543923660cb9c31602b15d7e42084696fbd3546/68747470733a2f2f69626269732e62696f2f77702d636f6e74656e742f75706c6f6164732f323032352f30362f434f4d4d45435f4c6f676f5f486f72697a5f436f6c6f725f49424249535f6f6e57686974652e706e67)

The `commec` package is a free, open-source tool for DNA sequence screening that is part of the [Common Mechanism for DNA Synthesis screening](https://ibbis.bio/common-mechanism/).

More information about the Common Mechanism project is available on the [International Biosecurity and Biosafety Initiative for Science](https://ibbis.bio/) website, including in the [Common Mechanism FAQ](https://ibbis.bio/our-work/common-mechanism/faq/).

## Getting Started

This [GitHub Wiki](https://github.com/ibbis-screening/common-mechanism/wiki) has documentation for the [package](https://github.com/ibbis-bio/common-mechanism/), including information about installing `commec` and interpreting screening results.

1. [Install Guide](https://github.com/DNA-Screening/CommonMechanism/wiki/install) - quickstart, set up commec and required databases
2. [Tutorial](https://github.com/DNA-Screening/CommonMechanism/wiki/tutorial) - Step-by-step guide to your first screening run
3. [Command Line Usage](https://github.com/DNA-Screening/CommonMechanism/wiki/Command-Line-Usage) - Detailed reference for all commands

## Features

`commec` is designed to:

* Screen sequences down to 50 base pairs in length.
* Sensitively identify sequences of concern known to contribute to pathogenicity
* Flag regulated pathogens, including those listed on the Australia Group Common Control List and
  a variety of national control lists, including those from India, China, and South Africa.

The `commec screen` command runs an input FASTA through the following screening steps:

1. **Biorisk search**: Fast HMM-based search against curated sequence profiles
2. **Taxonomy Search**: look for best matches to regulated pathogens using a two-step process:
   * **Protein search**: BLASTX/DIAMOND search against NCBI nr
   * **Nucleotide search**: BLASTN search against NCBI core\_nt
3. **Low concern search**: Clear earlier flags based on matches to common or conserved sequences

![Flowchart showing decision-making by the common mechanism flag module.](https://camo.githubusercontent.com/e23de7509f00284b3d466c75022e3b7d8331c957851e5b36cf4897c1b11f2857/68747470733a2f2f69626269732e62696f2f77702d636f6e74656e742f75706c6f6164732f323032352f30382f636f6d6d65632d73637265656e696e672d666c6f772d76312e6a7067 "Decision Flow")

Information about the databases supporting screening can be found in the [commec-databases](https://github.com/ibbis-bio/commec-databases/) repostiory.

## Installation

See [Install Guide](wiki/Install) for more details.

A lightweight version of commec, using only the biorisk search to identify certain sequences of concern, can be run in seconds on a laptop using under 1 Gb of curated databases. **This can be used to add resilience to existing screening pipelines, but does not meet full regulatory compliance**.

Comprehensive screening of all regulated pathogens requires the taxonomy search, and is expected to be run in a high-performance bioinformatics computing environment, requiring 275-650 Gb of reference databases and at least 20 Gb of RAM.
