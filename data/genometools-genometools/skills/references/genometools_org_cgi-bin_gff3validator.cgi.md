Annotation file: enable &quot;tidy&quot; mode (tries to fix errors) use Sequence Ontology to validate types and parent-child relationships This GFF3 validator is part of the GenomeTools distribution which you can download to your computer. Use the gt gff3validator tool to validate your own &ndash; possibly larger &ndash; GFF3 files (with option -typecheck so to use the Sequence Ontology).
