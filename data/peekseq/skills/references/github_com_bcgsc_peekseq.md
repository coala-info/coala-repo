Install Clone and enter the peekseq directory. git clone https://github.com/bcgsc/peekseq cd peekseq or using conda: conda install -c bioconda peekseq Dependencies If PERL is installed on your system, you're good to go (no additional libraries needed, nor dependencies). Documentation Refer to the README.md file on how to install and run peekseq. Citing peekseq Thank you for your and for using, developing and promoting this free software! FASTA file (*-codingDNA.fa) A multi-FASTA with sequence region for any given frames with a corresponding translation in -codingPROTEIN.fa, excised from the original input FASTA. DNA/RNA outside of the initiation codon are soft-masked (i.e., lower case bases). FASTA file (*-codingPROTEIN.fa) A multi-FASTA with protein translation corresponding the regions in -codingDNA.fa. Residues outside of the initiation amino acid are soft-masked (i.e., lower case residues). LOG file (.log) Captures the verbose STDOUT in a log file, showing the progress of the program. Algorithm design and implementation Peekseq uses words of length k (k-mers) to quickly assess the coding potential simultaneously over the 6 frames, by sliding over each k frame of the input FASTA.
