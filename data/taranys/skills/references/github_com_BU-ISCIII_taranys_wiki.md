Next-generation sequencing (NGS) is replacing classical Sanger sequencing. The cost reduction on Hardware computers and the increasing speed evolution of NGS makes whole genome sequencing (WGS) a very attractive alternative, for routine analyses, on outbreak surveillance. From WGS data, there are two main methodologies to obtain interpretable information: whole genome Single Nucleotide Polymorphism (wgSNP) analysis or whole genome Multi Locus Sequence Typing (wgMLST) analysis. Taranis uses mainly the wgMLST, where samples are analyzed to identify which allele matches for each gene defined in the cgMSLT schema. Taranis does not create any template report for the analysis, but it creates a lot of information, classified in different files, to be analyzed by bio-informatics.
