# RDML-Tools

Analyze qPCR data.

 [Get help](help.html)  ·  [RDML-Tools](index.html)  ·  [Citation](https://www.gear-genomics.com/citation)  ·  [Source](https://github.com/RDML-consortium/rdml-tools)

#### [RDML-Edit](edit.html)

RDML-Edit allows to view and edit RDML files.

#### [RDML-RunAnalysis](runanalysis.html)

RDML-RunAnalysis analyzes single runs of a RDML file.

#### [RDML-ExperimentAnalysis](experimentanalysis.html)

RDML-ExperimentAnalysis analyzes an experiments of a RDML file.

#### [RDES-TableShaper](tableshaper.html)

RDES-TableShaper reshapes a table in RDES format for the import into RDML.

#### [RDML-RunEdit](runedit.html)

RDML-RunEdit edits errors and notes of a single run.

#### [RDML-AnnotationEdit](annotationedit.html)

RDML-AnnotationEdit edits the annotations of the samples.

#### [RDML-Merge](merge.html)

RDML-Merge combines data from two separate RDML files.

#### [RDML-Validate](validate.html)

RDML-Validate is a tool to validate RDML files against the RDML schema.

#### [RDML-BarGraph](bargraph.html)

RDML-BarGraph creates bar graphs with individual values for publication.

#### [RDML-TestStat](teststat.html)

RDML-TestStat calculates basic statistics on groups.

#### [RDML-RunView](runview.html)

RDML-RunView views single runs of a RDML file.

#### [RDML-Statistics](statistics.html)

RDML-Statistics prints statistics about RDML-Tools usage.

#### [RDML-Tools Help](help.html)

RDML-Tools Help provides an introduction into the RDML-Tools.
