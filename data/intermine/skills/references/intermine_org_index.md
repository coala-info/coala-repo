Get Started With InterMine InterMine Features Dynamic Data Tables InterMine's tables allow you to easily drill down into data. It's easy to filter data, add additional columns, navigate to report pages. Try it yourself! Try clicking on the&nbsp; icon in the ontology term namespace and filtering the column to show only one term. Run queries in many languages Try clicking on the "Generate Python code" button - or use the dropdown to choose a different language. You can then easily copy/paste the queries to your scripts. There's an InterMine for everyone There are many different InterMines worldwide, covering a broad range of model organisms and life science research areas. With InterMine you can explore organism and other research data provided by many different organizations, moving between databases using criteria such as homology. View All Mines AllianceMine AllianceMine integrates many types of data for all the Model Organisms Fly, Worm, Zebrafish, Yeast, Rat, and Mouse.
