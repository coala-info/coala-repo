```text
Usage: wg-blimp run-snakemake [OPTIONS] FASTQ_DIR REFERENCE_FASTA GROUP1
                              GROUP2 OUTPUT_DIR

  as comma seperated lists or are read from text files if --use-sample-files
  parameter is set. Annotation files are automatically downloaded if

  --dry-run                       Only dry-run the pipeline.
  --use-sample-files              Load sample names from text files instead of

  --nodes INTEGER                 Number of nodes to use in cluster mode.
  --help                          Show this message and exit.
```
