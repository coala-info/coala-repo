# bioconductor-restfulsedata

## Overview

Use the Bioconductor R package **restfulSEData** for: The Bioconductor project aims to develop and share open source software for precise and repeatable analysis of biological data. We foster an inclusive and collaborative community of developers and data scientists.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("restfulSEData")
```
