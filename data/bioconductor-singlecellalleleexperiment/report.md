# bioconductor-singlecellalleleexperiment Bioconductor Package Report

## SingleCellAlleleExperiment

### Package Description
The package is part of a workflow named single-cell ImmunoGenomic Diversity (scIGD), that firstly incorporates allele-aware quantification data for immune genes. This new data can then be used with the here implemented data structure and functionalities for further data handling and data analysis.
