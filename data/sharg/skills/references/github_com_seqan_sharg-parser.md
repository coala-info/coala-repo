With Sharg, your command line interface can be customized to your specific needs and validated in a single line. Furthermore, the library provides helpful templates and makes even subcommands feel effortless. For more information, details, and a tutorial on how to use our features, please see our online documentation . If you are working in the field of sequence analysis, we recommend using the SeqAn3 library , which offers a wide range of bioinformatics-related C++ features. Sneak Peek: An application with one option parsing an integer from the command line can be written in only 5 lines of code: # include &lt; sharg/all.hpp &gt; int main ( int argc, char ** argv) { int val{}; sharg::parser parser{ " Eat-Me-App " , argc, argv}; parser. add_subsection ( " Eating Numbers " ); parser. add_option (val, sharg::config{. short_id = ' i ' , . long_id = " int " , . description = " Desc. " }); parser. parse (); return 0 ; } Done. val is automatically filled with the value provided by -i or --int .
