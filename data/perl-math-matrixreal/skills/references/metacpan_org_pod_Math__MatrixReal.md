FUNCTIONS Constructor Methods And Such use Math::MatrixReal; Makes the methods and overloaded operators of this module available to your program. $new_matrix = new Math::MatrixReal($rows,$columns); The matrix object constructor method. A new matrix of size $rows by $columns will be created, with the value 0.0 for all elements. Note that this method is implicitly called by many of the other methods in this module. $new_matrix = $some_matrix-&gt;new($rows,$columns); Another way of calling the matrix object constructor method. However, all must be of the same dimension--no padding happens automatically.
