## Description

Annotation: Evaluating and ranking a set of pathways based on multiple metrics.
Given a set of pathways generated by RetroPath2.0, this workflow informs the user as to the theoretically best performing ones based on the four criteria (target product flux, thermodynamic feasibility, pathway length, and enzyme availability).
