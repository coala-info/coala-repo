cd rotate ; make To run: ./rotate rotate [options] &lt;input fasta filename&gt; -x &lt;pos&gt; position to become new start [0] -rc reverse complement after rotating [false] -s &lt;string&gt; rotate to start of search string -m &lt;n_mismatch&gt; maximum number of mismatches allowed in search [0] -o &lt;outfilename&gt; output file [standard output] use either -x and optionally -rc, or -s use filename '-' for standard input can read .fa.gz files as well as .fa files ./rotate -s tcta test.fa &gt; output.fa sequence 1 length 11 forward 4 nerr 0 sequence 2 length 11 forward 10 nerr 0 sequence 3 length 12 sequence 4 length 11 reverse 10 nerr 0 cat output.fa &gt;1 TCTACGGAACG &gt;2 TCTACGGAAGG &gt;4 TCTACGGAAGG composition You also get for free a handy utility to inspect a (optionally gzipped) fasta or fastq file and return the number of sequences and their lengths, with optional additional information. ./composition Usage: composition [opts] &lt;filename&gt; will read fasta, fastq, optionally gzipped.
