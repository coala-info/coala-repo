## IsoGeneGUI

### Package Description
The package ORCME is based on delta-clustering method (Cheng and Church, 2000) and ORIClust on Order Restricted Information Criterion (Liu et al., 2009), both perform same task but from different perspective and their outputs are clusters of genes. Additionally, profile selection for given gene based on Generalized ORIC (Kuiper et al., 2014) from package goric and permutation test for E2 based on package orQA are included in IsoGene-GUI. None of these four packages has GUI.
