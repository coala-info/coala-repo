Documentation Full documentation is available here ( http://ipyrad.readthedocs.io ), where you will find installation instructions, example tutorials, and a full user guide. Quick Install ipyrad is available through the Python package installer conda with all dependencies included. If you have conda installed you can simply run: conda install ipyrad -c conda-forge -c bioconda If you do not have conda installed then you will enjoy our complete installation instructions .
