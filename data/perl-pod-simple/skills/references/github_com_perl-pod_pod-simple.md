The Pod format is explained in the &lt;perlpod&gt; file shipped with Perl5; the most common formatter is called &lt;perldoc&gt;. Pod formatters can use Pod::Simple to parse Pod documents and render them into plain text, HTML, or any number of other formats. Installation To install this module, type the following: perl Makefile.PL make make test make install Support Questions or discussion about POD and Pod::Simple should be sent to the pod-people@perl.org mail list. Send an empty email to pod-people-subscribe@perl.org to subscribe. This module is managed in an open GitHub repository, &lt; https://github.com/perl-pod/pod-simple/ &gt;. Feel free to fork and contribute, or to clone it and send patches! Patches against Pod::Simple are welcome. Please send bug reports to &lt;bug-pod-simple@rt.cpan.org&gt;.
