Utility code for AToL's Genome Engine. This package provides modules for launching assemblies and annotations based on metadata ingested by the atol-bpa-datamapper . bpa-file-downloader Downloads a file from bioplatforms_url to file_name . Requires the environment variable BPA_APIKEY to be set.
