### 🚀 Features

* Add Sage PSM and fragment parsers for EasyPQP conversion
* Add Sage input options for library generation
* Enhance SagePSMParser with protein token parsing and unique accessions handling
* Add max delta mass option for UniMod annotation in library generation
* Implement table reading function for TSV and Parquet files in SagePSMParser
* Add mz precision option to SagePSMParser and conversion function
* Add convertsage cli command
* Add DataFrame parsing methods for Sage PSM and fragment data to support streaming
* Add streaming options to convertsage for improved performance with large inputs
* Enhance convert\_sage\_streaming for improved memory efficiency and processing speed
* Update streaming threshold to 2GB
* Add EasyPQP In-Silico Library Generation command
* Add EasyPQP In-Silico Library Generation command
* Enhance insilico\_library options with RT scaling, report generation, Parquet output, and threading support
* Add insilico library generation test and configuration files
* Add matched\_fragments and results data files for testing
* Implement optional in-silico library generation with rust backend support
* Add CI workflow for testing across multiple Python versions
* Add concurrency configuration to CI workflow for improved efficiency

### 🐛 Bug Fixes

* Correct documentation for output columns in SagePSMParser
* Handle import for transform\_pi0\_lambda across PyProphet versions
* Precursor mz calculcation
* Enhance \_basename\_wo\_ext to handle common compression extensions
* Improve logging messages for streaming conversion in convert\_sage
* Add missing easypqp\_rs dependency in pyproject.toml
* Remove timestamps from convertsage output for deterministic testing
* Enhance output cleaning in \_run\_convertsage for deterministic testing
* Pyprophet import error
* Update version to 0.1.54 in pyproject.toml
* Update changelog workflow to allow any tag and improve version input description

* Update README to clarify installation of optional features and in-silico library generation
