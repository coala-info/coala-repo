# bustools

## Overview
The `bustools` skill provides a specialized workflow for handling the BUS format, typically generated by `kallisto`. It transforms raw pseudoalignment output into analysis-ready digital expression matrices. This tool is essential for single-cell pipelines requiring high-speed processing, memory efficiency, and modularity in handling barcodes and UMIs.

## Core Workflow
A standard `bustools` pipeline follows a specific sequence to ensure data integrity and biological relevance:

1.  **Sort**: Organize the BUS file by barcode, UMI, and equivalence class.
2.  **Correct**: Perform barcode error correction against a technology-specific on-list.
3.  **Sort (Again)**: Re-sort the corrected file to group identical corrected barcodes.
4.  **Count**: Aggregate the processed records into a gene or TCC (Transcript Compatibility Count) matrix.

## Command Reference and Best Practices

### Sorting BUS Files
Sorting is a prerequisite for almost all other `bustools` commands.
```bash
bustools sort -t 4 -m 4G -o output_sorted.bus input.bus
```
- **Tip**: Use the `-T` flag to specify a temporary directory if your system's default `/tmp` has limited space, as sorting large datasets creates significant temporary overhead.

### Barcode Error Correction
Correct sequencing errors in barcodes using a whitelist (on-list).
```bash
bustools correct -w whitelist.txt -o output_corrected.bus input_sorted.bus
```
- **Requirement**: The input must be sorted. The on-list should be a simple text file with one barcode per line.

### Generating Count Matrices
Convert BUS records into a matrix format (Market Matrix `.mtx`).
```bash
bustools count -o matrix_folder/ -g transcripts_to_genes.txt -e matrix.ec -t transcripts.txt --genecounts input_corrected_sorted.bus
```
- **Gene Mapping**: The `-g` file is a tab-separated file mapping transcript IDs to gene IDs.
- **Equivalence Classes**: The `-e` file (ecmap) is generated during the `kallisto bus` step.
- **Output**: This command creates three files in the output folder: `genes.mtx`, `genes.genes.txt`, and `genes.barcodes.txt`.

### Inspection and Quality Control
Generate a summary report of the BUS file to check sequencing depth and library complexity.
```bash
bustools inspect -o report.json -e matrix.ec -w whitelist.txt input_sorted.bus
```
- **Performance**: Running `inspect` without `-e` and `-w` is significantly faster if you only need basic record counts.

### Conversion to Text
For manual inspection or custom scripting, convert binary BUS files to tab-delimited text.
```bash
bustools text -o output.txt input.bus
```
- **Warning**: Text files are much larger than binary BUS files; use this only for small subsets or debugging.

## Expert Tips
- **Piping**: Many `bustools` commands support the `-p` or `--pipe` flag, allowing you to stream data between commands (e.g., `sort` to `correct`) to save disk I/O.
- **Memory Management**: When using `sort`, the `-m` flag should be set to roughly 70-80% of available RAM to prevent OOM (Out of Memory) errors while maintaining speed.
- **Multi-threading**: Use the `-t` flag in `sort` to leverage multiple cores, as this is the most computationally intensive step.



## Subcommands

| Command | Description |
|---------|-------------|
| bustools allowlist | Generates an allowlist (on-list) of barcodes from a sorted BUS file based on a frequency threshold. |
| bustools capture | Capture records from a BUS file based on a list of transcripts, UMIs, barcodes, or flags. |
| bustools compress | Compress a BUS file, which should be sorted by barcode-umi-ec. |
| bustools correct | Correct barcodes in BUS files using an on-list |
| bustools count | Count gene matrix files from sorted BUS files |
| bustools decompress | Decompress or inflate a compressed BUS file |
| bustools extract | Extract reads from FASTQ files based on a sorted BUS file. Note: BUS file should be sorted by flag using bustools sort --flag. |
| bustools fromtext | Convert text files to BUS format |
| bustools inspect | Inspects a sorted BUS file and provides statistics |
| bustools sort | Sort BUS files by barcode, UMI, ec, and flag. |
| bustools text | Convert BUS files to text format |
| bustools umicorrect | Correct UMIs in sorted BUS files |
