DLCpar is a reconciliation package that maps a gene tree to a species tree by inferring gene duplications, losses, and coalescence (accounting for incomplete lineage sorting). DLCpar uses the labeled coalescent tree (LCT) to infer the species and locus to which a gene belongs. More detail can be found at the project website , manual , and examples . Authors Yi-Chieh Wu Morgan Carothers, Joseph Gardi, Gianluca Gross, Tatsuki Kuze, Nuo Liu, Fiona Plunkett, Julia Qian (ILP formulation) Ross Mawhorter and Nuo Liu (cost landscapes) Haoxing Du and Yi Sheng Ong (multiple optimal reconciliations) Matthew Rasmussen (compbio libraries) Citations If you use dlcpar, please cite Wu, Rasmussen, Bansal, Kellis.
