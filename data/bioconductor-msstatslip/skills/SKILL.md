# bioconductor-msstatslip

## Overview
MSstatsLiP is an R package designed for the analysis of Limited Proteolysis-coupled Mass Spectrometry (LiP-MS) experiments. It provides tools to identify structural changes in proteins by comparing proteolytic patterns across different conditions. The package handles both LiP data (peptides generated by limited proteolysis followed by trypsin digestion) and TrP data (trypsin-only control for protein abundance), allowing for the adjustment of LiP peptide changes based on total protein levels.

## Typical Workflow

### 1. Data Preprocessing and Conversion
The package requires data in a specific format. Use built-in converters for tools like Spectronaut.

```r
library(MSstatsLiP)

# Convert Spectronaut output to MSstatsLiP format
# Requires LiP raw data, FASTA file, and optional TrP (Protein) raw data
msstats_data <- SpectronauttoMSstatsLiPFormat(
  LiPRawData, 
  fasta_file = "path/to/file.fasta", 
  TrPRawData = TrPRawData
)

# Ensure condition names match between LiP and TrP datasets if they differ
# Example: msstats_data[["LiP"]]$Condition <- "Ctrl"
```

### 2. Data Summarization
Summarize feature intensities into run-level values.

```r
MSstatsLiP_Summarized <- dataSummarizationLiP(
  msstats_data,
  MBimpute = FALSE,
  normalization.LiP = "equalizeMedians"
)

# Access summarized data
lip_protein_data <- MSstatsLiP_Summarized[["LiP"]]$ProteinLevelData
trp_protein_data <- MSstatsLiP_Summarized[["TrP"]]$ProteinLevelData
```

### 3. Differential Analysis (Modeling)
Perform group comparisons to find significant structural changes. This function automatically calculates:
1. **LiP Model**: Peptide changes without protein correction.
2. **TrP Model**: Changes in total protein abundance.
3. **Adjusted LiP Model**: Peptide changes corrected for protein abundance.

```r
MSstatsLiP_model <- groupComparisonLiP(
  MSstatsLiP_Summarized,
  fasta = "path/to/file.fasta"
)

# View results
head(MSstatsLiP_model$Adjusted.LiP.Model)
```

### 4. Proteolytic Resistance Analysis
Calculate the ratio of fully tryptic peptides in LiP vs. TrP conditions to measure protease accessibility.

```r
# Calculate resistance ratios
Accessibility <- calculateProteolyticResistance(
  MSstatsLiP_Summarized, 
  fasta_file = "path/to/file.fasta", 
  differential_analysis = TRUE
)

# View differential resistance results
head(Accessibility$groupComparison)
```

## Visualization Functions

- **Tryptic Histogram**: `trypticHistogramLiP(MSstatsLiP_Summarized, fasta_file)`
- **Correlation Plot**: `correlationPlotLiP(MSstatsLiP_Summarized)`
- **QC/Profile Plots**: `dataProcessPlotsLiP(MSstatsLiP_Summarized, type = 'QCPLOT')` or `type = 'ProfilePlot'`
- **PCA Plot**: `PCAPlotLiP(MSstatsLiP_Summarized, comparison.pca = TRUE)`
- **Volcano/Heatmap**: `groupComparisonPlotsLiP(MSstatsLiP_model, type = "VolcanoPlot")`
- **Structural Barcode**: `StructuralBarcodePlotLiP(MSstatsLiP_model, fasta_file, model_type = "Adjusted", which.prot = "P12345")`
- **Resistance Barcode**: `ResistanceBarcodePlotLiP(Accessibility, fasta_file, which.prot = "P12345", differential_analysis = TRUE)`

## Key Tips
- **FULL_PEPTIDE Column**: Ensure the `FULL_PEPTIDE` column in LiP data contains both Protein and Peptide info (e.g., `ProteinID_PeptideSequence`) to allow proper summarization.
- **Trypticity**: Use `calculateTrypticity()` to annotate peptides as fully or half-tryptic based on the provided FASTA file.
- **Protein Correction**: Always provide TrP data when possible to ensure that observed LiP changes are due to structural alterations rather than just changes in protein concentration.
