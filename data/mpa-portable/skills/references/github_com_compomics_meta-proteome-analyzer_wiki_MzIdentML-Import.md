This feature is particularly useful for analyzing/reprocessing protein result data that have been generated using external tools or for data obtained from the public domain PRIDE database. The following tools already support the mzIdentML format (MPA is not listed there yet...): http://www.psidev.info/tools-implementing-mzidentml Project Creation and MzIdentML File Import: For the actual file import, an MPA project needs to be created first (in the Project tab). It is then required to navigate to the menu called "Import" at the top and click on the menu item "MzIdentML File" (see screenshot below). This will allow to import database search engine results in mzIdentML format and the respective result data. Please note that further protein analysis steps (e.g. grouping of proteins to meta-proteins) can be performed under the "View Results" tab.
