# bioconductor-cosmiq

## Overview

Use the Bioconductor R package **cosmiq** for: the package xcms as an alternative preprocessing step.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("cosmiq")
```
