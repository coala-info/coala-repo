# Installation[¶](#installation "Permalink to this headline")

## Via pip[¶](#via-pip "Permalink to this headline")

```
$ pip install enasearch
```

## Via Conda[¶](#via-conda "Permalink to this headline")

```
$ conda install -c bioconda enasearch
```

## On Galaxy[¶](#on-galaxy "Permalink to this headline")

ENASearch is also available as tool for [Galaxy](https://galaxyproject.org/). It is available on the [ToolShed](https://toolshed.g2.bx.psu.edu/view/iuc/suite_enasearch/).
