# HEASoft

A Unified Release of the FTOOLS and X-ray Analysis Software Packages

The current version of HEASoft is **[6.36](download.html)**

(30 September 2025)

---

## Included Packages

* [FTOOLS](/docs/software/ftools/ftools_menu.html):
  General and mission-specific tools to manipulate FITS files.
* [FITSIO](/docs/software/fitsio/fitsio.html):
  Core library responsible for reading and writing FITS files.
* [FV](/docs/software/ftools/fv/):
  General FITS file browser/editor/plotter with a graphical user interface.
* [Xspec](/docs/software/xspec/):
  X-ray astronomical spectral data analysis.
* [Xronos](/docs/software/xronos/xronos.html):
  X-ray astronomical timing data analysis.
* [Ximage](/docs/software/ximage/ximage.html):
  X-ray astronomical imaging data analysis.
* [XSTAR](/docs/software/xstar/xstar.html):
  Tool for calculating the physical conditions and emission spectra of photoionized gases.

---

## [Download](download.html)

Users may download either the source code for the software and compile the tools themselves,
or download precompiled executables for the most widely used computer platforms.

The [HEASoft Release Notes 6.36](release_notes.html) give an overview
of changes since HEASoft version 6.35.2.

For a list of known issues in the latest release, visit the [HEASoft Known Issues](issues.html) page.

## Install

For help [installing HEASoft](install.html) on your system, please visit the appropriate link here:

* [PC Linux](linux.html)
* [macOS](macos.html)
* [conda](/docs/software/conda.html) (Linux or Mac)

See also the [video walkthrough](install-heasoft-mac-tutorial.html)
describing the installation process on a Mac.

For guidance regarding removal of HEASoft from your system, visit the
[uninstall](uninstall.html) page.
