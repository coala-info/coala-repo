```text
usage: linkedsv.py [-h] -i input.phased_possorted_bam.bam -d output_directory
                   -r ref.fa [-v version] [--gap_region_bed BED]
                   [--black_region_bed BED] [-t num_thread]
                   [--min_fragment_length INT] [--min_reads_in_fragment INT]
                   [--min_supp_barcodes INT] [--samtools path/to/samtools]
                   [--bedtools path/to/bedtools] [--wgs] [--targeted]
                   [--germline_mode] [--somatic_mode] [--target_region BED]
                   [--gap_distance_cut_off INT] [--save_temp_files]
                   [--version]

Detection of SVs from linked-read sequencing data

optional arguments:
  -h, --help            show this help message and exit
  -i input.phased_possorted_bam.bam, --bam input.phased_possorted_bam.bam
                        input bam file (should be the phased_possorted_bam.bam
                        generated by Longranger
  -d output_directory, --out_dir output_directory
                        output directory
  -r ref.fa, --ref ref.fa
                        reference FASTA file
  -v version, --ref_version version
                        version of reference fasta file. Current supported
                        versions are: hg19, b37, hg38
  --gap_region_bed BED  reference gap region in bed format, required if
                        --ref_version is not specified
  --black_region_bed BED
                        black region in bed format, required if --ref_version
                        is not specified
  -t num_thread, --n_thread num_thread
                        number of threads (default: 4)
  --min_fragment_length INT
                        minimal fragment length considered for SV calling
  --min_reads_in_fragment INT
                        minimal number of confidently mapped reads in one
                        fragment
  --min_supp_barcodes INT
                        minimal number of shared barcodes between two SV
                        breakpoints (default: 10)
  --samtools path/to/samtools
                        path to samtools (default: find in environmental path)
  --bedtools path/to/bedtools
                        path to bedtools (default: find in environmental path)
  --wgs                 the input is whole-genome sequencing data
  --targeted            the input is targeted region sequencing data (such as
                        WES)
  --germline_mode       detect germline SVs
  --somatic_mode        detect somatic SVs (with low variant allele frequency)
  --target_region BED   bed file of target regions (required if --targeted is
                        specified)
  --gap_distance_cut_off INT
                        max distance between two reads in a HMW DNA molecule
                        (default: automatically determined)
  --save_temp_files     Do not remove intermediate files after the run. Use in
                        debug mode. Default: False
  --version             show program's version number and exit
```
