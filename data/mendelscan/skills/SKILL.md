# mendelscan

## Overview

MendelScan is a Java-based bioinformatics tool designed to streamline the identification of disease-causing variants in Mendelian disorder studies. It functions by integrating raw variant calls (VCF) with family relationship data (PED), functional annotations from the Variant Effect Predictor (VEP), and gene expression profiles. By combining these disparate data sources, it provides a prioritized list of candidates, significantly reducing the manual effort required to filter through thousands of exome variants.

## Command Line Usage

The primary function of MendelScan is the `score` subcommand, which evaluates variants based on the provided input files.

### Basic Scoring Pattern

```bash
java -jar MendelScan.jar score <variants.vcf> \
    --vep-file <annotation.vep> \
    --ped-file <family.ped> \
    --output-file <results.tsv> \
    --output-vcf <results.vcf>
```

### Advanced Prioritization

To further refine results using gene expression data, include the `--gene-file` flag:

```bash
java -jar MendelScan.jar score variants.vcf \
    --vep-file annotation.vep \
    --ped-file family.ped \
    --gene-file gene-expression.txt \
    --output-file mendelscan_prioritized.tsv
```

## Key Parameters and Inputs

- **VCF File**: The standard input containing variant calls for the individual or family.
- **--vep-file**: Functional annotations generated by Ensembl's Variant Effect Predictor. Ensure the VEP version is compatible with the reference genome used for variant calling.
- **--ped-file**: A standard pedigree file defining family relationships, sex, and phenotype status. This is critical for filtering based on inheritance patterns (e.g., autosomal recessive, de novo).
- **--gene-file**: A text file containing gene expression data, used to weight variants in genes that are relevant to the tissue or condition being studied.
- **--output-file**: Generates a tab-separated (TSV) file containing the scored and ranked variants.
- **--output-vcf**: Produces a VCF file containing the original variants with MendelScan scores added to the INFO field.

## Expert Tips

- **Build from Source**: If the JAR file is not provided, use Apache Ant to build the tool:
  - `ant compile` to build the Java source.
  - `ant dist` to generate the `MendelScan.jar` file in the `dist/` directory.
- **Memory Management**: Since MendelScan processes large VCF and VEP files in Java, you may need to increase the heap size for large datasets using the `-Xmx` flag (e.g., `java -Xmx4g -jar MendelScan.jar ...`).
- **IBD Thresholds**: For specific identity-by-descent analysis, the tool supports an `ibd-score-threshold` within related subcommands like `sibd` to filter shared segments between siblings.
