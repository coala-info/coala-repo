# msweep

## Overview

mSWEEP (metagenomic Sweep) is a specialized tool for estimating the composition of bacterial communities from metagenomic sequencing data. By processing pseudoalignments—most commonly generated by Themisto—mSWEEP applies variational inference to provide high-resolution abundance estimates, even at the within-species level. It is particularly useful for researchers needing to quantify specific lineages or bin reads into taxonomic groups for downstream analysis like assembly or quality checking.

## Installation

The recommended way to install mSWEEP is via Bioconda:

```bash
conda install -y -c bioconda -c conda-forge msweep
```

## Common CLI Patterns

### Basic Abundance Estimation
To estimate relative abundances from paired-end pseudoalignments:

```bash
mSWEEP --themisto-1 fwd.txt --themisto-2 rev.txt -i clustering.txt -t 8 -o result_abundances
```
*   `-i`: Path to the group indicators (clustering) file.
*   `-t`: Number of threads to use.
*   `-o`: Prefix for the output file.

### Binning Reads
To extract reads belonging to specific lineages or those exceeding an abundance threshold:

```bash
# Bin reads for all lineages with >1% relative abundance
mSWEEP --themisto fwd.txt,rev.txt -i clustering.txt --min-abundance 0.01

# Extract reads only for specific target lineages
mSWEEP --themisto fwd.txt,rev.txt -i clustering.txt --target-groups lineage_A,lineage_B
```

### Handling Large or Sparse Datasets
For complex samples or large-scale data, use these optimizations:

*   **Compressed Input**: mSWEEP can read compressed `.aln` files directly to save disk space.
*   **Sparse Alignments**: Use `--min-hits` to ignore reference groups with few or no alignments, significantly reducing memory usage.
    ```bash
    mSWEEP --themisto compressed.aln -i clustering.txt --min-hits 1
    ```

### Reliability and Quality Control
To assess the confidence of the abundance estimates, use the experimental RATE method:

```bash
mSWEEP --themisto compressed.aln -i clustering.txt --run-rate
```
*   This appends **RATE** and **KLD** columns to the output.
*   **Tip**: A lineage is generally considered reliably estimated if its RATE value exceeds `1 / (number of lineages)`.

## Expert Tips

*   **Themisto Integration**: mSWEEP is designed to work seamlessly with Themisto. Ensure your pseudoalignments are generated against the same reference clustering file used in the `-i` argument.
*   **Memory Management**: If running out of memory on large references, ensure you are using the `--min-hits 1` flag. This prunes the likelihood matrix of groups that have no evidence in your specific sample.
*   **GPU Acceleration**: If your version is compiled with LibTorch support, you can use `--algorithm rcggpu` for faster processing on compatible hardware.
*   **Downstream Validation**: For binned reads, it is best practice to run `demix_check` or `checkm` on the resulting bins to evaluate the accuracy of the community reconstruction.
