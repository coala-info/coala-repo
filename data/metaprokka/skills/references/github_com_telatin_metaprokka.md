Prokka is a software tool to annotate bacterial, archaeal and viral genomes quickly and produce standards-compliant output files. Installation (WIP) If you use Conda (or mamba ) you can use the Bioconda channel : mamba install -c conda-forge -c bioconda metaprokka Usage prokka-gv Prokka with prodigal-gv is called prokka-gv , this is particularly useful for phage genomes (and will detect the genetic code). You can download the Phrogs HMM, which will be automatically placed in the conda folder. get-phrogs and then run prokka-gv (which will detect the Phrogs HMM): prokka-gv --outdir OUT_DIR --prefix PREFIX --locustag LOCUSTAG --cpus 16 input.fa metaprokka Metaprokka is optimised for metagenomes, is better suited at fragmented assemblies and will produce less output files (see wiki) Licence GPL v3 Citation This tool is a modified version of Prokka, so if you use it please cite: Seemann T.
