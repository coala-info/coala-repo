## Overview

`nwkit info` prints various information on the input tree.

By default, species counts are derived from leaf labels that follow `GENUS_SPECIES` or `GENUS_SPECIES_SUFFIX`. If your tree uses a different label convention, provide `--species_regex`. When the regex contains capture groups, non-empty captured groups are joined with underscores.

## Examples

### Command

```
nwkit info --infile input.nwk
```

## Custom species parsing

```
nwkit info --infile input.nwk \
  --species_regex '^([A-Za-z]+)\.([A-Za-z]+)\|'
```

With labels such as `Homo.sapiens|GENE1`, this reports species names such as `Homo_sapiens`.
