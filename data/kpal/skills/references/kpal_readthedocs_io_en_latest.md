# Welcome to kPAL (k-mer profile analysis library)![¶](#welcome-to-kpal-k-mer-profile-analysis-library "Permalink to this headline")

kPAL is an analysis toolkit and Python programming library for *k*-mer
profiles.

Note

Please cite the following paper if you use kPAL in your own work:

Anvar et al., [Determining the quality and complexity of next-generation
sequencing data without a reference
genome](http://genomebiology.com/2014/15/12/555). *Genome Biology* 2014,
**15**:555. doi:10.1186/s13059-014-0555-3
