# dmtools

## Overview
dmtools is a high-performance suite designed to handle the large-scale data generated by various bisulfite sequencing methods. It utilizes a specific binary format (.dm) that allows for rapid random access and efficient storage of methylation information. This skill provides the necessary command-line patterns to transform raw alignment data into actionable methylation metrics, enabling downstream analysis like differential methylation calling and visualization of methylation patterns across genomic features.

## Core Workflows

### 1. Data Conversion and Indexing
The primary entry point is converting aligned BAM files into the `.dm` format.
- **From BAM**: `dmtools parse -i input.bam -o output.dm`
- **From SAM**: `dmtools parse -S -i input.sam -o output.dm`
- **Note**: Ensure the BAM file is sorted by coordinate before parsing to ensure proper indexing.

### 2. Methylation Level Calculation
Extract methylation levels for specific genomic regions or at single-base resolution.
- **Single-base resolution**: `dmtools view -i input.dm > output.txt`
- **Region-based**: `dmtools region -i input.dm -r regions.bed -o levels.txt`
- **Stats by chromosome**: `dmtools stats -i input.dm`

### 3. Differential Methylation Analysis
Compare two or more `.dm` files to identify differentially methylated regions (DMRs).
- **Basic DMR calling**: `dmtools dmr -1 control.dm -2 treatment.dm -o dmr_results.txt`
- **Parameters**: Use `-c` to set minimum coverage and `-p` to adjust the p-value threshold for statistical significance.

### 4. Profile Generation
Generate profiles for visualization (e.g., around TSS or gene bodies).
- **Profile**: `dmtools profile -i input.dm -r genes.bed -o profile.txt`

## Best Practices
- **Memory Management**: For extremely large datasets, use the `-m` flag during parsing to limit memory usage, though this may increase processing time.
- **Strand Specificity**: By default, dmtools handles strand information. If your library prep is non-directional, ensure you use the appropriate flags during the `parse` step to avoid miscalculating levels.
- **Filtering**: Use the `-q` flag to filter out low-quality alignments (MAPQ) during the initial parsing phase to improve the accuracy of methylation calls.

| dmtools | A collection of tools for DNA methylation analysis. |
| dmtools align | Align reads to a genome. |

| dmtools ebsrate | Calculate bisulfite conversion rate from DM file. |
| dmtools index | Index a genome for dmtools. |
| dmtools mr2dm | Convert methylation ratio files to DM format. |
| dmtools overlap | Calculate overlap between two DM files. |
| dmtools regionstats | Calculate methylation statistics for regions in a DM file. |
| dmtools view | View mode parameters |
| dmtools_addzm | add zoom levels for dm |

| dmtools_merge | Merge multiple DM files |

| dmtools_stats | Calculate statistics from DM files. |
| dmtools_viewheader | View header of a DM file |
