# decenttree

## Overview
DecentTree is a high-performance phylogenetic inference tool optimized for the genomic era. It provides a scalable alternative to traditional distance-based methods by implementing parallelized and vectorized versions of algorithms like Neighbor-Joining and BIONJ. It is specifically designed to handle massive datasets that would otherwise be computationally prohibitive for standard phylogenetic software, producing trees in standard Newick format.

## Installation
The most reliable way to install DecentTree is via Bioconda:
```bash
conda install bioconda::decenttree
```

## Common CLI Patterns

### Basic Tree Inference
To build a tree from a FASTA alignment using the default algorithm (typically NJ or BIONJ):
```bash
decenttree -f input_alignment.fasta > output_tree.nwk
```

### Working with Distance Matrices
If you already have a distance matrix in PHYLIP format:
```bash
decenttree -p input_matrix.dist > output_tree.nwk
```

### Algorithm Selection
DecentTree supports several distance-based methods. While specific flags may vary by version, the tool generally allows selection between:
- **NJ**: Standard Neighbor-Joining.
- **BIONJ**: An improved version of NJ (often the preferred default for accuracy).
- **UPGMA**: Unweighted Pair Group Method with Arithmetic Mean.
- **UNJ**: Unweighted Neighbor-Joining.

### Performance and Parallelization
Since DecentTree uses OpenMP, you should specify the thread count to take advantage of multi-core processors:
```bash
decenttree -nt <number_of_threads> -f input.fasta
```

### Handling Large Datasets
For datasets exceeding 10,000 sequences, ensure you are using the optimized versions (often referred to as "Rapid" or "Vectorized" variants within the tool) to minimize memory overhead and execution time.

## Expert Tips
- **Algorithm Choice**: Use **BIONJ** over standard NJ whenever possible, as it better accounts for the variance of evolutionary distance estimates.
- **UPGMA Warning**: Be aware that UPGMA trees generated by DecentTree may not always be strictly ultrametric; verify results if your downstream analysis requires strict ultrametricity.
- **Input Pre-processing**: DecentTree can handle compressed distance and PHYLIP files directly, which is useful for saving disk space on very large projects.
- **Integration with IQ-TREE**: DecentTree's core logic is derived from IQ-TREE 2. It is an excellent tool for generating high-quality starting trees for more intensive Maximum Likelihood (ML) searches.
- **Sequence Comparison**: The tool includes custom vectorized sequence comparison functions. If building from FASTA, it will calculate the distance matrix internally before tree construction.
