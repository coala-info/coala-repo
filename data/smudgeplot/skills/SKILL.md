# smudgeplot

## Overview
Smudgeplot is a bioinformatic tool designed for reference-free genome profiling. It extracts heterozygous k-mer pairs from sequencing reads and analyzes the relationship between the sum of their coverages (CovA + CovB) and their relative coverage (CovB / (CovA + CovB)). This analysis produces a 2D histogram where "smudges" represent different haplotype structures. The position and intensity of these smudges allow researchers to disentangle complex genome structures, including those with duplications or high ploidy levels.

## Core Workflow

The standard smudgeplot workflow requires a k-mer database (typically generated by FastK) as input.

### 1. K-mer Pair Extraction
Use the `hetmers` task to find unique k-mer pairs. This is the most computationally intensive step.

```bash
smudgeplot hetmers -L <lower_cutoff> -t <threads> -o <output_prefix> <FastK_Table>
```
- `-L`: The lower coverage cutoff. A common rule of thumb is 0.5 times the haploid k-mer coverage.
- `-o`: Prefix for the output `.smu` file.

### 2. Ploidy Inference and Plotting
Use the `all` task to process the `.smu` file, infer ploidy, and generate the final visualizations.

```bash
smudgeplot all -o <output_prefix> <kmerpairs.smu>
```
- This generates three PDFs, summary tables, and logs.
- If the y-axis (coverage) needs adjustment, use the `-ylim` parameter (default is 100).

## Expert Tips and Best Practices

### Integration with GenomeScope
Smudgeplot should be used jointly with GenomeScope. 
1. Run GenomeScope on your k-mer histogram first to estimate the haploid k-mer coverage (1n).
2. Use the 1n estimate to set the `-L` (lower) and `-U` (upper) cutoffs for `smudgeplot hetmers`.
3. **Lower Cutoff (-L):** Typically set to 0.5 * 1n to exclude sequencing errors.
4. **Upper Cutoff (-U):** Typically set to 8.5 * 1n to exclude high-copy repeats.

### Handling High Coverage
For very high coverage datasets, the `hetmers` task can be memory-intensive. Ensure you have sufficient RAM or consider subsampling if the k-mer database is excessively large.

### Interpreting the Plot
- Each smudge represents a unique haplotype structure.
- The "heat" (color intensity) of a smudge indicates the frequency of that structure in the genome.
- A triploid genome, for example, will show distinct smudges corresponding to AAB and ABB structures.

### Task-Specific Commands
If you need to run steps individually rather than using `smudgeplot all`:
- `smudgeplot cutoff`: Helps calculate meaningful values for the lower k-mer histogram cutoff.
- `smudgeplot plot`: Generates the 2D histogram and performs inference (use this if you already have the aggregated data).
- `smudgeplot peak_aggregation`: Uses a local aggregation algorithm to group smudges.
