# ucsc-axttopsl

## Overview
The `axtToPsl` utility is a specialized tool from the UCSC Genome Browser "kent" source suite. It transforms AXT alignment files—which typically contain the actual sequence of the alignment blocks—into PSL (Pattern Space Layout) files. PSL files are the standard format for BLAT and other UCSC tools, describing the alignment coordinates and match statistics without storing the sequence itself. This conversion is essential for integrating custom alignments into the UCSC Genome Browser or for calculating alignment coverage and identity metrics.

## Usage Patterns

### Basic Conversion
The tool requires the input AXT file, the sizes of the target and query sequences, and the output filename.

```bash
axtToPsl input.axt target.sizes query.sizes output.psl
```

*   **input.axt**: The source alignment file.
*   **target.sizes**: A tab-separated file containing `<chrom> <size>` for the target genome.
*   **query.sizes**: A tab-separated file containing `<chrom> <size>` for the query genome.

### Generating Chromosome Sizes
If you do not have the `.sizes` files, you can generate them from 2bit or FASTA files using `faSize`:

```bash
# For a 2bit file
twoBitInfo genome.2bit stdout | sort -k2nr > genome.sizes

# For a FASTA file
faSize -detailed genome.fa > genome.sizes
```

## Expert Tips and Best Practices

### Handling Large Alignments
AXT files can be extremely large because they contain sequence data. Converting to PSL significantly reduces file size, making it easier to perform downstream operations like sorting or filtering.

### Downstream Processing
PSL files generated by `axtToPsl` are typically unsorted. For use in the Genome Browser or with other UCSC utilities, you should sort them:

```bash
pslSort dirs stdout tmp/ input.psl | pslCDnaFilter -minId=0.9 stdin output.filtered.psl
```

### Validation
Always ensure that the chromosome names in your AXT file exactly match the names in your `.sizes` files. Discrepancies in naming (e.g., "chr1" vs "1") will cause the tool to fail or produce incorrect coordinate mappings.

### Permission Issues
If running the binary directly from a download (rather than via Conda), ensure the execution bit is set:
```bash
chmod +x axtToPsl
```
