# unitem

## Overview
UniteM is a specialized toolkit designed for "ensemble binning," a process that merges and refines genomic bins generated by different algorithms. Instead of relying on a single binning method, UniteM evaluates the outputs of multiple methods against marker gene sets derived from the Genome Taxonomy Database (GTDB). It then selects the highest quality, non-redundant representations of microbial genomes from the aggregate pool, effectively leveraging the strengths of different binning heuristics to produce superior MAGs.

## Command Line Usage and Best Practices

### Core Workflow
The primary entry point for ensemble binning is the `bin` sub-command. The general workflow involves:
1. Generating initial bins using several independent binning tools.
2. Providing these bin sets to UniteM for evaluation and de-replication.
3. Outputting a refined, non-redundant set of bins.

### Common CLI Patterns
Access the primary help menu to see available sub-commands:
```bash
unitem -h
```

To view specific arguments for the ensemble binning process:
```bash
unitem bin -h
```

### Expert Tips for Ensemble Binning
*   **Marker Gene Evaluation**: Since version 1.0.0, UniteM has moved away from CheckM in favor of a custom evaluation pipeline. It uses **Prodigal** for gene prediction and **HMMER** to identify marker genes across GTDB species representatives. Ensure these dependencies are in your PATH if running in a custom environment.
*   **Version 1.1.0+ Improvements**: If working with novel or closely related species, ensure you are using version 1.1.0 or higher. This version includes HMMs from closely related protein families to ensure more accurate annotation of marker genes, which directly impacts the quality scores used to select the best bins.
*   **Input Preparation**: When combining outputs from multiple binners, ensure that the contig headers are consistent across all input bin sets. Discrepancies in naming will prevent UniteM from correctly identifying overlapping contigs between different binning methods.
*   **Quality Estimation**: UniteM calculates completeness and contamination based on the presence/absence of lineage-specific marker genes. Use the output statistics to filter your final MAG set based on standard thresholds (e.g., >50% completeness, <10% contamination for medium-quality MAGs).

### Installation
UniteM is a Python-based tool and can be installed via pip:
```bash
pip install unitem
```
It requires `numpy` (>=1.24) and `svgwrite` (>=1.1.9) for its operations and visualization components.



## Subcommands

| Command | Description |
|---------|-------------|
| unitem bin | Apply binning methods to an assembly. |
| unitem profile | Identify marker genes and calculate assembly statistics. |
| unitem unanimous | Unanimous bin filtering across multiple binning methods. |
| unitem_consensus | Consensus clustering across multiple binning methods. |
| unitem_greedy | Greedy bin selection across multiple binning methods. |
