It works on Linux, OS X and Windows, and was created for Python programs but can package and distribute any software. Miniconda is a small version that includes only conda, Python, and the packages they depend on. Over 720 scientific packages and their dependencies can be installed individually from the Continuum repository with the “conda install” command.
