Install mamba ( recommended ) Install conda Bioconda lets you install thousands of software packages related to biomedical research using the conda package manager. One-time setup of bioconda Install with bioconda: Given that you already have a conda environment in which you want to have this package, install with: mamba install stellarscope To create a new environment, run: mamba create --name myenvname stellarscope with myenvname being a reasonable name for the environment.
