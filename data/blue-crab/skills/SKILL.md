# blue-crab

## Overview
The `blue-crab` skill enables efficient bidirectional conversion between ONT's proprietary POD5 format and the SLOW5/BLOW5 format. While POD5 uses a column-based architecture, SLOW5/BLOW5 is designed for high-performance row-based reading, which is often preferred for downstream signal analysis. This tool ensures data parity during conversion, allowing researchers to leverage the SLOW5 ecosystem without losing metadata or signal integrity.

## Core Workflows

### POD5 to SLOW5/BLOW5 Conversion (`p2s`)
The primary use case is converting raw ONT data for use in SLOW5-based pipelines.

- **Single File**: `blue-crab p2s input.pod5 -o output.blow5`
- **Directory Processing**: `blue-crab p2s pod5_dir -d blow5_dir`
- **Compression**: By default, the tool uses `zlib`. For better performance and smaller file sizes, ensure `zstd` development libraries are installed and use the `.blow5` extension.

### SLOW5/BLOW5 to POD5 Conversion (`s2p`)
Use this to return data to the ONT-native format if required by specific ONT proprietary tools.

- **Single File**: `blue-crab s2p input.blow5 -o output.pod5`
- **Directory Processing**: `blue-crab s2p blow5_dir -d pod5_dir`

## Expert Tips and Best Practices

- **Integrity Verification**: Because ONT frequently updates the POD5 specification, always verify converted files using `slow5tools quickcheck` or `slow5tools index`.
- **Performance Optimization**: When installing from source or pip, set `export PYSLOW5_ZSTD=1` before installation to enable ZSTD support, which significantly improves read/write speeds for BLOW5 files.
- **Real-time Conversion**: For active sequencing runs, use the `realtime-p2s` script found in the repository's `scripts/` directory to convert data as it is generated by the sequencer.
- **Environment Requirements**: Ensure you are using Python 3.8 or higher. If using a virtual environment, the wrapper script in `scripts/blue-crab` can be added to your PATH to handle environment activation automatically.



## Subcommands

| Command | Description |
|---------|-------------|
| blue-crab p2s | Convert POD5 -> SLOW5/BLOW5 |
| s2p | Convert SLOW5/BLOW5 -> POD5 |
