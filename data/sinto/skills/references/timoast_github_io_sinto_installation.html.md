# Installation[](#installation "Link to this heading")

Install from conda:

```
conda install sinto
```

Install from PyPI:

```
pip install sinto
```

Install from source:

```
git clone https://github.com/timoast/sinto.git
cd sinto
python -m build
pip install .
```

## Testing[](#testing "Link to this heading")

```
pip install pytest
pytest
```
