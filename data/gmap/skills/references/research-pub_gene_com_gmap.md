The new mailing list for issues relating to both GMAP and GSNAP is now at Google Groups . To sign up, go to your list , select "All groups and messages", and search for "gsnap-users". Click on that group, and then "Join group". You can join there to receive release notices, ask questions, or see previous messages. If you have a bug to report or a feature to request, I believe you can also send email to gsnap-users@googlegroups.com. (You may have to subscribe to the list first, though, or the message will be held for me to approve) Genome databases You can build your own genome database with the gmap_build program included with this software. Please see instructions in the README file. For the human genome, you may want to retrieve UCSC hg19/GRCh37 Ensembl GRCh37 Documentation README file from the source code distribution.
