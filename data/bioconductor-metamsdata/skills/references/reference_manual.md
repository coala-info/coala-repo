metaMSdata-package

Example CDF data for the metaMS package

Description

Example CDF data for the metaMS package. For both GC-MS and LC-MS data, four data files are
provided, all in the inst/extdata directory of the package. In addition, this directory contains a
csv file that can be used as a template for building in-house databases for GC-MS. No further R
code is present.
