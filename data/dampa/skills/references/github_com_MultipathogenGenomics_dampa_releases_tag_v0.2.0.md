[v0.2.0](/MultipathogenGenomics/dampa/tree/v0.2.0)

[`91fd2b0`](/MultipathogenGenomics/dampa/commit/91fd2b0bd834e4bd30ef3df0fc04d017776b12dd)

* Added target generation function to produce a fasta file that can be used in downstream mapping analysis. Sequencing data generated by capture using a given panel design can be mapped to the target file to determine the presence of target organisms. i.e. using [castanet](https://github.com/MultipathogenGenomics/castanet)
* Target generation is currently in development!!
* Added remove\_outliers flag. This will cluster input sequences at a low cutoff (i.e. 80% identity) then remove sequences that are in singleton clusters. The rationale is that these are likely to be another species or possibly artificially modified (i.e. codon optimised). caution should be used if you have a sparse, diverse genome collection.
