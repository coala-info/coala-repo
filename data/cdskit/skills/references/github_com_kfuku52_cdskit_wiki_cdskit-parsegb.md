# cdskit parsegb

[Jump to bottom](#wiki-pages-box)

Kenji Fukushima edited this page Mar 10, 2023
·
[4 revisions](/kfuku52/cdskit/wiki/cdskit-parsegb/_history)

`cdskit parsegb` converts a [GenBank-formatted file](https://www.ncbi.nlm.nih.gov/Sitemap/samplerecord.html) with the sequence name formatting.

### Example

#### Command

```
cdskit parsegb -s input.gb -o output.fasta
```
