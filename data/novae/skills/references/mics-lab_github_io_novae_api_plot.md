Quality control This plot is useful to check the quality of the spatial connectivities obtained via novae.spatial_neighbors . Make sure few cells (e.g., less than 5%) have a number of neighbors below ngh_threshold . If too many cells are isolated, you may want to increase the radius parameter in novae.spatial_neighbors . Conversely, if there are some less that are really far from each other , but still connected, so may want to decrease the radius parameter to disconnect them.
