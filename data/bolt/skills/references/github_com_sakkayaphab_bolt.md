Installing Bolt from source Requirements GCC &gt;=v5.4 HTSlib &gt;=v1.9 CMake &gt;=v3.9 Intel TBB &gt;=v2019.9 git clone https://github.com/sakkayaphab/bolt.git mkdir build cd build cmake .. -DINSTALL_BIN_PREFIX=(directory path for the binary executable file of Bolt) -DINCLUDE_LIBRARY_PREFIX=(directory path for include of libraries) -DLIBRARY_LINK_PREFIX=(directory path for lib of libraries) make make install 2. Installing Bolt with conda conda install -c intel tbb conda install -c bioconda bolt 3. Run with docker docker run -v /hostpath/:/opt/mount --name bolt sakkayaphab/bolt:latest /project/build/bolt call -b /opt/mount/sample.bam -r /opt/mount/ref.fa -t 100 -o /opt/mount/outputfolder Usage bolt call -b (aligment) -r (reference) -t (number of threads) -o (output) for user guide License Bolt is distributed under the GPLv3 .
