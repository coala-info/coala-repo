# WF7201 Workflow Files

This directory contains the workflow files for **WF7201**, a computational workflow described using the **Common Workflow Language (CWL)** and enriched with metadata following the **RO-Crate** standard.

## File Overview

- **WF7201.cwl File**  
  This is the main entry point of the workflow. It defines the global inputs, outputs, and the connections between the steps. 

- **ro-crate-metadata.json**  
  A metadata template generated from the CWL description. It should list all the entities in the workflow.
