#### Trees as Python objects

Load, create, traverse, search,
prune, or modify hierarchical tree
structures with ease using the ETE Python API.](/docs/latest/tutorial/tutorial_trees.html)

[![box image](/static/img/piechart_pic.jpg)

#### Programmatic tree visualization

Get full control
of your tree images. Browse them interatively or
render SVG, PNG of PDF images.](/docs/latest/tutorial/tutorial_drawing.html)

[![box image](/static/img/heatmap_pic.jpg)

#### Tree annotation

Custom node
attributes can be rendered as graphical
elements. Choose among external images,
charts, symbols, text labels, and more!](/gallery/)

[![box image](/static/img/ipython_pic.jpg)

#### Jupyter notebook support

Prototype your methods using the Jupyter notebook framework including inline visualization of trees.](/ipython_notebook/)

## Phylogenomic Tools

[Tools overview](/documentation/tools/)
[Cookbook](/cookbook/)

[![box image](/static/img/etebuild_pic.jpg)

#### Automated reconstruction of gene and species trees.

Run phylogenetic workflows with ease.](/documentation/ete-build/)

[![box image](/static/img/omega_pic.jpg)

#### Test evolutionary hypotheses

You can run CodeML and SLR easly using ETE, as well as visualizing results.](/documentation/ete-evol/)

[![box image](/static/img/taxonomy_pic.jpg)

#### Annotate trees with taxonomic data.

Use the NCBI taxonomy database to perform queries efficiently or to annotate your trees.](/documentation/ete-ncbiquery/)

[![box image](/static/img/phylogeny_pic.jpg)

#### Visualize trees and sequence alignments

Choose among detailed view, block-based, domain-based or condensed alignment formats.](/documentation/ete-view/)

[![box image](/static/img/phylogeny_pic.jpg)

#### Compare tree topologies

You can estimate distances between gene and species trees of a different size and with duplicated events.](/documentation/ete-compare/)

![box image](/static/img/annotation_pic.jpg)

#### Summarize phylogenetic signal

You can summarize phylogentic signal from multiple gene trees into a single species tree.
