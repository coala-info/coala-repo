# bioconductor-metaneighbor

## Overview

MetaNeighbor is a tool for measuring the similarity of cell types across different scRNA-seq datasets. It uses a neighbor voting algorithm based on cell-cell correlations within specific gene sets. The primary output is an **AUROC (Area Under the Receiver Operating Characteristic) score**. 

- **AUROC = 0.5**: The cell type is no more similar to itself across datasets than to a random cell type (random guessing).
- **AUROC > 0.9**: Indicates high replicability and strong evidence that the cell type identity is consistent across datasets.

## Data Requirements

MetaNeighbor requires data in a `SummarizedExperiment` or `SingleCellExperiment` object.

1.  **Expression Matrix**: An assay containing gene-by-sample data (e.g., RPKM, CPM, or log-transformed counts).
2.  **Experiment Labels**: A vector in `colData` indicating the source/study of each sample (e.g., `study_id`).
3.  **Cell Type Labels**: A vector in `colData` indicating the cell type of each sample.
4.  **Gene Sets**: A list of vectors containing gene symbols that match the row names of the expression matrix.

## Supervised MetaNeighbor

Use this workflow when you have known, corresponding cell type labels across datasets and want to quantify their similarity.

```r
library(MetaNeighbor)
library(SummarizedExperiment)

# Required inputs:
# dat: SummarizedExperiment object
# experiment_labels: numeric or factor vector of study IDs
# celltype_labels: binary matrix (samples x cell types) where 1 = member
# genesets: list of gene vectors (e.g., GO terms)

AUROC_scores <- MetaNeighbor(
  dat = mn_data,
  experiment_labels = as.numeric(factor(mn_data$study_id)),
  celltype_labels = metadata(colData(mn_data))[["cell_labels"]],
  genesets = GOmouse,
  bplot = TRUE
)
```

## Unsupervised MetaNeighbor (Data Exploration)

Use this workflow to identify potential cell type overlaps between datasets when labels are ambiguous or different.

1.  **Identify Variable Genes**: Use `variableGenes` to find genes that are highly variable across datasets.
2.  **Run MetaNeighborUS**: Run the unsupervised version.
3.  **Identify Top Hits**: Use `topHits` to find reciprocal best matches.

```r
# 1. Identify variable genes
var_genes <- variableGenes(dat = mn_data, exp_labels = mn_data$study_id)

# 2. Run Unsupervised MetaNeighbor
celltype_NV <- MetaNeighborUS(
  var_genes = var_genes,
  dat = mn_data,
  study_id = mn_data$study_id,
  cell_type = mn_data$cell_type
)

# 3. Find reciprocal top hits (AUROC > 0.9)
top_hits <- topHits(
  cell_NV = celltype_NV,
  dat = mn_data,
  study_id = mn_data$study_id,
  cell_type = mn_data$cell_type,
  threshold = 0.9
)
```

## Handling Large Datasets

For datasets with >10,000 cells, use the low-memory "fast" version. This version approximates the correlation network to save time and RAM.

- Set `fast_version = TRUE` in `MetaNeighbor()` or `MetaNeighborUS()`.
- Use `SingleCellExperiment` objects with sparse matrices for better efficiency.

```r
celltype_NV <- MetaNeighborUS(
  var_genes = var_genes,
  dat = large_pancreas_obj,
  study_id = large_pancreas_obj$study_id,
  cell_type = large_pancreas_obj$cell_type,
  fast_version = TRUE
)
```

## Visualization

- **Beanplots**: Generated by default in `MetaNeighbor()` (`bplot = TRUE`) to show the distribution of AUROC scores across gene sets.
- **Heatmaps**: Use `gplots::heatmap.2` to visualize the `celltype_NV` matrix from unsupervised runs. High AUROC (red) indicates strong similarity between the training (row) and testing (column) cell types.
