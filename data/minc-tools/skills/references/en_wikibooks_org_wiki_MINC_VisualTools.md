# Documentation for Visual/Interactive MINC Tools

[[edit](/w/index.php?title=MINC/VisualTools&veaction=edit&section=1 "Edit section: Documentation for Visual/Interactive MINC Tools") | [edit source](/w/index.php?title=MINC/VisualTools&action=edit&section=1 "Edit section's source code: Documentation for Visual/Interactive MINC Tools")]

## Manual Registration (alignment)

[[edit](/w/index.php?title=MINC/VisualTools&veaction=edit&section=2 "Edit section: Manual Registration (alignment)") | [edit source](/w/index.php?title=MINC/VisualTools&action=edit&section=2 "Edit section's source code: Manual Registration (alignment)")]

In cases when manual alignment of two volumes is needed, [register](/wiki/MINC/VisualTools/register "MINC/VisualTools/register") can be used.

## Manual Segmentation (labeling, painting)

[[edit](/w/index.php?title=MINC/VisualTools&veaction=edit&section=3 "Edit section: Manual Segmentation (labeling, painting)") | [edit source](/w/index.php?title=MINC/VisualTools&action=edit&section=3 "Edit section's source code: Manual Segmentation (labeling, painting)")]

[Display](/wiki/MINC/VisualTools/Display "MINC/VisualTools/Display") is the tool developed for manual labeling of MRI volumes and visualization of 3D objects. Alternatively, [mincified version of ITK-SNAP](/wiki/MINC/EZMINC/MincifiedTools "MINC/EZMINC/MincifiedTools") can be used for the same purpose.

## The rest of the tools (unsorted heap)

[[edit](/w/index.php?title=MINC/VisualTools&veaction=edit&section=4 "Edit section: The rest of the tools (unsorted heap)") | [edit source](/w/index.php?title=MINC/VisualTools&action=edit&section=4 "Edit section's source code: The rest of the tools (unsorted heap)")]

[xdisp](/w/index.php?title=MINC/VisualTools/xdisp&action=edit&redlink=1 "MINC/VisualTools/xdisp (does not exist)") - useful for quantitative measurements

[brain-view](/wiki/MINC/VisualTools/brain-view "MINC/VisualTools/brain-view") - visualization of statistical results

[viewnup](/w/index.php?title=MINC/VisualTools/viewnup&action=edit&redlink=1 "MINC/VisualTools/viewnup (does not exist)") - visualization of multiple volumes simultaneously
