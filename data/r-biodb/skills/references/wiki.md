# biodb

*biodb* is an R package for accessing chemical and biological databases.

This package mainly focusses on:

* Databases access to retrieve entries.
* Search for compounds by names or mass.
* Search for MS spectra.
* MSMS spectra matching.

The current version is *1.0*.
