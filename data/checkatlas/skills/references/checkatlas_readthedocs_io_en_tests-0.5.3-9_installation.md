The checkatlas pythn module which can be downloaded with PyPi, and the checkatlas workflow which can be downloaded with nextflow. pip install checkatlas nextflow pull becavin-lab/nf-core-checkatlas You need also to install a version of MultiQC with checkatlas capability (for the moment). This version of MultiQC is available at checkatlas branch of github.com:becavin-lab/MultiQC. git clone git@github.com:becavin-lab/MultiQC.git cd MultiQC/ git checkout checkatlas pip install . Finally, checkatlas comes with rpy2 to perform the interface between python and R. But, it does not automatically install Seurat. So if you want to screen Seurat atlases you need to perfrom this last installation % R &gt; install.packages('Seurat') &gt; library(Seurat) Previous Next Built with MkDocs using a theme provided by Read the Docs .
