Installation instructions for CONUS
------------------------------------------
uncompress conus-1.0.tar.gz Uncompresses the archive.
tar xf conus-1.0.tar Unpacks the archive.
(makes a new directory, conus-1.0/)
cd conus-1.0 Moves into the distribution directory.
./configure Configures the software for your system.
make Builds the binaries.
make install Moves binaries to directory conus-1.0/bin/
