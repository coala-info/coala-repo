## mitoODE

### Package Description
The package contains the methods to fit a cell-cycle model on cell count data and the code to reproduce the results shown in our paper "Dynamical modelling of phenotypes in a genome-wide RNAi live-cell imaging assay" by Pau, G., Walter, T., Neumann, B., Heriche, J.-K., Ellenberg, J., & Huber, W., BMC Bioinformatics (2013), 14(1), 308. doi:10.
