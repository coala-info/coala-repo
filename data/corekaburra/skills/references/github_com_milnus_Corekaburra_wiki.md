Contributions and community guidelines Let us know if you have any issues with the code: bugs, feature-requests, undesired behaviour, or anything else. The easiest way to let us know is to report an issue via the Github issues page . If you have a desired feature you want and can implement it yourself, feel free. It does not need to come with tests, but we would like pull requests to include some description of desired behaviour and any data (can be a simple toy example) you have used for tests during developement.
