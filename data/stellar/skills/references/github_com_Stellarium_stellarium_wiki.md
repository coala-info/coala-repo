It should be enough to say for the last version only how to install the files, and list the available files with URL where to download them Landscapes: There is an extensive set of Landscapes available for Stellarium at http://stellarium.org/en/landscapes.html . The set is currently being transferred from the older wiki at http://stellarium.sourceforge.net/wiki/index.php/Landscapes . Some landscapes may not be available yet, so please consult both sites. Nebula images: See Stellarium User Guide, chapter 8. Scripts: A nice collection of user-contributed scripts can be found at http://stellarium.org/scripts.html . Solar System Objects: Please use the Solar System Editor plugin to add, update or remove orbital elements or minor planets and comets. The User Guide explains the data format if you want to write your own data. Sky cultures: See how people across the world look at the heavens above. See Stellarium User Guide, chapter 9.
