## Overview
The `svtopovz` tool is designed to visualize complex structural variants (SVs) identified in sequencing data, with a specific focus on PacBio HiFi data. It takes connected breakend information (typically generated from VCF and BAM files, often in conjunction with variant callers like sawfish) and transforms it into high-quality graphical representations of structural rearrangements. This allows for a deeper understanding of deletions, duplications, inversions, and other complex genomic alterations.

## Usage Instructions

The `svtopovz` tool is a Python utility used for plotting structural variants. It primarily takes JSON output from SVTopo and generates image files.

### Core Functionality

The main purpose of `svtopovz` is to convert structural variant breakend data into visual plots.

### Command Line Usage

The typical command structure involves specifying input and output files.

**Basic Command:**

```bash
svtopovz --input <input.json> --output <output.png>
```

**Key Arguments:**

*   `--input <input.json>`: Path to the JSON file containing the connected breakend data. This file is usually generated by the SVTopo analysis pipeline.
*   `--output <output.png>`: Path where the generated visualization image (e.g., PNG) will be saved.

### Advanced Options and Best Practices

*   **Input Data Preparation**: Ensure your input JSON file is correctly formatted and contains the structural variant breakend information. This data is typically derived from variant callers like `sawfish` and processed by `SVTopo`.
*   **Gene Annotations**: The tool supports optional gene annotations. If your input data includes gene information, `svtopovz` can incorporate these into the visualization, providing genomic context. Refer to the SVTopo documentation for how to prepare input JSONs with gene annotations.
*   **Plot Interpretation**: The visualization highlights connected aligned sequences as blocks, indicating their order and orientation relative to a reference genome. Block weight often represents alignment support strength, and block sizes are encoded in a legend. The bottom section of the plot chains together deletions, duplications, and inversions to illustrate the rearranged genome structure.
*   **Output Formats**: While PNG is common, check the tool's capabilities for other supported image formats if needed.

### Example Workflow

1.  **Run SVTopo**: Process your VCF and BAM files with `SVTopo` to generate the connected breakend JSON output.
    ```bash
    # Example command (actual command may vary based on SVTopo usage)
    # svtopo --vcf input.vcf --bam input.bam --output svtopo_output.json
    ```
2.  **Visualize with svtopovz**: Use the generated JSON file as input for `svtopovz`.
    ```bash
    svtopovz --input svtopo_output.json --output svtopo_visualization.png
    ```
