It contains two core modules, and use a stepwise fashion to correct the error bases in reference genome. To correct/assemble the raw third-generation sequencing (TGS) long reads with approximately 10-15% sequencing errors, please use NextDenovo . To further improve the consensus accuracy of genomes assembled using HiFi long-reads, please use NextPolish2 .
