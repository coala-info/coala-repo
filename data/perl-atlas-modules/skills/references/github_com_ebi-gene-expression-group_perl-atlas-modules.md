Install perl-atlas-modules can be installed from Bioconda: conda install -c bioconda perl-atlas-modules Configuration perl-atlas-modules will look for configuration files in a directory specified by the enviironment variable ATLAS_META_CONFIG if that variable is set, and will otherwise look at the supporting_files directory, which is installed along with the perl modules as part of the Bioconda install method above. The '.default' files are used to initialise default configurations where not already present.
