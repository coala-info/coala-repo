# R packages

These are the *R* libraries that you will need to install to analyze data output from the *hecatomb* workflow. Some are installable directly, and other optional libraries will require [BioConductor](https://bioconductor.org/) that we will also install.

## Recommended libraries:

These are libraries we have found essential for working with hecatomb data.

To install the recommended libraries, the command below can be entered directly into an R console, preferably accessed using [Rstudio](https://rstudio.com), or optionally using R from the command line:

```
install.packages(c("BiocManager", "tidyverse",  "tidylog", "rstatix"), dependencies = TRUE)
```

* [BiocManager](https://cran.r-project.org/web/packages/BiocManager/index.html): Allows access to Bioconductor Project packages
* [tidyverse](https://www.tidyverse.org): A standard to work with data in a *tidy* way in R. Includes [ggplot](https://ggplot2.tidyverse.org), [dplyr](https://dplyr.tidyverse.org) and more
* [tidylog](https://github.com/elbersb/tidylog): Provides additional details about your tidy operations in the RStudio console (optional but highly recommended)
* [rstatix](https://github.com/kassambara/rstatix): tidy statistics in R (also technically optional, but used extensively for statistical analysis of hecatomb data in our lab)

## Optional libraries:

These are libraries that we have found to be useful for analyzing data output by the *hecatomb* workflow:

* [data.table](https://github.com/Rdatatable/data.table): A high-performance version of base R's data.frame. Really useful for fast, memory efficient operations, particularly for large data tables. If you are struggling with slow i/o performance or other table operations such as joins then consider using data.table
* [phyloseq](https://joey711.github.io/phyloseq/): Hecatomb's data format is largely based on Phyloseq. As such, many of the built in functions of phyloseq may be applicable to your analysis.
* [plotly](https://plotly.com/r/): An interface to the brilliant interactive [plotly](https://plotly.com) library from the plots you create from ggplot. Useful if you want to use your mouse to select specific parts of a plot for visualization.
