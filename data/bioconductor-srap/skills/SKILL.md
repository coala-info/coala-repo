## Overview

Use the Bioconductor R package **sRAP** for: The Bioconductor project aims to develop and share open source software for precise and repeatable analysis of biological data. We foster an inclusive and collaborative community of developers and data scientists.

## Installation

```r
if (!require("BiocManager", quietly = TRUE))

BiocManager::install("sRAP")
```
