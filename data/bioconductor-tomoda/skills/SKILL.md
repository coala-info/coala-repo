# bioconductor-tomoda

## Overview

Use the Bioconductor R package **tomoda** for: the package is to find zones with similar transcriptional profiles and spatially expressed genes in a tomo-seq sample. Several visulization functions are available to create easy-to-modify plots.

## Installation
