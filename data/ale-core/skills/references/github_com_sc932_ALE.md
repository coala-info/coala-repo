Plotting -------- The authors recommend using IGV to view the output. http://www.broadinstitute.org/igv/ Just import the assembly, bam and ALE scores. You can convert the .ale file to a set of .wig files with ale2wiggle.py and IGV can read those directly. Depending on your genome size you may want to convert the .wig files to the BigWig format.
