To install, simply use the conda installer: $ conda install -c bioconda peptides GitHub + pip # If, for any reason, you prefer to download the library from GitHub, you can clone the repository and install the repository by running (with the admin rights): $ git clone --recursive https://github.com/althonos/peptides.py $ pip install --user ./peptides.py Caution Keep in mind this will install always try to install the latest commit, which may not even build, so consider using a versioned release instead. GitHub + setuptools # If you do not want to use pip , you can still clone the repository and run the setup.py file manually, although you will need to install the build dependencies (mainly astor ): $ git clone --recursive https://github.com/althonos/peptides.py $ cd peptides.py # python setup.py build_py install Danger Installing packages without pip is strongly discouraged, as they can only be uninstalled manually, and may damage your system. previous User Guide next Contributing to peptides.py On this page PyPi Conda GitHub + pip GitHub + setuptools Edit on GitHub This Page Show Source © Copyright 2021-2025, Martin Larralde.
