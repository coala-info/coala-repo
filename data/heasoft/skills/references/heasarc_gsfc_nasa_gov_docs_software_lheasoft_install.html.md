# Installing HEASoft

HEASoft can be installed from source or using conda. For help installing HEASoft on your system, please visit the appropriate link here:

* [PC Linux](linux.html), including on WSL (Windows Subsystem for Linux)
* [macOS](macos.html)
* [conda](/docs/software/conda.html) (Linux or Mac)

HEASoft can also be installed in a [Docker container](docker.html).

See also the [video walkthrough](install-heasoft-mac-tutorial.html)
of the installation process on a Mac.

## Alternatives to Installing HEASoft

For those who prefer not to install HEASoft locally on their machine, the HEASARC offers multiple cloud platforms. HEASoft is already installed on these platforms and NASA data is available.

* [SciServer](/docs/sciserver/index.html)
* [Fornax](https://science.nasa.gov/astrophysics/programs/physics-of-the-cosmos/community/the-fornax-initiative/)

## Uninstalling HEASoft

For guidance regarding removal of HEASoft from your system, visit the
[uninstall](uninstall.html) page.
