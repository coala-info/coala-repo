Commands Currently wrapped scmap functions are described below. Each script has usage insructions available via --help, consult function documentation in scmap for further details. Preprocess SCE object for scmap pipeline This script makes the necessary changes to the SCE object required by the scmap workflow, including 'un-sparsing' and log-normalising the expression matrix. scmap-preprocess-sce.R --input-object &lt;path to the SCE object&gt;\ --output-sce-object &lt;path to the updated SCE object in .rds format&gt; Extract test data Input to the workflow will be a serialised single-cell experiment object. You can generate one for testing (derived from the package-provided test data) like: scmap-make-test-data.R --output-object-file &lt;output SingleCellExperiment in .rds format&gt; Find the most informative features (genes/transcripts) for projection scmap-select-features.R --input-object-file &lt;input SingleCellExperiment in .rds format&gt; \ --n-features &lt;number features to use&gt; --output-object-file &lt;output SingleCellExperiment in .rds format&gt; \ --output-plot-file &lt;optional file name in .png format, for feature selection plot&gt; Calculate centroids of each cell type and merge them into a single table. Here we generate a summary representation of each cluster in the indexed dataset: scmap-index-cluster.R --input-object-file &lt;input SingleCellExperiment in .rds format&gt; \ --cluster-col &lt;column name where cell types are stored&gt; \ --train-id &lt;Training dataset ID (optional)&gt; \ --remove-mat &lt;Should expression data be removed from the index? &gt;\ --output-object-file &lt;output SingleCellExperiment in .rds format&gt; \ --output-plot-file &lt;optional file name in .png format, for heatmap-style index visualisation&gt; Project one dataset to another In this step we find the cluster medoid of the index dataset closest to the cells of a query: scmap-scmap-cluster.R -i &lt;cluster-indexed SingleCellExperiment in .rds format&gt; \ -p &lt;query SingleCellExperiment in .rds format&gt; --threshold &lt;cluster similarity threshold&gt; \ --output-text-file &lt;csv-format file to store results&gt; \ --output-object-file &lt;output SingleCellExperiment in .rds format&gt; Create an index for a dataset to enable fast approximate nearest neighbour search Here we generate a cell-wise index: scmap-index-cell.R --input-object-file &lt;input SingleCellExperiment in .rds format&gt; \ --train-id &lt;Training dataset ID (optional)&gt; \ --number-chunks &lt;number of chunks into which the expr matrix is split&gt; \ --remove-mat &lt;Should expression data be removed from the index? &gt;\ --number-clusters &lt;number of clusters per group for k-means clustering&gt; \ --output-object-file &lt;output SingleCellExperiment in .rds format&gt; For each cell in a query dataset, search for the nearest neighbours by cosine distance within a collection of reference datasets Here we find the nearest 'n' neighbours in an index dataset for the cells of a query dataset. Optionally (when --cluster-col is set and corresponds to a column in the index dataset's colData()), generate a cluster identity for query cells via the cluster inenties in the index: scmap-scmap-cell.R -i $index_cell_sce -p &lt;input SingleCellExperiment in .rds format&gt; \ --number-nearest-neighbours &lt;number nearest neighbours&gt; \ --cluster-col &lt;column name where cell types are stored&gt; \ --output-object-file &lt;output SingleCellExperiment in .rds format&gt; \ --output-clusters-text-file &lt;file to store optional cluster identities&gt; \ --closest-cells-text-file &lt;csv file to store closest cells&gt; \ --closest-cells-similarities-text-file &lt;csv file to store similarity values&gt; Get standard output for downstream processing and analysis as part of various workflows scmap_get_std_output.R\ --predictions-file &lt;Path to the predictions file in text format&gt;\ --output-table &lt;Path to the final output file in text format&gt;\ --include-scores &lt;Boolean: Should prediction scores be included in output? Default: FALSE&gt;\ --tool &lt;What tool produced output? (scmap-cell or scmap-cluster)&gt;\ --index &lt;Path to the index object in .rds format (Optional; required to add dataset of origin to output table)&gt;\ --sim-col-name &lt;Column name of similarity scores&gt; About CLI scripts for the scmap Bioconductor package Resources Readme License Apache-2.0 license Uh oh!
